学完如何描述事物以及使用动词描述动作之后，我们来学习否定式（译者注：日语里面似乎没有「否定式」这种语法叫法，但为避免读者将其与动词其他活用形混淆，或者看其他书产生疑惑，本指南统一按照「否定式」来翻译）。换个说法，我们想表达某个动作未被执行。做法跟名词及形容词一样，活用动词，但是规则稍微复杂一些。

## 将动词活用为否定式 {#part2}

### 词汇学习

1. ある （う动词） - 存在（无生命）
2. いる （る动词） - 存在（有生命）
3. 食べる 【た・べる】 （る动词） - 吃
4. 買う 【か・う】 （う动词） - 买
5. 待つ 【ま・つ】 （う动词） - 等待
6. する （例外） - 做
7. 来る 【く・る】 （例外） - 来
8. 見る 【み・る】 （る动词） - 看
9. 寝る 【ね・る】 （る动词） - 睡觉
10. 起きる 【お・きる】 （る动词） - 叫醒；发生
11. 考える 【かんが・える】 （る动词） - 考虑
12. 教える 【おし・える】 （る动词） - 教；告知
13. 出る 【で・る】 （る动词） - 出来
14. 着る 【き・る】 （る动词） - 穿
15. 話す 【はな・す】 （う动词） - 说
16. 聞く 【き・く】 （う动词） - 问；听
17. 泳ぐ 【およ・ぐ】 （う动词） - 游泳
18. 遊ぶ 【あそ・ぶ】 （う动词） - 玩
19. 飲む 【の・む】 （う动词） - 喝
20. 帰る 【かえ・る】 （う动词） - 回家
21. 死ぬ 【し・ぬ】 （う动词） - 死去
22. お金 【お・かね】 - 钱
23. 私 【わたし】 - 我；我自己
24. 猫 【ねこ】 - 猫

我们现在要按照上一课学的分类法来学习活用规则。但首先我们要说一个非常重要的活用规则例外：「<span class="jp-word" title="ある - 存在（无生命）">ある</span>」。

- ある （う动词） - 存在（无生命）
- いる （る动词） - 存在（有生命）

「<span class="jp-word" title="ある - 存在（无生命）">ある</span>」是一个表示无生命物体存在的う动词，相应的，表示有生命物体（比如人或动物）的存在则用「<span class="jp-word" title="いる - 存在（有生命）">いる</span>」，它是一个正常的る动词。比如你想说屋里有张椅子，就要用动词「<span class="jp-word" title="ある - 存在（无生命）">ある</span>」，但说屋里有个人就要用「<span class="jp-word" title="いる - 存在（有生命）">いる</span>」。「<span class="jp-word" title="ある - 存在（无生命）">ある</span>」和「<span class="jp-word" title="いる - 存在（有生命）">いる</span>」这两个动词跟其他动词很不一样，因为它们仅仅描述事物的存在而不代表具体的动作。你要根据对象是否有生命来选用其中一个。

说这个插曲的原因是因为「<span class="jp-word" title="ある - 存在（无生命）">ある</span>」的否定式是「<span class="jp-word" title="ある - 存在（无生命）">ない</span>」（意思是某物不存在）。其他所有动词的否定活用规则如下，附示例。

＊ = 这种活用的例外

<div class="sumbox">
<div class="sumbox-title">动词否定式的活用规则</div>

- <b>对る动词：</b> 把「る」去掉换成「ない」<br>
<span class="jp-trans">例：食べ る + ない = 食べ <em>ない</em></span>
- <b>＊ 对以「う」结尾的う动词：</b> 把「う」换成「わ」，再加上「ない」<br>
<span class="jp-trans">例：買 う + わ + ない = 買 <em>わない</em></span>
- <b>其他う动词：</b> 把末假名换成同行的あ段假名，再加上「ない」<br>
<span class="jp-trans">例：待 つ + た = 待 <em>たない</em></span>
- <b>例外：</b>
  1. する → しない
  2. くる → こない
  3. ＊ある → ない
- 译者注：加上「ない」之前的活用形在语法上称为「未然形」
</div>

### 示例

以下是用动词否定式造的例句。

1. アリスは <em><span class="jp-word" title="たべる - 吃">食べない</span></em> 。<br>
<span class="jp-trans">对 Alice 来说，不吃。</span>
2. ジムが <em><span class="jp-word" title="あそぶ - 玩">遊ばない</span></em> 。<br>
<span class="jp-trans">Jim 是不玩的那个人。</span>
3. ボブも <em><span class="jp-word" title="する - 做">しない</span></em> 。<br>
<span class="jp-trans">Bob 也不做。</span>
4. <span class="jp-word" title="おかね - 钱">お金</span> が <em><span class="jp-word" title="ある - 存在（无生命）">ない</span></em> 。<br>
<span class="jp-trans">没有钱。（字面：钱是不存在的那样东西。）</span>
5. <span class="jp-word" title="わたし -我；我自己">私</span> は <em><span class="jp-word" title="かう - 买">買わない</span></em> 。<br>
<span class="jp-trans">对我来说，不买。</span>
6. <span class="jp-word" title="ねこ - 猫">猫</span> は <em><span class="jp-word" title="いる- 存在（有生命）">いない</span></em> 。<br>
<span class="jp-trans">没有猫。（字面：对于猫来说，不存在。）</span>
