## 在日本如何达到自己的目的 {#part1}

现在我们学习如何表达自己的愿望——你可以直截了当说出来，也可以隐晦的提建议。这里主要的知识点是「たい」活用形以及意向形。我们也将学习如何用条件语「たら」和「ば」来提建议。

## 用「たい」表达你想做的事 {#part2}

### 词汇学习

1. 行く 【い・く】 （う动词） - 走
2. 何 【なに】 - 什么
3. する （例外） - 做
4. 温泉 【おん・せん】 - 温泉
5. ケーキ - 蛋糕
6. 食べる 【た・べる】 （る动词） - 吃
7. ずっと - 长；远
8. 一緒 【いっ・しょ】 - 一起
9. いる （る动词） - 存在（有生命）
10. 犬 【いぬ】 - 狗
11. 遊ぶ 【あそ・ぶ】 （う动词） - 玩

你可以通过「たい」形来表达你<i>想要</i>做的事，很简单，在词根后面加上「たい」就行。不过不像前面学过的大多数活用规则那样把动词变为了る动词，「たい」形实际上把动词变成了一个い形容词（注意「たい」很配合的以「い」结尾）。这是说得通的，因为这种活用是用来形容你想做的事。变为「たい」形后，你可以像其他い形容词那样活用它。不过跟普通い形容词不同的是，「たい」形是从动词变化而来的，这意味着那些配合动词使用的助词例如「を」、「に」、「へ」或「で」都可以配合「たい」形使用，这一点い形容词是做不到的。下面我为你准备了图表。

### 示例

1. <span class="jp-word" title="なに - 什么">何</span> を <em><span class="jp-word" title="する - 做">したい</span></em> ですか。<br>
<span class="jp-trans">你想做什么？</span>
2. <span class="jp-word" title="おんせん - 温泉">温泉</span> に <em><span class="jp-word" title="いく - 走">行きたい</span></em> 。<br>
<span class="jp-trans">我想去泡温泉。</span>
3. <span class="jp-word" title="ケーキ - 蛋糕">ケーキ</span> 、 <em><span class="jp-word" title="たべる - 吃">食べたくない</span></em> の？<br>
<span class="jp-trans">你不想吃蛋糕？</span>
4. <em><span class="jp-word" title="たべる - 吃">食べたくなかった</span></em> けど <em><span class="jp-word" title="たべる - 吃">食べたく</span></em> <span class="jp-word" title="なる - 成为">なった</span> 。<br>
<span class="jp-trans">我过去不想吃，但后来变得想吃了。</span>

例四很难翻译，但在日语里面却是很简单的结构，可以参考[与い形容词合用「<span class="jp-word" title="なる - 成为">なる</span>」](/grammar/surunaru/#part3)一节。这里动词「<span class="jp-word" title="なる - 成为">なる</span>」的过去形 用来表示「（过去）变得想吃」。这个句子还可以变成一个绕口令，即组合「～たくない」和「<span class="jp-word" title="なる - 成为">なる</span>」的过去形：「<span class="jp-word" title="たべる - 吃">食べたくなくなった</span>」，意思是「（过去）变得不想吃」。

还有一点也许很明显了，就是「<span class="jp-word" title="ある - 存在（无生命）">ある</span>」不能活用为「たい」形，因为无生命的物体不可能想要什么。不过「<span class="jp-word" title="いる - 存在（有生命）">いる</span>」就可以，如下所示。

- <span class="jp-word" title="ずっと - 长时间">ずっと</span> <span class="jp-word" title="いっしょ - 一起">一緒</span> に <em><span class="jp-word" title="いる - 存在（有生命）">いたい</span></em> 。<br>
<span class="jp-trans">我想要永远在一起（字面：想要一起存在很久）。</span>

另外，你只能对第一人称使用「たい」形，因为你不可能知道别人的心思。如果主语是别人，通常要用类似「我想他要…」或「她说她想要…」之类的表达，这我们在后面会学。当然，如果你是在问问题，那可以随便用「たい」形，因为提问是不设前提的。

- <span class="jp-word" title="いぬ - 狗">犬</span> と <em><span class="jp-word" title="あそぶ - 玩">遊びたい</span></em> ですか。<br>
<span class="jp-trans">你想跟狗一起玩吗？</span>

## 用「<span class="jp-word" title="ほしい - 要，渴望">欲しい</span>」表示你想要的东西，或想要完成的事 {#part3}

### 词汇学习

1. 欲しい 【ほ・しい】 （い形容词） - 要，渴望
2. 好き 【す・き】 （な形容词） - 惹人喜欢的
3. 大きい 【おお・きい】（い形容词） - 大的
4. 縫いぐるみ 【ぬ・いぐるみ】 - 公仔
5. 全部 【ぜん・ぶ】 - 所有东西
6. 食べる 【た・べる】 （る动词） - 吃
7. 部屋 【へ・や】 - 房间
8. きれい （な形容词） - 好看；干净

英语里面我们会用一个动词来表示我们想要什么东西。日语里面，「想要」其实是一个い形容词而非动词。我们曾经见过类似的单词「<span class="jp-word" title="すき - 喜欢">好き</span>」，它就是个形容词，而英语里面「喜欢」是个动词。虽然我并没有深入介绍「<span class="jp-word" title="すき - 喜欢">好き</span>」，但这里我却要用整一个小节来介绍「<span class="jp-word" title="ほしい - 要，渴望">欲しい</span>」，因为跟动词て形合用的时候，整个词的意思是「想要某事（被）完成」。我们会在[「做出请求」](/grammar/requests/)一课介绍一种更为礼貌和得体的说法来发出请求，而不仅仅是说「我想搞定这事」。

另外虽然不是硬性要求，那些粘黏到动词て形用来实现特定语法功能的字符习惯上会用平假名写。这是因为动词本身就是用汉字书写的了，而粘黏的字符则成为了单词的一部分。

### 示例

1. <span class="jp-word" title="おおきい - 大的">大きい</span> <span class="jp-word" title="ぬいぐるみ - 公仔">縫いぐるみ</span> が <em><span class="jp-word" title="ほしい - 要，渴望">欲しい</span></em> ！<br>
<span class="jp-trans">我想要个大公仔！</span>
2. <span class="jp-word" title="ぜんぶ - 全部">全部</span> <em><span class="jp-word" title="たべる - 吃">食べて</span><span class="jp-word" title="ほしい - 要，渴望">ほしい</span></em> んだけど・・・。<br>
<span class="jp-trans">想全部被吃掉，不过…</span>
3. <span class="jp-word" title="へや - 房间">部屋</span> を <span class="jp-word" title="きれい - neat, pretty">きれい</span> に <em><span class="jp-word" title="する - 做">して</span><span class="jp-word" title="ほしい - 要，渴望">ほしい</span></em> のよ。<br>
<span class="jp-trans">我想要这房间被打扫干净，你懂的。</span>

上面说过，发出请求还有更得体的说法，现在先不提。这种语法用的不是很多，但考虑到完整性还是加上。

## 用意向形来提议做某事（口语） {#part4}

### 词汇学习

1. 食べる 【た・べる】 （る动词） - 吃
2. 入る 【はい・る】 （う动词） - 进入
3. 着る 【き・る】 （る动词） - 穿
4. 信じる 【しん・じる】 （る动词） - 相信
5. 寝る 【ね・る】 （る动词） - 睡
6. 起きる 【お・きる】 （る动词） - 起
7. 出る 【で・る】 る动词 - 出来
8. 掛ける 【か・ける】 （る动词） - 挂
9. 捨てる 【す・てる】 （る动词） - 扔掉
10. 調べる 【しら・べる】 （る动词） - 调查
11. 話す 【はな・す】 （う动词） - 说
12. 書く 【か・く】 （う动词） - 写
13. 待つ 【ま・つ】 （う动词） - 等待
14. 飲む 【の・む】 （う动词） - 喝
15. 取る 【と・る】 （う动词） - 取
16. 聞く 【き・く】 （う动词） - 问；听
17. 泳ぐ 【およ・ぐ】 （う动词） - 游泳
18. 遊ぶ 【あそ・ぶ】 （う动词） - 玩
19. 直る 【なお・る】 （う动词） - 修好
20. 死ぬ 【し・ぬ】 （う动词） - 死
21. 買う 【か・う】 （う动词）- 买
22. する （例外） - 做
23. 来る 【く・る】 （例外）- 来
24. 今日 【きょう】 - 今天
25. 何 【なに】 - 什么
26. テーマパーク - 主题公园
27. 行く 【い・く】 （う动词） - 走
28. 明日 【あした】 - 明天
29. カレー - 咖喱

「意向」这个词在这里的意思是做某事的意愿。换句话说，意向形代表某人准备着手做某事。它最常见的翻译就是 "let's" 或 "shall we?"，不过后面我们也将学到如何用这种活用形来表达为做某事付出的努力。

口语里面要将动词活用为意向形的话：对る动词，只要去掉「る」再加上「よう」；对う动词，把末尾假名从 / u / 段改为同行 / o / 段，再加上「う」。

<div class="sumbox">
<div class="sumbox-title">口语里面意向形的活用规则</div>

- <b>对る动词：</b> 去掉「る」再加上「よう」<br>
<span class="jp-trans">例： <span class="jp-word" title="たべる - 吃">食べ<em>る</em></span> → <span class="jp-word" title="たべる - 吃">食べ</span> + <em>よう</em> → <span class="jp-word" title="たべる - 吃">食べよう</span></span>
- <b>对う动词：</b> 把末尾假名从 / u / 段改为同行 / o / 段，再加上「う」。<br>
<span class="jp-trans">例： <span class="jp-word" title="はいる - 进入">入<em>る</em></span> → <span class="jp-word" title="はいる - 进入">入<em>ろ</em></span> + <em>う</em> → <span class="jp-word" title="はいる - 进入">入ろう</span></span>
</div>

## 用意向形提议做某事（丁宁语） {#part5}

### 词汇学习

1. 食べる 【た・べる】 （る动词） - 吃
2. 入る 【はい・る】 （う动词） - 进入
3. する （例外） - 做
4. 来る 【く・る】 （例外）- 来
5. 寝る 【ね・る】 （る动词） - 睡
6. 行く 【い・く】 （う动词） - 走
7. 遊ぶ 【あそ・ぶ】 （う动词） - 玩
8. 今日 【きょう】 - 今天
9. 何 【なに】 - 什么
10. テーマパーク - 主题公园
11. 明日 【あした】 - 明天
12. カレー - 咖喱

丁宁语里面意向形的活用规则就更简单了，只要在词根后面加上「～ましょう」就可以。类似ます形，活用出来的动词必须出现在句尾。实际上，如我们前面所见一样，丁宁语里面所有这些动词都必须放在句尾。

<div class="sumbox">
<div class="sumbox-title">丁宁语里面的意向形活用规则</div>

- <b>对所有动词：</b> 在词根后面加上「～ましょう」
  1. <span class="jp-word" title="たべる - 吃">食べ<em>る</em></span> → <span class="jp-word" title="たべる - 吃">食べ</span> + <em>ましょう</em> → <span class="jp-word" title="たべる - 吃">食べましょう</span>
  2. <span class="jp-word" title="はいる - 进入">入<em>る</em></span> → <span class="jp-word" title="はいる - 进入">入<em>り</em></span> + <em>ましょう</em> → <span class="jp-word" title="はいる - 进入">入りましょう</span>
</div>

## 用条件语「ば」或「たら」提建议 {#part6}

### 词汇学习

1. 銀行 【ぎん・こう】 - 银行
2. 行く 【い・く】 （う动词） - 走
3. たまに - 时不时
4. 両親【りょう・しん】 - 父母
5. 話す 【はな・す】 （う动词） - 说

你可以用条件语「ば」或「たら」提意见，后面再加上「<span class="jp-word" title="どう - 如何">どう</span>」。字面上意思是「如果你做某事，如何？」，用英语说就是 "How about doing [X]?"。这里也没什么新语法，不过这是一个常用的搭配。

### 示例

1. <span class="jp-word" title="ぎんこう - 银行">銀行</span> に <em><span class="jp-word" title="いく - 走">行ったら</span><span class="jp-word" title="どう - 如何">どう</span></em> ですか。<br>
<span class="jp-trans">去银行如何？</span>
2. <span class="jp-word" title="たまに - 时不时">たまに</span> ご <span class="jp-word" title="りょうしん - 父母">両親</span> と <em><span class="jp-word" title="はなす - 说">話せば</span><span class="jp-word" title="どう - 如何">どう</span></em> ？<br>
<span class="jp-trans">偶尔跟你父母聊聊如何？</span>
