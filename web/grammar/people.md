对不同的人不仅要注意选用不同的语言，同时还要注意恰当的称呼对方。另外称呼自己也需要特别注意。日语的特别之处就在于「我」和「你」这种简单的单词都却有好多说法。下面我们介绍如何称呼自己和别人。

## 称呼自己 {#part2}

### 词汇学习

- 名前 【な・まえ】 - 名字

日语里面说「我」有很多种方式。其中一些用的不多，而另一些则濒临绝迹。我们要学习的是现如今常用的那些。这里面又要按照礼貌程度和说话人性别加以区分。换句话说，有些词一般是男性使用，而另一些一般只是女性使用，这全都要视场合而定。

开始前先说一下单词「<span class="jp-word" title="わたし - 我；我自己">私</span>」。这个汉字的官方读法是「<span class="jp-word" title="わたくし - 我；我自己">わたくし</span>」，在正式场合（比如公司总裁讲话）就需要这么用。一般如此发音的时候都是在用尊敬语和谦逊语，后面会介绍。在其他场合，一般就读做「<span class="jp-word" title="わたし - 我；我自己">わたし</span>」，它是最泛泛的第一人称指代，所以通常也是日语学生首先学的。

以下是「我」的常用说法，以及使用场合

1. <span class="jp-word" title="わたくし - 我；我自己 （正式）">私</span> 【わたくし】 - 正式场合使用，男女通用。
2. <span class="jp-word" title="わたし - 我；我自己">私</span> 【わたし】 - 普通场合的礼貌用语，男女通用。
3. <span class="jp-word" title="ぼく - 我；我自己">僕</span> 【ぼく】 - 主要是男性使用，相当讲礼貌或相当随意的场合都可以用。
4. <span class="jp-word" title="おれ - 我；我自己">俺</span> 【おれ】 - 非常随意的口语，基本只有男性使用。
5. <span class="jp-word" title="あたし - 我；我自己">あたし</span> - 很女性化很口语的说法。但很多女生会用「 <span class="jp-word" title="わたし - 我；我自己">わたし</span> 」，因为「 <span class="jp-word" title="あたし - 我；我自己">あたし</span> 」听起来太娘了。
6. 自己的名字 - 这也是很女性化同时很幼稚的自我称呼。
7. <span class="jp-word" title="わし - 我；我自己">わし</span> - 通常是中年大叔使用。

我们来看如何在不同的句子里用不同的「我」。这里没提「わたくし」，因为我们还没有介绍更为正式的表达方式。

1. <span class="jp-word" title="わたし - 我；我自己">私</span> の <span class="jp-word" title="なまえ - 名字">名前</span> はキムです。<br>
<span class="jp-trans">我的名字是 Kim。（中性，礼貌）</span>
2. <span class="jp-word" title="ぼく - 我；我自己">僕</span> の <span class="jp-word" title="なまえ - 名字">名前</span> はキムです。<br>
<span class="jp-trans">我的名字是 Kim。（男性化，礼貌）</span>
3. <span class="jp-word" title="ぼく - 我；我自己">僕</span> の <span class="jp-word" title="なまえ - 名字">名前</span> はボブだ。<br>
<span class="jp-trans">我的名字是 Bob。（男性化，口语）</span>
4. <span class="jp-word" title="おれ - 我；我自己">俺</span> の <span class="jp-word" title="なまえ - 名字">名前</span> はボブだ。<br>
<span class="jp-trans">我的名字是 Bob。（男性化，口语）</span>
5. <span class="jp-word" title="あたし - 我；我自己">あたし</span> の <span class="jp-word" title="なまえ - 名字">名前</span> はアリス。<br>
<span class="jp-trans">我的名字是 Alice。（女性化，口语）</span>

## 用名字称呼别人 {#part3}

### 词汇学习

1. 社長 【しゃ・ちょう】 - 公司总裁
2. 課長 【か・ちょう】 - 课长
3. 先生 【せん・せい】 - 老师
4. 田中 【た・なか】 - 田中（名）

跟英语相反，日语里很少用到「你」。希望前面例子里的 Bob、Alice 和 Jim 能让你意识到人们互相之间都是用名字称呼的，即使是在面对面谈话。另外一个常用的对别人的称呼是他们的头衔，例如「<span class="jp-word" title="しゃちょう - 公司总裁">社長</span>」、「<span class="jp-word" title="かちょう - 课长">課長</span>」、「<span class="jp-word" title="せんせい - 老师">先生</span>」等等。单词「<span class="jp-word" title="せんせい - 老师">先生</span>」一般用来称呼某方面很有经验或知识的人，例如人们经常用「<span class="jp-word" title="せんせい - 老师">先生</span>」来称呼医生或者老师（废话）。你也可以加上对方的姓氏，比如「田中<span class="jp-word" title="せんせい - 老师">先生</span>」（田中老师）。如果对方没有什么头衔，你可以用对方的名字（通常是姓）加上「さん」来尊称他或她。如果叫姓有点过于礼貌而显生疏的话，把「さん」加在对方名后面也是可以的。「さん」更为随意的说法包括「くん」和「ちゃん」。「くん」通常用在平等或社会地位低于自己的男性名字后面（比如我老板有时候就叫我「キムくん」）。「ちゃん」则是一种比较可爱的说法，通常用在平等或社会地位低于自己的女性名字后面。

## 用「你」来称呼别人 {#part4}

请不要像在英语里面用 "you" 那样用「<span class="jp-word" title="あなた - 你">あなた</span>」称呼对方。在和别人对话时，有三种礼貌级别的称呼：1、对方的名字加合适的头衔；2、什么也不用；3、用「<span class="jp-word" title="あなた - 你">あなた</span>」。实际上如果用到第三种，那你基本上是准备爆粗口了。大多数时候，你什么也不用说，因为你就是在跟对方说话。每个句子里都带着「你」的话会让人觉得你在指责他什么。

「<span class="jp-word" title="あなた - 你">あなた</span>」同时也是以前女性称呼丈夫或者情人的方式。除非你是一个有着日本老公的中年妇女，否则你应该不会这么用「<span class="jp-word" title="あなた - 你">あなた</span>」。

以下是等同于英语里面 "you" 的一些说法。基本上你是用不到它们的，尤其是后半部分那些。

1. <span class="jp-word" title="あなた - 你">あなた</span> - 只有在没办法直接跟人说话，或者不知道别人名字的时候才用。比如纸面上向读者提出而读者又必须回答的问题，就可能会用「 <span class="jp-word" title="あなた - 你">あなた</span> 」。
2. <span class="jp-word" title="きみ - 你">君</span> 【きみ】 - 男生对女生亲密而又傲慢的称呼，可能会显得粗鲁。
3. <span class="jp-word" title="おまえ - 你">お前</span> 【お・まえ】 - 很粗鲁的称呼对方的方法。通常是男人使用，且会说成「 <span class="jp-word" title="おめえ - 你">おめえ</span> 」。
4. <span class="jp-word" title="あんた - 你">あんた</span> - 熟人之间很不客气的称呼，说话的人一般是生气了。
5. <span class="jp-word" title="てまえ - 你">手前</span> 【て・めえ】 - 非常粗鲁的说法，类似「 <span class="jp-word" title="おまえ - 你">お前</span> 」，并且更凶，一般会说成「てめ～～」，听起来像是准备开打。我只在动漫和电影里见过这种用法。实际上如果你这么称呼你的朋友，他们反而可能会哈哈大笑，说你看漫画书看多了。
6. <span class="jp-word" title="きさま - 你">貴様</span> 【き・さま】 - 非常、非常粗鲁的说法。听起来像是你要干掉对方。我还是只在漫画书里见过，这里介绍只是为了让你能看懂漫画书。

## 第三人称 {#part5}

### 词汇学习

1. 彼 【かれ】 - 他；男朋友
2. 彼女 【かの・じょ】 - 她；女朋友
3. ガールフレンド - 女朋友
4. ボーイフレンド - 男朋友

你可以用「<span class="jp-word" title="かれ - 他；男朋友">彼</span>」和「<span class="jp-word" title="かのじょ - she, girlfriend">彼女</span>」来表达「他」和「她」。注意「<span class="jp-word" title="かれ - 他；男朋友">彼</span>」和「<span class="jp-word" title="かのじょ - she, girlfriend">彼女</span>」也有「男朋友」和「女朋友」的意思。所以你该怎么区别呢？当然靠上下文了。比如别人问你「<span class="jp-word" title="かのじょ - she, girlfriend">彼女</span>ですか？」，显然意思是「她是你女朋友吗？」，不然「她是她吗？」这种问题没什么意义。另一个用的不多的说法是「<span class="jp-word" title="ガールフレンド - 女朋友">ガールフレンド</span>」和「<span class="jp-word" title="ボーイフレンド - 男朋友">ボーイフレンド</span>」，你读一下应该就知道是什么意思了。

## 称呼家庭成员 {#part6}

1. 母 【はは】 - 母亲
2. お母さん 【お・かあ・さん】 - 母亲（尊称）
3. 両親 【りょう・しん】 - 父母
4. 父 【ちち】 - 父亲
5. お父さん 【お・とう・さん】 - 父亲（尊称）
6. 妻 【つま】 - 妻子
7. 奥さん 【おく・さん】 - 妻子（尊称）
8. 夫 【おっと】 - 丈夫
9. 主人 【しゅ・じん】 - 丈夫
10. 姉 【あね】 - 姐姐
11. お姉さん 【お・ねえ・さん】 - 姐姐（尊称）
12. 兄 【あに】 - 哥哥
13. お兄さん 【お・にい・さん】 - 哥哥（尊称）
14. 妹 【いもうと】 - 妹妹
15. 弟 【おとうと】 - 弟弟
16. 息子 【むす・こ】 - 儿子
17. 娘 【むすめ】 - 女儿

日语里面称呼家庭成员比英语麻烦点（不过你试试泡菜语看！），为简洁起见（因为这可是个语法指南），我们只说直系亲属。日语里面称呼别人的家人要比称呼你自己（当着外人的面）的礼貌。比如在外人面前称呼自己母亲可以用「<span class="jp-word" title="はは - 母亲">母</span>」，而跟家人说话时可以用「<span class="jp-word" title="おかあさん - 母亲">お母さん</span>」。另外称呼同辈人也要分长幼。下面是一些常见用法，不一定全面。
