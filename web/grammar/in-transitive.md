日语里面，有两种相同动作的动词通常被称为<i>他动词</i>和<i>自动词</i>。两者的区别在于前者是由一个活动主体完成的动作，而后者则是在没有直接主体的情况下发生的动作。英语里面这两种情况一般用相同的动词来表示，比如 "The ball dropped" 和 "I dropped the ball"，但在日语里面就要写成「<span class="jp-word" title="ボール - 球">ボール</span>が<span class="jp-word" title="おちる - 被扔掉">落<em>ちた</em></span>」和「<span class="jp-word" title="ボール - 球">ボール</span>を<span class="jp-word" title="おとす  - 扔掉">落<em>とした</em></span>」。有时候翻译成英语时不大一样，例如 "To put it in the box" (<span class="jp-word" title="はこ - 盒子">箱</span>に<span class="jp-word" title="いれる - 放入">入れる</span>）和 "To enter the box"（<span class="jp-word" title="はこ - 盒子">箱</span>に<span class="jp-word" title="はいる - 进入">入る</span>），但这只是两种语言表达方式的不同所造成的。你可以认为在日语里面他动词和自动词的意思是一样的，只不过一个动作里面有直接的手（直接对象）而另一个没有。话说回来，术语并不重要，重要的是知道哪个是哪个，从而使用正确的助词。

## 注意助词的选择！ {#part2}

本节最重要的是学习根据不同的动词选用正确的助词。一开始可能很难掌握二者的区别，甚至都搞不清楚某个动词是否有他自动的两个版本。不确定的话，你可以查字典，比如 [jisho.org](http://jisho.org/)。

### 示例

1. <span class="jp-word" title="わたし - 我；我自己">私</span> が <span class="jp-word" title="でんき - 电；电灯">電気</span> <em>を</em> <span class="jp-word" title="つける - 附加上">つけた</span> 。<br>
<span class="jp-trans">把灯打开的人是我。</span>
2. <span class="jp-word" title="でんき - 电；电灯">電気</span> <em>が</em> <span class="jp-word" title="つく - 被附上">ついた</span> 。<br>
<span class="jp-trans">灯亮了。</span>
3. <span class="jp-word" title="でんき - 电；电灯">電気</span> <em>を</em> <span class="jp-word" title="けす - 擦除">消す</span> 。<br>
<span class="jp-trans">关掉灯。</span>
4. <span class="jp-word" title="でんき - 电；电灯">電気</span> <em>が</em> <span class="jp-word" title="きえる - 消失">消える</span> 。<br>
<span class="jp-trans">灯灭了。</span>
5. <span class="jp-word" title="だれ - 谁">誰</span> が <span class="jp-word" title="まど - 窗户">窓</span> <em>を</em> <span class="jp-word" title="あける - 打开">開けた</span> ？<br>
<span class="jp-trans">谁开的窗？</span>
6. <span class="jp-word" title="まど - 窗户">窓</span> <em>が</em> <span class="jp-word" title="どうして - 为什么">どうして</span> <span class="jp-word" title="あく - 被打开">開いた</span> ？<br>
<span class="jp-trans">窗户为什么开了？</span>

很重要的一点是，自动词<i>不能</i>带直接对象，因为没有直接的动作主体。下面的句子语法是错误的。

1. <span class="jp-word" title="でんき - 电；电灯">電気</span> <em>を</em> <span class="jp-word" title="つく - 被附上">ついた</span> 。<br>
<span class="jp-trans">（「を」应该换成「が」或者「は」）</span>
2. <span class="jp-word" title="でんき - 电；电灯">電気</span> <em>を</em> <span class="jp-word" title="きえる - 消失">消える</span> 。<br>
<span class="jp-trans">（「を」应该换成「が」或者「は」）</span>
3. <span class="jp-word" title="どうして - 为什么">どうして</span> <span class="jp-word" title="まど - 窗户">窓</span> <em>を</em> <span class="jp-word" title="あく - 被打开">開いた</span> ？<br>
<span class="jp-trans">（「を」应该换成「が」或者「は」）</span>

唯一可以和自动词合用助词「を」的场景：地点是动作动词的直接对象，这在上一节有简单说过。

1. <span class="jp-word" title="へや - 房间">部屋</span> <em>を</em> <span class="jp-word" title="でる - 出来；离开">出た</span> 。<br>
<span class="jp-trans">我离开了房间。</span>
