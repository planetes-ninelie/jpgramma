在[讲修饰关系从句的那一节](/grammar/clause/)里，我们学习了如何将关系从句作为形容词来修饰名词。现在我们把关系从句的作用扩展到作为动作的宾语。很显然，我们不能直接把「を」加到关系从句末尾，因为「を」只能用于名词词组。我们得想办法把关系从句封装成一个小单元以对其施加动作，也就是引用词组。

英语里面你用引号和一个逗号就可以进行引用，但日语是在引用结尾加「と」。这跟助词「と」以及条件语「と」是完全不一样的。与英语里面的引用不同，除了「他说」、「她说」之外，我们还可以在引用上做很多动作。比如我们可以用「认为」或者「听说」，这样就变成了「我认为（从句）」或者「我听说（从句）」。这在日语里面很重要，因为日本人很少断言什么事情。因此，我们在后面还会介绍很多种语法来表达不确定性及可能性。

## 直接引用 {#part2}

### 词汇学习

1. 言う 【い・う】 （う动词）- 说
2. 聞く 【き・く】 （う动词） - 问；听
3. 叫ぶ 【さけ・ぶ】 （う动词） - 惊叫
4. 呼ぶ 【よ・ぶ】 （う动词） - 呼叫
5. 呟く 【つぶや・く】 （う动词） - 嘀咕
6. 寒い 【さむ・い】 （い形容词） - 冷
7. 今日 【きょう】 - 今天
8. 授業 【じゅ・ぎょう】 - 班
9. 先生 【せん・せい】 - 老师
10. 田中 【た・なか】 - 田中（名）

我们先学最简单的引用法，也就是直接引用。基本上你就是在直接引用别人的话，只要在头尾加上引号，然后后面加上「と」以及合适的动词就可以了。这种引用最常搭配的动词就是「<span class="jp-word" title="いう- 说">言う</span>」和「<span class="jp-word" title="きく - 问；听">聞く</span>」了，不过你可以用其他动词例如「<span class="jp-word" title="さけぶ - 惊叫">叫ぶ</span>」、「<span class="jp-word" title="よぶ - 呼叫">呼ぶ</span>」和「<span class="jp-word" title="つぶやく - 嘀咕">呟く</span>」等等。这种引用经常出现在小说的对话，以及其他叙事体文章中。

### 示例

1. アリスが、 <em>「<span class="jp-word" title="さむい - 冷">寒い</span>」と</em> <span class="jp-word" title="いう- 说">言った</span> 。<br>
<span class="jp-trans">Alice 说「冷」。</span>
2. <em>「<span class="jp-word" title="きょう - 今天">今日</span>は<span class="jp-word" title="じゅぎょう - 班">授業</span>が<span class="jp-word" title="ある - 存在（无生命）">ない</span>」と</em> <span class="jp-word" title="せんせい - 老师">先生</span> から <span class="jp-word" title="きく - 问；听">聞いた</span> んだけど。<br>
<span class="jp-trans">我听老师说「今天没课」。</span>

这个动词不必紧跟关系从句，只要出现在其他动词前面就行。中间可以插入任何形容词、副词或者名词。

- <em>「<span class="jp-word" title="さむい - 冷">寒い</span>」と</em> アリスが <span class="jp-word" title="たなか - 田中">田中</span> に <em><span class="jp-word" title="いう- 说">言った</span></em> 。<br>
<span class="jp-trans">「冷」，Alice 对田中说。</span>

## 转述引用 {#part3}

### 词汇学习

1. 先生 【せん・せい】 - 老师
2. 今日 【きょう】 - 今天
3. 授業 【じゅ・ぎょう】 - 班
4. 聞く 【き・く】 （う动词） - 问；听
5. これ - 这
6. 日本語 【に・ほん・ご】 - 日语
7. 何 【なに／なん】 - 什么
8. 言う 【い・う】 （う动词）- 说
9. 私 【わたし】 - 我；我自己
10. カレー - 咖喱
11. 食べる 【た・べる】 （る动词） - 吃
12. 思う 【おも・う】 （う动词） - 思考，认为
13. 時間 【じ・かん】 - 时间
14. 今 【いま】- 现在
15. どこ - 哪里
16. 行く 【い・く】 （う动词） - 走
17. 考える 【かんが・える】 （る动词） - 考虑
18. 彼 【かれ】 - 他；男朋友
19. 高校生 【こう・こう・せい】 - 高中生
20. 信じる 【しん・じる】 （る动词） - 相信

第二种引用是转述别人说的话，它不是逐字逐句的引用，所以也不用加引号。你也可以用这种形式表达想法，只要用动词「<span class="jp-word" title="おもう - 思考，认为">思う</span>」，你就可以说你认为怎样怎样。你会随时听到日本人说这个词。你也可以用动词「<span class="jp-word" title="かんがえる - 考虑">考える</span>」表达你在考虑某事。

### 示例

1. <span class="jp-word" title="せんせい - 老师">先生</span> から <em><span class="jp-word" title="きょう - 今天">今日</span>は<span class="jp-word" title="じゅぎょう - 班">授業</span>が<span class="jp-word" title="ある - 存在（无生命）">ない</span>と</em> <span class="jp-word" title="きく - 问；听">聞いた</span> んだけど。<br>
<span class="jp-trans">我听老师说今天没课。</span>
2. <span class="jp-word" title="これ - 这">これ</span> は、 <span class="jp-word" title="にほんご - 日语">日本語</span> で <em><span class="jp-word" title="なん - 什么">何</span>と</em> <span class="jp-word" title="いう- 说">言います</span> か。<br>
<span class="jp-trans">这个用日语怎么说？（字面：关于这个，你用日语怎么说？）</span>
3. <span class="jp-word" title="わたし - 我；我自己">私</span> は、アリス <em>と<span class="jp-word" title="いう- 说">言います</span></em> 。<br>
<span class="jp-trans">别人叫我 Alice。（字面：至于我，你可以说 Alice）</span>

在转述引用里，「<span class="jp-word" title="いう- 说">言う</span>」的意思可能会变化，如第二句和第三句所示。但实际上，从字面直译可以看出，日语的意思还是一样的，只不过翻译成英语以后变化了（下一课我们会学习「<span class="jp-word" title="いう- 说">いう</span>」的更多用法。)

下面是一些使用引用的关系从句表达想法的例子。在第二句里，提问词和意向形组成了一个[嵌入的问题](/grammar/question/#part3)。

1. <span class="jp-word" title="カレー - 咖喱">カレー</span> を <span class="jp-word" title="たべる - 吃">食べよう</span> <em>と<span class="jp-word" title="おもう - 思考，认为">思った</span></em> けど、 <span class="jp-word" title="たべる - 吃">食べる</span> <span class="jp-word" title="じかん - 时间">時間</span> が <span class="jp-word" title="ある - 存在（无生命）">なかった</span> 。<br>
<span class="jp-trans">我想过出去吃咖喱，但是没有时间去吃。</span>
2. <span class="jp-word" title="いま- 现在">今</span> 、 <span class="jp-word" title="どこ - 哪里">どこ</span> に <span class="jp-word" title="いく - 走">行こう</span> か <em>と<span class="jp-word" title="かんがえる - 考虑">考えている</span></em> 。<br>
<span class="jp-trans">现在我在想该去哪里。</span>

与直接引用不同，虽然你可以逐字引用，但如果所引用的关系从句是名词或者な形容词形成的状态表示的话，你必须显式的加上表陈述的「だ」。

1. <span class="jp-word" title="かれ - 他">彼</span> は、 <span class="jp-word" title="これ - 这">これ</span> は <span class="jp-word" title="なん - 什么">何</span> <em>だと</em> <span class="jp-word" title="いう- 说">言いました</span> か。<br>
<span class="jp-trans">他说这 <em>是</em> 什么?</span>
2. <span class="jp-word" title="かれ - 他">彼</span> は <span class="jp-word" title="こうこうせい - 高中生">高校生</span> <em>だと</em> <span class="jp-word" title="きく - 问；听">聞いた</span> けど、 <span class="jp-word" title="しんじる - 相信">信じられない</span> 。<br>
<span class="jp-trans">我听说他 <em>是</em> 个高中生，但我没法相信。</span>

注意到显式声明状态的「だ」在译文里面高亮标出了。从下面两个例句的比较中你可以看出这个「だ」有多重要。

- <span class="jp-word" title="これ - 这">これ</span> は <span class="jp-word" title="なん - 什么">何</span> <em>だと</em> <span class="jp-word" title="いう- 说">言いました</span> か。<br>
<span class="jp-trans">（他）说这 <em>是</em> 什么?</span>

- <span class="jp-word" title="なん - 什么">何</span> <em>と</em> <span class="jp-word" title="いう- 说">言いました</span> か。<br>
<span class="jp-trans">（他）说什么？</span>

## 「と」的口语版本：「って」 {#part4}

### 词汇学习

1. 智子 【とも・こ】 - 智子（姓）
2. 来年 【らい・ねん】 - 明年
3. 海外 【かい・がい】 - 海外
4. もう - 已经
5. お金 【お・かね】 - 钱
6. ある （う动词） - 存在（无生命）
7. 本当 【ほん・とう】 - 真的
8. 明日 【あした】 - 明天
9. 雨 【あめ】 - 雨
10. 降る 【ふ・る】（う动词） - 沉淀
11. すごい （い形容词） - 很大程度上
12. いい （い形容词） - 好的
13. 人 【ひと】 - 人

你也许会觉得惊讶，引用关系从句还有更短更随意的版本？因为本来就只有一个假名「と」了啊。不过重点在于使用这种口语版本以后，你可以扔掉句子的其他部分，让对方靠上下文来明白你的意思。

### 示例

1. <span class="jp-word" title="ともこ - 智子">智子</span> は <span class="jp-word" title="らいねん - 明年">来年</span> 、 <span class="jp-word" title="かいがい - 海外">海外</span> に <span class="jp-word" title="いく - 走">行く</span> んだ <em>って</em> 。<br>
<span class="jp-trans">智子说她明年要出国。</span>
2. <span class="jp-word" title="もう - 已经">もう</span> お <span class="jp-word" title="おかね - 钱">金</span> が <span class="jp-word" title="ある - 存在（无生命）">ない</span> <em>って</em> 。<br>
<span class="jp-trans">我已经告诉你我没钱了。</span>
3. え？ <span class="jp-word" title="なん - 什么">何</span> だ <em>って</em> ？<br>
<span class="jp-trans">嗯？你说什么来着？</span>
4. <span class="jp-word" title="いま- 现在">今</span> 、 <span class="jp-word" title="じかん - 时间">時間</span> が <span class="jp-word" title="ある - 存在（无生命）">ない</span> <em>って</em> <span class="jp-word" title="きく - 问；听">聞いた</span> んだけど、 <span class="jp-word" title="ほんとう - 真的">本当</span> ？<br>
<span class="jp-trans">我听说你现在没时间，是吗？</span>
5. <span class="jp-word" title="いま- 现在">今</span> 、 <span class="jp-word" title="じかん - 时间">時間</span> が <span class="jp-word" title="ある - 存在（无生命）">ない</span> <em>って</em> 、 <span class="jp-word" title="ほんとう - 真的">本当</span> ？<br>
<span class="jp-trans">你现在没时间（听说），对吧？</span>

「って」 基本上可以用来谈论任何事情，不仅仅是引用别人的话。日常对话中，你随时可以听到日本人说「って」。大多数时候它是用来代替「は」以提出一个新的话题。

1. <span class="jp-word" title="あした - 明天">明日</span> <em>って</em> 、 <span class="jp-word" title="あめ - 雨">雨</span> が <span class="jp-word" title="ふる - 沉淀">降るん</span> だ <em>って</em> 。<br>
<span class="jp-trans">关于明天，我听说要下雨。</span>
2. アリス <em>って</em> 、 <span class="jp-word" title="すごい - 很大程度上">すごく</span> <span class="jp-word" title="いい - 好的">いい</span> <span class="jp-word" title="ひと - 人">人</span> でしょ？<br>
<span class="jp-trans">关于 Alice，她是个很好的人，对吧？</span>
