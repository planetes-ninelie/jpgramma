## 日本人喜欢礼物 {#part1}

### 词汇学习

1. お歳暮 【お・せい・ぼ】 - 年末礼
2. お中元 【お・ちゅう・げん】 - 盂兰盆节礼物
3. あげる （る动词） - 给；举
4. くれる （る动词） - 给
5. もらう （う动词） - 接受

日本人有个特点就是狠喜欢礼物，很多节日(<span class="jp-word" title="おせいぼ - 年末礼">お歳暮</span>、<span class="jp-word" title="おちゅうげん - 盂兰盆节礼物">お中元</span>等等)都有送礼和收礼的习俗。他们出去旅行一定会带纪念品回来送人。甚至在参加红白喜事的时候，也习惯送红包。你慢慢就会知道，学会如何表达给出和接受帮助、物品是很重要的。一直以来，「<span class="jp-word" title="あげる - 给，举">あげる</span>」、「<span class="jp-word" title="くれる - 给">くれる</span>」和「<span class="jp-word" title="もらう - 接受">もらう</span>」的用法总是让日语学习者伤透脑筋，觉得无从下手。但我认为从概念上来说，授受动词是很直观而简单的，希望我的讲解能证明这一点。

## 何时该用「<span class="jp-word" title="あげる - 给，举">あげる</span>」 {#part2}

### 词汇学习

1. あげる （る动词） - 给；举
2. 私 【わたし】 - 我；我自己
3. 友達 【とも・だち】- 朋友
4. プレゼント - 礼物
5. これ - 这
6. 先生 【せん・せい】 - 老师
7. 車 【くるま】 - 车
8. 買う 【か・う】 （う动词）- 买
9. 代わり 【か・わり】 - 代替
10. 行く 【い・く】 （う动词） - 走
11. 学生 【がく・せい】 - 学生
12. 父【ちち】 - 父亲
13. いい （い形容词） - 好的
14. こと - 事情
15. 教える 【おし・える】 （る动词） - 教；通知

「<span class="jp-word" title="あげる - 给，举">あげる</span>」是站在说话人角度，表示「给」的意思。当你想给别人东西，或者想为别人做什么的时候就必须用这个动词。

### 示例

1. <span class="jp-word" title="わたし - 我；我自己">私</span> が <span class="jp-word" title="ともだち- 朋友">友達</span> に <span class="jp-word" title="プレゼント - 礼物">プレゼント</span> を <em><span class="jp-word" title="あげる - 给，举">あげた</span></em> 。<br>
<span class="jp-trans">我给了朋友礼物。</span>
2. <span class="jp-word" title="これ - 这">これ</span> は <span class="jp-word" title="せんせい - 老师">先生</span> に <em><span class="jp-word" title="あげる - 给，举">あげる</span></em> 。<br>
<span class="jp-trans">我要把这个给老师。</span>

要想表达帮别人干什么的意思的话，就要用万能的动词て形再加上「<span class="jp-word" title="あげる - 给，举">あげる</span>」。这一点适用于本节中其他想表达帮忙的语法。

1. <span class="jp-word" title="くるま - 车">車</span> を <em><span class="jp-word" title="かう - 买">買って</span><span class="jp-word" title="あげる - 给，举">あげる</span></em> よ。<br>
<span class="jp-trans">我会帮你买车。</span>
2. <span class="jp-word" title="かわり - 代替">代わり</span> に <em><span class="jp-word" title="いく - 走">行って</span><span class="jp-word" title="あげる - 给，举">あげる</span></em> 。<br>
<span class="jp-trans">我会帮你顶替的。</span>

如果是第三者使用这个动词，那说话者是站在付出一方的角度上说的。后面学习「<span class="jp-word" title="くれる - 给">くれる</span>」的时候你就会明白这一点的重要性。

1. <span class="jp-word" title="がくせい - 学生">学生</span> が <span class="jp-word" title="これ - 这">これ</span> を <span class="jp-word" title="せんせい - 老师">先生</span> に <em><span class="jp-word" title="あげる - 给，举">あげる</span></em> 。<br>
<span class="jp-trans">学生把这个给了老师。（站在学生的角度）</span>
2. <span class="jp-word" title="ともだち- 朋友">友達</span> が <span class="jp-word" title="ちち - 父亲">父</span> に <span class="jp-word" title="いい - 好的">いい</span> <span class="jp-word" title="こと - 事情">こと</span> を <em><span class="jp-word" title="おしえる- 教，通知">教えて</span><span class="jp-word" title="あげる - 给，举">あげた</span></em> 。<br>
<span class="jp-trans">朋友帮忙教了我爸好东西。（站在朋友的角度）</span>

### 用「<span class="jp-word" title="やる - 做">やる</span>」表达「<span class="jp-word" title="あげる - 给，举">あげる</span>」的意思 {#yaru}

### 词汇学习

1. 犬 【いぬ】 - 狗
2. 餌 【えさ】 - 饵（动物吃的东西）
3. やる （う动词） - 做

对象是动物、宠物之类的时候，你可以用「<span class="jp-word" title="やる - 做">やる</span>」（通常意思是「做」）来代替「<span class="jp-word" title="あげる - 给，举">あげる</span>」。你可不能对人用「やる」，否则会有看不起人的意思，会冒犯别人。

- <span class="jp-word" title="いぬ - 狗">犬</span> に <span class="jp-word" title="えさ - 饵（动物吃的东西）">餌</span> を <em><span class="jp-word" title="やる - 做">やった</span></em> ？<br>
<span class="jp-trans">你投喂这只狗了吗？</span>

这里「<span class="jp-word" title="やる - 做">やる</span>」的意思不是「做」而是「给」。很明显，「对狗做食物」是没意义的。

## 何时该用「<span class="jp-word" title="くれる - 给">くれる</span>」 {#part3}

### 词汇学习

1. くれる （る动词） - 给
2. 友達 【とも・だち】- 朋友
3. 私 【わたし】 - 我；我自己
4. プレゼント - 礼物
5. これ - 这
6. 先生 【せん・せい】 - 老师
7. 車 【くるま】 - 车
8. 買う 【か・う】 （う动词）- 买
9. 代わり 【か・わり】 - 代替
10. 行く 【い・く】 （う动词） - 走
11. 学生 【がく・せい】 - 学生
12. 父【ちち】 - 父亲
13. いい （い形容词） - 好的
14. こと - 事情
15. 教える 【おし・える】 （る动词） - 教；通知
16. あげる （る动词） - 给；举
17. 全部 【ぜん・ぶ】 - 所有东西
18. 食べる 【た・べる】 （る动词） - 吃

「<span class="jp-word" title="くれる - 给">くれる</span>」是另一个表示「给」的东西，不过与「<span class="jp-word" title="あげる - 给，举">あげる</span>」不同，它是用在接受者身上的。<i>别人</i>给你什么，或者为你做什么的时候就必须用这个动词（其实就是「<span class="jp-word" title="あげる - 给，举">あげる</span>」的反方向）。

### 示例

1. <span class="jp-word" title="ともだち- 朋友">友達</span> が <span class="jp-word" title="わたし - 我；我自己">私</span> に <span class="jp-word" title="プレゼント - 礼物">プレゼント</span> を <em><span class="jp-word" title="くれる - 给">くれた</span></em> 。<br>
<span class="jp-trans">朋友给了我礼物。</span>
2. <span class="jp-word" title="これ - 这">これ</span> は、 <span class="jp-word" title="せんせい - 老师">先生</span> が <em><span class="jp-word" title="くれる - 给">くれた</span></em> 。<br>
<span class="jp-trans">老师把这个给了我。</span>
3. <span class="jp-word" title="くるま - 车">車</span> を <em><span class="jp-word" title="かう - 买">買って</span><span class="jp-word" title="くれる - 给">くれる</span></em> の？<br>
<span class="jp-trans">你会帮我买车吗？</span>
4. <span class="jp-word" title="かわり - 代替">代わり</span> に <em><span class="jp-word" title="いく - 走">行って</span><span class="jp-word" title="くれる - 给">くれる</span></em> ？<br>
<span class="jp-trans">能帮忙顶替我吗？</span>

类似的，第三者使用的时候，是站在接受者、而非赠与者的角度说的。

1. <span class="jp-word" title="せんせい - 老师">先生</span> が <span class="jp-word" title="これ - 这">これ</span> を <span class="jp-word" title="がくせい - 学生">学生</span> に <em><span class="jp-word" title="くれる - 给">くれる</span></em> 。<br>
<span class="jp-trans">老师把这个给了学生（站在学生的角度）。</span>
2. <span class="jp-word" title="ともだち- 朋友">友達</span> が <span class="jp-word" title="ちち - 父亲">父</span> に <span class="jp-word" title="いい - 好的">いい</span> <span class="jp-word" title="こと - 事情">こと</span> を <em><span class="jp-word" title="おしえる- 教，通知">教えて</span><span class="jp-word" title="くれる - 给">くれた</span></em> 。<br>
<span class="jp-trans">朋友帮忙教了我爸好东西。（站在父亲的角度）</span>

下面的图站在说话者角度说明了授受关系。

从说话者的角度来看，所有给出的东西都是「向上」的，而别人给自己的东西则是「向下」的。这也许可以解释，为何一个跟あげる完全相同的动词「<span class="jp-word" title="あげる - to raise">上げる</span>」（意思是「举起」）里面有<span class="jp-word" title="うえ - 上">上</span>字，而「<span class="jp-word" title="くれる - 给">くれる</span>」的敬语版本则是「<span class="jp-word" title="くださる - 给 （尊敬语）">下さる</span>」，里面带了个<span class="jp-word" title="した - below">下</span>字。这就有助于我们理解一些本来模棱两可的句子，例如：

- <span class="jp-word" title="せんせい - 老师">先生</span> が <span class="jp-word" title="おしえる- 教，通知">教えて</span> <em><span class="jp-word" title="あげる - 给，举">あげる</span></em> んですか。<br>
<span class="jp-trans">老师，你能不能帮忙教…（除了说话者以外的任何人）？</span>

因为所有作用于说话者的赠与都必须用「<span class="jp-word" title="くれる - 给">くれる</span>」，我们就能知道老师必然是要帮别人，但<i>除说话者</i>以外。另外，说话者此时也是站在老师的角度说的。

- <span class="jp-word" title="せんせい - 老师">先生</span> が <span class="jp-word" title="おしえる- 教，通知">教えて</span> <em><span class="jp-word" title="くれる - 给">くれる</span></em> んですか。<br>
<span class="jp-trans">老师，你能不能帮忙教…（任何人，包括说话者）？</span>

因为赠与者不是说话者，所以老师可以教别人，也可以教说话者。说话者此时是站在接受者的角度，等着接受老师给出的帮忙。

下面看一些可能犯的错误。

- <span class="jp-word" title="わたし - 我；我自己">私</span> が <span class="jp-word" title="ぜんぶ - 全部">全部</span> <span class="jp-word" title="たべる - 吃">食べて</span> <em><span class="jp-word" title="くれる - 给">くれました</span></em> 。<br>
<span class="jp-trans">「 <span class="jp-word" title="くれる - 给">くれる</span> 」这里用来表示说话者做出的赠与（错误）。</span>
- <span class="jp-word" title="わたし - 我；我自己">私</span> が <span class="jp-word" title="ぜんぶ - 全部">全部</span> <span class="jp-word" title="たべる - 吃">食べて</span> <em><span class="jp-word" title="あげる - 给，举">あげました</span></em> 。<br>
<span class="jp-trans">我帮忙全吃了（正确）。</span>

- <span class="jp-word" title="ともだち- 朋友">友達</span> が <span class="jp-word" title="プレゼント - 礼物">プレゼント</span> を <span class="jp-word" title="わたし - 我；我自己">私</span> に <em><span class="jp-word" title="あげる - 给，举">あげた</span></em> 。<br>
<span class="jp-trans">「 <span class="jp-word" title="あげる - 给，举">あげる</span> 」这里用来表示说话者接受的赠与（错误）。</span>
- <span class="jp-word" title="ともだち- 朋友">友達</span> が <span class="jp-word" title="プレゼント - 礼物">プレゼント</span> を <span class="jp-word" title="わたし - 我；我自己">私</span> に <em><span class="jp-word" title="くれる - 给">くれた</span></em> 。<br>
<span class="jp-trans">朋友给了我礼物（正确）。</span>

## 何时该用「<span class="jp-word" title="もらう - 接受">もらう</span>」 {#part4}

### 词汇学习

1. 私 【わたし】 - 我；我自己
2. 友達 【とも・だち】- 朋友
3. プレゼント - 礼物
4. もらう （う动词） - 接受
5. これ - 这
6. 買う 【か・う】 （う动词）- 买
7. 宿題 【しゅく・だい】 - 作业
8. チェック - 检查
9. する （例外） - 做
10. 時間 【じ・かん】 - 时间
11. ある （う动词） - 存在（无生命）
12. 無理 【む・り】 - 不可能
13. その - 那个 （それの的缩写）
14. 時計 【と・けい】 - 钟表

「<span class="jp-word" title="もらう - 接受">もらう</span>」的意思是「接受」，它只有这么一个版本，这跟「<span class="jp-word" title="あげる - 给，举">あげる</span>／<span class="jp-word" title="くれる - 给">くれる</span>」不同，所以没啥好说的。需要说明的是，既然是<i>从</i>别人那里接受了什么，那么除了搭配目标助词「に」，用「から」也是合适的。

### 示例

1. <span class="jp-word" title="わたし - 我；我自己">私</span> が <span class="jp-word" title="ともだち- 朋友">友達</span> <em>に</em> <span class="jp-word" title="プレゼント - 礼物">プレゼント</span> を <em><span class="jp-word" title="もらう - 接受">もらった</span></em> 。<br>
<span class="jp-trans">我从朋友那里收到了礼物。</span>
2. <span class="jp-word" title="ともだち- 朋友">友達</span> <em>から</em> <span class="jp-word" title="プレゼント - 礼物">プレゼント</span> を <em><span class="jp-word" title="もらう - 接受">もらった</span></em> 。<br>
<span class="jp-trans">我从朋友那里收到了礼物。</span>
3. <span class="jp-word" title="これ - 这">これ</span> は <span class="jp-word" title="ともだち- 朋友">友達</span> に <em><span class="jp-word" title="かう - 买">買って</span><span class="jp-word" title="もらう - 接受">もらった</span></em> 。<br>
<span class="jp-trans">关于这个，是朋友帮忙买的（字面：从朋友那里得到了买的帮助）。</span>
4. <span class="jp-word" title="しゅくだい - 作业">宿題</span> を <span class="jp-word" title="チェック - 检查">チェック</span> <em><span class="jp-word" title="する - 做">して</span><span class="jp-word" title="もらう - 接受">もらいたかった</span></em> けど、 <span class="jp-word" title="じかん - 时间">時間</span> が <span class="jp-word" title="ある - 存在（无生命）">なくて</span> <span class="jp-word" title="むり - 不可能">無理</span> だった。<br>
<span class="jp-trans">我想别人帮我检查作业，不过没时间了，所以不可能了。</span>

「もらう」是站在接受者的角度用的，所以对于第一人称的说话者来说，别人通常不是从你那里收到东西的。不过你可以用「<span class="jp-word" title="わたし - 我；我自己">私</span>から<span class="jp-word" title="もらう - 接受">もらう</span>」来强调这个事实。比如如果你想说，「喂，我<em>给了</em>你那东西！」，那可以用「<span class="jp-word" title="あげる - 给，举">あげる</span>」，但你也可以用「<span class="jp-word" title="もらう - 接受">もらう</span>」来表示，「喂，你是从我这里<em>得到</em>那东西的！」

- <span class="jp-word" title="その - 那个">その</span> <span class="jp-word" title="とけい - 手表">時計</span> は <span class="jp-word" title="わたし - 我；我自己">私</span> から <span class="jp-word" title="もらう - 接受">もらった</span> のよ。<br>
<span class="jp-trans">（他）从我这里收到了手表。</span>

## 用「<span class="jp-word" title="くれる - 给">くれる</span>」或「<span class="jp-word" title="もらう - 接受">もらえる</span>」请人帮忙 {#part5}

### 词汇学习

1. 千円 【せん・えん】 - 1,000 日元
2. 貸す 【か・す】 （う动词） - 借
3. する （例外） - 做
4. くれる （る动词） - 给
5. もらう （う动词） - 接受
6. あなた - 你
7. 私 【わたし】 - 我；我自己
8. ちょっと - 一点
9. 静か 【しず・か】 （な形容词）- 安静
10. 漢字 【かん・じ】 - 汉字
11. 書く 【か・く】 （う动词） - 写

想让别人帮忙，可以用「<span class="jp-word" title="くれる - 给">くれる</span>」以及「<span class="jp-word" title="もらう - 接受">もらう</span>」的可能形（我是否可以收到…的帮助）。我们已经在「<span class="jp-word" title="くれる - 给">くれる</span>」小节的第四个例句见过例子了。因为请求是从说话者发出的，所以这时候就不能用「<span class="jp-word" title="あげる - 给，举">あげる</span>」了。

### 示例

1. <span class="jp-word" title="せんえん - 1000日元">千円</span> を <span class="jp-word" title="かす - 借">貸して</span> <span class="jp-word" title="くれる - 给">くれる</span> ？<br>
<span class="jp-trans">能否帮忙借 1000 日元?</span>
2. <span class="jp-word" title="せんえん - 1000日元">千円</span> を <span class="jp-word" title="かす - 借">貸して</span> <span class="jp-word" title="もらう - 接受">もらえる</span> ？<br>
<span class="jp-trans">我能收到借出 1000 日元的帮忙吗?</span>

注意两个句子的意思是一样的，只不过赠与者和接受者被省略了，因为从上下文来看很明显。如果一定要写出来，那就是这样：

1. <em><span class="jp-word" title="あなた - 你">あなた</span>が</em> 、 <em><span class="jp-word" title="わたし - 我；我自己">私</span>に</em> <span class="jp-word" title="せんえん - 1000日元">千円</span> を <span class="jp-word" title="かす - 借">貸して</span> <span class="jp-word" title="くれる - 给">くれる</span> ？<br>
<span class="jp-trans">你能否帮忙借 1000 日元给我?</span>
2. <em><span class="jp-word" title="わたし - 我；我自己">私</span>が</em> 、 <em><span class="jp-word" title="あなた - 你">あなた</span>に</em> <span class="jp-word" title="せんえん - 1000日元">千円</span> を <span class="jp-word" title="かす - 借">貸して</span> <span class="jp-word" title="もらう - 接受">もらえる</span> ？<br>
<span class="jp-trans">我能收到你借出 1000 日元的帮忙吗?</span>

当面跟别人说话的时候通常不会这样把主语宾语说出来，但是要注意使用「<span class="jp-word" title="くれる - 给">くれる</span>」和「<span class="jp-word" title="もらう - 接受">もらえる</span>」的时候主语、宾语是相反的。

你可以用否定式来让请求变得温柔些。后面你会发现，很多语法里面都是这样的。

1. <span class="jp-word" title="ちょっと - 一点">ちょっと</span> <span class="jp-word" title="しずか- 安静">静か</span> に <span class="jp-word" title="する - 做">して</span> <em><span class="jp-word" title="くれる - 给">くれない</span></em> ？<br>
<span class="jp-trans">你可以安静一点吗？</span>
2. <span class="jp-word" title="かんじ - 汉字">漢字</span> で <span class="jp-word" title="かく - 写">書いて</span> <em><span class="jp-word" title="もらう - 接受">もらえません</span></em> か。<br>
<span class="jp-trans">你能帮我用汉字写这个吗？</span>

### 让别人别做什么 {#dontdo}

### 词汇学习

1. 全部 【ぜん・ぶ】 - 所有东西
2. 食べる 【た・べる】 （る动词） - 吃
3. くれる （る动词） - 给
4. 高い 【たか・い】 （い形容词） - 高
5. 物 【もの】 - 东西
6. 買う 【か・う】 （う动词）- 买

要想让别人<i>别</i>做什么，你只要把「で」加到动词否定式后面就可以了。

1. <span class="jp-word" title="ぜんぶ - 全部">全部</span> <span class="jp-word" title="たべる - 吃">食べない</span> <em>で</em> <span class="jp-word" title="くれる - 给">くれます</span> か。<br>
<span class="jp-trans">你能不能别全吃了？</span>
2. <span class="jp-word" title="たかい - expensive">高い</span> <span class="jp-word" title="もの - 东西">物</span> を <span class="jp-word" title="かう - 买">買わない</span> <em>で</em> <span class="jp-word" title="くれる - 给">くれる</span> ？<br>
<span class="jp-trans">你能不能别买贵的东西？</span>
