## 那些必须做和不能做的事 {#part1}

生活里面总有一些不得不做或不能做的事，例如倒垃圾，又例如做作业。下面我们来学习如何用日语来表达这些意思，它不但很有用，而且跟上一节紧密相关。本节最后，我们会介绍如何表达「你不必…」。

## 用「<span class="jp-word" title="だめ - 不好">だめ</span>」、「<span class="jp-word" title="いけない - 不能做；不好">いけない</span>」和「ならない」表示不能做的事 {#part2}

### 词汇学习

1. 駄目 【だめ】 - 不好
2. ここ - 这里
3. 入る 【はい・る】 （う动词） - 进入
4. それ - 那
5. 食べる 【たべ・る】 （る动词） - 吃
6. 夜 【よる】 - 夜
7. 遅い 【おそ・い】 （い形容词） - 迟
8. ～まで（助词）- 直到~
9. 電話 【でん・わ】 - 电话
10. する （例外） - 做
11. 早い 【はや・い】 （い形容词） - 快；早
12. 寝る 【ね・る】 （る动词） - 睡

先介绍一下「<span class="jp-word" title="だめ - 不好">だめ</span>」（<span class="jp-word" title="だめ - 不好">駄目</span>）这个单词，它的用法很多，但本质上的意思都是「不好」。这一小节里面另外两个关键单词是「<span class="jp-word" title="いけない - 不能做；不好">いけない</span>」和「ならない」，它们的意思基本上跟「<span class="jp-word" title="だめ - 不好">だめ</span>」是一样的，只不过「<span class="jp-word" title="いけない - 不能做；不好">いけない</span>」可以单独使用，而「ならない」则必须配合下面介绍的语法使用。另外虽然「<span class="jp-word" title="いけない - 不能做；不好">いけない</span>」和「ならない」和い形容词的活用规则类似，但它们其实并非形容词。下面介绍如何用这几个词表示不能做的事。

<div class="sumbox">
<div class="sumbox-title">如何说：不能做什么</div>

- 将动词活用为て形，加上助词「は」，再加上「 <span class="jp-word" title="だめ - 不好">だめ</span> 」、「 <span class="jp-word" title="いけない - 不能做；不好">いけない</span> 」或者「ならない」。<br>
<span class="jp-trans">例</span><br>
<span class="jp-trans"><span class="jp-word" title="はいる - 进入">入る</span> → <span class="jp-word" title="はいる - 进入">入って</span> + は + <span class="jp-word" title="だめ - 不好">だめ</span> ／ <span class="jp-word" title="いけない - 不能做；不好">いけない</span> ／ならない = <span class="jp-word" title="はいる - 进入">入って</span> は <span class="jp-word" title="だめ - 不好">だめ</span> ／ <span class="jp-word" title="はいる - 进入">入って</span> は <span class="jp-word" title="いけない - 不能做；不好">いけない</span> ／ <span class="jp-word" title="はいる - 进入">入って</span> はならない</span>
</div>

1. <span class="jp-word" title="ここ - 这里">ここ</span> に <em><span class="jp-word" title="はいる - 进入">入って</span>は</em> <span class="jp-word" title="いけない - 不能做；不好">いけません</span> 。<br>
<span class="jp-trans">你不能进到这里。</span>
2. <span class="jp-word" title="それ - 那">それ</span> を <em><span class="jp-word" title="たべる - 吃">食べて</span>は</em> <span class="jp-word" title="だめ - 不好">だめ</span> ！<br>
<span class="jp-trans">你不能吃那个！</span>
3. <span class="jp-word" title="よる - 夜">夜</span> 、 <span class="jp-word" title="おそい - 迟">遅く</span> <span class="jp-word" title="まで - until">まで</span> <span class="jp-word" title="でんわ - telephone">電話</span> <em><span class="jp-word" title="する - 做">して</span>は</em> ならない。<br>
<span class="jp-trans">晚上不能打电话到很迟。</span>
4. <span class="jp-word" title="はやい - 快；早">早く</span> <em><span class="jp-word" title="ねる - sleep">寝て</span>は</em> なりませんでした。<br>
<span class="jp-trans">（以前）不被允许早睡。</span>

「<span class="jp-word" title="だめ - 不好">だめ</span>」、「<span class="jp-word" title="いけない - 不能做；不好">いけない</span>」和「ならない」的区别在于「<span class="jp-word" title="だめ - 不好">だめ</span>」比较口语化，「<span class="jp-word" title="いけない - 不能做；不好">いけない</span>」和「ならない」基本上一样，但「ならない」更常用在那些适用于众人的要求，例如政策和规定。

## 表达必须做的事 {#part3}

### 词汇学习

1. 毎日 【まい・にち】 - 每天
2. 学校 【がっ・こう】 - 学校
3. 行く 【い・く】 （う动词） - 走
4. 宿題 【しゅく・だい】 - 作业
5. する （例外） - 做

你也许以为和「不能做」相反的意思应该用「<span class="jp-word" title="いける - is possible, will work out">いける</span>」或「<span class="jp-word" title="なる - 成为">なる</span>」来说吧，因为它们看起来像是「<span class="jp-word" title="いけない - 不能做；不好">いけない</span>」和「ならない」的反义词。可惜，「<span class="jp-word" title="いけない - 不能做；不好">いけない</span>」和「ならない」必须总是以否定的形式出现，所以这样说是不对的。实际上，我们还是要用「<span class="jp-word" title="だめ - 不好">だめ</span>／<span class="jp-word" title="いけない - 不能做；不好">いけない</span>／ならない」，但在前面用否定式动词。一开始你可能会不习惯这种双重否定的表达，但你要通过练习来学会适应。活用「<span class="jp-word" title="だめ - 不好">だめ</span>／<span class="jp-word" title="いけない - 不能做；不好">いけない</span>／ならない」前面动词的方法有三种，其中两种用到了上一节学的条件表达。有没有觉得我安排得很巧妙呢？

<div class="sumbox">
<div class="sumbox-title">如何说：必须做什么</div>

1. 否定式的て形 + 助词「は」 + <span class="jp-word" title="だめ - 不好">だめ</span> ／ <span class="jp-word" title="いけない - 不能做；不好">いけない</span> ／ならない
2. 动词否定式 + 表示条件的「と」 + <span class="jp-word" title="だめ - 不好">だめ</span> ／ <span class="jp-word" title="いけない - 不能做；不好">いけない</span> ／ならない
3. 动词否定式 + 表示条件的「ば」 + <span class="jp-word" title="だめ - 不好">だめ</span> ／ <span class="jp-word" title="いけない - 不能做；不好">いけない</span> ／ならない
</div>

第一种方法跟说「不能做」是一样的，只不过用的是动词的否定て形 。

1. <span class="jp-word" title="まいにち - 每天">毎日</span> <span class="jp-word" title="がっこう - 学校">学校</span> に <em><span class="jp-word" title="いく - 走">行かなくて</span>は</em> なりません。<br>
<span class="jp-trans">必须每天上学。</span>
2. <span class="jp-word" title="しゅくだい - 作业">宿題</span> を <em><span class="jp-word" title="する - 做">しなくて</span>は</em> <span class="jp-word" title="いけない - 不能做；不好">いけなかった</span><br>
<span class="jp-trans">不得不做了作业。</span>

第二种方法用了上一节学过的表达自然结果的条件语。字面上看，意思是如果你不做某事，那么自然就会带来不好的结果（换句话说你必须那么做）。不过实际生活中往往不限于这种自然结果的场景，人们用它只是因为它比其他两种简短罢了。

1. <span class="jp-word" title="まいにち - 每天">毎日</span> <span class="jp-word" title="がっこう - 学校">学校</span> に <em><span class="jp-word" title="いく - 走">行かない</span>と</em> <span class="jp-word" title="だめ - 不好">だめ</span> です。<br>
<span class="jp-trans">必须每天上学。</span>
2. <span class="jp-word" title="しゅくだい - 作业">宿題</span> を <em><span class="jp-word" title="する - 做">しない</span>と</em> <span class="jp-word" title="いけない - 不能做；不好">いけない</span><br>
<span class="jp-trans">不得不做了作业。</span>

第三种方法跟第二种类似，只不过用了另一种条件语「ば」，并且因此适用面更广。注意因为动词是否定式的，所以用条件语「ば」之前要把「い」去掉再加上「ければ」。

1. <span class="jp-word" title="まいにち - 每天">毎日</span> <span class="jp-word" title="がっこう - 学校">学校</span> に <em><span class="jp-word" title="いく - 走">行かなければ</span></em> <span class="jp-word" title="いけない - 不能做；不好">いけません</span> 。<br>
<span class="jp-trans">必须每天上学。</span>
2. <span class="jp-word" title="しゅくだい - 作业">宿題</span> を <em><span class="jp-word" title="する - 做">しなければ</span></em> <span class="jp-word" title="だめ - 不好">だめ</span> だった。<br>
<span class="jp-trans">不得不做了作业。</span>

貌似我一下子扔了一大堆知识点给你，因为三种语法再搭配「<span class="jp-word" title="だめ - 不好">だめ</span>／<span class="jp-word" title="いけない - 不能做；不好">いけない</span>／ならない」，我们就有了九种选择。不过一些用法肯定是比其他用法常见的，但我不告诉你，因为任意一种用法在语法上都是正确的，并且这么走马观花的介绍也不至于让你混淆。另外要记住，从活用规则角度来讲上面并没有任何新知识，上一节我们已经介绍了条件语，而第一种用法也已在第一小节介绍了。

### ※ 回到现实生活

尽管我们在上面介绍了「～なければ」和「～なくては」，但实际上因为它们太长了，日常对话中鲜有人用。虽然书面语中仍然要经常使用它们，但口语里面人们通常用条件语「と」或者下面将要介绍的简略说法。口语里面，「と」是最为通用的条件语，虽然我很详细的介绍了它的适用场景，但要知道人性本懒，有些东西知道就行了。

## 各种各样偷懒的说法 {#part4}

### 词汇学习

1. 勉強 【べん・きょう】 - 学习
2. する （例外） - 做
3. ご飯 【ご・はん】 - 米；饭
4. 食べる 【た・べる】 （る动词） - 吃
5. 学校 【がっ・こう】 - 学校
6. 行く 【い・く】 （う动词） - 走
7. ここ - 这里
8. 入る 【はい・る】 （う动词） - 进入
9. 駄目 【だめ】 - 不好
10. 死ぬ 【し・ぬ】 （う动词） - 死

也许你已经开始抱怨，为什么我只想表达要做什么，但却要说这么一长串！为了说「我必须…」，竟然要发八个音节的音！

哈，其他人也是这么想的，所以人们会在口语中用「なくては」和「なければ」的缩写版本。因为用起来简单，老师们一般也懒得教这些整天挂在嘴边的说法，不过当用缩略语不合适的时候你就知道学正经用法的必要性了。不过呢，要是不学这些口语表达，你可能又无法理解朋友们说的话（或者因此无法认识新朋友！）。所以请看下面的介绍，但要记住多练习正规表达方法，以便在不同的场合用合适的说法。

<div class="sumbox">
<div class="sumbox-title">表达必须做什么的口语说法</div>

1. 把「なくて」换成「なくちゃ」就行
2. 把「なければ」换成「なきゃ」就行
</div>

你可能会脱口而出「我去…」，因为看起来这些所谓「简写」的长度没什么变化呀。秘密在于，与前面学过的表达方法不同，你可以完全不用在后面加「<span class="jp-word" title="だめ - 不好">だめ</span>／<span class="jp-word" title="いけない - 不能做；不好">いけない</span>／ならない」！

1. <span class="jp-word" title="べんきょう - 学习">勉強</span> <span class="jp-word" title="する - 做">し<em>なくちゃ</em></span> 。<br>
<span class="jp-trans">得学习。</span>
2. <span class="jp-word" title="ごはん - meal">ご飯</span> を <span class="jp-word" title="たべる - 吃">食べ<em>なきゃ</em></span> 。<br>
<span class="jp-trans">该吃饭。</span>

条件语「と」本身也可以用来隐含表达「<span class="jp-word" title="だめ - 不好">だめ</span>／<span class="jp-word" title="いけない - 不能做；不好">いけない</span>／ならない」的意思。

- <span class="jp-word" title="がっこう - 学校">学校</span> に <span class="jp-word" title="いく - 走">行かない</span> <em>と</em> 。<br>
<span class="jp-trans">得去上学。</span>

还有一种用「ちゃ」来表示<u>不能</u>做的事。不过这里你就不能把「<span class="jp-word" title="だめ - 不好">だめ</span>／<span class="jp-word" title="いけない - 不能做；不好">いけない</span>／ならない」省掉了，但既然<i>是</i>口语，自然「<span class="jp-word" title="だめ - 不好">だめ</span>」用的最多。

这种口语表达形式有一点很重要，即那些以「む」、「ぶ」和「ぬ」结尾的动词要用「じゃ」而不是「ちゃ」。也就是说，那些过去形以「んだ」结尾的动词都要这么用。

<div class="sumbox">
<div class="sumbox-title">表达不能做什么的口语</div>

1. 把「ては」换成「ちゃ」
2. 把「では」换成「じゃ」
</div>

1. <span class="jp-word" title="ここ - 这里">ここ</span> に <em><span class="jp-word" title="はいる - 进入">入っちゃ</span></em> <span class="jp-word" title="だめ - 不好">だめ</span> だよ。<br>
<span class="jp-trans">你不能进这里。</span>
2. <em><span class="jp-word" title="しぬ - 死">死んじゃ</span></em> <span class="jp-word" title="だめ - 不好">だめ</span> だよ！<br>
<span class="jp-trans">你不能死！</span>

最后说明一点，总体来说「ちゃ」听起来比较可爱或者说娘娘腔，就像「ちゃん」一样。类似的「なくちゃ」听起来也是有点可爱或者说幼稚。

## 表达可以做或可以不做什么 {#part5}

### 词汇学习

1. 全部 【ぜん・ぶ】 - 所有东西
2. 食べる 【た・べる】 （る动词） - 吃
3. いい （い形容词） - 好的
4. 飲む 【の・む】 （う动词） - 喝
5. 大丈夫 【だい・じょう・ぶ】 （な形容词） - 没事
6. 構う 【かま・う】 （う动词） - 介意；关注
7. もう - 已经
8. 帰る 【かえ・る】 （う动词） - 回家
9. これ - 这
10. ちょっと - 就一点
11. 見る 【み・る】 （る动词） - 看

现在我们来学如何表达可以做什么，或可以不做什么。之所以放在这里介绍，是因为在日语里面我们要用它来表达你不必做什么事（也就是可以不做什么）。这种语法也相对简单，所以适合作为一个小节放这里。

把て形加上助词「も」，你就是在表达「即使做了什么…」，后面一般可以跟「<span class="jp-word" title="いい - 好的">いい</span>」、「<span class="jp-word" title="だいじょうぶ - 没事">大丈夫</span>」或「<span class="jp-word" title="かまう - 介意；关注">構わない</span>」。比如下面的例句。

1. <span class="jp-word" title="ぜんぶ - 全部">全部</span> <span class="jp-word" title="たべる - 吃">食べ<em>ても</em></span> <span class="jp-word" title="いい - 好的">いい</span> よ。<br>
<span class="jp-trans">你可以放心全吃了（字面：即使你全吃了，也可以，嗯）</span>
2. <span class="jp-word" title="ぜんぶ - 全部">全部</span> <span class="jp-word" title="たべる - 吃">食べなく<em>ても</em></span> <span class="jp-word" title="いい - 好的">いい</span> よ。<br>
<span class="jp-trans">你不用全吃了（字面：即使你不全吃掉，也可以，嗯）</span>
3. <span class="jp-word" title="ぜんぶ - 全部">全部</span> <span class="jp-word" title="のむ - 喝">飲ん<em>でも</em></span> <span class="jp-word" title="だいじょうぶ - 没事">大丈夫</span> だよ。<br>
<span class="jp-trans">全喝了吧没事（字面：即使你全喝了，也没问题，嗯）</span>
4. <span class="jp-word" title="ぜんぶ - 全部">全部</span> <span class="jp-word" title="のむ - 喝">飲ん<em>でも</em></span> <span class="jp-word" title="かまう - 介意；关注">構わない</span> よ。<br>
<span class="jp-trans">我不介意你全喝掉（字面：即使你全喝了，我也不介意，嗯）</span>

口语里面，「～ても<span class="jp-word" title="いい - 好的">いい</span>」有时候还省略成成「～て<span class="jp-word" title="いい - 好的">いい</span>」（或「～で<span class="jp-word" title="いい - 好的">いい</span>」而非「～でも<span class="jp-word" title="いい - 好的">いい</span>」 ）。

1. <span class="jp-word" title="もう - 已经">もう</span> <span class="jp-word" title="かえる - 回家">帰っ<em>て</em></span> <em><span class="jp-word" title="いい - 好的">いい</span></em> ？<br>
<span class="jp-trans">我是不是已经可以回家了？</span>
2. <span class="jp-word" title="これ - 这">これ</span> 、 <span class="jp-word" title="ちょっと - 就一点">ちょっと</span> <span class="jp-word" title="みる - 看">見<em>て</em></span> <em><span class="jp-word" title="いい - 好的">いい</span></em> ？<br>
<span class="jp-trans">能否让我扫一眼这个？</span>
