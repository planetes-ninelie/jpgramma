## 表达做某事的能力 {#part1}

日语里面不像英语那样通过加上 "can"、"able" 这样的单词来表达做某事的能力，而是直接活用动词。所有动词活用为可能形之后都变为る动词。

## 可能形 {#part2}

### 词汇学习

1. 見る 【み・る】 （る动词） - 看
2. 遊ぶ 【あそ・ぶ】 （う动词） - 玩
3. する （例外） - 做
4. 来る 【く・る】 （例外）- 来
5. 出来る 【で・き・る】 （る动词） - 做好；做出；出现
6. 食べる 【た・べる】 （る动词） - 吃
7. 着る 【き・る】 （る动词） - 穿
8. 信じる 【しん・じる】 （る动词） - 相信
9. 寝る 【ね・る】 （る动词） - 睡
10. 起きる 【お・きる】 （る动词） - 起
11. 出る 【で・る】 る动词 - 出来
12. 掛ける 【か・ける】 （る动词） - 挂
13. 調べる 【しら・べる】 （る动词） - 调查
14. 話す 【はな・す】 （う动词） - 说
15. 書く 【か・く】 （う动词） - 写
16. 待つ 【ま・つ】 （う动词） - 等待
17. 飲む 【の・む】 （う动词） - 喝
18. 取る 【と・る】 （う动词） - 取
19. 死ぬ 【し・ぬ】 （う动词） - 死
20. 買う 【か・う】 （う动词）- 买
21. 漢字 【かん・じ】 - 汉字
22. 残念 【ざん・ねん】 （な形容词） - 不幸的
23. 今週末 【こん・しゅう・まつ】 - 这周末
24. 行く 【い・く】 （う动词） - 走
25. もう - 已经

可能形的活用规则又要按る动词, う动词和例外动词来分别讨论。不过，动词「<span class="jp-word" title="する - 做">する</span>」（「做」）)的可能形又是一个特殊的例外，因为它要被一个完全不同的单词所替代：「<span class="jp-word" title="できる -  做好；做出；出现">できる</span>」 （<span class="jp-word" title="できる -  做好；做出；出现">出来る</span>）

<div class="sumbox">
<div class="sumbox-title">可能形活用规则</div>

- <b>对る动词：</b> 将「る」替换为为「られる」。<br>
<span class="jp-trans">例： <span class="jp-word" title="みる - 看">見<em>る</em></span> → <span class="jp-word" title="みる - 看">見<em>られる</em></span></span>
- <b>对う动词：</b> 将最后的假名从 / u / 段改为同行 / e / 段假名，再加上「る」。<br>
<span class="jp-trans">例： <span class="jp-word" title="あそぶ - 玩">遊<em>ぶ</em></span> → <span class="jp-word" title="あそぶ - 玩">遊<em>べ</em></span> → <span class="jp-word" title="あそぶ - 玩">遊べ<em>る</em></span></span>
- <b>例外：</b>
  1. 「 <span class="jp-word" title="する - 做">する</span> 」变为「 <span class="jp-word" title="できる -  做好；做出；出现">できる</span> 」
  2. 「 <span class="jp-word" title="くる- 来">くる</span> 」变为「 <span class="jp-word" title="くる- 来">こられる</span> 」
※ 记住，所有可能形动词都是る动词。
</div>

对于る动词，也可以把「られる」简写为「れる」。例如，「<span class="jp-word" title="たべる - 吃">食べる</span>」的可能形可以是「<span class="jp-word" title="たべる - 吃">食べれる</span>」，这是「<span class="jp-word" title="たべる - 吃">食べられる</span>」的简写。但我建议还是先学习正式的活用方式「られる」，因为一旦养成了懒惰的习惯就很难改过来了，并且这种缩写虽然常用，却被认为是俚语。

### 示例

1. <span class="jp-word" title="かんじ - 汉字">漢字</span> は <em><span class="jp-word" title="かく - 写">書けます</span>か</em> ？<br>
<span class="jp-trans">你会写汉字吗？</span>
2. <span class="jp-word" title="ざんねん - 不幸的">残念</span> だが、 <span class="jp-word" title="こんしゅうまつ - 这周末">今週末</span> は <em><span class="jp-word" title="いく - 走">行けない</span></em> 。<br>
<span class="jp-trans">不幸的是，我这周末不能去了。</span>
3. <span class="jp-word" title="もう - 已经">もう</span> <em><span class="jp-word" title="しんじる - 相信">信じられない</span></em> 。<br>
<span class="jp-trans">我已经不相信了。</span>

## 可能形动词不会有直接对象 {#part3}

### 词汇学习

1. 富士山 【ふ・じ・さん】 - 富士山
2. 登る 【のぼ・る】 （う动词） - 登
3. 重い 【おも・い】 （い形容词） - 重
4. 荷物 【に・もつ】 - 行李
5. 持つ 【も・つ】 （う动词） - 拿，带

可能形表达的是可能性，但实际的动作并没有发生。所以尽管可能形动词仍然是动词，但它描述的是状态，所以你不能用直接对象助词「を」。下面是错误的用法举例。

1. <span class="jp-word" title="ふじさん - 富士山">富士山</span> <em>を</em> <span class="jp-word" title="のぼる - 登">登れた</span> 。
2. <span class="jp-word" title="おもい - 重">重い</span> <span class="jp-word" title="にもつ - 行李">荷物</span> <em>を</em> <span class="jp-word" title="もつ - 拿，带">持てます</span> 。

正确的说法应该是：

1. <span class="jp-word" title="ふじさん - 富士山">富士山</span> <em>が</em> <span class="jp-word" title="のぼる - 登">登れた</span> 。<br>
<span class="jp-trans">曾经可以爬富士山。</span>
2. <span class="jp-word" title="おもい - 重">重い</span> <span class="jp-word" title="にもつ - 行李">荷物</span> <em>は</em> <span class="jp-word" title="もつ - 拿，带">持てます</span> 。<br>
<span class="jp-trans">可以拿重的行李。</span>

## 「<span class="jp-word" title="みえる - 变得可见">見える</span>」和「<span class="jp-word" title="きこえる - 变得可听见">聞こえる</span>」是例外吗？ {#part4}

### 词汇学习

1. 見える 【み・える】 （る动词） - 变得可见
2. 聞こえる 【き・こえる】 （る动词） - 变得可听见
3. 今日 【きょう】 - 今天
4. 晴れる 【は・れる】 （る动词） - 变晴
5. 富士山 【ふ・じ・さん】 - 富士山
6. 友達 【とも・だち】- 朋友
7. おかげ - 多亏
8. 映画 【えい・が】 - 电影
9. ただ - 免费；仅仅
10. 見る 【み・る】 （る动词） - 看
11. こと - 事情
12. 出来る 【で・き・る】 （る动词） - 做好；做出；出现
13. 久しぶり 【ひさ・しぶり】 - 好久以后
14. 彼 【かれ】 - 他；男朋友
15. 声 【こえ】 - 声音
16. 聞く 【き・く】 （う动词） - 问；听
17. 周り 【まわ・り】 - 周围
18. うるさい （い形容词） - 吵闹
19. 言う 【い・う】 （う动词）- 说
20. あまり／あんまり - 不是很（用在否定句）

「<span class="jp-word" title="みえる - 变得可见">見える</span>」和「<span class="jp-word" title="きこえる - 变得可听见">聞こえる</span>」这两个单词的意思分别表示某物是可见的、可听见的。如果你想说你可以看到或者听到什么，就得用这个两个词。但是，如果你想说的是你被赋予看见或听见某物的机会的话，就得用普通的可能形，不过这种情况下，一般会用例三的表达方式。

### 示例

1. <span class="jp-word" title="きょう - 今天">今日</span> は <span class="jp-word" title="はれる - 变晴">晴れて</span> 、 <span class="jp-word" title="ふじさん - 富士山">富士山</span> が <span class="jp-word" title="みえる - 变得可见">見える</span> 。<br>
<span class="jp-trans">今天放晴了，富士山都能看得见。</span>
2. <span class="jp-word" title="ともだち- 朋友">友達</span> の <span class="jp-word" title="おかげ - 多亏">おかげ</span> で、 <span class="jp-word" title="えいが - 电影">映画</span> は <span class="jp-word" title="ただ - 免费">ただ</span> で <span class="jp-word" title="みる - 看">見られた</span> 。<br>
<span class="jp-trans">多亏了（我的）朋友，（我）才有机会免费看了电影。</span>
3. <span class="jp-word" title="ともだち- 朋友">友達</span> の <span class="jp-word" title="おかげ - 多亏">おかげ</span> で、 <span class="jp-word" title="えいが - 电影">映画</span> を <span class="jp-word" title="ただ - 免费">ただ</span> で <span class="jp-word" title="みる - 看">見る</span> <span class="jp-word" title="こと - 事情">こと</span> が <span class="jp-word" title="できる -  做好；做出；出现">できた</span> 。<br>
<span class="jp-trans">多亏了（我的）朋友，（我）才有机会免费看了电影。</span>

可以看到例三用了通用名词来表示一件事，字面意思就是「看电影这件事得以被完成」，这本质上跟「<span class="jp-word" title="みる - 看">見られる</span>」的意思是一样的。你也可以这么说：

1. <span class="jp-word" title="ともだち- 朋友">友達</span> の <span class="jp-word" title="おかげ - 多亏">おかげ</span> で、 <span class="jp-word" title="えいが - 电影">映画</span> を <span class="jp-word" title="ただ - 免费">ただ</span> で <span class="jp-word" title="みる - 看">見る</span> <em>の</em> が <span class="jp-word" title="できる -  做好；做出；出现">できた</span> 。

下面再用「<span class="jp-word" title="きく - 听；问">聞く</span>」造几个句子，能看出区别来吗？注意「<span class="jp-word" title="きこえる - 变得可听见">聞こえる</span>」都是「可听见」而不是「能问」的意思。

1. <span class="jp-word" title="ひさしぶり - 好久以后">久しぶり</span> に <span class="jp-word" title="かれ - 他">彼</span> の <span class="jp-word" title="こえ - 声音">声</span> が <em><span class="jp-word" title="きく - 听；问">聞けた</span></em> 。<br>
<span class="jp-trans">这么久以来我第一次能听到他的声音。</span>
2. <span class="jp-word" title="まわり - 周围">周り</span> が <span class="jp-word" title="うるさい - 吵闹">うるさくて</span> 、 <span class="jp-word" title="かれ - 他">彼</span> が <span class="jp-word" title="いう- 说">言っている</span> <span class="jp-word" title="こと - 事情">こと</span> が <span class="jp-word" title="あんまり - 不是很">あんまり</span> <em><span class="jp-word" title="きこえる - 变得可听见">聞こえなかった</span></em> 。<br>
<span class="jp-trans">周围很吵，他说什么我听得不是很清楚。</span>

## 「<span class="jp-word" title="ある - 存在（无生命）">ある</span>」，另一个例外 {#part5}

### 词汇学习

1. そんな - 那种
2. こと - 事情
3. 有り得る 【あ・り・え・る／あ・り・う・る】 （る动词） - 可能存在
4. 彼 【かれ】 - 他；男朋友
5. 寝坊【ね・ぼう】 - 睡过头
6. する （例外） - 做
7. それ - 那
8. 話 【はなし】 - 故事

如果想说某物可能存在，你可以把「<span class="jp-word" title="ある - 存在（无生命）">ある</span>」和动词「<span class="jp-word" title="える - 获得">得る</span>」组合起来得到「<span class="jp-word" title="ありうる／ありえる - 可能">あり得る</span>」，这本质上的意思是「<span class="jp-word" title="ある - 存在（无生命）">ある</span><span class="jp-word" title="こと - 事情">こと</span>が<span class="jp-word" title="できる -  做好；做出；出现">できる</span>」，只不过没人这么说，大家都直接说「<span class="jp-word" title="ありうる／ありえる - 可能">あり得る</span>」。这个动词很奇怪，因为既可以读作「<span class="jp-word" title="ありうる - 可能">ありうる</span>」，也可以读作「<span class="jp-word" title="ありえる - 可能">ありえる</span>」。<i>但是</i>其活用形例如「<span class="jp-word" title="ありえる - 可能">ありえない</span>」、「<span class="jp-word" title="ありえる - 可能">ありえた</span>」和「<span class="jp-word" title="ありえる - 可能">ありえなかった</span>」都只有一种读法，中间是「え」。

### 示例

1. <span class="jp-word" title="そんな - 那种">そんな</span> <span class="jp-word" title="こと - 事情">こと</span> は <em><span class="jp-word" title="ありうる - 可能">ありうる</span></em> 。<br>
<span class="jp-trans">那种事情是可能的（字面：可能存在）。</span>
2. <span class="jp-word" title="そんな - 那种">そんな</span> <span class="jp-word" title="こと - 事情">こと</span> は <em><span class="jp-word" title="ありえる - 可能">ありえる</span></em> 。<br>
<span class="jp-trans">那种事情是可能的（字面：可能存在）。</span>
3. <span class="jp-word" title="そんな - 那种">そんな</span> <span class="jp-word" title="こと - 事情">こと</span> は <em><span class="jp-word" title="ありえる - 可能">ありえない</span></em> 。<br>
<span class="jp-trans">那种事情是不可能的（字面：不可能存在）。</span>
4. <span class="jp-word" title="かれ - 他；男朋友">彼</span> が <span class="jp-word" title="ねぼう - 睡过头">寝坊</span> <span class="jp-word" title="する - 做">した</span> <span class="jp-word" title="こと - 事情">こと</span> も <em><span class="jp-word" title="ありうる - 可能">ありうる</span></em> ね。<br>
<span class="jp-trans">也可能是他睡过头了（字面：他睡过头了那种情况也可能存在）。</span>
5. <span class="jp-word" title="それ - 那">それ</span> は、 <em><span class="jp-word" title="ありえる - 可能">ありえない</span></em> <span class="jp-word" title="はなし - 故事">話</span> だよ。<br>
<span class="jp-trans">那是不可能的事（字面：那事不可能存在）。</span>
