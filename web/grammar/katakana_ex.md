## 把英语单词改写成片假名 {#part4}

为了好玩，我们来试着猜一下一些单词的片假名写法。我在下面列了一些常见的模式，但这只是辅助用，有些单词并不适用。

你也知道，因为日语发音基本上都是「辅音加元音」的组合，按此派生出来的英语单词会出现问题，你也许已经注意到这一点了。

看过《迷失东京》你就知道， / l / 和 / r / 是无法区分的。

（１） Ready -> <em>レ</em>ディ<br>
<span class="jp-trans">（２） Lady -> <em>レ</em>ディ</span>

对于连续的元音以及以 / r / 结尾的原因，日语里通常变为长元音。

（１） <em>Tar</em>get -> <em>ター</em>ゲット<br>
<span class="jp-trans">（２） <em>Shoo</em>t -> <em>シュー</em>ト</span>

由 / t / 或 / c / 产生的去声发音使用小号「ッ」表示。

（１） Ca<em>t</em>ch -> キャ<em>ッ</em>チ<br>
<span class="jp-trans">（２） Ca<em>c</em>he -> キャ<em>ッ</em>シュ</span>

以辅音结尾的单词需要用另一个元音来补充「辅音加元音」的结构（"n" 和 "m" 除外，因为我们有「ン」）<br>
<span class="jp-trans">对于 "t" 和 "d" ，通常用 "o" 补充；其他基本上用 "u"。</span>

（１） Goo<em>d</em> -> グッ<em>ド</em><br>
<span class="jp-trans">（２） To<em>p</em> -> トッ<em>プ</em></span><br>
<span class="jp-trans">（３） Jac<em>k</em> -> ジャッ<em>ク</em></span>
