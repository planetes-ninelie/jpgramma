到现在为止，我们一直在讲已经发生的事情或者因某些事情而发生的改变。现在我们要学一些简单的语法来表达<i>缺少</i>改变的意思。

## 用「まま」表示没有变化 {#part2}

### 词汇学习

1. この - 这个（これの的缩写）
2. 宜しい 【よろ・しい】 （い形容词） - 好的 （正式）
3. 半分 【はん・ぶん】 - 一半
4. 食べる 【た・べる】 （る动词） - 吃
5. 捨てる 【す・てる】 （る动词） - 扔掉
6. 駄目 【だめ】 - 不好
7. いる （る动词） - 存在（有生命）
8. 今日 【きょう】 - 今天
9. 悲しい 【かな・しい】 （い形容词） - 悲伤
10. その - 那个 （それの的缩写）
11. 格好 【かっ・こう】 - 样子
12. クラブ - 俱乐部；夜店
13. 入る 【はい・る】 （う动词） - 进入

不要把「まま」跟小孩子叫的妈妈（ママ）搞混了，这是一个用来表示某物缺少改变的词。语法上看，它算是一个名词。如果你在便利店买一个小东西，就很可能听到这个词。因为店员通常都会说很快的超尊敬敬语，学习这个表达方式会让你有所准备（当然了，当听不明白的时候，人一般会重复同样的词组…用同样的速度）

### 示例

- <span class="jp-word" title="この - 这">この</span> <em>まま</em> で <span class="jp-word" title="よろしい - 好的">宜しい</span> ですか？<br>
<span class="jp-trans"><em>像这样</em> 就可以吗？</span>

说这句话时，店员想要知道你是否要胶袋，还是直接带走。如果我没讲过「<span class="jp-word" title="よろしい - 好的">宜しい</span>」的话，这其实就是「<span class="jp-word" title="いい - 好的">いい</span>」的敬语版本。注意到「まま」语法上来说跟名词一样，所以你可以用动词短语或形容词修饰它。

- <span class="jp-word" title="はんぶん - 一半">半分</span> しか <em><span class="jp-word" title="たべる - 吃">食べてない</span>まま</em> で <span class="jp-word" title="すてる - 扔掉">捨てちゃ</span> <span class="jp-word" title="だめ - 坏；不好的">ダメ</span> ！<br>
<span class="jp-trans">你不能吃一半就扔掉！</span>

上面的意思是食物处在一个吃了一半的不变状态的时候，不能把它扔掉。

下面是我 Google 到的一个很好的例子<br>
<span class="jp-trans">提示：「<span class="jp-word" title="いる - 存在（有生命）">いさせる</span>」是「<span class="jp-word" title="いる - 存在（有生命）">いる</span>」的使役形，意思是「让我存在」。</span>

- <span class="jp-word" title="きょう - 今天">今日</span> だけは <em><span class="jp-word" title="かなしい - 悲伤">悲しい</span>まま</em> で <span class="jp-word" title="いる - 存在（有生命）">いさせて</span> <span class="jp-word" title="ほしい - want">ほしい</span> 。<br>
<span class="jp-trans">你让我自己伤心吧，就今天。</span>

最后顺便举一个用名词来修饰它的例子。

- <span class="jp-word" title="その - 那个">その</span> <em><span class="jp-word" title="かっこう - 样子">格好</span>のまま</em> で <span class="jp-word" title="クラブ - 俱乐部">クラブ</span> に <span class="jp-word" title="はいる - 进入">入れない</span> よ。<br>
<span class="jp-trans">你不能穿成那样（不换）就去俱乐部。</span>

## 用「っぱなし」让某物保持不变 {#part3}

### 词汇学习

1. 放す 【はな・す】 （う动词） - 放
2. くれる （る动词） - 给
3. ほったらかす （う动词） - 忽视
4. テレビ - 电视机
5. 開ける 【あ・ける】 （る动词） - 开
6. 書く 【か・く】 （う动词） - 写
7. つける （る动词） - 附加；打开
8. する （例外） - 做
9. 眠れる 【ねむ・れる】 （る动词） - 睡着
10. 人 【ひと】 - 人
11. 結構 【けっ・こう】 - 相当的，合理的
12. いる （る动词） - 存在（有生命）
13. 窓 【まど】 - 窗户
14. 蚊 【か】 - 蚊子
15. いっぱい - 满
16. 入る 【はい・る】 （う动词） - 进入
17. しまう （う动词） - 无意的举动，完全完成

动词「<span class="jp-word" title="はなす - 放">放す</span>」的意思是「放开」，它可以被用在很多地方，表示让某物保持不变。比如它的变形「<span class="jp-word" title="ほっとく - 留下">放っとく</span>」用在你想说类似「别管我（Leave me alone）」的地方——比如用请求（<span class="jp-word" title="くれる - 给">くれる</span>）的命令形说「<span class="jp-word" title="ほっとく - 留下">ほっといて</span><span class="jp-word" title="くれる - 给">くれ</span>！」（别管我！）。另一个变形「<span class="jp-word" title="ほったらかす - 忽视">ほったらかす</span>」的意思是「忽视」。

我在这里想介绍的语法是后缀「っぱなし」，你可以把它加到任何动词词根后面来形容做某件事，然后保持那个状态。新的单词可以当作名词使用。

这个链接有更多[这个语法的示例](http://jisho.org/sentences?jap=っぱなし)。可以看到，它暗示保持原样是因为忽视。下面是这个语法的一些（简单的）活用规则。

<div class="sumbox">
<div class="sumbox-title">用「っぱなし」来完成一个动作，然后保持原样</div>

- 词根后面加上「っぱなし」.<br>
<span class="jp-trans">例</span>
  1. <span class="jp-word" title="あける - 开">開け<em>る</em></span> → <span class="jp-word" title="あける - 开">開け</span> <em>っぱなし</em>
  2. <span class="jp-word" title="かく - 写">書<em>く</em></span> → <span class="jp-word" title="かく - 写">書<em>き</em></span> → <span class="jp-word" title="かく - 写">書き</span> <em>っぱなし</em>
</div>

### 示例

1. <span class="jp-word" title="テレビ - 电视">テレビ</span> を <em><span class="jp-word" title="つける - 附加；打开">つけ</span>っぱなし</em> に <span class="jp-word" title="する - 做">しなければ</span> <span class="jp-word" title="ねむれる - 睡着">眠れない</span> <span class="jp-word" title="ひと - 人">人</span> は、 <span class="jp-word" title="けっこう - 相当">結構</span> <span class="jp-word" title="いる - 存在（有生命）">いる</span> 。<br>
<span class="jp-trans">很多人非得开着电视才能睡着。</span><br>
<span class="jp-trans">（译者注：好难啊——2/22/2014）</span>
2. <span class="jp-word" title="まど - 窗户">窓</span> が <em><span class="jp-word" title="あける - 开">開け</span>っ放し</em> だったので、 <span class="jp-word" title="か - 蚊子">蚊</span> が <span class="jp-word" title="いっぱい - 很多; full">いっぱい</span> <span class="jp-word" title="はいる - 进入">入って</span> しまった。<br>
<span class="jp-trans">窗户就这么开着，很多蚊子飞了进来。</span>
