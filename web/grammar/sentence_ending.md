8. 込む 【こ・む】 （う动词） - 变得拥挤

学过「よ」和「ね」之后，「さ」和「な」是使用频率仅次于前两者的常用句尾助词。

「さ」基本上是「よ」的一种更加随意的说法，意思跟英语里面的 "like" 比较像，有些人习惯于在每句话后面都带这么个词。当然了，这并不一定是一种很熟练的说话技巧，不过不得不承认，它还是满顺口的。由于被滥用，这个助词本身已经没什么具体意义了，你也许听到过类似以下的对话：

Ａ：<span class="jp-word" title="あの - 喂">あの</span><em>さ</em>・・・<br>
<span class="jp-trans">A：那啥…</span>

Ｂ：<span class="jp-word" title="うん - 嗯">うん</span>。<br>
<span class="jp-trans">B：嗯。</span>

Ａ：<span class="jp-word" title="このあいだ - 不久前">この間</span><em>さ</em>・・・<br>
<span class="jp-trans">A：前两天呢….</span>

Ｂ：<span class="jp-word" title="うん - 嗯">うん</span>。<br>
<span class="jp-trans">B：嗯。</span>

Ａ：<span class="jp-word" title="ディズニーランド - 迪士尼乐园">ディズニーランド</span>に<span class="jp-word" title="いく - 走">行った</span>んだけど<em>さ</em>、<span class="jp-word" title="なんか - something like that">なんか</span><em>さ</em>、<span class="jp-word" title="すごい - 很大程度上">すごい</span><span class="jp-word" title="こむ - 变得拥挤">込んで</span>て・・・<br>
<span class="jp-trans">A：我去了迪斯尼乐园，人好多…</span>

Ｂ：<span class="jp-word" title="うん - 嗯">うん</span><br>
<span class="jp-trans">B：嗯。</span>

Ａ：<span class="jp-word" title="なにも - 什么也没有">何も</span><span class="jp-word" title="できる -  做好；做出；出现">できなくて</span><em>さ</em>・・・<br>
<span class="jp-trans">A：什么也没玩到，你懂的…</span>

对话还可以继续下去，B 也许会突然开始回应。

如果觉得对于你想说的话或者你的听众来说，「ね」听起来太软，你也可以用「な」代替它。通常是男性会这么用，不过也不是那么严格。

### 例一

<span class="jp-word" title="ようすけ - 洋介">洋介</span>：<span class="jp-word" title="いま- 现在">今</span>、<span class="jp-word" title="としょかん - 图书馆">図書館</span>に<span class="jp-word" title="いく - 走">行く</span>んだよ<em>な</em>。<br>
<span class="jp-trans">洋介：你这是要去图书馆呢啊？（想让对方解释）</span>

<span class="jp-word" title="ともこ - 智子">智子</span>：<span class="jp-word" title="うん - 嗯">うん</span>、<span class="jp-word" title="なんで - 为什么">なんで</span>？<br>
<span class="jp-trans">智子：嗯，怎么？</span>

### 例二

ボブ：<span class="jp-word" title="にほんご - 日语">日本語</span>は、<span class="jp-word" title="たくさん - 很多">たくさん</span><span class="jp-word" title="べんきょう - 学习">勉強</span><span class="jp-word" title="する - 做">した</span>けど<em>な</em>。<span class="jp-word" title="まだ - 还">まだ</span><span class="jp-word" title="ぜんぜん - 完全">全然</span><span class="jp-word" title="わかる - 明白">わからない</span>。<br>
<span class="jp-trans">Bob：我学日语很努力了，对吧？但我还是搞不明白。</span>

アリス：<span class="jp-word" title="だいじょうぶ - 没事">大丈夫</span>よ。<span class="jp-word" title="きっと - 当然">きっと</span><span class="jp-word" title="わかる - 明白">わかる</span>ように<span class="jp-word" title="なる - 成为">なる</span>から<em>さ</em>。<br>
<span class="jp-trans">Alice：没问题，你一定会搞明白的，好吗？</span>

ボブ：なら<span class="jp-word" title="いい - 好的">いい</span>けど<em>な</em>。<br>
<span class="jp-trans">Bob：要这样就好了。</span>

句尾助词「な」经常跟提问词「か」合用，表示说话者正在考虑什么。

1. <span class="jp-word" title="きょう - 今天">今日</span> は <span class="jp-word" title="あめ - 雨">雨</span> が <span class="jp-word" title="ふる - 沉淀">降る</span> <em>かな</em> ？<br>
<span class="jp-trans">我在想今天会不会下雨。</span>
2. <span class="jp-word" title="いい - 好的">いい</span> <span class="jp-word" title="だいがく - 大学">大学</span> に <span class="jp-word" title="いく - 走">行ける</span> <em>かな</em> ？<br>
<span class="jp-trans">我在想自己能否上个好大学。</span>

## 句尾助词「かい」和「だい」 {#kaidai}

### 词汇学习

1. おい - 喂
2. どこ - 哪里
3. 行く 【い・く】 （う动词） - 走
4. 呼ぶ 【よ・ぶ】 （う动词） - 呼叫
5. いい （い形容词） - 好的
6. 一体 【いったい】 - 到底，究竟
7. 何時 【なん・じ】 - 何时
8. 帰る 【かえ・る】 （う动词） - 回家
9. つもり - 意向，计划
10. 俺 【おれ】 - 我；我自己 （男性化）
11. 土曜日 【ど・よう・び】 - 周六
12. 映画 【えい・が】 - 电影
13. 見る 【み・る】 （る动词） - 看
14. 一緒 【いっ・しょ】 - 一起

「かい」和「だい」是用来提问的句尾助词，非常男性化。「かい」用来问是非题，而「だい」则用于开放性问题。

### 示例

1. おい、どこに行くん <em>だい</em> ？<br>
<span class="jp-trans">喂，（你）要去哪里？</span>
2. さきちゃんって呼んでもいい <em>かい</em> ？<br>
<span class="jp-trans">（我）可以叫你 Saki-chan 吗？</span>
3. 一体何時に帰ってくるつもりだったん <em>だい</em> ？<br>
<span class="jp-trans">（你）到底打算何时回家？</span>
4. 俺は土曜日、映画を見に行くけど、一緒に行く <em>かい</em> ？<br>
<span class="jp-trans">俺打算周六去看电影，同去？</span>

## 适用于不同性别的句尾助词 {#part3}

下面这些句尾助词都是用作强调用的，并没有具体的含义。不过，它们能让你说的话更强硬，也更性别分明。「わ」的用法跟「よ」差不多，只不过非常女性化 (this is a different sound from the 「わ」 used in 关西方言)。「かしら」则是刚学过的「かな」的女性化版本。「ぞ」和「ぜ」跟「よ」差不多，只不过听起来更酷更 man（或者只是说话者希望带来这种感觉）。下面的例子看看就行了，要听到才有感觉。

### 词汇学习

1. もう - 已经
2. 時間 【じ・かん】 - 时间
3. ある （う动词） - 存在（无生命）
4. おい - 喂
5. 行く 【い・く】 （う动词） - 走
6. これ - 这
7. 終わり 【お・わり】 - 结束
8. いい （い形容词） - 好的
9. 大学 【だい・がく】 - 大学
10. 入る 【はい・る】 （う动词） - 进入

1. <span class="jp-word" title="もう - 已经">もう</span> <span class="jp-word" title="じかん - 时间">時間</span> が <span class="jp-word" title="ある - 存在（无生命）">ない</span> <em>わ</em> 。<br>
<span class="jp-trans">没时间了。</span>
2. <span class="jp-word" title="おい - 喂!">おい</span> 、 <span class="jp-word" title="いく - 走">行く</span> <em>ぞ</em> ！<br>
<span class="jp-trans">嗨，我们走吧！</span>
3. <span class="jp-word" title="これ - 这">これ</span> で、 <span class="jp-word" title="もう - 已经">もう</span> <span class="jp-word" title="おわり - 结束">終わり</span> だ <em>ぜ</em> 。<br>
<span class="jp-trans">这样的话，那就已经结束了。</span>
4. <span class="jp-word" title="いい - 好的">いい</span> <span class="jp-word" title="だいがく - 大学">大学</span> に <span class="jp-word" title="はいれる - 能进入">入れる</span> <em>かしら</em> ？<br>
<span class="jp-trans">我在想自己能否上个好大学。</span>

### 例一

<span class="jp-trans">アリス：「Hello」を<span class="jp-word" title="にほんご - 日语">日本語</span>で<em><span class="jp-word" title="なん - 什么">何</span>と<span class="jp-word" title="いう- 说">言えば</span></em><span class="jp-word" title="いい - 好的">いい</span>ですか。</span><br>
<span class="jp-trans"><span class="jp-word" title="かが - Kaga">加賀</span><span class="jp-word" title="せんせい - 老师">先生</span>： <span class="jp-word" title="そう - 那's so">そう</span>ね。<span class="jp-word" title="だいたい - 基本上">大体</span>、「<span class="jp-word" title="こんにちは - 好的 afternoon">こんにちは</span>」<em>と<span class="jp-word" title="いう- 说">言う</span>と<span class="jp-word" title="おもう - 思考，认为">思います</span></em>よ。<span class="jp-word" title="ただし - 但是，只是">ただし</span>、<span class="jp-word" title="かく - 写">書く</span><span class="jp-word" title="とき - 时间">時</span>は「<span class="jp-word" title="こんにちわ - 好的 afternoon">こんにちわ</span>」<em>じゃなくて</em>、「<span class="jp-word" title="こんにちは - 好的 afternoon">こんにちは</span>」と<span class="jp-word" title="かく - 写">書かなくて</span>はなりません。</span><br>
