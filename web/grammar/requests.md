## 礼貌地（或不大礼貌地）做出请求 {#part1}

上一课我们已经学过请别人帮忙的说法。与此类似，日语里面还有很多种做出请求的说法，这也就是用日语说「请做 xxx」的方法。我们会先学用动词「<span class="jp-word" title="くださる - （尊敬语）给">くださる</span>」的一种特殊活用法来做出请求，这是最常见的用法；然后是更坚决一点的「<span class="jp-word" title="なさる - 做 （尊敬语）">なさる</span>」。最后，为了完整性，我们还会学习一种很少用到的命令形。如果你不想看漫画的话跳过这一小节完全没问题。

## 「～<span class="jp-word" title="ください - 请给、请做">ください</span>」－ 「<span class="jp-word" title="くださる - （尊敬语）给">くださる</span>」的特殊活用法 {#part2}

### 词汇学习

1. それ - 那
2. くれる （る动词） - 给
3. 漢字 【かん・じ】 - 汉字
4. 書く 【か・く】 （う动词） - 写
5. ここ - 这里
6. 来る 【く・る】 （例外）- 来
7. 日本語 【に・ほん・ご】 - 日语
8. 話す 【はな・す】 （う动词） - 说
9. 消しゴム 【け・し・ごむ】 - 橡皮
10. 貸す 【か・す】 （う动词） - 借
11. 遠い 【とお・い】 （い形容词） - 远
12. 所 【ところ】 - 地方
13. 行く 【い・く】 （う动词） - 走
14. お父さん【お・とう・さん】 - 父亲 （礼貌）
15. 時計 【と・けい】 - 钟表
16. 壊れる 【こわ・れる】 （る动词） - 弄坏
17. 言う 【い・う】 （う动词）- 说

「<span class="jp-word" title="ください - 请给、请做">ください</span>」是「<span class="jp-word" title="くださる - （尊敬语）给">くださる</span>」的一种[特殊活用形](/grammar/honorific/#part5)，而后者是「<span class="jp-word" title="くれる - 给">くれる</span>」的敬语版本。我们会在下一章开头学敬语和谦逊语时讲到，这里先讲「<span class="jp-word" title="ください - 请给、请做">ください</span>」是因为它跟「<span class="jp-word" title="くれる - 给">くれる</span>」及其敬语版本「<span class="jp-word" title="くださる - （尊敬语）给">くださる</span>」有些区别。「<span class="jp-word" title="ください - 请给、请做">ください</span>」的特殊之处在于：

1. <span class="jp-word" title="それ - 那">それ</span> を <em><span class="jp-word" title="ください - 请给、请做">ください</span></em> 。<br>
<span class="jp-trans">请把那个给我。</span>
2. <span class="jp-word" title="それ - 那">それ</span> を <em><span class="jp-word" title="くれる - 给">くれる</span></em> ？<br>
<span class="jp-trans">可以把那个给我吗？</span>

可以看到，「<span class="jp-word" title="ください - 请给、请做">ください</span>」是直接的请求，而「<span class="jp-word" title="くれる - 给">くれる</span>」则以问句的形式请别人给自己什么。不过与「<span class="jp-word" title="くれる - 给">くれる</span>」相同的是，你可以把它加在动词て形后面来请别人做什么。

1. <span class="jp-word" title="かんじ - 汉字">漢字</span> で <em><span class="jp-word" title="かく - 写">書いて</span></em> <span class="jp-word" title="ください - 请给、请做">ください</span> 。<br>
<span class="jp-trans">请用汉字写。</span>
2. <span class="jp-word" title="ゆっくり - 慢慢的">ゆっくり</span> <em><span class="jp-word" title="はなす - 说">話して</span></em> <span class="jp-word" title="ください - 请给、请做">ください</span> 。<br>
<span class="jp-trans">请慢慢讲。</span>

否定请求的用法也跟「<span class="jp-word" title="くれる - 给">くれる</span>」一样。

1. <span class="jp-word" title="らくがき - 乱写，涂鸦">落書き</span> を <em><span class="jp-word" title="かく - 写">書かない</span>で</em> <span class="jp-word" title="ください - 请给、请做">ください</span> 。<br>
<span class="jp-trans">请不要涂鸦。</span>
2. <span class="jp-word" title="ここ - 这里">ここ</span> に <em><span class="jp-word" title="くる- 来">こないで</span></em> <span class="jp-word" title="ください - 请给、请做">ください</span> 。<br>
<span class="jp-trans">请不要来这里。</span>

口语里面通常会直接去掉「<span class="jp-word" title="ください - 请给、请做">ください</span>」这部分。

1. <span class="jp-word" title="にほんご - 日语">日本語</span> で <em><span class="jp-word" title="はなす - 说">話して</span></em> 。<br>
<span class="jp-trans">请说日语。</span>
2. <span class="jp-word" title="けしゴム - 橡皮">消しゴム</span> を <em><span class="jp-word" title="かす - 借">貸して</span></em> 。<br>
<span class="jp-trans">请把橡皮借我。</span>
3. <span class="jp-word" title="とおい - 远">遠い</span> <span class="jp-word" title="ところ - 地方">所</span> に <em><span class="jp-word" title="いく - 走">行かない</span>で</em> 。<br>
<span class="jp-trans">请不要去远的地方。</span>

对于那些想要显得很强硬很 man 的人，也可以用「<span class="jp-word" title="くれる - 给">くれる</span>」，但把「る」去掉。

1. <span class="jp-word" title="にほんご - 日语">日本語</span> で <em><span class="jp-word" title="はなす - 说">話して</span><span class="jp-word" title="くれる - 给">くれ</span></em> 。<br>
<span class="jp-trans">说日语。</span>
2. <span class="jp-word" title="けしゴム - 橡皮">消しゴム</span> を <em><span class="jp-word" title="かす - 借">貸して</span><span class="jp-word" title="くれる - 给">くれ</span></em> 。<br>
<span class="jp-trans">橡皮借我。</span>
3. <span class="jp-word" title="とおい - 远">遠い</span> <span class="jp-word" title="ところ - 地方">所</span> に <em><span class="jp-word" title="いく - 走">行かない</span>で<span class="jp-word" title="くれる - 给">くれ</span></em> 。<br>
<span class="jp-trans">别走太远。</span>

因为「<span class="jp-word" title="ください - 请给、请做">ください</span>」跟ます形一样总是出现在句尾，你不用用它来直接修饰名词。例如下面就不能用「<span class="jp-word" title="ください - 请给、请做">ください</span>」。

- <span class="jp-word" title="おとうさん - 父亲">お父さん</span> が <em><span class="jp-word" title="くれる - 给">くれた</span></em> <span class="jp-word" title="とけい - 钟">時計</span> が <span class="jp-word" title="こわれる - 成为 broken">壊れた</span> 。<br>
<span class="jp-trans">父亲给的钟坏了。</span>

当然，因为直接引用只不过是逐字逐句重复别人的话，你可以引用任何东西。

- 「 <span class="jp-word" title="それ - 那">それ</span> を <span class="jp-word" title="ください - 请给、请做">ください</span> 」 <em>と</em> <span class="jp-word" title="おとうさん - 父亲">お父さん</span> が <span class="jp-word" title="いう- 说">言った</span> 。<br>
<span class="jp-trans">父亲说，「把那个给我」。</span>

## 口语里用「～<span class="jp-word" title="ちょうだい - 接受，接收（尊敬语）">ちょうだい</span>」发出请求 {#part3}

### 词汇学习

1. 頂戴 【ちょうだい】 - 接收（谦逊语）
2. 致す 【いたす】 （う动词） - 做（谦逊语）
3. スプーン - 调羹
4. ここ - 这里
5. 名前 【な・まえ】 - 名字
6. 書く 【か・く】 （う动词） - 写

口语里面「<span class="jp-word" title="ください - 请给、请做">ください</span>」的替代词是「<span class="jp-word" title="ちょうだい - 接受，接收（尊敬语）">ちょうだい</span>」。尽管谁都能用，但它还是有点娘和孩子气，并且总是用平假名书写。写成汉字的时候，它一般用在非常正式的表达里面，例如「<span class="jp-word" title="ちょうだい - 接受，接收（尊敬语）">頂戴</span><span class="jp-word" title="いたす - 做（谦逊语）">致します</span>」。语法上它跟「<span class="jp-word" title="ください - 请给、请做">ください</span>」的用法一样。

### 示例

1. <span class="jp-word" title="スプーン - 调羹">スプーン</span> を <em><span class="jp-word" title="ちょうだい - 接受，接收（尊敬语）">ちょうだい</span></em> 。<br>
<span class="jp-trans">请把调羹给我。</span>
2. <span class="jp-word" title="ここ - 这里">ここ</span> に <span class="jp-word" title="なまえ - 名字">名前</span> を <em><span class="jp-word" title="かく - 写">書いて</span><span class="jp-word" title="ちょうだい - 接受，接收（尊敬语）">ちょうだい</span></em> 。<br>
<span class="jp-trans">请把名字写在这里。</span>

## 用「～<span class="jp-word" title="なさる - 做 （尊敬语）">なさい</span>」做出礼貌而坚决的请求 {#part4}

### 词汇学习

1. 食べる 【たべ・る】 （る动词） - 吃
2. 飲む 【の・む】 （う动词） - 喝
3. する （例外） - 做
4. いい （い形容词） - 好的
5. 聞く 【き・く】 （う动词） - 问；听
6. ここ - 这里
7. 座る 【すわ・る】 （る动词） - 坐
8. まだ - 还
9. いっぱい - 满
10. ある （う动词） - 存在（无生命）
11. たくさん - （数量）很多
12. それ - 那
13. 思う 【おも・う】 （う动词） - 思考，认为
14. そう - 那样

「<span class="jp-word" title="なさる - 做 （尊敬语）">なさい</span>」是「<span class="jp-word" title="する - 做">する</span>」的一种特别的敬语活用形。这是一种较温和但又坚决的命令方式，通常用于母亲教训小孩，或者老师教训学生等场合。与「<span class="jp-word" title="ください - 请给、请做">ください</span>」不同，「<span class="jp-word" title="なさる - 做 （尊敬语）">なさい</span>」只能用在肯定动词，而且用的是动词的词根而非て形。它不能单独使用。

<div class="sumbox">
<div class="sumbox-title">用「なさい」做出礼貌而坚决的请求</div>

- 把动词活用为词根，再加上「 <span class="jp-word" title="なさる - 做 （尊敬语）">なさい</span> 」<br>
<span class="jp-trans">例</span>
  1. <span class="jp-word" title="たべる - 吃">食べ<em>る</em></span> → <span class="jp-word" title="たべる - 吃">食べ</span> <em><span class="jp-word" title="なさる - 做 （尊敬语）">なさい</span></em>
  2. <span class="jp-word" title="のむ - 喝">飲<em>む</em></span> → <span class="jp-word" title="のむ - 喝">飲<em>み</em></span> → <span class="jp-word" title="のむ - 喝">飲み</span> <em><span class="jp-word" title="なさる - 做 （尊敬语）">なさい</span></em>
  3. <span class="jp-word" title="する - 做">する</span> → <em><span class="jp-word" title="する - 做">し</span></em> → <span class="jp-word" title="する - 做">し</span> <em><span class="jp-word" title="なさる - 做 （尊敬语）">なさい</span></em>
</div>

### 示例

1. <span class="jp-word" title="いい - 好的">よく</span> <em><span class="jp-word" title="きく - 问；听">聞き</span></em> <span class="jp-word" title="なさる - 做 （尊敬语）">なさい</span> ！<br>
<span class="jp-trans">听好了！</span>
2. <span class="jp-word" title="ここ - 这里">ここ</span> に <em><span class="jp-word" title="すわる - 坐">座り</span></em> <span class="jp-word" title="なさる - 做 （尊敬语）">なさい</span> 。<br>
<span class="jp-trans">坐这里。</span>

你也可以把「<span class="jp-word" title="なさる - 做 （尊敬语）">なさい</span>」里面的「さい」去掉，口语中会这样用。

1. <span class="jp-word" title="まだ - 还">まだ</span> <span class="jp-word" title="いっぱい - 满, a lot">いっぱい</span> <span class="jp-word" title="ある - 存在（无生命）">ある</span> から、 <span class="jp-word" title="たくさん - 很多">たくさん</span> <em><span class="jp-word" title="たべる - 吃">食べな</span></em> 。<br>
<span class="jp-trans">还是有很多，请多吃点。</span>
2. <span class="jp-word" title="それ - 那">それ</span> で <span class="jp-word" title="いい - 好的">いい</span> と <span class="jp-word" title="おもう - 思考，认为">思う</span> なら、 <span class="jp-word" title="そう - 像那样">そう</span> <em><span class="jp-word" title="する - 做">し</span>な</em> よ。<br>
<span class="jp-trans">觉得没问题的话，就放手去做吧。</span>

## 命令形 {#part5}

### 词汇学习

1. くれる （る动词） - 给
2. 死ぬ 【し・ぬ】 （う动词） - 死
3. する （例外） - 做
4. 来る 【く・る】 （例外）- 来
5. 食べる 【た・べる】 （る动词） - 吃
6. 着る 【き・る】 （る动词） - 穿
7. 信じる 【しん・じる】 （る动词） - 相信
8. 寝る 【ね・る】 （る动词） - 睡
9. 起きる 【お・きる】 （る动词） - 起
10. 出る 【で・る】 る动词 - 出来
11. 掛ける 【か・ける】 （る动词） - 挂
12. 捨てる 【す・てる】 （る动词） - 扔掉
13. 話す 【はな・す】 （う动词） - 说
14. 聞く 【き・く】 （う动词） - 问；听
15. 遊ぶ 【あそ・ぶ】 （う动词） - 玩
16. 待つ 【ま・つ】 （う动词） - 等待
17. 飲む 【の・む】 （う动词） - 喝
18. 直る 【なお・る】 （う动词） - 修好
19. 買う 【か・う】 （う动词）- 买
20. 好き 【す・き】 （な形容词） - 讨人喜欢的
21. あっち - （远的）那里（あちら的缩写）
22. 行く 【い・く】 （う动词） - 走
23. 早い 【はや・い】 （い形容词） - 快；早
24. 酒 【さけ】 - 酒
25. 持つ 【も・つ】 （う动词） - 拿，带

为了全面介绍动词的活用形，我们在这里介绍一下命令形。日常生活里面日本人是非常礼貌的，因此极少对别人下命令。同时，这种粗鲁的说法比较罕见。当生气或被冒犯时，如果确实需要用到，女性一般会使用「<span class="jp-word" title="なさる - 做 （尊敬语）">なさい</span>」或是愤怒的「<span class="jp-word" title="くれる - 给">くれる</span>」来代替。只有在看虚构作品的时候这种语法才能派上用场，你可能会经常在动漫里听到或看到「<span class="jp-word" title="しぬ - 死">死ね</span>！」（去死！），但在现实生活中可是听不到的（但愿）。

需要注意的是除了我们熟悉的例外动词「<span class="jp-word" title="する - 做">する</span>」和「<span class="jp-word" title="くる- 来">くる</span>」，命令形里面「<span class="jp-word" title="くれる - 给">くれる</span>」也是个例外。

<div class="sumbox">
<div class="sumbox-title">构建命令形的规则</div>

- <b>对る动词：</b> 把「る」换成「ろ」
- <b>对う动词：</b> 把末尾假名从 / u / 段改为同行 / e / 段假名。
- <b>例外：</b>
  1. <span class="jp-word" title="する - 做">する</span> → <span class="jp-word" title="する - 做">しろ</span>
  2. <span class="jp-word" title="くる- 来">くる</span> → <span class="jp-word" title="くる- 来">こい</span>
  3. <span class="jp-word" title="くれる - 给">くれる</span> → <span class="jp-word" title="くれる - 给">くれ</span>
</div>

## 否定命令 {#part6}

### 词汇学习

1. 行く 【い・く】 （う动词） - 走
2. する （例外） - 做
3. それ - 那
4. 食べる 【た・べる】 （る动词） - 吃
5. 変 【へん】 （な形容词） - 奇怪的
6. こと - 事情
7. 言う 【い・う】 （う动词）- 说

否定命令形很简单，把「な」加在る动词或う动词后面即可。不要把它跟句末助词「な」（本章结尾会讲到）搞混了，两者的语调完全不同

<div class="sumbox">
<div class="sumbox-title">使用否定命令形</div>

- 动词后面加「な」<br>
<span class="jp-trans">例</span>
  1. 行く → 行く <em>な</em>
  2. する → する <em>な</em>
</div>

### 示例

1. <span class="jp-word" title="それ - 那">それ</span> を <span class="jp-word" title="たべる - 吃">食べる</span> <em>な</em> ！<br>
<span class="jp-trans">别吃那个！</span>
2. <span class="jp-word" title="へん - 奇怪的">変</span> な <span class="jp-word" title="こと - 事情">こと</span> を <span class="jp-word" title="いう- 说">言う</span> <em>な</em> ！<br>
<span class="jp-trans">别说这种奇怪的话！</span>

另外不要跟上一小节刚学的简写的「～なさい」混淆了。两者最明显（另外语调的区别也很明显）的区别是用「～なさい」的时候动词要活用成词根，而否定命令形则没有活用动词。例如对于「<span class="jp-word" title="する - 做">する</span>」，「<span class="jp-word" title="する - 做">しな</span>」是「<span class="jp-word" title="する - 做">しなさい</span>」的简写，而「<span class="jp-word" title="する - 做">する</span>な」则是否定命令形。
