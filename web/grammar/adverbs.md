## 副词的属性 {#part2}

### 词汇学习

1. 早い 【はや・い】 （い形容词） - 快；早
2. きれい （な形容词） - 好看；干净
3. 朝ご飯 【あさ・ご・はん】 - 早饭
4. 食べる 【た・べる】（る动词） - 吃
5. 自分 【じ・ぶん】 - 自己
6. 部屋 【へ・や】 - 房间
7. 映画 【えい・が】 - 电影
8. たくさん - 大量
9. 見る 【み・る】 - 看；观看
10. 最近 【さい・きん】 - 最近
11. 全然 【ぜん・ぜん】 - 全然（否定时用）
12. 声 【こえ】 - 声音
13. 結構 【けっ・こう】 - 不需要；不错的
14. 大きい 【おお・きい】（い形容词） - 大
15. この - 这 （これの的缩写）
16. 町 【まち】 - 镇
17. 変わる 【か・わる】（う动词） - 改变
18. 図書館 【と・しょ・かん】 - 图书馆
19. 中 【なか】 - 里面
20. 静か 【しず・か】（な形容词） - 安静

与英语不同，日语里面把形容词变为副词很简单很直观。而且因为助词的使用使得句子的组合顺序具有很大的灵活性，副词可以放在它所修饰的动词<i>前面</i>的任何地方。对于い形容词和な形容词，我们还是两套规则。

<div class="sumbox">
<div class="sumbox-title">如何将形容词变为副词</div>

- <b>对い形容词：</b> 把「い」换成「く」。<br>
<span class="jp-trans">例： <span class="jp-word" title="はやい - 快；早">早</span> <em>い</em> → <span class="jp-word" title="はやい - 快；早">早<em>く</em></span></span>
- <b>对な形容词：</b> 后面加「に」。<br>
<span class="jp-trans">例： <span class="jp-word" title="きれい - 好看；干净">きれい</span> → <span class="jp-word" title="きれい - 好看；干净">きれい</span> <em>に</em></span>
</div>

- ボブは <span class="jp-word" title="あさごはん - 早饭">朝ご飯</span> を <em><span class="jp-word" title="はやい - 快；早">早く</span></em> <span class="jp-word" title="たべる - 吃">食べた</span> 。<br>
<span class="jp-trans">Bob 很快的吃了早饭。</span>

副词「<span class="jp-word" title="はやい - 快；早">早く</span>」跟英语里面「快」的意思有点不一样，因为它既可指速度（快），也可以指时间（早）。换句话说，Bob 可能很早吃了他的早饭，也可能很快的吃了早饭，这要看上下文了。在另外的句子例如「<span class="jp-word" title="はやい - 快；早">早く</span><span class="jp-word" title="はしる - 跑">走った</span>」里面，很显然它应该是快而不是早的意思（但还是跟上下文有关）。

- アリスは <span class="jp-word" title="じぶん - 自己">自分</span> の <span class="jp-word" title="へや - 房间">部屋</span> を <em><span class="jp-word" title="きれい - 好看；干净">きれい</span>に</em> <span class="jp-word" title="する - 做">した</span> 。<br>
<span class="jp-trans">Alice 向着干净整了她的房间。</span>

上面这种直译应该能让你理解这里为何要用目标助词。其实这种变化是否应该称为副词还是有争议的，但我们这么介绍就可以把两种形容词的变化统一起来了。如果把它看作是副词的话，我们可以把句子翻译成「Alice 把她的房间整干净了」，或者更顺畅一点，「Alice 清理了她的房间」。（「<span class="jp-word" title="きれい - 好看；干净">きれい</span>」字面意思是「漂亮」，不过你可以把它想成是「Alice 把她的房间弄得漂亮了起来」"Alice prettied up her own room."）

注意：不是所有副词都是从形容词变化而来的，比如「<span class="jp-word" title="ぜんぜん - 全然">全然</span>」和「<span class="jp-word" title="たくさん - 大量">たくさん</span>」本身就是副词，非活用而来。这些词可以像其他副词一样不带助词使用。

1. <span class="jp-word" title="えいが - 电影">映画</span> を <em><span class="jp-word" title="たくさん - 大量">たくさん</span></em> <span class="jp-word" title="みる - 看">見た</span> 。<br>
<span class="jp-trans">看了很多电影。</span>
2. <span class="jp-word" title="さいきん - 最近">最近</span> 、 <em><span class="jp-word" title="ぜんぜん - 全然">全然</span></em> <span class="jp-word" title="たべる - 吃">食べない</span> 。<br>
<span class="jp-trans">最近什么都没吃。</span>

### 示例

以下是更多的副词使用示例。

1. ボブの <span class="jp-word" title="こえ - 声音">声</span> は、 <em><span class="jp-word" title="けっこう - 相当">結構</span></em> <span class="jp-word" title="おおきい - 大">大きい</span> 。<br>
<span class="jp-trans">Bob 的声音 <em>相当</em> 大。</span>
2. <span class="jp-word" title="この - 这">この</span> <span class="jp-word" title="まち - 镇">町</span> は、 <span class="jp-word" title="さいきん - 最近">最近</span> <em><span class="jp-word" title="おおきい - 大">大きく</span></em> <span class="jp-word" title="かわる - 改变">変わった</span> 。<br>
<span class="jp-trans">这个镇子最近变化 <em>很大</em> 。</span>
3. <span class="jp-word" title="としょかん - 图书馆">図書館</span> の <span class="jp-word" title="なか - 里面">中</span> では、 <em><span class="jp-word" title="しずか - 安静">静か</span>に</em> <span class="jp-word" title="する - 做">する</span> 。<br>
<span class="jp-trans">在图书馆里面，（我们） <em>安静的</em> 做事情。</span>

## 用来结束句子的助词 {#part3}

### 词汇学习

1. いい （い形容词） - 好
2. 天気 【てん・き】 - 天气
3. そう - 那样
4. 面白い 【おも・しろ・い】（い形容词） - 有趣
5. 映画 【えい・が】 - 电影
6. 全然 【ぜん・ぜん】 - 全然（否定时用）
7. 時間 【じ・かん】 - 时间
8. ある （う动词） - 存在（无生命）
9. 大丈夫 【だい・じょう・ぶ】 （な形容词） - 好的
10. 今日 【きょう】 - 今天
11. うん - 是（口语）
12. でも - 但是
13. 明日 【あした】 - 明天
14. 雨 【あめ】 - 雨
15. 降る 【ふ・る】（う动词） - 沉淀
16. 魚 【さかな】 - 鱼
17. 好き 【す・き】 （な形容词） - 讨人喜欢的

用于结束句子的助词是那些出现在句尾，用来改变句子「音调」或者「感觉」的助词。这一节里面我们介绍两个最常用的。

## 「ね」结尾 {#part4}

人们希望别人赞同自己的时候，经常会在句尾加一个「ね」，意思类似「对吧？」、「不是吗？」

### 例一

ボブ：<span class="jp-word" title="いい - 好">いい</span><span class="jp-word" title="てんき - 天气">天気</span>だ<em>ね</em>。<br>
<span class="jp-trans">Bob：好天气，<em>是吧</em>?</span>

アリス：<span class="jp-word" title="そう - 那样">そう</span><em>ね</em>。<br>
<span class="jp-trans">Alice：是那样， <em>不是吗</em>?</span>

直译「<span class="jp-word" title="そう - 那样">そう</span>ね」听起来有点奇怪，但基本上它的意思类似「没错」。男性会说成「<span class="jp-word" title="そう - 那样">そう</span>だね」。

### 例二

アリス：<span class="jp-word" title="おもしろい - 有趣">おもしろい</span><span class="jp-word" title="えいが - 电影">映画</span>だった<em>ね</em>。<br>
<span class="jp-trans">Alice：电影挺好看的，<em>对吧</em>?</span>

ボブ：え？<span class="jp-word" title="ぜんぜん - 全然">全然</span><span class="jp-word" title="おもしろい - 有趣">おもしろくなかった</span>。<br>
<span class="jp-trans">Bob：啊？不好看，一点也不好看。</span>

因为 Alice 期待一个赞同的回答，所以 Bob 有点吃惊，因为他完全不这么认为。（「え」 是一个表达惊讶和困惑的象声词）

## 「よ」 结尾 {#part5}

当把「よ」加到句子结尾的时候，说话的人是想让对方知道新的信息。英语里面我们会说 "You know..." ，比如 "You know, I'm actually a genius."

### 例一

アリス：<span class="jp-word" title="じかん - 时间">時間</span>が<span class="jp-word" title="ある - 存在（无生命）">ない</span><em>よ</em>。<br>
<span class="jp-trans">Alice：<em>知道不</em>，没时间了。</span>

ボブ：<span class="jp-word" title="だいじょうぶ - 好的">大丈夫</span>だ<em>よ</em>。<br>
<span class="jp-trans">Bob：没事，<em>you know</em>.</span>

### 例二

アリス：<span class="jp-word" title="きょう - 今天">今日</span>は<span class="jp-word" title="いい - 好">いい</span><span class="jp-word" title="てんき - 天气">天気</span>だね。<br>
<span class="jp-trans">Alice：今天好天气，是吧？</span>

ボブ：<span class="jp-word" title="うん - 嗯">うん</span>。<span class="jp-word" title="でも - 但是">でも</span>、<span class="jp-word" title="あした - 明天">明日</span><span class="jp-word" title="あめ - 雨">雨</span>が<span class="jp-word" title="ふる - 沉淀">降る</span><em>よ</em>。<br>
<span class="jp-trans">Bob：嗯，不过明天会下雨，<em>知道吧</em>。</span>

## 合用「よね」 {#part6}

你也可以把这两个助词合起来用，变成「よね」。当你需要告诉别人新的信息，同时又期望对方同意的时候，就可以这么用。不过要注意顺序不能调整，必须是「よね」。

### 示例

アリス：ボブは、<span class="jp-word" title="さかな - 鱼">魚</span>が<span class="jp-word" title="すき - 喜欢">好き</span>なんだ<em>よね</em>。<br>
<span class="jp-trans">Alice：<em>知道不</em>，你喜欢吃鱼，<em>不是吗</em>?</span>

ボブ：<span class="jp-word" title="そう - 那样">そう</span>だね。<br>
<span class="jp-trans">Bob：这样<em>哈</em>?</span>
