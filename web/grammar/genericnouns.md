我们已经介绍过如何用通用名词修饰名词。现在我们再来介绍通用名词的一些特殊用法。

## 用「<span class="jp-word" title="こと - 事情">こと</span>」表达某件事是否发生过 {#part2}

### 词汇学习

1. こと - 事情
2. ある （う动词） - 存在（无生命）
3. 徹夜 【てつ・や】 - 通宵
4. 宿題 【しゅく・だい】 - 作业
5. する （例外） - 做
6. 一人 【ひとり】 - 一人；独自
7. 行く 【い・く】 （う动词） - 走
8. パリ - 巴黎
9. お寿司 【お・す・し】- 寿司
10. 食べる 【たべ・る】 （る动词） - 吃
11. 日本 【に・ほん】 - 日本
12. 映画 【えい・が】 - 电影
13. 観る 【み・る】 （る动词） - 观看
14. ヨーロッパ - 欧洲
15. いい （い形容词） - 好的
16. そう - 那样
17. 言う 【い・う】 （う动词）- 说
18. 見る 【み・る】 （る动词） - 看
19. ～度 【～ど】 - 次数的量词

如果把泛指事情的「<span class="jp-word" title="こと - 事情">こと</span>」与「<span class="jp-word" title="ある - 存在（无生命）">ある</span>」合用，你就可以谈论某件事是否存在。

### 示例

1. <span class="jp-word" title="てつや - 通宵">徹夜</span> <span class="jp-word" title="する - 做">して</span> 、 <span class="jp-word" title="しゅくだい - 作业">宿題</span> <em><span class="jp-word" title="する - 做">する</span><span class="jp-word" title="こと - 事情">こと</span></em> は <em><span class="jp-word" title="ある - 存在（无生命）">ある</span></em> 。<br>
<span class="jp-trans">我有时候会通宵做作业。</span>
2. <span class="jp-word" title="ひとり - alone">一人</span> で <em><span class="jp-word" title="いく - 走">行く</span><span class="jp-word" title="こと - 事情">こと</span></em> は <em><span class="jp-word" title="ある - 存在（无生命）">ありません</span></em> 。<br>
<span class="jp-trans">我从不独自走。</span>

如果将动词过去形与「<span class="jp-word" title="こと - 事情">こと</span>」合用，你就可以谈论某件事是否发生过。这基本上是你用日语说 "have done" 的唯一办法了，所以非常有用。当你要提及某人是否做过某事的时候，就得用这种语法。

### 示例

1. <span class="jp-word" title="パリ - 巴黎">パリ</span> に <em><span class="jp-word" title="いく - 走">行った</span><span class="jp-word" title="こと - 事情">こと</span></em> は <em><span class="jp-word" title="ある - 存在（无生命）">あります</span></em> か。<br>
<span class="jp-trans">你去过巴黎吗？</span>
2. <span class="jp-word" title="おすし - 寿司">お寿司</span> を <em><span class="jp-word" title="たべる - 吃">食べた</span><span class="jp-word" title="こと - 事情">こと</span></em> が <em><span class="jp-word" title="ある - 存在（无生命）">ある</span></em> 。<br>
<span class="jp-trans">我以前吃过寿司。</span>
3. <span class="jp-word" title="にほん - 日本">日本</span> の <span class="jp-word" title="えいが - 电影">映画</span> を <em><span class="jp-word" title="みる - 看">観た</span><span class="jp-word" title="こと - 事情">こと</span><span class="jp-word" title="ある - 存在（无生命）">ない</span></em> の？<br>
<span class="jp-trans">你从没看过日本电影？</span>
4. <span class="jp-word" title="ヨーロッパ - 欧洲">ヨーロッパ</span> に <em><span class="jp-word" title="いく - 走">行った</span><span class="jp-word" title="こと - 事情">こと</span></em> が <em><span class="jp-word" title="ある - 存在（无生命）">あったら</span></em> <span class="jp-word" title="いい - 好的">いい</span> な。<br>
<span class="jp-trans">我要是去过欧洲就好了。</span>
5. <span class="jp-word" title="そういう - 那种东西">そういう</span> のを <em><span class="jp-word" title="みる - 看">見た</span><span class="jp-word" title="こと - 事情">こと</span></em> が <em><span class="jp-word" title="ある - 存在（无生命）">なかった</span></em> 。<br>
<span class="jp-trans">我从没见过那种东西。</span>
6. <span class="jp-word" title="いちど - 一次">一度</span> <em><span class="jp-word" title="いく - 走">行った</span><span class="jp-word" title="こと - 事情">こと</span></em> も <em><span class="jp-word" title="ある - 存在（无生命）">ない</span></em> んです。<br>
<span class="jp-trans">我没去过，一次也没有。</span>

## 用「<span class="jp-word" title="ところ - 地方">ところ</span>」泛指地方 {#part3}

### 词汇学习

1. 所 【ところ】 - 地方
2. 早い 【はや・い】 （い形容词） - 快；早
3. 来る 【く・る】 （例外）- 来
4. 映画 【えい・が】 - 电影
5. 今 【いま】- 现在
6. ちょうど - 刚刚好；正好
7. いい - 好的
8. 彼 【かれ】 - 他；男朋友
9. 優しい 【やさ・しい】 （い形容词） - 温柔；友善
10. ある （う动词） - 存在（无生命）
11. 授業 【じゅ・ぎょう】 - 班
12. 終わる 【お・わる】 （う动词） - 结束
13. これ - 这
14. 行く 【い・く】 （う动词） - 走

「<span class="jp-word" title="ところ - 地方">ところ</span>」（<span class="jp-word" title="ところ - 地方">所</span>）通常用来泛指某个物理位置。但是它也能引申出更抽象的含义，例如个性，或者某个时期。

### 示例

1. <span class="jp-word" title="はやい - 快；早">早く</span> <span class="jp-word" title="くる- 来">きて</span> 。 <span class="jp-word" title="えいが - 电影">映画</span> は <span class="jp-word" title="いま- 现在">今</span> <span class="jp-word" title="ちょうど - 正好">ちょうど</span> <span class="jp-word" title="いい - 好的">いい</span> <em><span class="jp-word" title="ところ - 地方">ところ</span></em> だよ。<br>
<span class="jp-trans">快点来。我们正看到电影的精彩处。</span>
2. <span class="jp-word" title="かれ - 他，男朋友">彼</span> は <span class="jp-word" title="やさしい - 温柔；友善">優しい</span> <em><span class="jp-word" title="ところ - 地方">ところ</span></em> も <span class="jp-word" title="ある - 存在（无生命）">ある</span> よ。<br>
<span class="jp-trans">他个性里也有温柔的一面。</span>
3. <span class="jp-word" title="いま- 现在">今</span> は <span class="jp-word" title="じゅぎょう - 班">授業</span> が <span class="jp-word" title="おわる - 结束">終った</span> <em><span class="jp-word" title="ところ - 地方">ところ</span></em> です。<br>
<span class="jp-trans">刚刚下课。</span>
4. <span class="jp-word" title="これから - from now">これから</span> <span class="jp-word" title="いく - 走">行く</span> <em><span class="jp-word" title="ところ - 地方">ところ</span></em> でした。<br>
<span class="jp-trans">我当时正准备要走。</span>

## 「<span class="jp-word" title="もの - 东西，人">もの</span>」——口语里女性化的强调 {#part4}

### 词汇学习

1. 物 【もの】 - 东西
2. どうして - 为什么
3. 来る 【く・る】 （例外）- 来
4. 授業 【じゅ・ぎょう】 - 班
5. ある （う动词） - 存在（无生命）

通用名词「<span class="jp-word" title="もの - 东西，人">もの</span>」在口语里面也可以女性化的表示强调。这跟女性化的、解释性的强调助词「の」是一样的。与「の」一样，「の」也经常变成「ん」，从而出现「もん」。用「もん」给人的感觉非常娘，而且有点撒娇耍赖的感觉。

### 示例

- <span class="jp-word" title="どうして - 为什么">どうして</span> <span class="jp-word" title="くる- 来">こなかった</span> の？<br>
<span class="jp-trans">（你）为什么没来？</span>

1. <span class="jp-word" title="じゅぎょう - 班">授業</span> が <span class="jp-word" title="ある - 存在（无生命）">あった</span> <em>の</em> 。<br>
<span class="jp-trans">（我）有课（女性化、解释口吻）</span>
2. <span class="jp-word" title="じゅぎょう - 班">授業</span> が <span class="jp-word" title="ある - 存在（无生命）">あった</span> <em><span class="jp-word" title="もの - 东西，人">もの</span></em> 。<br>
<span class="jp-trans">（我）有课（女性化、解释口吻）</span>
3. <span class="jp-word" title="じゅぎょう - 班">授業</span> が <span class="jp-word" title="ある - 存在（无生命）">あった</span> <em>もん</em> 。<br>
<span class="jp-trans">（我）有课，就酱（女性化、解释口吻）</span>
