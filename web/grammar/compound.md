这一节我们学习把简单句组合成复合句的几种方法。比如，我们将学习如何将不同的句子串起来表达多重动作或状态。换句话说，如果两个句子有相同的主语，比如「我跑了」和「我吃了」，我们会学到如何表达「我跑了也吃了」。我们还会类似「他高、帅、富」的表达方式。

## 表达一系列的状态 {#part2}

### 词汇学习

1. 一般的 【いっ・ぱん・てき】 - 一般的
2. 静か 【しず・か】 （な形容词）- 安静
3. 狭い 【せま・い】 （い形容词） - 狭窄
4. 彼女 【かの・じょ】 - 她；女朋友
5. いい （い形容词） - 好的
6. 私 【わたし】 - 我；我自己
7. 部屋 【へ・や】 - 房间
8. きれい （な形容词） - 好看；干净
9. とても - 非常
10. 好き 【す・き】 （な形容词） - 惹人喜欢的
11. 学生 【がく・せい】 - 学生
12. 先生 【せん・せい】 - 老师
13. 田中 【た・なか】 - 田中（名）
14. お金持ち 【お・かね・も・ち】 - 富有
15. かっこいい （い形容词） - 酷；帅
16. 魅力的 【み・りょく・てき】 - 迷人

把一堆形容词、名词串起来形容一个人或一个东西很简单。比如英语里要说 "He is X. He is Y. He is Z." 的话，因为主语一样，通常就合并为 "He is X, Y, and Z"。日语里面也可以通过活用名词或形容词来达到同样的效果。但最后一个名词或形容词不需要变化。

<div class="sumbox">
<div class="sumbox-title">如何把名词或形容词串起来</div>

- <b>对名词和な形容词：</b> 后面加「で」<br>
<span class="jp-trans">例</span>
  1. <span class="jp-word" title="いっぱんてき - 一般的">一般的</span> → <span class="jp-word" title="いっぱんてき - 一般的">一般的</span> <em>で</em>
  2. <span class="jp-word" title="しずか- 安静">静か</span> → <span class="jp-word" title="しずか- 安静">静か</span> <em>で</em>
- <b>对 い形容词、否定式名词或形容词：</b> 把「い」换成「くて」<br>
<span class="jp-trans">※对「 <span class="jp-word" title="いい - 好的">いい</span> 」和「 <span class="jp-word" title="かっこいい - 好的-looking, cool">かっこいい</span> 」，原先介绍的例外规则「い→よ」依然适用。例：</span>
  1. <span class="jp-word" title="せまい - 狭窄">狭<em>い</em></span> → <span class="jp-word" title="せまい - 狭窄">狭<em>くて</em></span>
  2. <span class="jp-word" title="かのじょ - she, girlfriend">彼女</span> じゃな <em>い</em> → <span class="jp-word" title="かのじょ - she, girlfriend">彼女</span> じゃな <em>くて</em>
  3. <span class="jp-word" title="いい - 好的"><em>いい</em></span> → <span class="jp-word" title="いい - 好的"><em>よくて</em></span>
</div>

### 示例

1. <span class="jp-word" title="わたし - 我；我自己">私</span> の <span class="jp-word" title="へや - 房间">部屋</span> は、 <span class="jp-word" title="きれい - 干净，漂亮">きれい</span> <em>で</em> 、 <span class="jp-word" title="しずか- 安静">静か</span> <em>で</em> 、 <span class="jp-word" title="とても - 非常">とても</span> <span class="jp-word" title="すき - 喜欢">好き</span> 。<br>
<span class="jp-trans">我的房间干净、安静，我非常喜欢。</span>
2. <span class="jp-word" title="かのじょ - she, girlfriend">彼女</span> は、 <span class="jp-word" title="がくせい - 学生">学生</span> じゃな <em>くて</em> 、 <span class="jp-word" title="せんせい - 老师">先生</span> だ。<br>
<span class="jp-trans">她不是学生，她是老师。</span>
3. <span class="jp-word" title="たなか - 田中">田中</span> さんは、 <span class="jp-word" title="おかねもち - 富有">お金持ち</span> <em>で</em> 、 <span class="jp-word" title="かっこいい - 酷，帅">かっこ<em>よくて</em></span> 、 <span class="jp-word" title="みりょくてき - 迷人">魅力的</span> ですね。<br>
<span class="jp-trans">田中桑是一个富有、帅气且有魅力的人，不是吗？</span>

可以看到，把「で」加到「お金持ち」后面显然表达的不是助词「で」的意思，因为这里没有动词。你可以把「で」的作用看成跟「だ」类似，只不过是用来连接用的版本。

## 用て形连接一串动词 {#part3}

### 词汇学习

1. 学生 【がく・せい】 - 学生
2. 買う 【か・う】 （う动词）- 买
3. 食べる 【た・べる】 （る动词） - 吃
4. 行く 【い・く】 （う动词） - 走
5. する （例外） - 做
6. 遊ぶ 【あそ・ぶ】 （う动词） - 玩
7. 飲む 【の・む】 （う动词） - 喝
8. 食堂 【しょく・どう】 - 小饭馆
9. 昼ご飯 【ひる・ご・はん】 - 午餐
10. 昼寝 【ひる・ね】 - 午睡
11. 時間 【じ・かん】 - 时间
12. ある （う动词） - 存在（无生命）
13. 映画 【えい・が】 - 电影
14. 見る 【み・る】 （る动词） - 看

用类似的方法，你可以把多个动作连起来，这通常用来表达做一系列的动作（我做了 [X]，又做了 [Y]，最后做了 [Z]）。这里有两种形式，肯定和否定，取决于最后一个动词的形态。

<div class="sumbox">
<div class="sumbox-title">如何把动词串起来</div>

- <b>肯定：</b> 把动词活用成过去形，然后把「た」换成「て」，或把「だ」换成「で」。这通常被称为 <b><i>て形</i></b> ——虽然也可能是で。
- <b>否定：</b> 跟い形容词一样，把「い」换成「くて」。<br>
<span class="jp-trans">这条规则也适用于以「です」和「ます」结尾的丁宁语。</span><br>
<span class="jp-trans">例</span>
  1. <span class="jp-word" title="がくせい - 学生">学生</span> で <em>す</em> → <span class="jp-word" title="がくせい - 学生">学生</span> <em>でした</em> → <span class="jp-word" title="がくせい - 学生">学生</span> でし <em>て</em>
  2. <span class="jp-word" title="かう- 买">買いま<em>す</em></span> → <span class="jp-word" title="かう- 买">買いま<em>した</em></span> → <span class="jp-word" title="かう- 买">買いまし<em>て</em></span>
</div>

<span class="jp-trans">还有时间，我去看了电影。</span>

## 用「から」和「ので」表达原因或因果关系 {#part4}

### 词汇学习

1. 時間 【じ・かん】 - 时间
2. ある （う动词） - 存在（无生命）
3. パーティー - 聚会
4. 行く 【い・く】 （う动词） - 走
5. 友達 【とも・だち】- 朋友
6. プレゼント - 礼物
7. 来る 【く・る】 （例外）- 来
8. 田中 【た・なか】 - 田中（名）
9. どうして - 为什么
10. 山田 【や・まだ】 - 山田（名）
11. 一郎 【いち・ろう】 - 一郎（姓）
12. 直子 【なお・こ】 - 直子（姓）
13. ちょっと - 一点
14. 忙しい 【いそが・しい】 （い形容词）- 忙
15. そろそろ - 逐渐；即将
16. 失礼 【しつ・れい】 - 失礼
17. する （例外） - 做
18. 学生 【がく・せい】 - 学生
19. お金 【お・かね】 - 钱
20. ここ - 这里
21. 静か 【しず・か】 （な形容词）- 安静
22. とても - 非常
23. 穏やか 【おだ・やか】 （な形容词） - 冷静，平稳
24. 会う 【あ・う】 （う动词） - 会见

你可以用「から」连接两个句子来表示因果关系，二者的顺序必须是 [ 原因 ] から [ 结果 ]。如果原因是字典形名词或な形容词，后面 <em>必须</em>加「だ」来显式的表示它是原因，即「( 名词 / な形容词 ) <em>だ</em>から」。如果你忘了在「から」前面加表陈述的「だ」的话，听起来「から」就是「从…」的意思了，前面讲助词的时候说过这个用法。

### 示例

1. <span class="jp-word" title="じかん - 时间">時間</span> が <span class="jp-word" title="ある - 存在（无生命）">なかった</span> <em>から</em> <span class="jp-word" title="パーティー - 聚会">パーティーに</span> <span class="jp-word" title="いく - 走">行きませんでした</span> 。<br>
<span class="jp-trans">没时间了，所以我没去聚会。</span>
2. <span class="jp-word" title="ともだち- 朋友">友達</span> <em>から</em> <span class="jp-word" title="プレゼント - 礼物">プレゼント</span> が <span class="jp-word" title="くる- 来">来た</span> 。<br>
<span class="jp-trans">礼物来 <em>自</em> 朋友。</span>
3. <span class="jp-word" title="ともだち- 朋友">友達</span> <em>だから</em> <span class="jp-word" title="プレゼント - 礼物">プレゼント</span> が <span class="jp-word" title="くる- 来">来た</span> 。<br>
<span class="jp-trans"><em>因为</em> （那个人是）朋友，所以收到礼物。（有点奇怪听起来）</span>

有上下文的时候，原因和结果都可以省略掉。另外如果是在用丁宁语的话，应该把「から」当成普通名词，在后面加「です」。

### 例一

<span class="jp-word" title="たなか - 田中">田中</span>さん：<span class="jp-word" title="どうして - 为什么">どうして</span><span class="jp-word" title="パーティー - 聚会">パーティー</span>に<span class="jp-word" title="いく - 走">行きませんでしたか</span>。<br>
<span class="jp-trans">田中桑：你为什么没去聚会？</span>

<span class="jp-word" title="やまだ - 山田">山田</span>さん：<span class="jp-word" title="じかん - 时间">時間</span>が<span class="jp-word" title="ある - 存在（无生命）">なかった</span><em>から</em>です。<br>
<span class="jp-trans">山田桑：因为我没时间。</span>

### 例二

<span class="jp-word" title="いちろう - 一郎">一郎</span>：<span class="jp-word" title="パーティー - 聚会">パーティー</span>に<span class="jp-word" title="いく - 走">行かなかった</span>の？<br>
<span class="jp-trans">一郎：你没去聚会？</span>

<span class="jp-word" title="なおこ - 直子">直子</span>：うん、<span class="jp-word" title="じかん - 时间">時間</span>が<span class="jp-word" title="ある - 存在（无生命）">なかった</span><em>から</em>。<br>
<span class="jp-trans">直子：嗯，因为我没时间。</span>

### 例三

省略原因的时候，必须加上表陈述的「だ」或「です」。

<span class="jp-word" title="なおこ - 直子">直子</span>：<span class="jp-word" title="じかん - 时间">時間</span>が<span class="jp-word" title="ある - 存在（无生命）">なかった</span>。<br>
<span class="jp-trans">直子：我没时间。</span>

<span class="jp-word" title="いちろう - 一郎">一郎</span>：<em>だから</em><span class="jp-word" title="パーティー - 聚会">パーティー</span>に<span class="jp-word" title="いく - 走">行かなかった</span>の？<br>
<span class="jp-trans">一郎：所以你没去聚会？</span>

注意我们也可以使用表解释的「の」来表达同样的意思。换句话说，<span class="jp-word" title="やまだ - 山田">山田</span>さん也可以这么说，「<span class="jp-word" title="じかん - 时间">時間</span>が<span class="jp-word" title="ある - 存在（无生命）">なかった</span>のです」或「<span class="jp-word" title="じかん - 时间">時間</span>が<span class="jp-word" title="ある - 存在（无生命）">なかった</span>んです」，而<span class="jp-word" title="なおこ - 直子">直子</span>则可以说「<span class="jp-word" title="じかん - 时间">時間</span>が<span class="jp-word" title="ある - 存在（无生命）">なかった</span>の」（这里假设她喜欢使用女性化的口吻说话）。实际上，这大概也是「ので」这个词的由来。比如你想把这两个句子，「<span class="jp-word" title="じかん - 时间">時間</span>が<span class="jp-word" title="ある - 存在（无生命）">なかった</span>のだ」和「<span class="jp-word" title="パーティー - 聚会">パーティー</span>に<span class="jp-word" title="いく - 走">行かなかった</span>」连起来。记住我们可以把「の」当作名词对待，所以可以用上第一小节学到的知识。

<span class="jp-word" title="じかん - 时间">時間</span>が<span class="jp-word" title="ある - 存在（无生命）">なかった</span>のだ ＋ <span class="jp-word" title="パーティー - 聚会">パーティー</span>に<span class="jp-word" title="いく - 走">行かなかった</span><br>
<span class="jp-trans">变成了：</span><br>
<span class="jp-trans"><span class="jp-word" title="じかん - 时间">時間</span>が<span class="jp-word" title="ある - 存在（无生命）">なかった</span><em>ので</em><span class="jp-word" title="パーティー - 聚会">パーティー</span>に<span class="jp-word" title="いく - 走">行かなかった</span>。</span>

实际上，「ので」基本上跟「から」可以互换使用，除了一些微妙的区别。「から」显式的表明了前面的句子是原因，而「ので」这仅仅是把两个句子连了起来，然后把前一个句子变为了解释的口吻。我称之为因果关系，即发生了 [X]，导致发生了 [Y]。这跟「から」有点不同，用后者的时候，我们声明发生了 [Y] 的<i>直接原因</i>是发生了 [X]。这种区别使得「ので」听起来口气更柔和，也更礼貌，所以在解释做了失礼之事的原因的时候，人们更倾向于用它而非「から」。

- <span class="jp-word" title="ちょっと - 一点">ちょっと</span> <span class="jp-word" title="いそがしい- 忙">忙しい</span> <em>ので</em> 、 <span class="jp-word" title="そろそろ - 逐渐；即将">そろそろ</span> <span class="jp-word" title="しつれい - 失礼">失礼</span> <span class="jp-word" title="する - 做">します</span> 。<br>
<span class="jp-trans">我有点忙，所以一会儿就要先离开了。</span>

（「<span class="jp-word" title="しつれい - 失礼">失礼</span><span class="jp-word" title="する - 做">します</span>」的字面意思是「失礼」，但却通常用来礼貌的表示「失陪」、「叨扰」的意思。）

<b>切记：</b>别忘了解释口气的「の」前面如果是字典形名词或者な形容词的话，一定要加「な」。忘了的话请复习[助词「の」用作解释](/grammar/nounparticles/)小节。

1. <span class="jp-word" title="わたし - 我；我自己">私</span> は <span class="jp-word" title="がくせい - 学生">学生</span> <em>な</em> ので、 <span class="jp-word" title="おかね - 钱">お金</span> が <span class="jp-word" title="ある - 存在（无生命）">ない</span> んです。<br>
<span class="jp-trans">我是学生，所以我没有钱（字面意思：没有钱）。</span>
2. <span class="jp-word" title="ここ - 这里">ここ</span> は <span class="jp-word" title="しずか- 安静">静か</span> <em>な</em> ので、 <span class="jp-word" title="とても - 非常">とても</span> <span class="jp-word" title="おだやか - 冷静，平稳">穏やか</span> です。<br>
<span class="jp-trans">因为安静，所以这里很平静。</span>
3. <em>な</em> ので、 <span class="jp-word" title="ともだち- 朋友">友達</span> に <span class="jp-word" title="あう - 会见">会う</span> <span class="jp-word" title="じかん - 时间">時間</span> が <span class="jp-word" title="ある - 存在（无生命）">ない</span> 。<br>
<span class="jp-trans">那就是没时间见朋友的原因。</span>

如同解释口气的「の」可以简写成「ん」一样，说话的时候「ので」也可以改成「んで」，因为这样可以省略音节 / o / 的发音，形成连读。

1. <span class="jp-word" title="じかん - 时间">時間</span> が <span class="jp-word" title="ある - 存在（无生命）">なかった</span> <em>んで</em> <span class="jp-word" title="パーティー - 聚会">パーティー</span> に <span class="jp-word" title="いく - 走">行かなかった</span> 。<br>
<span class="jp-trans">因为没时间，所以没去聚会。</span>
2. <span class="jp-word" title="ここ - 这里">ここ</span> は <span class="jp-word" title="しずか- 安静">静か</span> な <em>んで</em> 、 <span class="jp-word" title="とても - 非常">とても</span> <span class="jp-word" title="おだやか - 冷静，平稳">穏やか</span> です。<br>
<span class="jp-trans">这里安静，所以很平静。</span>
3. <em>なんで</em> 、 <span class="jp-word" title="ともだち- 朋友">友達</span> に <span class="jp-word" title="あう - 会见">会う</span> <span class="jp-word" title="じかん - 时间">時間</span> が <span class="jp-word" title="ある - 存在（无生命）">ない</span> 。<br>
<span class="jp-trans">那就是没时间见朋友的原因。</span>

## 用「のに」表达「尽管」 {#part5}

### 词汇学习

1. 毎日 【まい・にち】 - 每天
2. 運動 【うん・どう】 - 运动
3. する （例外） - 做
4. 全然 【ぜん・ぜん】 - 全然 （用在否定句）
5. 痩せる 【や・せる】 （る动词） - 瘦身
6. 学生 【がく・せい】 - 学生
7. 彼女 【かの・じょ】 - 她；女朋友
8. 勉強 【べん・きょう】 - 学习

语法层面上，「のに」的用法跟「ので」完全一样，用来连接两个简单句的时候，意思是「尽管 [ 第二句 ]，但还是 [ 第一句 ]」。句子的顺序是 [ 第二句 ] のに [ 第一句 ]。

### 示例

1. <span class="jp-word" title="まいにち - 每天">毎日</span> <span class="jp-word" title="うんどう - 运动">運動</span> <span class="jp-word" title="する - 做">した</span> <em>のに</em> 、 <span class="jp-word" title="ぜんぜん - 全然">全然</span> <span class="jp-word" title="やせる - 瘦身">痩せなかった</span> 。<br>
<span class="jp-trans">尽管每天都运动，我还是没瘦下来。</span>
2. <span class="jp-word" title="がくせい - 学生">学生</span> な <em>のに</em> 、 <span class="jp-word" title="かのじょ - she, girlfriend">彼女</span> は <span class="jp-word" title="べんきょう - 学习">勉強</span> <span class="jp-word" title="する - 做">しない</span> 。<br>
<span class="jp-trans">尽管是学生，她却不学习。</span>

## 用「が」和「けど」表达转折 {#part6}

### 词汇学习

1. デパート - 百货
2. 行く 【い・く】 （う动词） - 走
3. 何 【なに／なん】 - 什么
4. 全然 【ぜん・ぜん】 - 全然 （用在否定句）
5. 欲しい 【ほ・しい】 （い形容词） - 令人想要的
6. 友達 【とも・だち】- 朋友
7. 聞く 【き・く】 （う动词） - 问；听
8. 知る 【し・る】 （う动词） - 知道
9. 今日 【きょう】 - 今天
10. 暇 【ひま】 有空
11. 明日 【あした】 - 明天
12. 忙しい 【いそが・しい】 （い形容词）- 忙
13. 彼 【かれ】 - 他；男朋友
14. まだ - 还
15. 好き 【す・き】 （な形容词） - 惹人喜欢的
16. いい （い形容词） - 好的
17. 物 【もの】 - 东西
18. たくさん - （数量）很多
19. ある （う动词） - 存在（无生命）
20. 見る 【み・る】 （る动词） - 看
21. 面白い 【おも・しろ・い】（い形容词） - 有趣

与「から」和「ので」的用法相同，「が」和「けど」也是用来连接两个句子，但表达的是转折的意思。和「から」一样，字典形名词和な形容词后面也要加表陈述的「だ」。同样类似「から」和「ので」，前后部分也都可以省略。

### 示例

1. <span class="jp-word" title="デパート - 百货">デパート</span> に <span class="jp-word" title="いく - 走">行きました</span> <em>が</em> 、 <span class="jp-word" title="なにも - 什么也没有">何も</span> <span class="jp-word" title="ほしい - 令人想要的">欲しくなかった</span> です。<br>
<span class="jp-trans">我去了百货，但却没有我想要的东西。</span>
2. <span class="jp-word" title="ともだち- 朋友">友達</span> に <span class="jp-word" title="きく - 听, to ask">聞いた</span> <em>けど</em> 、 <span class="jp-word" title="しる - 知道">知らなかった</span> 。<br>
<span class="jp-trans">我问了（或：听说自）一个朋友，但他（或我）不知道。</span>
3. <span class="jp-word" title="きょう - 今天">今日</span> は <span class="jp-word" title="ひま-  有空，闲暇">暇</span> <em>だけど</em> 、 <span class="jp-word" title="あした - 明天">明日</span> は <span class="jp-word" title="いそがしい- 忙">忙しい</span> 。<br>
<span class="jp-trans">我今天有空，但明天忙。。</span>
4. <em>だけど</em> 、 <span class="jp-word" title="かれ - 他">彼</span> が <span class="jp-word" title="まだ - 还">まだ</span> <span class="jp-word" title="すき - 喜欢">好き</span> なの。<br>
<span class="jp-trans">即便如此，还是喜欢他（解释的、女性口吻）</span>

「<span class="jp-word" title="きく - 听, to ask">聞く</span>」这个词既能表示「听」又能表示「问」，这可能让你觉得奇怪。但在上下文里，它的含义通常都很清楚。在第二个例子里，我们假设朋友不知道，所以说话人应该是在问她朋友。我们又一次看到了上下文对于日语的重要性，因为没有主语也没有话题的时候，这个句子也可以理解为「我从朋友那里听说了，但还是不知道」。

如「から」和「ので」的区别一样, 「が」相比「けど」口气更软，也更有礼貌。一个不成文的规矩是，「が」一般跟在「～ます」或「～です」后面，而「けど」则跟在口语句尾。「けど」还有一个稍微正式一点的版本「けれど」，以及一个更正式的版本「けれども」，这些在后面讲到正式表达语的时候会介绍。

与英语里面的转折词如 "but" 和 "however" 不同，「けど」和「が」不总是表达直接的转折。很多时候，特别是引入新话题时，它们被用来作为普通的连接词。在下面的例句里，我们看不到转折，「が」和「けど」只是用来连接句子。在这种情况下，用 "and" 而不是「但是」来翻译更合适（译者注：中文的话，似乎就不用翻译了）。

1. <span class="jp-word" title="デパート - 百货">デパート</span> に <span class="jp-word" title="いく - 走">行きました</span> <em>が</em> 、 <span class="jp-word" title="いい - 好的">いい</span> <span class="jp-word" title="もの - 东西">物</span> が <span class="jp-word" title="たくさん - 很多">たくさん</span> <span class="jp-word" title="ある - 存在（无生命）">ありました</span> 。<br>
<span class="jp-trans">我去了百货，那里有很多好东西。</span>
2. <span class="jp-word" title="マトリックス - matrix">マトリックス</span> を <span class="jp-word" title="みる - 看">見た</span> <em>けど</em> 、 <span class="jp-word" title="おもしろい - 有趣">面白かった</span> 。<br>
<span class="jp-trans">我看了《黑客帝国》，有意思。</span>

## 用「し」表示多种原因 {#part7}

### 词汇学习

1. どうして - 为什么
2. 友達 【とも・だち】- 朋友
3. 先生 【せん・せい】 - 老师
4. 年上 【とし・うえ】 - 年长
5. 彼 【かれ】 - 他；男朋友
6. 好き 【す・き】 （な形容词） - 讨人喜欢的
7. 優しい 【やさ・しい】 （い形容词） - 温柔；友善
8. かっこいい （い形容词） - 酷；帅
9. 面白い 【おも・し・ろい】 （い形容词） - 有趣

如果你想用多个状态或者动作来表示原因，可以在每个关系从句后面加上「し」。这跟助词「や」 很像，只不过它是用于动词和状态表达语。再次提醒，用在状态表达语时，对于字典形名词或な形容词.必须显式的加上「だ」。

### 例一

Ａ：<span class="jp-word" title="どうして - 为什么">どうして</span><span class="jp-word" title="ともだち- 朋友">友達</span>じゃないんですか？<br>
<span class="jp-trans">A：为什么（他/她）不是朋友？</span>

Ｂ：<span class="jp-word" title="せんせい - 老师">先生</span><em>だし</em>、<span class="jp-word" title="としうえ - 年长">年上</span><em>だし</em>・・・。<br>
<span class="jp-trans">B：这个，他/她是老师，比我老……</span>

### 例二

Ａ：<span class="jp-word" title="どうして - 为什么">どうして</span><span class="jp-word" title="かれ - 他">彼</span>が<span class="jp-word" title="すき - 喜欢">好き</span>なの？<br>
<span class="jp-trans">A：（你）为什么喜欢他？</span>

Ｂ：<span class="jp-word" title="やさしい - 温柔；友善">優しい</span><em>し</em>、<span class="jp-word" title="かっこいい - 酷，帅">かっこいい</span><em>し</em>、<span class="jp-word" title="おもしろい - 有趣">面白い</span>から。<br>
<span class="jp-trans">B：因为他人好，有魅力，还有趣（等等）。</span>

注意你也可以说「<span class="jp-word" title="やさしい - 温柔；友善">優しくて</span>、<span class="jp-word" title="かっこいい - 酷，帅">かっこよくて</span>、<span class="jp-word" title="おもしろい - 有趣">面白い</span>から。」，但就像助词「と」和「や」的区别一样，「し」暗示可能还有其他原因。

## 用「～たりする」表达多个动作或状态 {#part8}

### 词汇学习

1. する （例外） - 做
2. 食べる 【た・べる】 （る动词） - 吃
3. 飲む 【の・む】 （う动词） - 喝
4. 簡単 【かん・たん】 （な形容词） - 简单
5. 難しい 【むずか・しい】 （い形容词） - 困难
6. 映画 【えい・が】 - 电影
7. 見る 【み・る】 （る动词） - 看
8. 本 【ほん】 - 书
9. 読む 【よ・む】 （う动词） - 读
10. 昼寝 【ひる・ね】 - 午睡
11. この - 这个（これの的缩写）
12. 大学 【だい・がく】 - 大学
13. 授業 【じゅ・ぎょう】 - 班

这是助词「や」的动词版本。想从一堆动词里面挑几个连起来的话，只要挨个活用成过去形，然后加上「り」。在最后还要加上动词「<span class="jp-word" title="する - 做">する</span>」。跟助词「や」一样，时态取决于最后的动词，也就是「<span class="jp-word" title="する - 做">する</span>」（因为它必须加在末尾）。

对于状态表达语你也可以这么用，同样是活用成过去形再加「り」，然后在末尾加「<span class="jp-word" title="する - 做">する</span>」。

<div class="sumbox">
<div class="sumbox-title">用「～たりする」来表示一堆动词中的几个的规则</div>

- <b>对于动词：</b> 挨个活用为过去形并加「り」，然后在结尾加「する」。<br>
<span class="jp-trans">例</span><br>
<span class="jp-trans"><span class="jp-word" title="たべる - 吃">食べ<em>る</em></span> 、 <span class="jp-word" title="のむ - 喝">飲<em>む</em></span> → <span class="jp-word" title="たべる - 吃">食べ<em>た</em></span> 、 <span class="jp-word" title="のむ - 喝">飲<em>んだ</em></span> → <span class="jp-word" title="たべる - 吃">食べた<em>り</em></span> 、 <span class="jp-word" title="のむ - 喝">飲んだ<em>り</em></span> → <span class="jp-word" title="たべる - 吃">食べたり</span> 、 <span class="jp-word" title="のむ - 喝">飲んだり</span> <em><span class="jp-word" title="する - 做">する</span></em></span>
- <b>对状态表达语：</b> 挨个活用为过去形并加「り」，然后在结尾加「する」。<br>
<span class="jp-trans">例</span><br>
<span class="jp-trans"><span class="jp-word" title="かんたん - 简单">簡単</span> 、 <span class="jp-word" title="むずかしい - 困难">難し<em>い</em></span> → <span class="jp-word" title="かんたん - 简单">簡単</span> <em>だった</em> 、 <span class="jp-word" title="むずかしい - 困难">難し<em>かった</em></span> → <span class="jp-word" title="かんたん - 简单">簡単</span> だった <em>り</em> 、 <span class="jp-word" title="むずかしい - 困难">難しかった<em>り</em></span> → <span class="jp-word" title="かんたん - 简单">簡単</span> だったり、 <span class="jp-word" title="むずかしい - 困难">難しかったり</span> <em><span class="jp-word" title="する - 做">する</span></em></span>
</div>

1. <span class="jp-word" title="えいが - 电影">映画</span> を <em><span class="jp-word" title="みる - 看">見たり</span></em> 、 <span class="jp-word" title="ほん - 书">本</span> を <em><span class="jp-word" title="よむ - 读">読んだり</span></em> 、 <em><span class="jp-word" title="ひるね - 午睡">昼寝</span><span class="jp-word" title="する - 做">したり</span></em> する。<br>
<span class="jp-trans">我做一些事，例如看电影、看书和午睡（等等）。</span>
2. <span class="jp-word" title="この - 这">この</span> <span class="jp-word" title="だいがく - 大学">大学</span> の <span class="jp-word" title="じゅぎょう - 班">授業</span> は <span class="jp-word" title="かんたん - 简单">簡単</span> <em>だったり</em> 、 <em><span class="jp-word" title="むずかしい - 困难">難しかったり</span></em> <span class="jp-word" title="する - 做">する</span> 。<br>
<span class="jp-trans">这所大学的课程有时候简单，有时候难（可能还有其他情况）。</span>

可以看到，时态以及肯定、否定均由最后的「<span class="jp-word" title="する - 做">する</span>」的活用状态控制。

1. <span class="jp-word" title="えいが - 电影">映画</span> を <span class="jp-word" title="みる - 看">見たり</span> 、 <span class="jp-word" title="ほん - 书">本</span> を <span class="jp-word" title="読む - 读">読んだり</span> <em><span class="jp-word" title="する - 做">した</span></em> 。<br>
<span class="jp-trans">我 <em>做了</em> 一些事，例如看电影、看书（等等）。</span>
2. <span class="jp-word" title="えいが - 电影">映画</span> を <span class="jp-word" title="みる - 看">見たり</span> 、 <span class="jp-word" title="ほん - 书">本</span> を <span class="jp-word" title="読む - 读">読んだり</span> <em><span class="jp-word" title="する - 做">しない</span></em> 。<br>
<span class="jp-trans">我 <em>不做</em> 一些事，例如看电影、看书（等等）。</span>
3. <span class="jp-word" title="えいが - 电影">映画</span> を <span class="jp-word" title="みる - 看">見たり</span> 、 <span class="jp-word" title="ほん - 书">本</span> を <span class="jp-word" title="読む - 读">読んだり</span> <em><span class="jp-word" title="する - 做">しなかった</span></em> 。<br>
<span class="jp-trans">我 <em>没做</em> 一些事，例如看电影、看书（等等）。</span>
