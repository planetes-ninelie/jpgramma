日语里面有很多种表达相似性的说法，如何使用要取决于外表、行为或者结果。第一次学这些表达方式的时候，你很难搞明白他们之间的区别，因为翻译成英语都是一样的。本节的目的是讲解这些表达方式之间的区别，这样你可能就能入个门，知道想说什么的时候该用什么合适。

## 用<span class="jp-word" title="よう - 样子">よう</span> （<span class="jp-word" title="よう - 样子">様</span>）表达相似性 {#part2}

### 词汇学习

1. ここ - 这里
2. 誰 【だれ】 - 谁
3. いる （る动词） - 存在（有生命）
4. 映画 【えい・が】 - 电影
5. 観る 【み・る】 （る动词） - 观看
6. 学生 【がく・せい】 - 学生
7. 静か 【しず・か】 （な形容词）- 安静
8. あの - 远处那个（あれの的缩写）
9. 人 【ひと】 - 人
10. 見る 【み・る】 （る动词） - 看
11. 気 【き】 - 心情；意向
12. する （例外） - 做
13. 彼 【かれ】 - 他；男朋友
14. 雰囲気【ふん・い・き】 - 气氛；心情
15. ちょっと - 一点
16. 怒る 【おこ・る】 （う动词） - 生气
17. 聞こえる 【き・こえる】 （る动词） - 变得可听见
18. 何 【なに／なん】 - 什么
19. 起こる 【おこ・る】 （う动词） - 发生
20. 言う 【い・う】 （う动词）- 说

我们已经[简单介绍过](/grammar/surunaru/#part4)「<span class="jp-word" title="よう - 样子">よう</span>」，知道「<span class="jp-word" title="よう - 样子">よう</span>」是「样子」的意思，我们可以用它来说某物是什么样子，或处于什么状态。这个词也可以用来在多种语法中表示相似性。最简单的例子就是用它来修饰状语从句了，如果一个句子用「<span class="jp-word" title="よう - 样子">よう</span>」结尾，你必须加上显式表达状态的「だ」、「です」或「でございます」。

1. <span class="jp-word" title="ここ - 这里">ここ</span> には、 <span class="jp-word" title="だれも - nobody">誰も</span> <span class="jp-word" title="いる - 存在（有生命）">いない</span> <em><span class="jp-word" title="よう - 样子">よう</span>だ</em> 。<br>
<span class="jp-trans">看样子这里没人。</span>
2. <span class="jp-word" title="えいが - 电影">映画</span> を <span class="jp-word" title="みる - 观看">観た</span> <em><span class="jp-word" title="よう - 样子">よう</span>です</em> 。<br>
<span class="jp-trans">看样子（他）看了这部电影。</span>

直接修饰名词或な形容词时，你必须在名词后面加上助词「の」，在な形容词后面加上「な」。

1. <span class="jp-word" title="がくせい - 学生">学生</span> <em>の</em> <span class="jp-word" title="よう - 样子">よう</span> だ。<br>
<span class="jp-trans">看样子是个学生。</span>
2. <span class="jp-word" title="ここ - 这里">ここ</span> は <span class="jp-word" title="しずか- 安静">静か</span> <em>な</em> <span class="jp-word" title="よう - 样子">よう</span> だ。<br>
<span class="jp-trans">看样子很安静。</span>

注意例一并不是说那个人看着<i>像个学生</i>。更确切的，表状态的「だ」说明了那个人看上去<i>是个学生</i>。顺便说一下，你不能用「<span class="jp-word" title="おいしい - 可口">おいしい</span><span class="jp-word" title="よう - 样子">よう</span>だ」来说什么东西看起来好吃。这句话的意思是，「这道菜很明显是好吃的」，这其实很粗鲁。

你也可以把它作为な形容词，来形容某物像是另外的东西。

1. <span class="jp-word" title="あの - 那">あの</span> <span class="jp-word" title="ひと - 人">人</span> を <span class="jp-word" title="みる - 看">見た</span> <span class="jp-word" title="よう - 样子">よう</span> <em>な</em> 気がした。<br>
<span class="jp-trans">有一种以前见过那个人的感觉。</span>
2. <span class="jp-word" title="かれ - 他；男朋友">彼</span> は <span class="jp-word" title="がくせい - 学生">学生</span> の <span class="jp-word" title="よう - 样子">よう</span> <em>な</em> <span class="jp-word" title="ふんいき - 气氛">雰囲気</span> ですね。<br>
<span class="jp-trans">他有点学生气。</span>

最后，我们可以加上目标助词来表示「听起来像」或「说的好像」。

1. <span class="jp-word" title="ちょっと - 一点">ちょっと</span> <span class="jp-word" title="おこる - 生气">怒った</span> <span class="jp-word" title="よう - 样子">よう</span> <em>に</em> <span class="jp-word" title="きこえる - 变得可听见">聞こえた</span> 。<br>
<span class="jp-trans">能听出（她）好像有点生气了。</span>
2. <span class="jp-word" title="なにも - 什么也没有">何も</span> <span class="jp-word" title="おこる - 发生">起こらなかった</span> <span class="jp-word" title="よう - 样子">よう</span> <em>に</em> <span class="jp-word" title="いう- 说">言った</span> 。<br>
<span class="jp-trans">（把这事）说的好像什么都没发生过似的。</span>

## 用「みたい」表示某物看起来像是别的东西 {#part3}

### 词汇学习

1. 見る 【み・る】 （る动词） - 看
2. 犬 【いぬ】 - 狗
3. もう - 已经
4. 売り切れ 【う・り・き・れ】 - 售罄
5. 制服 【せい・ふく】 - 制服
6. 着る 【き・る】 （る动词） - 穿
7. 姿 【すがた】 - 姿态
8. 学生 【がく・せい】 - 学生
9. この - 这个（これの的缩写）
10. ピザ - pizza
11. お好み焼き 【お・この・み・や・き】 - 煎菜饼（日式煎饼）
12. 見える 【み・える】 （る动词） - 变得可见

另一种更为随意表示相似性的说法是「みたい」。别把它跟「<span class="jp-word" title="みる - 看">見る</span>」的「たい」活用搞混了，他们最主要的区别是「みたい」可以直接像助词那样加到名词、形容词和动词后面，而「～たい」这种い形容词显然不行。

### 示例

1. <span class="jp-word" title="もう - 已经">もう</span> <span class="jp-word" title="うりきれ - 售罄">売り切れ</span> <em>みたい</em> 。<br>
<span class="jp-trans">看样子已经卖光了。</span>
2. <span class="jp-word" title="せいふく - 制服">制服</span> を <span class="jp-word" title="きる - 穿">着ている</span> <span class="jp-word" title="すがた - 姿态">姿</span> を <span class="jp-word" title="みる - 看">みる</span> と、 <span class="jp-word" title="がくせい - 学生">学生</span> <em>みたい</em> です。<br>
<span class="jp-trans">那个穿制服的身影，（他）看着像个学生。</span>

这里隐含的意思是那个穿制服的人不是真正的学生，因为他（她）只不过<i>看起来</i>像个学生。这与上一小节「<span class="jp-word" title="よう - 样子">よう</span>」的例三不同，那句话隐含的意思是那人像是（但实际可能也不是）学生。再次提醒，我们也不能用「<span class="jp-word" title="おいしい - 可口">おいしい</span>みたい」来说什么东西看着挺好吃，因为这其中隐含的意思是实际上那东西不一定好吃。

别忘了「みたい」不能像い形容词的たい形那样活用。

- <span class="jp-word" title="この - 这">この</span> <span class="jp-word" title="ピザ - pizza">ピザ</span> は <span class="jp-word" title="おこのみやき - 日式煎菜饼">お好み焼き</span> みた <em>くない</em> ？<br>
<span class="jp-trans">（みたい要像な形容词那用活用）</span>

- <span class="jp-word" title="この - 这">この</span> <span class="jp-word" title="ピザ - pizza">ピザ</span> は <span class="jp-word" title="おこのみやき - 日式煎菜饼">お好み焼き</span> みたい <em>じゃない</em> ？<br>
<span class="jp-trans">这 pizza 不像日式煎菜饼吗？</span>

「みたい」只是在日常口语中使用，写比较正式的文章时不要用。你可以按下面的方式来用「<span class="jp-word" title="よう - 样子">よう</span>」替代。

1. <span class="jp-word" title="もう - 已经">もう</span> <span class="jp-word" title="うりきれ - 售罄">売り切れ</span> の <em><span class="jp-word" title="よう - 样子">よう</span>だ</em> 。<br>
<span class="jp-trans">看样子它已经卖光了。</span>
2. <span class="jp-word" title="この - 这">この</span> <span class="jp-word" title="ピザ - pizza">ピザ</span> は <span class="jp-word" title="おこのみやき - 日式煎菜饼">お好み焼き</span> の <em><span class="jp-word" title="よう - 样子">よう</span></em> に <span class="jp-word" title="みえる - to be in sight; to appear">見える</span> 。<br>
<span class="jp-trans">这 pizza 看起来像日式煎菜饼。</span>

## 用「～そう」猜测结果 {#part4}

### 词汇学习

1. いい （い形容词） - 好的
2. バランス - 平衡
3. 崩れる 【くず・れる】 （る动词） - 崩塌
4. 一瞬 【いっ・しゅん】 - 一瞬间
5. 倒れる 【たお・れる】 （る动词） - 倒塌
6. この - 这个（これの的缩写）
7. 辺り 【あた・り】 - 周边地区
8. ある （う动词） - 存在（无生命）
9. 漬物 【つけ・もの】 - 酱菜
10. おいしい （い形容词） - 可口
11. これ - 这
12. 結構 【けっ・こう】 - 相当的，合理的
13. やはり／やっぱり - 如我所料，果然
14. 高い 【たか・い】 （い形容词） - 高
15. お前 【お・まえ】 - 你 （口语）
16. 金髪 【きん・ぱつ】 - 金发
17. 女 【おんな】 - 女人
18. 好き 【す・き】 （な形容词） - 惹人喜欢的
19. もう - 已经
20. ～時 【～じ】 - 小时的量词
21. なる （う动词） - 成为
22. 来る 【く・る】 （例外）- 来
23. ただ - 免费；仅仅
24. 試合 【し・あい】 - 比赛
25. その - 那个 （それの的缩写）
26. 人 【ひと】 - 人
27. 学生 【がく・せい】 - 学生
28. かわいい （い形容词） - 可爱
29. かわいそう （い形容词） - 可怜的
30. 犬 【いぬ】 - 狗

英语里面 "seems like" 的问题在于含义太多了。它既可以表示外表相似，也可以表示行为相似，甚至可以表示目前的迹象表明某个结果可能出现。我们现在就要学习如何用日语表达第三个意思：根据目前的情况来看可能出现的结果。

就像这一节已经学过的语法一样，我们可以简单的把「そう」加到动词和形容词结尾，只不过有四种重要的例外。实际上我刚发现，它的活用规则跟我们上一节学过的「～<span class="jp-word" title="すぎる - to exeed；经过">すぎる</span>」语法是基本一致的，唯一区别在于对形容词「<span class="jp-word" title="いい  - 好的">いい</span>」，你必须用「<span class="jp-word" title="いい  - 好的">よさ</span>」加上「そう」来组成「<span class="jp-word" title="いい  - 好的">よさそう</span>」。

<div class="sumbox">
<div class="sumbox-title">活用规则</div>

1. 动词必须改为 <i>词根</i> 。
2. い形容词里的「い」必须去掉，只有「 <span class="jp-word" title="いい  - 好的">いい</span> 」是例外。
3. 「 <span class="jp-word" title="いい  - 好的">いい</span> 」必须先活用为「 <span class="jp-word" title="いい  - 好的">よさ</span> 」。
4. 所有否定式单词的「い」必须替换为「さ」。
5. 该语法不适用于普通名词。
</div>

### 1. 动词必须改为词根

对る动词，去掉「る」

- <span class="jp-word" title="バランス - 平衡">バランス</span> が <span class="jp-word" title="くずれる - 崩塌">崩れて</span> 、 <span class="jp-word" title="いっしゅん - 一瞬间">一瞬</span> <em><span class="jp-word" title="たおれる - 倒塌">倒れ</span></em> そうだった。<br>
<span class="jp-trans">失去了平衡的那一瞬间，我好像要倒下。</span>

对う动词，把末假名从 / u / 段改为同行 / i / 段

- <span class="jp-word" title="この - 这">この</span> <span class="jp-word" title="あたり - 周边地区">辺り</span> に <em><span class="jp-word" title="ある - 存在（无生命）">あり</span></em> そうだけどな。<br>
<span class="jp-trans">好像应该在这周围不过…</span>

### 2. い形容词的「い」必须去掉

下例中，「<span class="jp-word" title="おいしい - 可口">おいしい</span>」的「い」被去掉了

- <span class="jp-word" title="この - 这">この</span> <span class="jp-word" title="つけもの - 酱菜">漬物</span> は <em><span class="jp-word" title="おいしい - 可口">おいし</span></em> そう！<br>
<span class="jp-trans">我觉得这酱菜很好吃（酱菜看起来很好）！</span>

<b>例外</b>：<u>这条规则的唯一例外是「<span class="jp-word" title="いい  - 好的">いい</span>」，它必须改为「よさ」。</u>

- <span class="jp-word" title="これ - 这">これ</span> も <span class="jp-word" title="けっこう - 相当">結構</span> <em><span class="jp-word" title="いい - 好的">よさ</span>そう</em> だけど、 <span class="jp-word" title="やっぱり - 如我所料">やっぱり</span> <span class="jp-word" title="たかい - 高；贵">高い</span> よね。<br>
<span class="jp-trans">这一个看起来也不错，不过如我所想应该很贵，对吧？</span>

な形容词就不用动了

- <span class="jp-word" title="おまえ - 你">お前</span> なら、 <span class="jp-word" title="きんぱつ - 金发">金髪</span> の <span class="jp-word" title="おんな - 女人">女</span> が <em><span class="jp-word" title="すき - 讨人喜欢的，令人想要的">好き</span></em> そうだな。<br>
<span class="jp-trans">凭我对你的了解，我猜你喜欢金发女生。</span>

### 3. 所有否定式单词的「い」必须替换为「さ」。

「<span class="jp-word" title="くる- 来">来る</span>」的否定式是「<span class="jp-word" title="くる- 来">こない</span>」，所以用「～そう」语法的时候就成了「<span class="jp-word" title="くる- 来">こなさ</span>そう」。

1. <span class="jp-word" title="もう - 已经">もう</span> <span class="jp-word" title="じゅうじ - 10 点">10時</span> に <span class="jp-word" title="なる - 成为">なった</span> から、 <em><span class="jp-word" title="くる- 来">来なさ</span>そう</em> だね。<br>
<span class="jp-trans">既然已经 10:00 了，看来那（人）不会来了。</span>
2. <span class="jp-word" title="これ - 这">これ</span> は <span class="jp-word" title="ただ - 仅仅">ただ</span> の <span class="jp-word" title="しあい - 比赛">試合</span> <em>じゃなさそうだ</em> 。<br>
<span class="jp-trans">看样子这不会是一场普通的比赛。</span>

与「～<span class="jp-word" title="すぎる - to exeed；经过">すぎる</span>」语法一样，从「～ない」衍生出来的い形容词，例如「<span class="jp-word" title="もったいない - 浪费的">もったいない</span>」或「<span class="jp-word" title="なさけない - 可怜的">情けない</span>」也要遵循这条规则（本例中也就是「<span class="jp-word" title="もったいない - 浪费的">もったいなさ</span>そう」和「<span class="jp-word" title="なさけない - 可怜的">情けなさ</span>そう）。

### 4. 该语法不适用于普通名词。

- <span class="jp-word" title="その - 那个">その</span> <span class="jp-word" title="ひと - 人">人</span> は <span class="jp-word" title="がくせい - 学生">学生</span> <em>そう</em> 。

我们已经介绍过的语法可以用来表示某物像某物。

1. <span class="jp-word" title="その - 那个">その</span> <span class="jp-word" title="ひと - 人">人</span> は <span class="jp-word" title="がくせい - 学生">学生</span> <em>でしょう</em> 。<br>
<span class="jp-trans">那人也许是学生。</span>
2. <span class="jp-word" title="その - 那个">その</span> <span class="jp-word" title="ひと - 人">人</span> は <span class="jp-word" title="がくせい - 学生">学生</span> <em>だろう</em> 。<br>
<span class="jp-trans">那人也许是学生。</span>

注意不要将这种语法用于「<span class="jp-word" title="かわいい - 可爱">かわいい</span>」。「<span class="jp-word" title="かわいそう - 可怜的">かわいそう</span>」是一个完全不同的单词，用来表示可怜的。「<span class="jp-word" title="かわいい - 可爱">かわいい</span>」的意思已经包含了「看起来」的含义，所以你不必再用本节的任何语法来重复这层意思了。

1. <span class="jp-word" title="この - 这">この</span> <span class="jp-word" title="いぬ - 狗">犬</span> は <span class="jp-word" title="かわいそう - 可怜的">かわいそう</span> 。<br>
<span class="jp-trans">噢，可怜的小狗。</span>
2. <span class="jp-word" title="この - 这">この</span> <span class="jp-word" title="いぬ - 狗">犬</span> は <span class="jp-word" title="かわいい - 可爱">かわいい</span> 。<br>
<span class="jp-trans">这狗可爱。</span>

## 用「～そうだ」表示传闻 {#part5}

### 词汇学习

1. 明日 【あした】 - 明天
2. 雨 【あめ】 - 雨
3. 降る 【ふ・る】（う动词） - 沉淀
4. 毎日 【まい・にち】 - 每天
5. 会う 【あ・う】 （う动词） - 会见
6. 行く 【い・く】 （う动词） - 走
7. 彼 【かれ】 - 他；男朋友
8. 高校生 【こう・こう・せい】 - 高中生
9. 今日 【きょう】 - 今天
10. 田中 【た・なか】 - 田中（名）
11. 来る 【く・る】 （例外）- 来

「～そう」语法之所以有那么复杂的变化，是为了跟这一小节要介绍的语法区分开来：它用来表示你所听闻的事情，但却不一定跟你自己的想法有关系，很有用。这个语法用起来也简单，只要把「そうだ」加到动词和い形容词后面即可。对な形容词和名词，你必须先加上表状态的「だ」。另外请注意，「そう」本身必须紧接「だ」、「です」或「でございます」，否则就可能跟上一小节的语法混淆了。这种语法没有时态。

1. <span class="jp-word" title="あした - 明天">明日</span> 、 <span class="jp-word" title="あめ - 雨">雨</span> が <span class="jp-word" title="ふる - 沉淀">降る</span> <em>そうだ</em> 。<br>
<span class="jp-trans">我听说明天要下雨。</span>
2. <span class="jp-word" title="まいにち - 每天">毎日</span> <span class="jp-word" title="あう - 会见">会い</span> に <span class="jp-word" title="いく - 走">行った</span> <em>そうです</em> 。<br>
<span class="jp-trans">我听说他每天都去见面。</span>

别忘了名词或な形容词要先加上「だ」

- <span class="jp-word" title="かれ - 他；男朋友">彼</span> は、 <span class="jp-word" title="こうこうせい - 高中生">高校生</span> <em>だ</em> そうです。<br>
<span class="jp-trans">我听说他是个高中生。</span>

如果句子以这个语法开头，那么要先加上一个表状态的「だ」，就跟[「だから」](/grammar/compound/#part4)一样。

Ａ：<span class="jp-word" title="きょう - 今天">今日</span>、<span class="jp-word" title="たなか - 田中">田中</span><span class="jp-word" title="さん - 桑">さん</span>は<span class="jp-word" title="くる- 来">こない</span>の？<br>
<span class="jp-trans">A：田中桑今天不来吗？</span>

Ｂ：<em>だそうです</em>。<br>
<span class="jp-trans">B：我听说是的。</span>

## 用「～らしい」表示根据传闻推断的事实 {#part6}

### 词汇学习

1. 今日 【きょう】 - 今天
2. 田中 【た・なか】 - 田中（名）
3. 来る 【く・る】 （例外）- 来
4. あの - 远处那个（あれの的缩写）
5. 人 【ひと】 - 人
6. 何 【なん】 - 什么
7. 美由紀 【み・ゆ・き】 - 美由纪（姓）
8. 友達 【とも・だち】- 朋友
9. 子 【こ】 - 小孩
10. 子供 【こ・ども】 - 小孩
11. 大人 【おとな】 - 成人
12. する （例外） - 做
13. つもり - 意向，计划
14. 大騒ぎ 【おお・さわ・ぎ】 - 大骚乱

「らしい」可以直接黏着到名词、形容词或动词后面，用来表示根据你所听闻的信息，事情看来是确定的。它跟「～そうだ」的区别是「～そうだ」表示你所听说的具体的事情，而「らしい」的意思是根据你所你所听说的一些情报，这事看起来是确定的。「らしい」跟普通い形容词的活用规则一样。

### 例一

Ａ：<span class="jp-word" title="きょう - 今天">今日</span>、<span class="jp-word" title="たなか - 田中">田中</span><span class="jp-word" title="さん - 桑">さん</span>は<span class="jp-word" title="くる- 来">こない</span>の？<br>
<span class="jp-trans">A：田中桑今天不来了吗？</span>

Ｂ：<span class="jp-word" title="くる- 来">こない</span><em>らしい</em>。<br>
<span class="jp-trans">B：（根据我所了解的信息）看样子是。</span>

### 例二

Ａ：<span class="jp-word" title="あの - 那">あの</span><span class="jp-word" title="ひと - 人">人</span>は<span class="jp-word" title="なん - 什么">何</span>なの？<br>
<span class="jp-trans">A：那边那个人是干嘛的？</span>

Ｂ：<span class="jp-word" title="みゆき - 美由纪">美由紀</span><span class="jp-word" title="さん - 桑">さん</span>の<span class="jp-word" title="ともだち- 朋友">友達</span><em>らしい</em>ですよ。<br>
<span class="jp-trans">B：（根据我所了解的信息）看起来是美由纪桑的朋友。</span>

「らしい」的另一个用法是用来表示根据行为判断一个人应该是干什么的。

1. <span class="jp-word" title="あの - 那">あの</span> <span class="jp-word" title="こ - 小孩">子</span> は <span class="jp-word" title="こども - 小孩">子供</span> <em>らしくない</em> 。<br>
<span class="jp-trans">那小孩的举动看上去不像是小孩。</span>
2. <span class="jp-word" title="おとな - 成人">大人</span> <em>らしく</em> <span class="jp-word" title="する - 做">する</span> <span class="jp-word" title="つもり - 计划，意愿">つもり</span> だったのに、 <span class="jp-word" title="おおさわぎ - 大骚动">大騒ぎ</span> <span class="jp-word" title="する - 做">して</span> しまった。<br>
<span class="jp-trans">虽然我想表现得像个成年人，但结果我还是引发了大骚动。</span>

## 「っぽい」：用来表示相似性的俚语 {#part7}

### 词汇学习

1. あの - 远处那个（あれの的缩写）
2. 人 【ひと】 - 人
3. 韓国人 【かん・こく・じん】 - 韩国人
4. 皆 【みんな】 - 所有人
5. もう - 已经
6. 全部 【ぜん・ぶ】 - 所有东西
7. 食べる 【た・べる】 （る动词） - 吃
8. 恭子 【きょう・こ】 - 恭子（姓）
9. 全然 【ぜん・ぜん】 - 全然 （用在否定句）
10. 女 【おんな】 - 女人

有一种非常随意的表示相似性的语法是把「っぽい」加到相似对象的后面。因为它是一种很随意的俚语，上面介绍过的所有表相似的语法都可以用它来替代。

「 っぽい」与普通い形容词的活用规则一致，如例句三所示。

1. <span class="jp-word" title="あの - 那">あの</span> <span class="jp-word" title="ひと - 人">人</span> は <span class="jp-word" title="ちょっと - 一点">ちょっと</span> <span class="jp-word" title="かんこくじん - 韩国人">韓国人</span> <em>っぽい</em> よね。<br>
<span class="jp-trans">那人看上去有点像韩国人，是吧？</span>
2. <span class="jp-word" title="みんな - 所有人">みんな</span> で、 <span class="jp-word" title="もう - 已经">もう</span> <span class="jp-word" title="ぜんぶ - 全部">全部</span> <span class="jp-word" title="たべる - 吃">食べて</span> しまった <em>っぽい</em> よ。<br>
<span class="jp-trans">看样子大家已经把东西全吃光了。</span>
3. <span class="jp-word" title="きょうこ - 恭子">恭子</span> は <span class="jp-word" title="ぜんぜん - 全然">全然</span> <span class="jp-word" title="おんな - 女人">女</span> <em>っぽくない</em> ね。<br>
<span class="jp-trans">恭子一点也没有女人样，是吧？</span>
