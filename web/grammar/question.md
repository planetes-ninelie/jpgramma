## 用丁宁语提问 {#part1}

### 词汇学习

1. 田中 【た・なか】 - 田中（名）
2. お母さん【お・かあ・さん】 - 母亲（尊敬）
3. どこ - 哪里
4. 鈴木 【すず・き】 - 铃木（名）
5. 母 【はは】 - 母亲
6. 買い物 【か・い・もの】 - 购物
7. 行く 【い・く】 （う动词） - 走
8. イタリア - 意大利
9. 料理 【りょう・り】 - 料理
10. 食べる 【た・べる】 （る动词） - 吃
11. すみません - 对不起（礼貌）
12. ちょっと - 一点
13. お腹 【お・なか】 - 胃
14. いっぱい - 满
15. ごめんなさい - 对不起（礼貌）
16. ごめん - 对不起

把提问词放在这里介绍，是因为它主要用来在敬语里面标明问题。虽然在丁宁语里面也完全可以只用声调来表示提问，但人们一般总在结尾加一个提问词。提问词很简单，就是平假名「か」，后面不用跟问号。前面说过，这时候就不能再用表陈述的「だ」了。

### 例一

<span class="jp-word" title="たなか - 田中">田中</span>さん：<span class="jp-word" title="おかあさん - 母亲">お母さん</span>は<span class="jp-word" title="どこ - 哪里">どこ</span>です<em>か</em>。<br>
<span class="jp-trans">田中桑：（你）母亲在哪里？</span>

<span class="jp-word" title="すずき - 铃木">鈴木</span>さん：<span class="jp-word" title="はは - 母亲">母</span>は<span class="jp-word" title="かいもの - 购物">買い物</span>に<span class="jp-word" title="いく - 走">行きました</span>。<br>
<span class="jp-trans">铃木桑：（我）妈去买东西了。</span>

### 例二

キムさん：<span class="jp-word" title="イタリア - 意大利">イタリア</span><span class="jp-word" title="りょうり - 料理">料理</span>を<span class="jp-word" title="たべる - 吃">食べ</span>に<span class="jp-word" title="いく - 走">行きません</span><em>か</em>。<br>
<span class="jp-trans">Kim 桑：去吃意大利菜吗？</span>

鈴木さん：<span class="jp-word" title="すみません - 对不起">すみません</span>。<span class="jp-word" title="ちょっと - 一点">ちょっと</span>、<span class="jp-word" title="おなか - 胃">お腹</span>が<span class="jp-word" title="いっぱい - 满">いっぱい</span>です。<br>
<span class="jp-trans">铃木桑：对不起，（我的）肚子有点饱。</span>

这里问题是像英语里面 "Won't you come in for a drink?" 一样，以邀请的方式提出的。 「<span class="jp-word" title="すみません - 对不起">すみません</span>」是礼貌的道歉方式，稍微不正式一点的说法是「<span class="jp-word" title="ごめんなさい - 对不起">ごめんなさい</span>」，口语的时候就简化为「<span class="jp-word" title="ごめん - 对不起">ごめん</span>」。

## 口语里面的提问词 {#part2}

### 词汇学习

1. こんな - 这种
2. 本当 【ほん・とう】 - 真的
3. 食べる 【た・べる】 （る动词） - 吃
4. そんな - 那种
5. ある （う动词） - 存在（无生命）

也许你会认为提问词在口语里面的意思跟丁宁语里面一样。但是，事实<b>并非</b>如此。提问词「か」用在口语里面的时候一般不是用来提问，而通常用来表示对某件事真假的不确定。根据不同的上下文，它也可以用作反问或表示讽刺。口语里面用「か」要小心，因为可能会让人觉得粗鲁。

### 示例

1. <span class="jp-word" title="こんな - 这种">こんな</span> のを <span class="jp-word" title="ほんとう - 真的">本当</span> に <span class="jp-word" title="たべる - 吃">食べる</span> <em>か</em> ？<br>
<span class="jp-trans">你真以为（他/她）会吃这种东西？</span>
2. <span class="jp-word" title="そんな - 那种">そんな</span> のは、 <span class="jp-word" title="ある - 存在（无生命）">ある</span> <em>か</em> よ！<br>
<span class="jp-trans">我看起来像是会有那种东西的样子吗？！</span>

口语里真正的问题不是用「か」，而是用表解释的助词の或直接用声调来表示的，这一点前面已经说过。

1. <span class="jp-word" title="こんな - 这种">こんな</span> のを <span class="jp-word" title="ほんとう - 真的">本当</span> に <span class="jp-word" title="たべる - 吃">食べる</span> ？<br>
<span class="jp-trans">你真的要吃这种东西吗？</span>
2. <span class="jp-word" title="そんな - 那种">そんな</span> のは、 <span class="jp-word" title="ある - 存在（无生命）">ある</span> <em>の</em> ？<br>
<span class="jp-trans">你有那种东西吗？</span>

## 用在关系从句里的「か」 {#part3}

### 词汇学习

1. 昨日【きのう】 - 昨天
2. 何【なに】 - 什么
3. 食べる 【た・べる】 （る动词） - 吃
4. 忘れる 【わす・れる】 （る动词） - 忘记
5. 彼 【かれ】 - 他；男朋友
6. 言う 【い・う】 （う动词）- 说
7. 分かる 【わ・かる】 （う动词） - 明白
8. 先生 【せん・せい】 - 老师
9. 学校 【がっ・こう】 - 学校
10. 行く 【い・く】 （う动词） - 走
11. 教える 【おし・える】 （る动词） - 教；通知
12. どう - 如何
13. 知る 【し・る】 （う动词） - 知道

提问词的另一个用法是语法性质的，跟礼节无关。当提问词加在关系从句末尾的时候，就构成了整句里面的一个迷你问题。这样我们就可以谈论这个问题。例如，你可以跟人讨论「我昨天吃什么了？」这个问题。下面例子里，被谈论的问题用红字标出。

1. <em><span class="jp-word" title="きのう - 昨天">昨日</span><span class="jp-word" title="なに - 什么">何</span>を<span class="jp-word" title="たべる - 吃">食べた</span>か</em> <span class="jp-word" title="わすれる - 忘记">忘れた</span> 。<br>
<span class="jp-trans">忘记了昨天吃了什么。</span>
2. <span class="jp-word" title="かれ - 他">彼</span> は <em><span class="jp-word" title="なに - 什么">何</span>を<span class="jp-word" title="いう- 说">言った</span>か</em> <span class="jp-word" title="分かる - 明白">分からない</span> 。<br>
<span class="jp-trans">不明白他说了什么。</span>
3. <em><span class="jp-word" title="せんせい - 老师">先生</span>が<span class="jp-word" title="がっこう - 学校">学校</span>に<span class="jp-word" title="いく - 走">行った</span>か</em> <span class="jp-word" title="おしえる - 教">教えない</span> ？<br>
<span class="jp-trans">你能告诉我老师去没去学校吗？</span>

对于像上面第三个例句这样用「是/否」来回答的问题，一般会加上「どうか」，这基本上跟英语里面 "whether or not" 一个意思。你也可以把正反两种猜测都加上来表示相同的意思。

1. <span class="jp-word" title="せんせい - 老师">先生</span> が <span class="jp-word" title="がっこう - 学校">学校</span> に <span class="jp-word" title="いく - 走">行った</span> か <em><span class="jp-word" title="どう - 如何">どう</span>か</em> <span class="jp-word" title="しる - 知道">知らない</span> 。<br>
<span class="jp-trans">不知道老师是否去了学校。</span>
2. <span class="jp-word" title="せんせい - 老师">先生</span> が <span class="jp-word" title="がっこう - 学校">学校</span> に <span class="jp-word" title="いく - 走">行った</span> か <em><span class="jp-word" title="いく - 走">行かなかった</span>か</em> <span class="jp-word" title="しる - 知道">知らない</span> 。<br>
<span class="jp-trans">不知道老师是去学校了还是没去。（这里用的是行く的过去否定式加か）</span>

<span class="jp-trans">（解释口吻）你要从这里面选一个出来。</span>

## 表示包含的疑问词 {#part5}

### 词汇学习

1. 全部 【ぜん・ぶ】 - 所有东西
2. 皆 【みんな】 - 所有人
3. 皆さん 【みな・さん】 - 所有人（礼貌）
4. この - 这个（これの的缩写）
5. 質問 【しつ・もん】 - 问题
6. 答え 【こた・え】 - 回答
7. 知る 【し・る】 （う动词） - 知道
8. 友達 【とも・だち】- 朋友
9. 遅れる 【おく・れる】 （る动词） - 迟到
10. ここ - 这里
11. ある （う动词） - 存在（无生命）
12. レストラン - 餐馆
13. おいしい （い形容词） - 可口
14. 今週末 【こん・しゅう・まつ】 - 这周末
15. 行く 【い・く】 （う动词） - 走

上面表格里的疑问词也可以改为加「も」，用在疑问句里，表示「没人」（<span class="jp-word" title="だれも - 没人">誰も</span>）、「没东西」（<span class="jp-word" title="なにも - 什么也没有">何も</span>）和「无处」（<span class="jp-word" title="どこも - everywhere">どこも</span>）等意思。

「<span class="jp-word" title="だれも - 没人">誰も</span>」和「<span class="jp-word" title="なにも - 什么也没有">何も</span>」主要只用在否定句里。奇怪的是，疑问词无法表示「所用东西」这个意思。取而代之，一般习惯用「<span class="jp-word" title="ぜんぶ - 全部">全部</span>」来表达这个意思。另外虽然「誰も」有时可以用来表达「所有人」这个意思，习惯上却是用「<span class="jp-word" title="みんな - 所有人">皆</span>」或「<span class="jp-word" title="みなさん - 所有人">皆さん</span>」。

剩下三个「<span class="jp-word" title="いつも - 总是">いつも</span>」（总是）、「<span class="jp-word" title="どれも - 任一">どれも</span>」（任一）和「<span class="jp-word" title="どこも - everywhere">どこも</span>」（到处）都可以在肯定、否定句里用。

### 示例

1. <span class="jp-word" title="この - 这">この</span> <span class="jp-word" title="しつもん - 问题">質問</span> の <span class="jp-word" title="こたえ - 回答">答え</span> は、 <em><span class="jp-word" title="だれも - 没人">誰も</span></em> <span class="jp-word" title="しる - 知道">知らない</span> 。<br>
<span class="jp-trans">没人知道这个问题的答案。</span>
2. <span class="jp-word" title="ともだち- 朋友">友達</span> は <em><span class="jp-word" title="いつも - 总是">いつも</span></em> <span class="jp-word" title="おくれる - 迟到">遅れる</span> 。<br>
<span class="jp-trans">朋友总是迟到。</span>
3. <span class="jp-word" title="ここ - 这里">ここ</span> に <span class="jp-word" title="ある - 存在（无生命）">ある</span> <span class="jp-word" title="レストラン - 餐馆">レストラン</span> は <em><span class="jp-word" title="どれも - 任一（都）">どれも</span></em> <span class="jp-word" title="おいしい - 可口">おいしくない</span> 。<br>
<span class="jp-trans">这里的餐馆任一家都不好吃。</span>
4. <span class="jp-word" title="こんしゅうまつ - 这周末">今週末</span> は、 <em><span class="jp-word" title="どこにも- 哪里也没有，nowhere">どこにも</span></em> <span class="jp-word" title="いく - 走">行かなかった</span> 。<br>
<span class="jp-trans">这周末哪里也没去。</span>

（语法上，这里的「も」也是主题助词「も」，所以目标助词「に」必须放在它前面。）

### 示例

1. <span class="jp-word" title="この - 这">この</span> <span class="jp-word" title="しつもん - 问题">質問</span> の <span class="jp-word" title="こたえ - 回答">答え</span> は、 <em><span class="jp-word" title="だれでも - 任何人">誰でも</span></em> <span class="jp-word" title="わかる - 明白">分かる</span> 。<br>
<span class="jp-trans">任何人都明白这个问题的答案。</span>
2. <span class="jp-word" title="ひるごはん - 午餐">昼ご飯</span> は、 <em><span class="jp-word" title="どこでも - 任何地方">どこでも</span></em> <span class="jp-word" title="いい - 好的">いい</span> です。<br>
<span class="jp-trans">关于午饭，在哪里吃都好。</span>
3. <span class="jp-word" title="あの - 远方那个">あの</span> <span class="jp-word" title="ひと - 人">人</span> は、 <span class="jp-word" title="ほんとう - 真的">本当</span> に <em><span class="jp-word" title="なんでも - 任何东西">何でも</span></em> <span class="jp-word" title="たべる - 吃">食べる</span> 。<br>
<span class="jp-trans">那个人真的什么都吃。</span>
