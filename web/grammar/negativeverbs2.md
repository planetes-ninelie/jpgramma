我们已经学习了最常用的否定动词类型：那些以「ない」结尾的。但是，否定动词却不止这些。下面介绍的，最有用的当属前两种，它们表示没完成另一个动作就完成了这个动作。其他的就用的不多，或只在很随意的场合使用。不过如果你学习日语一段时间之后是<i>必然</i>会遇到的。

## 没做一件事就做另一件事 {#part2}

### 词汇学习

1. 食べる 【た・べる】 （る动词） - 吃
2. 寝る 【ね・る】 （る动词） - 睡
3. 何 【なに／なん】 - 什么
4. 歯 【は】 - 牙齿
5. 磨く 【みが・く】 （う动词） - 刷；磨
6. 学校 【がっ・こう】 - 学校
7. 行く 【い・く】 （う动词） - 走
8. 宿題 【しゅく・だい】 - 作业
9. する （例外） - 做
10. 授業 【じゅ・ぎょう】 - 班
11. 止める 【や・める】 （る动词） - 停止
12. 方 【1) ほう; 2) かた】 - 1) 方向；立场 2) 人；方法
13. いい （い形容词） - 好的
14. 先生 【せん・せい】 - 老师
15. 相談 【そう・だん】 - 咨询
16. この - 这个（これの的缩写）
17. 取る 【と・る】 （う动词） - 取
18. こと - 事情
19. 出来る 【で・き・る】 （る动词） - 做好；做出；出现
20. 彼 【かれ】 - 他；男朋友
21. 言う 【い・う】 （う动词）- 说
22. 帰る 【かえ・る】 （う动词） - 回家
23. そんな - 那种
24. お酒 【お・さけ】 - 酒
25. 飲む 【の・む】 （う动词） - 喝
26. 当然 【とう・ぜん】 - 自然的
27. 酔っ払う 【よ・っ・ぱ・らう】 （う动词） - 喝醉
28. 勉強 【べん・きょう】 - 学习
29. 東大 【とう・だい】- 东大（「東京大学」的缩写）
30. 入る 【はい・る】 （う动词） - 进入
31. 思う 【おも・う】 （う动词） - 思考，认为

一开始我们学习了如何表示一连串的动作，这对肯定形和否定式动词都适用。例如「我没吃，然后去睡了」就是「<span class="jp-word" title="たべる - 吃">食べなくて</span><span class="jp-word" title="ねる - 睡">寝た</span>。」 但是这听起来有点奇怪，因为吃饭跟睡觉没关系。我们<i>实际</i>想说的是我们没有吃饭<i>就</i>去睡觉了。要想表达这种意思，我们需要用到[授受动词一节](/grammar/favors/#dontdo)最后介绍的否定请求语法的更一般化的形式。换句话说，不用把「い」改为「くて」，而是只要加上「で」就行。

<div class="sumbox">
<div class="sumbox-title">没做一件事就做另一件事</div>

- 要表示没做某个动作 <i>就</i> 做了另一个动作，把「で」加到 <u>没做</u> 的那个动作的否定式后面。
- 例<br>
<span class="jp-trans"><span class="jp-word" title="たべる - 吃">食べ<em>る</em></span> → <span class="jp-word" title="たべる - 吃">食べ<em>ない</em></span> → <span class="jp-word" title="たべる - 吃">食べない</span> <em>で</em></span>
</div>

### 示例

1. <span class="jp-word" title="なにも - 什么也没有">何も</span> <span class="jp-word" title="たべる - 吃">食べない</span> <em>で</em> <span class="jp-word" title="ねる - 睡">寝ました</span> 。<br>
<span class="jp-trans">没吃任何东西就去睡觉。</span>
2. <span class="jp-word" title="は - 牙齿">歯</span> を <span class="jp-word" title="みがく - to brush">磨かない</span> <em>で</em> 、 <span class="jp-word" title="がっこう - 学校">学校</span> に <span class="jp-word" title="いく - 走">行っちゃいました</span> 。<br>
<span class="jp-trans">（不小心）没刷牙就去上学。</span>
3. <span class="jp-word" title="しゅくだい - 作业">宿題</span> を <span class="jp-word" title="する - 做">しない</span> <em>で</em> 、 <span class="jp-word" title="じゅぎょう - 班">授業</span> に <span class="jp-word" title="いく - 走">行く</span> のは、 <span class="jp-word" title="やめる - 停止；退出">やめた</span> <span class="jp-word" title="ほう - 方向，立场">方</span> が <span class="jp-word" title="いい - 好的">いい</span> よ。<br>
<span class="jp-trans">最好不要再不做作业就去上学。</span>
4. <span class="jp-word" title="せんせい - 老师">先生</span> と <span class="jp-word" title="そうだん - 咨询">相談</span> <span class="jp-word" title="する - 做">しない</span> <em>で</em> 、 <span class="jp-word" title="この - 这">この</span> <span class="jp-word" title="じゅぎょう - 班">授業</span> を <span class="jp-word" title="とる - 取">取る</span> <span class="jp-word" title="こと - 事情">こと</span> は <span class="jp-word" title="できる -  做好；做出；出现">出来ない</span> 。<br>
<span class="jp-trans">你不能没跟老师谈过就上这课。</span>

希望对你来说不会太难。另一个表达这个意思的语法是把最后的「ない」换成「ず」。不过两个例外动词 「<span class="jp-word" title="する - 做">する</span>」和「<span class="jp-word" title="くる- 来">くる</span>」要变成「<span class="jp-word" title="する - 做">せず</span>」和「<span class="jp-word" title="くる- 来">こず</span>」。另外大家也经常把目标助词「に」跟这种语法合用。这种说法比「ないで」更正式，普通人说话中不常用。

<div class="sumbox">
<div class="sumbox-title">没做一件事就做另一件事</div>

- 另一种表达没做一件事就做另一件事的方法是把没做的动作的否定式最后的「ない」换成「ず」。<br>
<span class="jp-trans">例</span>
  1. <span class="jp-word" title="たべる - 吃">食べ<em>る</em></span> → <span class="jp-word" title="たべる - 吃">食べ<em>ない</em></span> → <span class="jp-word" title="たべる - 吃">食べ<em>ず</em></span>
  2. <span class="jp-word" title="いく - 走">行<em>く</em></span> → <span class="jp-word" title="いく - 走">行か<em>ない</em></span> → <span class="jp-word" title="いく - 走">行か<em>ず</em></span>
- <b>例外：</b>
  1. <em><span class="jp-word" title="する - 做">する</span></em> → <em><span class="jp-word" title="する - 做">せず</span></em>
  2. <em><span class="jp-word" title="くる- 来">くる</span></em> → <em><span class="jp-word" title="くる- 来">こず</span></em>
</div>

### 示例

1. <span class="jp-word" title="かれ - 他；男朋友">彼</span> は <span class="jp-word" title="なにも - 什么也没有">何も</span> <em><span class="jp-word" title="いう- 说">言わず</span></em> 、 <span class="jp-word" title="かえる - 回家">帰って</span> しまった。<br>
<span class="jp-trans">他什么也没说就回家了。</span>
2. <span class="jp-word" title="なにも - 什么也没有">何も</span> <em><span class="jp-word" title="たべる - 吃">食べず</span></em> に <span class="jp-word" title="そんな - 像那样">そんな</span> に <span class="jp-word" title="おさけ - 酒">お酒</span> を <span class="jp-word" title="のむ - 喝">飲む</span> と <span class="jp-word" title="とうぜん - 自然的">当然</span> <span class="jp-word" title="よっぱらう - 喝醉">酔っ払います</span> よ。<br>
<span class="jp-trans">你要是什么也不吃酒喝那么多，明显会醉。</span>
3. <span class="jp-word" title="べんきょう - 学习">勉強</span> <em><span class="jp-word" title="する - 做">せず</span></em> に <span class="jp-word" title="とうだい- 东大（東京大学）">東大</span> に <span class="jp-word" title="はいる - 进入">入れる</span> と <span class="jp-word" title="おもう - 思考，认为">思わない</span> な。<br>
<span class="jp-trans">我觉得你不可能不学习就进东大。</span>

## 一种随意、阳刚的否定式用法，以「ん」结尾 {#part3}

### 词汇学习

1. する （例外） - 做
2. 来る 【く・る】 （例外）- 来
3. すまん - 对不起 （男性化）
4. すみません - 对不起（礼貌）
5. 知る 【し・る】 （う动词） - 知道
6. 韓国人 【かん・こく・じん】 - 韩国人
7. 結婚 【けっ・こん】 - 婚姻
8. なる （う动词） - 成为
9. そんな - 那种
10. こと - 事情
11. 皆 【みんな】 - 所有人
12. 今日 【きょう】 - 今天
13. 行く 【い・く】 （う动词） - 走

最后我们再学一种主要是老人使用的否定式。因为 「ない」太长了不好发音（我在说反话），你可以把它缩短成「ん」。不过你不能用这种形式直接修饰其他单词；换句话说，不能用以修饰状语从句。跟前面一样，「<span class="jp-word" title="する - 做">する</span>」要变为「<span class="jp-word" title="する - 做">せん</span>」，而「<span class="jp-word" title="くる- 来">くる</span>」变为「<span class="jp-word" title="くる- 来">こん</span>」，不过我倒从没见过或听过「<span class="jp-word" title="くる- 来">こん</span>」。万一你听到「<span class="jp-word" title="すまん - 对不起">すまん</span>」不明白啥意思，那就是这种语法了。注意「<span class="jp-word" title="すみません - 对不起">すみません</span>」实际上是一种丁宁否定式，对应的字典形是「<span class="jp-word" title="すまない - 对不起">すまない</span>」，对吧？再进一步就是「<span class="jp-word" title="すまん - 对不起">すまん</span>」了。这个单词让我脑海里浮现出了<span class="jp-word" title="おじさん - 中年男子，大叔">おじさん</span>的形象，不过也许只有我这么觉得。不管怎么说，这是男性使用的。

<div class="sumbox">
<div class="sumbox-title">否定式的简短发音</div>

- 否定式的简短发音方式是用「ん」代替「ない」.<br>
<span class="jp-trans">例</span><br>
<span class="jp-trans"><span class="jp-word" title="しる - 知道">知<em>る</em></span> → <span class="jp-word" title="しる - 知道">知ら<em>ない</em></span> → <span class="jp-word" title="しる - 知道">知ら<em>ん</em></span></span>
- <b>例外：</b>
  1. <em><span class="jp-word" title="する - 做">する</span></em> → <em><span class="jp-word" title="する - 做">せん</span></em>
  2. <em><span class="jp-word" title="くる- 来">くる</span></em> → <em><span class="jp-word" title="くる- 来">こん</span></em>
</div>

### 示例

1. <span class="jp-word" title="すまん - 对不起">すまん</span> 。<br>
<span class="jp-trans">对不起。</span>
2. <span class="jp-word" title="かんこくじん - 韩国人">韓国人</span> と <span class="jp-word" title="けっこん - 婚姻">結婚</span> <span class="jp-word" title="する - 做">しなくて</span> は <em>ならん</em> ！<br>
<span class="jp-trans">你必须跟韩国人结婚！</span>
3. <span class="jp-word" title="そんな - 像那样">そんな</span> <span class="jp-word" title="こと - 事情">こと</span> は <em><span class="jp-word" title="する - 做">させん</span></em> ！<br>
<span class="jp-trans">我不会允许你这么做的！</span>

俚语里面你甚至可以加上「かった」表示过去式。

- <span class="jp-word" title="みんな - 所有人">皆</span> 、 <span class="jp-word" title="きょう - 今天">今日</span> <span class="jp-word" title="いく - 走">行く</span> って、 <span class="jp-word" title="しる - 知道">知らん<em>かった</em></span> よ。<br>
<span class="jp-trans">我不知道大家今天都去了。</span>

## 一种传统的以「ぬ」结尾的否定式动词活用法 {#part4}

### 词汇学习

1. する （例外） - 做
2. 来る 【く・る】 （例外）- 来
3. 知る 【し・る】 （う动词） - 知道
4. 韓国人 【かん・こく・じん】 - 韩国人
5. 結婚 【けっ・こん】 - 婚姻
6. なる （う动词） - 成为
7. 模擬 【も・ぎ】 - mock
8. 試験 【し・けん】 - 考试
9. 何回 【なん・かい】 - 多少次
10. 失敗 【しっ・ぱい】 - 失败
11. 実際 【じっ・さい】 - 实际的
12. 受ける 【う・ける】 （る动词） - 接受
13. 思う 【おも・う】 （う动词） - 思考，认为
14. 結果 【けっ・か】 - 结果
15. 出る 【で・る】 る动词 - 出来

还有一种否定式动词活用是用「ぬ」替代结尾的「ない」。虽然这是一种老式的传统用法，你还是会时不时遇到。其实我今天就在一个车站指示牌上看到了，所以并不是很罕见。

对任何否定式动词，把结尾的「ない」换成「ぬ」就变成这种传统发音了。与上一小节类似的，「<span class="jp-word" title="する - 做">する</span>」变成「<span class="jp-word" title="する - 做">せぬ</span>」，「<span class="jp-word" title="くる- 来">くる</span>」变成「<span class="jp-word" title="くる- 来">こぬ</span>」如果和你对话的是老人，或者对方想让自己听起来沧桑点，就可能用这种语法。

<div class="sumbox">
<div class="sumbox-title">一种老式的否定式动词读法</div>

- 一种老式的否定式动词活用法是用「ぬ」替代「ない」。<br>
<span class="jp-trans">例</span><br>
<span class="jp-trans"><span class="jp-word" title="しる - 知道">知<em>る</em></span> → <span class="jp-word" title="しる - 知道">知ら<em>ない</em></span> → <span class="jp-word" title="しる - 知道">知ら<em>ぬ</em></span></span>
- <b>例外：</b>
  1. <em><span class="jp-word" title="する - 做">する</span></em> → <em><span class="jp-word" title="する - 做">せぬ</span></em>
  2. <em><span class="jp-word" title="くる- 来">くる</span></em> → <em><span class="jp-word" title="くる- 来">こぬ</span></em>
</div>

### 示例

1. <span class="jp-word" title="かんこくじん - 韩国人">韓国人</span> と <span class="jp-word" title="けっこん - 婚姻">結婚</span> <span class="jp-word" title="する - 做">して</span> は <em>ならぬ</em> ！<br>
<span class="jp-trans">你不能跟韩国人结婚！</span>
2. <span class="jp-word" title="もぎ - mock">模擬</span> <span class="jp-word" title="しけん 测试">試験</span> に <span class="jp-word" title="なんかい - 多次">何回</span> も <span class="jp-word" title="しっぱい - 失败">失敗</span> <span class="jp-word" title="する - 做">して</span> 、 <span class="jp-word" title="じっさい - 实际的">実際</span> に <span class="jp-word" title="うける - 接受">受けて</span> みたら <em><span class="jp-word" title="おもう - 思考，认为">思わぬ</span></em> <span class="jp-word" title="けっか - 结果">結果</span> が <span class="jp-word" title="でる - 出来">出た</span> 。<br>
<span class="jp-trans">模拟考失败多次之后参加真正的考试时，一个从未想过的结果出现了。</span>
