这一节我们学习如何表达在各种特定时间窗口内发生的动作。具体来说是：1、刚被完成的动作；2、一个动作之后紧接发生的动作；3、一个动作进行中时发生的另一个动作；4、重复发生的动作。

## 用「～ばかり」表达刚刚发生的动作 {#part2}

### 词汇学习

1. 食べる 【たべ・る】 （る动词） - 吃
2. すみません - 对不起（礼貌）
3. 今 【いま】- 现在
4. お腹 【お・なか】 - 胃
5. いっぱい - 满
6. キロ - 千克
7. 走る 【はし・る】 （う动词） - 跑
8. 凄い 【すご・い】 （い形容词） - 很大程度上
9. 疲れる 【つか・れる】 （る动词） - 疲劳
10. 家 【1) うち; 2) いえ】 - 1）自己家；2）房子
11. 帰る 【かえ・る】 （う动词） - 回家
12. 昼ご飯 【ひる・ご・はん】 - 午餐
13. もう - 已经
14. 空く 【す・く】 （う动词） - 变空
15. まさか - 不会吧，不可能吧
16. 起きる 【お・きる】 （る动词） - 起

这个语法很有用，用来表示某人刚刚完成了某个动作。比如以前当我刚吃了东西而想礼貌拒绝别人请我吃东西时，就特希望会说这个语法。你只需要把你刚刚完成的动作换成过去形，然后加上「ばかり」。这里只能用过去形，另外要注意不要跟[前面讲数量时候介绍的「ばかり」](/grammar/amount/#part5)混淆了。

正如我们[之前介绍](/grammar/amount/#part5)「ばかり」时所说，口语里面有人会用「ばっか」代替「ばかり」。

<div class="sumbox">
<div class="sumbox-title">用「ばかり」表示刚刚完成的动作</div>

- 要表示一个动作刚刚完成，把它活用成过去形，然后加上「ばかり」。<br>
<span class="jp-trans">例: <span class="jp-word" title="たべる - 吃">食べ<em>る</em></span> → <span class="jp-word" title="たべる - 吃">食べ<em>た</em></span> → <span class="jp-word" title="たべる - 吃">食べた<em>ばかり</em></span></span>
- 口语里面可以把「ばかり」省略为「ばっか」<br>
<span class="jp-trans">例: <span class="jp-word" title="たべる - 吃">食べた</span> <em>ばかり</em> → <span class="jp-word" title="たべる - 吃">食べた</span> <em>ばっか</em></span>

</div>

### 示例

1. <span class="jp-word" title="すみません - 对不起">すみません</span> 、 <span class="jp-word" title="いま- 现在">今</span> <em><span class="jp-word" title="たべる - 吃">食べた</span>ばかり</em> なので、 <span class="jp-word" title="おなか - 胃">お腹</span> が <span class="jp-word" title="いっぱい - 满">いっぱい</span> です。<br>
<span class="jp-trans">不好意思，我刚吃了，肚子饱。</span>
2. <span class="jp-word" title="じゅっきろ - 10 kilos">10キロ</span> を <em><span class="jp-word" title="はしる - 跑">走った</span>ばかり</em> で、 <span class="jp-word" title="すごい - 非常棒；相当的">凄く</span> <span class="jp-word" title="つかれる - 疲劳">疲れた</span> 。<br>
<span class="jp-trans">我刚跑了十公里，很累。</span>
3. <span class="jp-word" title="いま- 现在">今</span> 、 <span class="jp-word" title="いえ - 家; うち - home">家</span> に <em><span class="jp-word" title="かえる - 回家">帰った</span>ばかり</em> です。<br>
<span class="jp-trans">我刚到家。</span>

下面用缩略版本造几个句子。

1. <span class="jp-word" title="ひるごはん - 午餐">昼ご飯</span> を <span class="jp-word" title="たべる - 吃">食べた</span> <em>ばっか</em> なのに、 <span class="jp-word" title="もう - 已经">もう</span> <span class="jp-word" title="おなか - 胃">お腹</span> が <span class="jp-word" title="すく - 变空">空いた</span> 。<br>
<span class="jp-trans">虽然刚吃了午饭，但已经饿了。</span>
2. <span class="jp-word" title="まさか - 不是吧…">まさか</span> 、 <span class="jp-word" title="いま- 现在">今</span> <span class="jp-word" title="おきる - 起床；发生">起きた</span> <em>ばっか</em> なの？<br>
<span class="jp-trans">不会吧，你刚起？</span>

## 用「とたん」表示紧接发生的动作 {#part3}

### 词汇学习

1. 開ける 【あ・ける】 （る动词） - 开
2. 取る 【と・る】 （う动词） - 取
3. 窓 【まど】 - 窗户
4. 猫 【ねこ】 - 猫
5. 跳ぶ 【と・ぶ】 （う动词） - 跳
6. 映画 【えい・が】 - 电影
7. 観る 【み・る】 （る动词） - 观看
8. トイレ - 卫生间
9. 行く 【い・く】 （う动词） - 走
10. 眠い 【ねむ・い】（い形容词） - 困
11. なる （う动词） - 成为

作为「ばかり」的补充，我们再介绍一下如何说一件事刚发生另一件事就发生了。这种语法需要在先发生的那个动作的过去形后面加上「とたん」。另外大家也经常加上目标助词「に」来表明那个特定的时间点。

<div class="sumbox">
<div class="sumbox-title">用「とたん」来描述紧接发生的事情</div>

- 把前面的动作活用为 <i>过去形</i> ，再加上「とたん」或「とたんに」。<br>
<span class="jp-trans">例</span>
  1. <span class="jp-word" title="あける - 开">開け<em>る</em></span> → <span class="jp-word" title="あける - 开">開け<em>た</em></span> → <span class="jp-word" title="あける - 开">開けた</span> <em>とたん（に）</em>
  2. <span class="jp-word" title="とる - 取">取<em>る</em></span> → <span class="jp-word" title="とる - 取">取<em>った</em></span> → <span class="jp-word" title="とる - 取">取った<em>とたん（に）</em></span>
- ※注：你只能在不受你控制的动作上使用这种语法。
</div>

### 示例

1. <span class="jp-word" title="まど - 窗户">窓</span> を <em><span class="jp-word" title="あける - 开">開けた</span>とたんに</em> 、 <span class="jp-word" title="ねこ - 猫">猫</span> が <span class="jp-word" title="とぶ - 跳">跳んで</span> <span class="jp-word" title="いく - 走">いった</span> 。<br>
<span class="jp-trans">我一开窗户，猫就跳了出去。</span>

想看更多例子可以上我们的老朋友 WWWJDIC：[例句](http://www.csse.monash.edu.au/cgi-bin/cgiwrap/jwb/wwwjdic?1Q%C5%D3%C3%BC_1_)。

这里很重要的一点是这种语法只能用在一个动作之后紧接发生的事情，而不能跟你自己做的动作。比如看下面两个例句：

- <span class="jp-word" title="えいが - 电影">映画</span> を <span class="jp-word" title="みる - 观看">観た</span> とたんに、 <em><span class="jp-word" title="トイレ - 厕所">トイレ</span>に<span class="jp-word" title="いく - 走">行きました</span></em> 。<br>
<span class="jp-trans">（去浴室这个动作是你做的，所以不对）</span>

- <span class="jp-word" title="えいが - 电影">映画</span> を <span class="jp-word" title="みる - 观看">観た</span> とたんに、 <em><span class="jp-word" title="ねむい - 困">眠く</span><span class="jp-word" title="なる - 成为">なりました</span></em> 。<br>
<span class="jp-trans">（因为变困是你无法控制的事情，所以这句话没问题）</span>

## 用「<span class="jp-word" title="ながら - 当">ながら</span>」表示两个并行的动作 {#part4}

### 词汇学习

1. 走る 【はし・る】 （う动词） - 跑
2. テレビ - 电视机
3. 観る 【み・る】 （る动词） - 观看
4. 宿題 【しゅく・だい】 - 作业
5. する （例外） - 做
6. 音楽 【おん・がく】 - 音乐
7. 聴く 【き・く】 （う动词） - 听（音乐等）
8. 学校 【がっ・こう】 - 学校
9. 歩く 【ある・く】 （う动词） - 步行
10. 好き 【す・き】 （な形容词） - 讨人喜欢的
11. 相手 【あい・て】 - 对方
12. 何 【なに／なん】 - 什么
13. 言う 【い・う】 （う动词）- 说
14. 自分 【じ・ぶん】 - 自己
15. 気持ち 【き・も・ち】 - 感觉
16. 分かる 【わ・かる】 （う动词） - 明白
17. 欲しい 【ほ・しい】 （い形容词） - 令人想要的
18. 単なる 【たん・なる】 - 简单的
19. わがまま （な形容词） - 自私
20. 思う 【おも・う】 （う动词） - 思考，认为
21. ポップコーン - 爆米花
22. 食べる 【た・べる】 （る动词） - 吃
23. 映画 【えい・が】 - 电影
24. 口笛 【くち・ぶえ】 - 口哨
25. 手紙 【て・がみ】 - 信
26. 書く 【か・く】 （う动词） - 写

你可以用「<span class="jp-word" title="ながら - 当">ながら</span>」来表达两个同时进行的动作。用「<span class="jp-word" title="ながら - 当">ながら</span>」的时候，必须把第一个动词活用为词根，再加上「<span class="jp-word" title="ながら - 当">ながら</span>」。你也可以把「<span class="jp-word" title="ながら - 当">ながら</span>」加到动词否定式后面来表示否定，不过很少人这么用。这个语法的时态取决于第二个动作。

<div class="sumbox">
<div class="sumbox-title">用「ながら」表示并行的动作</div>

- 把第一个动词活用为 <i>词根</i> ，再加上「ながら」<br>
<span class="jp-trans">例</span><br>
<span class="jp-trans"><span class="jp-word" title="はしる - 跑">走<em>る</em></span> → <span class="jp-word" title="はしる - 跑">走<em>り</em></span> → <span class="jp-word" title="はしる - 跑">走り</span> <em><span class="jp-word" title="ながら - 当">ながら</span></em></span>
- 否定式也可以加「 <span class="jp-word" title="ながら - 当">ながら</span> 」<br>
<span class="jp-trans">例</span><br>
<span class="jp-trans"><span class="jp-word" title="はしる - 跑">走<em>る</em></span> → <span class="jp-word" title="はしる - 跑">走<em>らない</em></span> → <span class="jp-word" title="はしる - 跑">走らない</span> <em><span class="jp-word" title="ながら - 当">ながら</span></em></span>
</div>

### 示例

1. <span class="jp-word" title="テレビ - 电视">テレビ</span> を <em><span class="jp-word" title="みる - 观看">観</span><span class="jp-word" title="ながら - 当">ながら</span></em> 、 <span class="jp-word" title="しゅくだい - 作业">宿題</span> を <span class="jp-word" title="する - 做">する</span> 。<br>
<span class="jp-trans">做作业的同时看电视。</span>
2. <span class="jp-word" title="おんがく - 音乐">音楽</span> を <em><span class="jp-word" title="きく - 听">聴き</span><span class="jp-word" title="ながら - 当">ながら</span></em> 、 <span class="jp-word" title="がっこう - 学校">学校</span> へ <span class="jp-word" title="あるく - 步行">歩く</span> のが <span class="jp-word" title="すき - 喜欢">好き</span> 。<br>
<span class="jp-trans">喜欢在走去学校的路上听音乐。</span>
3. <span class="jp-word" title="あいて - 对方">相手</span> に <span class="jp-word" title="なにも - 什么也没有">何も</span> <em><span class="jp-word" title="いう- 说">言わない</span><span class="jp-word" title="ながら - 当">ながら</span></em> 、 <span class="jp-word" title="じぶん - 自己">自分</span> の <span class="jp-word" title="きもち - 感觉">気持ち</span> を <span class="jp-word" title="わかる - 明白">わかって</span> ほしいのは <span class="jp-word" title="たんなる - 简单的">単なる</span> <span class="jp-word" title="わがまま - 自私">わがままだ</span> と <span class="jp-word" title="おもう - 思考，认为">思わない</span> ？<br>
<span class="jp-trans">你不觉得想让别人理解你的感受却又不说话显得很自私吗？</span>

注意句子还是要以主要的动词结尾，也就是说句子的主题动作是结尾那个动词。「<span class="jp-word" title="ながら - 当">ながら</span>」只不过是在描述另一个同一时间在发生的动作。比如把例一里的两个动作对调一下，「<span class="jp-word" title="しゅくだい - 作业">宿題</span>を<span class="jp-word" title="する - 做">し</span><span class="jp-word" title="ながら - 当">ながら</span>、 <span class="jp-word" title="テレビ - 电视">テレビ</span>を<span class="jp-word" title="みる - 观看">観る</span>。」，意思就变成了「看电视的同时做作业」。换句话说，句子的主要动作变为了看电视，而做作业只是同时发生的另一件事。（译者注：中文和英文里面好像不是很分主次的样子。）

句子的时态受主要动词的控制，因此「<span class="jp-word" title="ながら - 当">ながら</span>」本身就不能有时态了。

1. <span class="jp-word" title="ポップコーン - 爆米花">ポップコーン</span> を <span class="jp-word" title="たべる - 吃">食べ</span> <span class="jp-word" title="ながら - 当">ながら</span> 、 <span class="jp-word" title="えいが - 电影">映画</span> を <em><span class="jp-word" title="みる - 观看">観る</span></em> 。<br>
<span class="jp-trans"><em>看</em> 电影时吃爆米花。</span>
2. <span class="jp-word" title="ポップコーン - 爆米花">ポップコーン</span> を <span class="jp-word" title="たべる - 吃">食べ</span> <span class="jp-word" title="ながら - 当">ながら</span> 、 <span class="jp-word" title="えいが - 电影">映画</span> を <em><span class="jp-word" title="みる - 观看">観た</span></em> 。<br>
<span class="jp-trans"><em>之前看</em> 电影的时候在吃爆米花。</span>
3. <span class="jp-word" title="くちぶえ - 口哨">口笛</span> を <span class="jp-word" title="する - 做">し</span> <span class="jp-word" title="ながら - 当">ながら</span> 、 <span class="jp-word" title="てがみ - 信">手紙</span> を <em><span class="jp-word" title="かく - 写">書いて</span>いた</em> 。<br>
<span class="jp-trans"><em>之前写</em> 信的时候在吹口哨。</span>

## 将「<span class="jp-word" title="ながら - 当">ながら</span>」与状态表达语合用 {#part5}

### 词汇学习

1. 残念 【ざん・ねん】 （な形容词） - 不幸的
2. 貧乏 【びん・ぼう】 （な形容词） - 穷
3. 仕事 【し・ごと】 - 工作
4. いっぱい - 满
5. 入る 【はい・る】 （う动词） - 进入
6. 今日 【きょう】 - 今天
7. 行く 【い・く】 （う动词） - 走
8. なる （う动词） - 成为
9. 高級 【こう・きゅう】 （な形容词） - 高级e
10. バッグ - 包包
11. 買う 【か・う】 （う动词）- 买
12. 彼 【かれ】 - 他；男朋友
13. 初心者 【しょ・しん・しゃ】 - 初学者
14. 実力 【じつ・りょく】 - 实力
15. プロ - 专家
16. 同じ 【おな・じ】 - 一样

「<span class="jp-word" title="ながら - 当">ながら</span>」的一个更高级的用法是和隐含的状态表示一起用。换句话说，你可以把它跟名词、形容词合用，来表达这个是这样的同时那个是那样。「隐含的状态表示」的意思是不能加上显式表状态的「だ」，直接往名词或形容词后面加「<span class="jp-word" title="ながら - 当">ながら</span>」就行。比如一个常见的用法是用来表达「很不幸的同时…」，在日语里你可以这么用「<span class="jp-word" title="ざんねん - 不幸的">残念</span><em><span class="jp-word" title="ながら - 当">ながら</span></em>・・・」来表达这个意思。

你也可以把表示包含的助词「も」加到「<span class="jp-word" title="ながら - 当">ながら</span>」后面变成「<span class="jp-word" title="ながら - 当">ながら</span>も」。这时候它的意思就从「当」变为了「<u>即便</u>当」。

<div class="sumbox">
<div class="sumbox-title">将「ながら」或「ながらも」与状态表示语合用</div>

- 想说 [ 这个 ] 是这样，而那个是那样的话，把「 <span class="jp-word" title="ながら - 当">ながら</span> 」加到 [ 这个 ] 后面<br>
<span class="jp-trans">例</span><br>
<span class="jp-trans"><span class="jp-word" title="ざんねん - 不幸的">残念</span> → <span class="jp-word" title="ざんねん - 不幸的">残念</span> <em><span class="jp-word" title="ながら - 当">ながら</span></em></span>
- 想说即便那个是那样，[ 这个 ] 还是这样的话，将「 <span class="jp-word" title="ながらも - 即便，虽然">ながらも</span> 」加到 [ 这个 ] 后面<br>
<span class="jp-trans">例</span><br>
<span class="jp-trans"><span class="jp-word" title="びんぼう - 穷">貧乏</span> → <span class="jp-word" title="びんぼう - 穷">貧乏</span> <em><span class="jp-word" title="ながらも - 即便，虽然">ながらも</span></em></span>
</div>

### 示例

1. <span class="jp-word" title="しごと - 工作">仕事</span> が <span class="jp-word" title="いっぱい - 很多; full">いっぱい</span> <span class="jp-word" title="はいる - 进入">入って</span> 、 <span class="jp-word" title="ざんねん - 不幸的">残念</span> <em><span class="jp-word" title="ながら - 当">ながら</span></em> 、 <span class="jp-word" title="きょう - 今天">今日</span> は <span class="jp-word" title="いく - 走">行けなく</span> <span class="jp-word" title="なる - 成为">なりました</span> 。<br>
<span class="jp-trans">来了很多工作，所以很不幸，我今天没法去了。</span>
2. <span class="jp-word" title="びんぼう - 穷">貧乏</span> <em><span class="jp-word" title="ながらも - 即便，虽然">ながらも</span></em> 、 <span class="jp-word" title="こうきゅう - 高档">高級</span> な <span class="jp-word" title="バッグ - 包包">バッグ</span> を <span class="jp-word" title="かう- 买">買っちゃった</span> よ。<br>
<span class="jp-trans">虽然我穷，但我还是买了一个高档包包。</span>
3. <span class="jp-word" title="かれ - 他；男朋友">彼</span> は、 <span class="jp-word" title="しょしんしゃ - 初学者">初心者</span> <em><span class="jp-word" title="ながらも - 即便，虽然">ながらも</span></em> 、 <span class="jp-word" title="じつりょく - 实力">実力</span> は <span class="jp-word" title="プロ - 专家">プロ</span> と <span class="jp-word" title="おなじ - 一样">同じ</span> だ。<br>

## 用「まくる」表示不计后果地重复做某事 {#part6}

### 词汇学习

1. やる （う动词） - 做
2. ゲーム - 游戏
3. はまる （う动词） - 陷入
4. 最近 【さい・きん】 - 最近
5. パソコン - 电脑
6. 使う 【つか・う】 （う动词） - 使用
7. アメリカ - 美国
8. いる （る动词） - 存在（有生命）
9. 時 【とき】 - 时间
10. コーラ - 可乐
11. 飲む 【の・む】 （う动词） - 喝

WWWJDIC 非常简洁的将这个动词解释为一个「用来表示不计后果的动词后缀」。很不幸的，它并没有多说该如何使用它。实际上也没什么好解释的。在词根后面加上「まくる」就行。不过既然这是一个进行中的行为，那除非是将来才做的事，就得用<i>[进行时](/grammar/teform/#part2)</i>。这是一个很口语化的表达方式。

<div class="sumbox">
<div class="sumbox-title">用「まくる」表示频繁的动作</div>

- 将第一个动词活用为 <i>词根</i> ，再加上「まくっている」<br>
<span class="jp-trans">例</span><br>
<span class="jp-trans"><span class="jp-word" title="やる - 做">や<em>る</em></span> → <span class="jp-word" title="やる - 做">や<em>り</em></span> → <span class="jp-word" title="やる - 做">やり</span> <em>まくっている</em></span>

### 示例

1. <span class="jp-word" title="ゲーム - 游戏">ゲーム</span> に <span class="jp-word" title="はまる - 陷入">はまっちゃって</span> 、 <span class="jp-word" title="さいきん - 迟ly">最近</span> <span class="jp-word" title="パソコン - 电脑">パソコン</span> を <em><span class="jp-word" title="つかう - 使用">使い</span>まくっている</em> よ。<br>

<span class="jp-trans">我在美国的时候总是喝可乐。</span>
