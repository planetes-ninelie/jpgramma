如果你想知道日语里面如何做比较，现在告诉你。我们将要学习如何使用「<span class="jp-word" title="ほう - 方向，立场; かた - 人；方法">方</span>」和「より」来比较两个东西。我们也会学到「<span class="jp-word" title="ほう - 方向，立场; かた - 人；方法">方</span>」和「よる」的其他用法。

## 用「<span class="jp-word" title="ほう - 方向，立场; かた - 人；方法">方</span>」来做比较 {#part2}

### 词汇学习

1. 方 【1) ほう; 2) かた】 - 1) 方向；立场 2) 人；方法
2. ご飯 【ご・はん】 - 米；饭
3. おいしい （い形容词） - 可口
4. 鈴木 【すず・き】 - 铃木（名）
5. 若い 【わか・い】 （い形容词） - 年轻
6. 学生 【がく・せい】 - 学生
7. いい （い形容词） - 好的
8. 赤ちゃん 【あか・ちゃん】 - 婴儿
9. 静か 【しず・か】 （な形容词）- 安静
10. 好き 【す・き】 （な形容词） - 惹人喜欢的
11. ゆっくり - 慢慢的
12. 食べる 【た・べる】 （る动词） - 吃
13. 健康 【けん・こう】 - 健康
14. こちら - 这边
15. 行く 【い・く】 （う动词） - 走
16. 早い 【はや・い】 （い形容词） - 快；早
17. 怖い 【こわ・い】 （い形容词） - 恐怖的
18. 映画 【えい・が】 - 电影
19. 観る 【み・る】 （る动词） - 观看
20. そんな - 那种
21. 飲む 【の・む】 （う动词） - 喝

名词「<span class="jp-word" title="ほう - 方向，立场; かた - 人；方法">方</span>」表示方向的时候读作「ほう」，但作为「<span class="jp-word" title="ひと - 人">人</span>」的礼貌版本时它读作「かた」。

当我们用「<span class="jp-word" title="ほう - 方向，立场; かた - 人；方法">方</span>」来表示方向时，我们可以用它来做比较，比如说这样好过那样，或者不如。语法上来说，它跟其他普通名词用法一样。

### 示例

利用助词「の」与其他名词合用。

1. <span class="jp-word" title="ごはん - 米饭">ご飯</span> <em>の<span class="jp-word" title="ほう - 方向，立场">方</span></em> が <span class="jp-word" title="おいしい - 可口">おいしい</span><br>
<span class="jp-trans">米饭更好吃（字面：米饭那一方更好吃）</span>
2. <span class="jp-word" title="すずき - 铃木">鈴木</span> <span class="jp-word" title="さん - 桑">さん</span> <em>の<span class="jp-word" title="ほう - 方向，立场">方</span></em> が <span class="jp-word" title="わかい - 年轻">若い</span> 。<br>
<span class="jp-trans">铃木桑更年轻（字面：铃木桑那一方年轻）</span>

语法上它无异于一般名词。

1. <em><span class="jp-word" title="がくせい - 学生">学生</span>じゃない</em> <span class="jp-word" title="ほう - 方向，立场">方</span> が <span class="jp-word" title="いい - 好的">いい</span> よ。<br>
<span class="jp-trans">不是学生更好（字面：不是学生那一方是好的）</span>
2. <span class="jp-word" title="あかちゃん - 婴儿">赤ちゃん</span> は、 <em><span class="jp-word" title="しずか- 安静">静か</span>な</em> <span class="jp-word" title="ほう - 方向，立场">方</span> が <span class="jp-word" title="すき - 令人想要的">好き</span> 。<br>
<span class="jp-trans">更喜欢安静的宝宝（字面：关于宝宝，安静的那一方令人喜欢）</span>

对于非否定式动词，你也可以用过去形态加上更多肯定的口吻，特别是在你提建议的时候。

1. <span class="jp-word" title="ゆっくり - 慢慢的">ゆっくり</span> <em><span class="jp-word" title="たべる - 吃">食べた</span></em> <span class="jp-word" title="ほう - 方向，立场">方</span> が <span class="jp-word" title="けんこう - 健康">健康</span> に <span class="jp-word" title="いい - 好的">いい</span> よ。<br>
<span class="jp-trans">吃慢点对你健康更好。</span>
2. <span class="jp-word" title="こちら - 这边">こちら</span> から <em><span class="jp-word" title="いく - 走">行った</span></em> <span class="jp-word" title="ほう - 方向，立场">方</span> が <span class="jp-word" title="はやい - 快；早">早かった</span> 。<br>
<span class="jp-trans">从这边走更快。</span>

但对否定动词<u>不能</u>这么用。

- <span class="jp-word" title="こわい - 恐怖的">怖い</span> <span class="jp-word" title="えいが - 电影">映画</span> は <em><span class="jp-word" title="みる - 观看">観ない</span></em> <span class="jp-word" title="ほう - 方向，立场">方</span> が <span class="jp-word" title="いい - 好的">いい</span> よ。<br>
<span class="jp-trans">最好别看恐怖片。</span>

只有当比较的对象发生在过去形，否定式动词才能用过去形态。

- <span class="jp-word" title="そんな - 像那样">そんな</span> に <em><span class="jp-word" title="のむ - 喝">飲まなかった</span></em> <span class="jp-word" title="ほう - 方向，立场">方</span> が <em><span class="jp-word" title="いい - 好的">よかった</span></em> 。<br>
<span class="jp-trans">最好别喝（那次）那么多。</span>

## 用「より」来做比较 {#part3}

### 词汇学习

1. 方 【1) ほう; 2) かた】 - 1) 方向；立场 2) 人；方法
2. 花 【はな】 － 花
3. 団子 【だん・ご】 － 团子（饺子）
4. ご飯 【ご・はん】 - 米；饭
5. パン - 面包
6. おいしい （い形容词） - 可口
7. 若い 【わか・い】 （い形容词） - 年轻
8. 鈴木 【すず・き】 - 铃木（名）
9. 毎日 【まい・にち】 - 每天
10. 仕事 【し・ごと】 - 工作
11. 嫌 【いや】 （な形容词） 厌恶；不爽
12. ある （う动词） - 存在（无生命）
13. まし - （比）好〔强〕些；（与其）不如（宁可）；胜过，胜于
14. ゆっくり - 慢慢的
15. 食べる 【た・べる】 （る动词） - 吃
16. 早い 【はや・い】 （い形容词） - 快；早
17. いい （い形容词） - 好的

你可以把「より」想象为「方」的对立，它的意思是「与其…不如」或「与…相反」，使用时直接加在任意单词后面。它通常与「<span class="jp-word" title="ほう - 方向，立场">方</span>」一起使用来表达类似「相比那样，这样更好」的意思。

### 示例

1. <span class="jp-word" title="はな － 花">花</span> より <span class="jp-word" title="だんご - 团子">団子</span> 。<br>
<span class="jp-trans">与其要花，不如要团子（日本著名谚语）</span>
2. <span class="jp-word" title="ごはん - 米饭">ご飯</span> の <span class="jp-word" title="ほう - 方向，立场">方</span> が、 <em><span class="jp-word" title="パン - 面包">パン</span>より</em> <span class="jp-word" title="おいしい - 可口">おいしい</span> 。<br>
<span class="jp-trans">米饭比面包好吃（字面：米饭这一方好吃，相较于面包）</span>
3. <em><span class="jp-word" title="キム - Kim">キム</span><span class="jp-word" title="さん - 桑">さん</span>より</em> <span class="jp-word" title="すずき - 铃木">鈴木</span> <span class="jp-word" title="さん - 桑">さん</span> の <span class="jp-word" title="ほう - 方向，立场">方</span> が <span class="jp-word" title="わかい - 年轻">若い</span> 。<br>
<span class="jp-trans">铃木桑比 Kim 桑年轻（字面：铃木这一方是年轻的，相较于 Kim 桑）</span>

如果你对第一句谚语感兴趣的话，团子是一种通常在节日出售的甜糯米团，这句话的意思是人们喜欢在赏花的时候吃这种点心，这里指的是「<span class="jp-word" title="はなみ - 赏樱">花見</span>」，即樱花盛开的时候去赏樱（并被挤扁）。更深一层的意思，如同其他谚语一样，就要看你如何运用了。

当然，没人规定「より」一定要跟「<span class="jp-word" title="ほう - 方向，立场; かた - 人；方法">方</span>」一起用，从上下文你也可以猜出比较的东西来。

<span class="jp-word" title="すずき - 铃木">鈴木</span>：<span class="jp-word" title="まいにち - 每天">毎日</span><span class="jp-word" title="しごと - 工作">仕事</span>に<span class="jp-word" title="いく - 走">行く</span>のが<span class="jp-word" title="いや - 不爽的，不愉快">嫌</span>だ。<br>
<span class="jp-trans">铃木：我不喜欢每天上班。</span>

スミス：<span class="jp-word" title="しごと - 工作">仕事</span>が<span class="jp-word" title="ある - 存在（无生命）">ない</span><em>より</em><span class="jp-word" title="まし - （比）好〔强〕些，（与其…）不如（宁可）…；胜过，胜于">まし</span>だよ。<br>
<span class="jp-trans">Smith：这总好过没有工作。</span>

被「より」粘黏的单词不需要任何时态。注意在下面的例句里「より」前面的「<span class="jp-word" title="たべる - 吃">食べる</span>」是现在时，虽然「<span class="jp-word" title="ほう - 方向，立场; かた - 人；方法">方</span>」前面的「<span class="jp-word" title="たべる - 吃">食べる</span>」已经是过去形。

- <span class="jp-word" title="ゆっくり - 慢慢的">ゆっくり</span> <em><span class="jp-word" title="たべる - 吃">食べた</span></em> <span class="jp-word" title="ほう - 方向，立场">方</span> が <span class="jp-word" title="はやい - 快；早">早く</span> <em><span class="jp-word" title="たべる - 吃">食べる</span></em> より <span class="jp-word" title="いい - 好的">いい</span> 。<br>
<span class="jp-trans">吃慢点比吃快点好。</span>

### 用「より」表示最高级 {#yori}

### 词汇学习

1. 誰 【だれ】 - 谁
2. 何【なに】 - 什么
3. どこ - 哪里
4. 商品 【しょう・ひん】 - 商品
5. 品質 【ひん・しつ】 - 品质
6. 大切 【たい・せつ】 （な形容词） - 重要
7. する （例外） - 做
8. この - 这个（これの的缩写）
9. 仕事 【し・ごと】 - 工作
10. 早い 【はや・い】 （い形容词） - 快；早
11. 出来る 【で・き・る】 （る动词） - 做好；做出；出现

你也可以将「より」与疑问词例如「<span class="jp-word" title="だれ - 谁">誰</span>」、「<span class="jp-word" title="なに - 什么">何</span>」或「<span class="jp-word" title="どこ - 哪里">どこ</span>」合用来表示与其他人或其他东西相比最高的意思。这种情况下虽然不是必须，但一般会加上助词「も」。

### 示例

1. <span class="jp-word" title="しょうひん - 商品">商品</span> の <span class="jp-word" title="ひんしつ - 品质">品質</span> を <span class="jp-word" title="なに - 什么">何</span> より <span class="jp-word" title="たいせつ - 重要">大切</span> に <span class="jp-word" title="する - 做">しています</span> 。<br>
<span class="jp-trans">商品质量是最重要的。</span>
2. <span class="jp-word" title="この - 这">この</span> <span class="jp-word" title="しごと - 工作">仕事</span> は <span class="jp-word" title="だれ - 谁">誰</span> よりも <span class="jp-word" title="はやい - 快；早">早く</span> <span class="jp-word" title="できる -  做好；做出；出现">できます</span> 。<br>
<span class="jp-trans">这项工作能比其他人做的都快。</span>

## 用「<span class="jp-word" title="ほう - 方向，立场; かた - 人；方法">方</span>」表示方法 {#part4}

### 词汇学习

1. 方 【1) ほう; 2) かた】 - 1) 方向；立场 2) 人；方法
2. 行く 【い・く】 （う动词） - 走
3. 食べる 【た・べる】 （る动词） - 吃
4. 新宿 【しん・じゅく】 - 新宿
5. 分かる 【わ・かる】 （う动词） - 明白
6. そう - 那样
7. 言う 【い・う】 （う动词）- 说
8. 体 【からだ】 - 身体
9. いい （い形容词） - 好的
10. 漢字 【かん・じ】 - 汉字
11. 書く 【か・く】 （う动词） - 写
12. 教える 【おし・える】 （る动词） - 教；通知
13. くれる （る动词） - 给
14. パソコン - 电脑
15. 使う 【つか・う】 （う动词） - 使用
16. 皆 【みんな】 - 所有人
17. 知る 【し・る】 （う动词） - 知道

你也可以把「<span class="jp-word" title="ほう - 方向，立场; かた - 人；方法">方</span>」加到动词词根后面来表示做这个动作的方法。在这种用法里，「<span class="jp-word" title="ほう - 方向，立场; かた - 人；方法">方</span>」读作「かた」，新的单词是个名词。例如「<span class="jp-word" title="いく - 走">行き</span><span class="jp-word" title="かた - 人；方法">方</span>」（いきかた）的意思是「走法」，「<span class="jp-word" title="たべる - 吃">食べ</span><span class="jp-word" title="かた - 人；方法">方</span>」（たべかた）的意思是「吃法」。在你想问别人怎么做一件事的时候也许需要用到这种语法。

### 示例

1. <span class="jp-word" title="しんじゅく - 新宿">新宿</span> の <em><span class="jp-word" title="いく - 走">行き</span><span class="jp-word" title="かた - 人；方法">方</span></em> は <span class="jp-word" title="わかる - 知道；明白">分かります</span> か。<br>
<span class="jp-trans">你知道去新宿的走法吗？</span>
2. <span class="jp-word" title="そういう - 那种">そういう</span> <em><span class="jp-word" title="たべる - 吃">食べ</span><span class="jp-word" title="かた - 人；方法">方</span></em> は <span class="jp-word" title="からだ - 身体">体</span> に <span class="jp-word" title="いい - 好的">よくない</span> よ。<br>
<span class="jp-trans">那种吃法对你身体不好。</span>
3. <span class="jp-word" title="かんじ - 汉字">漢字</span> の <em><span class="jp-word" title="かく - 写">書き</span><span class="jp-word" title="かた - 人；方法">方</span></em> を <span class="jp-word" title="おしえる - 教">教えて</span> <span class="jp-word" title="くれる - 给">くれます</span> か？<br>
<span class="jp-trans">能教我汉字的写法吗？</span>
4. <span class="jp-word" title="パソコン - 个人电脑">パソコン</span> の <em><span class="jp-word" title="つかう - 使用">使い</span><span class="jp-word" title="かた - 人；方法">方</span></em> は、 <span class="jp-word" title="みんな - 所有人">みんな</span> <span class="jp-word" title="しる - 知道">知っている</span> でしょう。<br>
<span class="jp-trans">也许每个人都知道电脑的用法。</span>

当动词作此种变化后，结果是一个名词性质的状语从句，这样有时候就需要更改助词了。例如 「<span class="jp-word" title="いく - 走">行く</span>」通常跟着目标助词（「に」或「へ」），但「<span class="jp-word" title="いく - 走">行き</span><span class="jp-word" title="かた - 人；方法">方</span>」成了名词，于是例一就用了「<span class="jp-word" title="しんじゅく - 新宿">新宿</span><em>の</em><span class="jp-word" title="いく - 走">行き</span><span class="jp-word" title="かた - 人；方法">方</span>」，而非我们熟悉的「<span class="jp-word" title="しんじゅく - 新宿">新宿</span><em>に</em><span class="jp-word" title="いく - 走">行く</span>」。

## 用「によって」来表示前提 {#part5}

### 词汇学习

1. 人 【ひと】 - 人
2. 話 【はなし】 - 故事
3. 違う 【ちが・う】 （う动词） - 不同
4. 季節 【き・せつ】 - 季节
5. 果物 【くだ・もの】 - 水果
6. おいしい （い形容词） - 可口
7. なる （う动词） - 成为
8. まずい （い形容词） - 不愉快
9. 和子 【かず・こ】 - 和子（姓）
10. 今日 【きょう】 - 今天
11. 飲む 【の・む】 （う动词） - 喝
12. 行く 【い・く】 （う动词） - 走
13. 大樹 【だい・き】 - 大树（姓）
14. それ - 那
15. 裕子 【ゆう・こ】 - 裕子（姓）

如果你想说「取决于 [X]」，日语里面很简单，把「によって」加在 [X] 后面即可。

### 示例

1. <em><span class="jp-word" title="ひと - 人">人</span>によって</em> <span class="jp-word" title="はなし - 故事">話</span> が <span class="jp-word" title="ちがう - 区别于">違う</span> 。<br>
<span class="jp-trans">这个故事因人而异。</span>
2. <em><span class="jp-word" title="きせつ - 季节">季節</span>によって</em> <span class="jp-word" title="くだもの - 水果">果物</span> は <span class="jp-word" title="おいしい - 可口">おいしく</span> <span class="jp-word" title="なる - 成为">なったり</span> 、 <span class="jp-word" title="まずい - 不愉快">まずく</span> <span class="jp-word" title="なる - 成为">なったり</span> <span class="jp-word" title="する - 做">する</span> 。<br>
<span class="jp-trans">水果好吃难吃取决于季节。</span>

这其实是「よる」的て形，如下面的简单对话所示。

<span class="jp-word" title="かずこ - 和子">和子</span>：<span class="jp-word" title="きょう - 今天">今日</span>は<span class="jp-word" title="のむ - 喝">飲み</span>に<span class="jp-word" title="いく - 走">行こう</span>か？<br>
<span class="jp-trans">和子：今天去喝酒吗？</span>

<span class="jp-word" title="だいき - 大树">大樹</span>：<span class="jp-word" title="それ - 那">それ</span>は、<span class="jp-word" title="ゆうこ - 裕子">裕子</span>に<em>よる</em>ね。<br>
<span class="jp-trans">大树：这要看裕子了。</span>

## 用「によると」表示消息来源 {#part6}

### 词汇学习

1. 天気 【てん・き】 - 天气
2. 予報 【よ・ほう】 - 预报
3. 今日 【きょう】 - 今天
4. 雨 【あめ】 - 雨
5. 友達 【とも・だち】- 朋友
6. 話 【はなし】 - 故事
7. 朋子 【とも・こ】 - 智子（姓）
8. やっと - 终于
9. ボーイフレンド - 男朋友
10. 見つける 【み・つける】 （る动词） - 找

「よる」的另外一个用法是跟目标以及条件语「と」合用来表示消息来源。翻译成英语就是 "according to [X]" ，而「によると」则加在 [X] 后面。

### 示例

1. <em><span class="jp-word" title="てんき - 天气">天気</span><span class="jp-word" title="よほう - 预报">予報</span>によると</em> 、 <span class="jp-word" title="きょう - 今天">今日</span> は <span class="jp-word" title="あめ - 雨">雨</span> だそうだ。<br>
<span class="jp-trans">据天气预报说，今天会下雨。</span>
2. <span class="jp-word" title="ともだち- 朋友">友達</span> の <em><span class="jp-word" title="はなし - 故事">話</span>によると</em> 、 <span class="jp-word" title="ともこ - 智子">朋子</span> は <span class="jp-word" title="やっと - 终于">やっと</span> <span class="jp-word" title="ボーイフレンド - 男朋友">ボーイフレンド</span> を <span class="jp-word" title="みつける - 找">見つけた</span> らしい。<br>
<span class="jp-trans">据一个朋友说，智子好像终于找到男朋友了。</span>
