学会表达动词的过去形和过去否定式之后，动词的基本属性就可以说学完了。我先提前警告各位，本章节里面将要学习的的活用规则是日语里面<b>最为复杂</b>的规则。一方面来说，搞定本节，其他的活用规则就不在话下了；另一方面来说，学明白之前，你可能要反复复习本节。想要熟悉这些复杂的活用规则就必须大量的练习。

## る动词的过去形 {#part2}

### 词汇学习

1. 出る 【で・る】 （る动词） - 出来
2. 捨てる 【す・てる】 （る动词） - 扔掉
3. ご飯 【ご・はん】 - 米饭
4. 食べる 【た・べる】 （る动词） - 吃
5. 映画 【えい・が】 - 电影
6. 全部 【ぜん・ぶ】 - 全部
7. 見る 【み・る】 （る动词） - 看

我们先从容易的る动词开始。把る动词从字典形变为过去形，只需要把「る」变成「た」。

<div class="sumbox">
<div class="sumbox-title">将る动词活用为过去形</div>

- 去掉「る」，再加上「た」<br>
<span class="jp-trans">示例</span>
  1. <span class="jp-word" title="でる - 出来">出<em>る</em></span> → <span class="jp-word" title="でる - 出来">出<em>た</em></span>
  2. <span class="jp-word" title="すてる - 扔掉">捨て<em>る</em></span> → <span class="jp-word" title="すてる - 扔掉">捨て<em>た</em></span>
</div>

### 示例

1. <span class="jp-word" title="ごはん - 饭">ご飯</span> は、 <em><span class="jp-word" title="たべる - 吃">食べた</span></em> 。<br>
<span class="jp-trans">对于饭来说，吃过了。</span>
2. <span class="jp-word" title="えいが - 电影">映画</span> は、 <span class="jp-word" title="ぜんぶ - 全部">全部</span> <em><span class="jp-word" title="みる - 看">見た</span></em> 。<br>
<span class="jp-trans">对于电影来说，全看过了。</span>

## う动词的过去形 {#part3}

### 词汇学习

1. 話す 【はな・す】 （う动词） - 说
2. 書く 【か・く】 （う动词） - 写
3. 泳ぐ 【およ・ぐ】 （う动词） - 游泳
4. 飲む 【の・む】 （う动词） - 喝
5. 遊ぶ 【あそ・ぶ】 （う动词） - 玩
6. 死ぬ 【し・ぬ】 （う动词） - 死
7. 切る 【き・る】 （う动词） - 切
8. 買う 【か・う】 （う动词） - 买
9. 持つ 【も・つ】 （う动词） - 抓住
10. する （例外） - 做
11. 来る 【く・る】 （例外） - 来
12. 行く 【い・く】 （う动词） - 走
13. 今日 【きょう】 - 今天
14. 走る 【はし・る】 （う动词） - 跑
15. 友達 【とも・だち】 - 朋友
16. 私 【わたし】 - 我；我自己
17. 勉強 【べん・きょう】 - 学习

### 示例

1. <span class="jp-word" title="きょう - 今天">今日</span> は、 <em><span class="jp-word" title="はしる - 跑">走った</span></em> 。<br>
<span class="jp-trans">至于今天，跑了。</span>
2. <span class="jp-word" title="ともだち - 朋友">友達</span> が <em><span class="jp-word" title="くる - 来">来た</span></em> 。<br>
<span class="jp-trans">来过的那个是朋友。</span>
3. <span class="jp-word" title="わたし - 我；我自己">私</span> も <em><span class="jp-word" title="あそぶ - 玩">遊んだ</span></em> 。<br>
<span class="jp-trans">我也玩过了。</span>
4. <span class="jp-word" title="べんきょう - 学习">勉強</span> は、 <em><span class="jp-word" title="する - 做">した</span></em> 。<br>
<span class="jp-trans">关于学习，已经做了。</span>

## 所有动词的过去否定式 {#part4}

### 词汇学习

1. 捨てる 【す・てる】 （る动词） - 扔掉
2. 行く 【い・く】 （う动词） - 去
3. 食べる 【たべ・る】 （る动词） - 吃
4. する （例外） - 做
5. お金 【お・かね】 - 钱
6. ある （う动词） - 存在（无生命）
7. 私 【わたし】 - 我；我自己
8. 買う 【か・う】 （う动词） - 买
9. 猫 【ねこ】 - 猫
10. いる （る动词） - 存在（有生命）

所有动词的过去否定式活用规则都是一样的。你也许已经注意到，基本上所有的否定式都是以「ない」结尾。动词的过去否定式活用规则跟其他那些以「ない」结尾的否定式规则一样，只要把动词否定式结尾「ない」里的「い」去掉，换成「かった」就行。

<div class="sumbox">
<div class="sumbox-title">将动词活用为过去否定式</div>

- 将动词活用为否定式，然后把结尾的「い」换成「かった」<br>
<span class="jp-trans">示例</span>
  1. <span class="jp-word" title="すてる - 扔掉">捨て<em>る</em></span> → <span class="jp-word" title="すてる - 扔掉">捨てな<em>い</em></span> → <span class="jp-word" title="すてる - 扔掉">捨てな<em>かった</em></span>
  2. <span class="jp-word" title="いく - 去">行<em>く</em></span> → <span class="jp-word" title="いく - 去">行かな<em>い</em></span> → <span class="jp-word" title="いく - 去">行かな<em>かった</em></span>
</div>

### 示例

1. アリスは <span class="jp-word" title="たべる - 吃">食べな<em>かった</em></span> 。<br>
<span class="jp-trans">对 Alice 来说，没有吃过。</span>
2. ジムが <span class="jp-word" title="する - 做">しな<em>かった</em></span> 。<br>
<span class="jp-trans">Jim 是没有做过的那个。</span>
3. ボブも <span class="jp-word" title="いく - 去">行かな<em>かった</em></span> 。<br>
<span class="jp-trans">Bob 也没有去过。</span>
4. <span class="jp-word" title="おかね - 钱">お金</span> が <em><span class="jp-word" title="ある - 存在（无生命）">なかった</span></em> 。<br>
<span class="jp-trans">没有过钱。（字面：对于钱来说，不曾存在过。）</span>
5. <span class="jp-word" title="わたし - 我；我自己">私</span> は <em><span class="jp-word" title="かう - 买">買わなかった</span></em> 。<br>
<span class="jp-trans">对于我来说，没有去买。</span>
6. <span class="jp-word" title="ねこ - 猫">猫</span> は <em><span class="jp-word" title="いる- 存在（有生命）">いなかった</span></em> 。<br>
<span class="jp-trans">没有过猫。（对于猫来说，不曾存在过。）</span>
