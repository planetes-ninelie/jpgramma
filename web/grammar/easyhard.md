### 词汇学习

1. 食べる 【た・べる】 （る动词） - 吃
2. しゃべる （う动词） - 说话
3. この - 这个（これの的缩写）
4. 字 【じ】 - 字
5. 読む 【よ・む】 （う动词） - 读
6. カクテル - 鸡尾酒
7. ビール - 啤酒
8. 飲む 【の・む】 （う动词） - 喝
9. 部屋 【へ・や】 - 房间
10. 暗い 【くら・い】 （い形容词） - 暗
11. 見る 【み・る】 （る动词） - 看
12. 難しい 【むずか・しい】 （い形容词） - 困难
13. 易しい 【やさ・しい】 （い形容词） - 简单
14. 簡単 【かん・たん】 （な形容词） - 简单
15. 容易 【よう・い】 （な形容词） - 简单
16. その - 那个 （それの的缩写）
17. 肉 【にく】 - 肉

这一节比较短也比较容易，介绍的是如何将动词改写为形容它容易做还是难做的形容词。基本上，规则是改写成词根，然后对于容易的加上「やすい」，困难的加上「にくい」。新单词是一个普通的い形容词。容易吧哈？

<div class="sumbox">
<div class="sumbox-title">用「～やすい、～にくい」来形容动作的难易</div>

- 要描述一个动作容易做，将其改为词根并加上「やすい」。要描述动作难做，则在词根后加上「にくい」。<br>
<span class="jp-trans">例</span>
  1. <span class="jp-word" title="たべる - 吃">食べ<em>る</em></span> → <span class="jp-word" title="たべる - 吃">食べ</span> <em>やすい</em>
  2. <span class="jp-word" title="しゃべる - 说话">しゃべ<em>る</em></span> → <span class="jp-word" title="しゃべる - 说话">しゃべ<em>り</em></span> → <span class="jp-word" title="しゃべる - 说话">しゃべり</span> <em>にくい</em>

### 示例

1. <span class="jp-word" title="この - 这">この</span> <span class="jp-word" title="じ - 字">字</span> は <em><span class="jp-word" title="よむ - 读">読み</span>にくい</em><br>
<span class="jp-trans">这个字迹难读懂。</span>
2. <span class="jp-word" title="カクテル - 鸡尾酒">カクテル</span> は <span class="jp-word" title="ビール - 啤酒">ビール</span> より <em><span class="jp-word" title="のむ - 喝">飲み</span>やすい</em> 。<br>
<span class="jp-trans">鸡尾酒比啤酒容易下口。</span>
3. <span class="jp-word" title="へや - 房间">部屋</span> が <span class="jp-word" title="くらい - 暗">暗かった</span> ので、 <em><span class="jp-word" title="みる - 看">見</span>にくかった</em> 。<br>
<span class="jp-trans">因为房间里暗，所以难看清。</span>

顺便说一句：要小心「<span class="jp-word" title="みる - 看">見</span>にくい」，因为「<span class="jp-word" title="みにくい - ugly">醜い</span>」是一个罕用的形容词，意思是「丑陋」。我在想「看不清楚」和「丑陋」这两个单词发音相同是否仅仅是巧合呢？

当然，你也可以用已经学过的其他的语法结构来表达相同的意思，只需要选用合适的形容词例如「<span class="jp-word" title="むずかしい - 困难">難しい</span>」、「<span class="jp-word" title="やさしい - 简单">易しい</span>」、 「<span class="jp-word" title="かんたん - 简单">簡単</span>」、「<span class="jp-word" title="ようい - 简单">容易</span>」等等。下面两个句子的意思是相同的。

1. <span class="jp-word" title="その - 那个">その</span> <span class="jp-word" title="にく - 肉">肉</span> は <em><span class="jp-word" title="たべる - 吃">食べ</span>にくい</em> 。<br>
<span class="jp-trans">那肉难吃。</span>
2. <span class="jp-word" title="その - 那个">その</span> <span class="jp-word" title="にく - 肉">肉</span> を <em><span class="jp-word" title="たべる - 吃">食べる</span>のは<span class="jp-word" title="むずかしい - 困难">難しい</span></em> 。<br>
<span class="jp-trans">吃那块肉这件事是困难的。</span>

## 「～にくい」的变形：「～がたい」和「～づらい」 {#part2}

### 词汇学习

1. 彼 【かれ】 - 他；男朋友
2. 忘れる 【わす・れる】 （る动词） - 忘记
3. 思い出 【おも・い・で】 - 回忆
4. 大切 【たい・せつ】 （な形容词） - 重要
5. する （例外） - 做
6. とても - 非常
7. 信じる 【しん・じる】 （る动词） - 相信
8. 話 【はなし】 - 故事
9. 本当 【ほん・とう】 - 真的
10. 起こる 【おこ・る】 （う动词） - 发生
11. 辛い【1) から・い; 2) つら・い】 （い形容词） - 1、辣；2、艰辛
12. 日本語 【に・ほん・ご】 - 日语
13. 読む 【よ・む】 （う动词） - 读
14. 待ち合わせ 【ま・ち・あわ・せ】 - 会议安排
15. 分かる 【わ・かる】 （う动词） - 明白
16. 場所 【ば・しょ】 - 场所

「にくい」的汉字写法其实来自「<span class="jp-word" title="にくい／かたい - 困难">難い</span>」，它也可以被读作「かたい」。所以，你也可以把「～がたい」作为动词的后缀来表达跟「にくい」一样的意思。「にくい」更常用于口语，而「がたい」则更适合用在书面。「にくい」通常用来与物理动作合用，而「がたい」则用于那些不怎么需要移动的动作。不过应该没有什么死规矩说哪个单词用哪个更合适，所以如果你觉得没把握，最好 google 一把看哪个出现的多。另外为了防止别人读错，这个后缀应该用平假名书写。

### 示例

1. <span class="jp-word" title="かれ - 他；男朋友">彼</span> との <em><span class="jp-word" title="わすれる - 忘记">忘れ</span>がたい</em> <span class="jp-word" title="おもいで - 回忆">思い出</span> を <span class="jp-word" title="たいせつ - 重要">大切</span> に <span class="jp-word" title="する - 做">している</span> 。<br>
<span class="jp-trans">我把和他的难忘的记忆看得很重要。</span>
2. <span class="jp-word" title="とても - 非常">とても</span> <em><span class="jp-word" title="しんじる - 相信">信じ</span>がたい</em> <span class="jp-word" title="はなし - 故事">話</span> だが、 <span class="jp-word" title="ほんとう - 真的">本当</span> に <span class="jp-word" title="おこる - to occur">起こった</span> らしい。<br>
<span class="jp-trans">这个故事很难让人信服，不过看样子（听闻）它是真的发生过。</span>

另一个更为随意的变种是「づらい」，它源自「<span class="jp-word" title="つらい - 痛苦的">辛い</span>」（つらい）。不要把它跟「<span class="jp-word" title="からい - 辣">辛い</span>」的另一个意思「辣」（からい）混淆了！

### 示例

1. <span class="jp-word" title="にほんご - 日语">日本語</span> は <em><span class="jp-word" title="よむ - 读">読み</span>づらい</em> な。<br>
<span class="jp-trans">天，日语真难读。</span>
2. <span class="jp-word" title="まちあわせ - 会议安排">待ち合わせ</span> は、 <em><span class="jp-word" title="わかる - 明白">分かり</span>づらい</em> <span class="jp-word" title="ばしょ - 场所">場所</span> に <span class="jp-word" title="する - 做">しない</span> でね。<br>
<span class="jp-trans">请不要挑一个不好明白的地点来安排会议。</span>
