这一节要介绍各种表达不同<i>数量</i>的说法。比如你想说「我<u>只</u>吃了一个」、「剩下的<u>都</u>在这里了」、「这里<u>只有</u>老人」，或者「我吃<u>太多</u>了」，这些表示的都是很多或者很少的意思。与英语使用单词不同，日语里这些基本上都是用助词来表示的。

## 用「だけ」表示全部都在这里的意思 {#part2}

### 词汇学习

1. りんご - 苹果
2. これ - 这
3. それ - 那
4. 食べる 【たべ・る】 （る动词） - 吃
5. この - 这个（これの的缩写）
6. 歌 【うた】 - 歌
7. 歌う 【うた・う】 （う动词） - 唱
8. その - 那个 （それの的缩写）
9. 人 【ひと】 - 人
10. 好き 【す・き】 （な形容词） - 惹人喜欢的
11. 販売機 【はん・ばい・き】 - 售货机
12. 五百円玉 【ご・ひゃく・えん・だま】 - 500 日元硬币
13. 小林 【こ・ばやし】 - 小林（姓）
14. 返事 【へん・じ】 - 答复
15. 来る 【く・る】 （例外）- 来
16. 準備 【じゅん・び】 - 准备
17. 終わる 【お・わる】 （う动词） - 结束
18. ここ - 这里
19. 名前 【な・まえ】 - 名字
20. 書く 【か・く】 （う动词） - 写
21. いい （い形容词） - 好的

助词「だけ」用来表示都在这里了的意思。与其他介绍过的助词一样，直接将它黏着在需要修饰的单词后面就可以。

### 示例

1. <span class="jp-word" title="りんご - 苹果">りんご</span> <em>だけ</em> 。<br>
<span class="jp-trans">只有苹果了（没别的东西）。</span>
2. <span class="jp-word" title="これ - 这">これ</span> と <span class="jp-word" title="それ - 那">それ</span> <em>だけ</em> 。<br>
<span class="jp-trans">只有那个和这个了（没别的东西）。</span>

如果还有用到其他的几个格助词，那么必须跟在「だけ」后面。事实上，对于多个助词的排列顺序，通常是最具体的先来，最笼统的放最后。

1. <span class="jp-word" title="それ - 那">それ</span> <em>だけは</em> 、 <span class="jp-word" title="たべる - 吃">食べないで</span> <span class="jp-word" title="ください - 请给、请做">ください</span> 。<br>
<span class="jp-trans">别吃那个就行了（吃其他的没问题）。</span>
2. <span class="jp-word" title="この - 这">この</span> <span class="jp-word" title="うた - 歌">歌</span> <em>だけを</em> <span class="jp-word" title="うたう - 唱">歌わなかった</span> 。<br>
<span class="jp-trans">唯独没唱这首歌。</span>
3. <span class="jp-word" title="その - 那个">その</span> <span class="jp-word" title="ひと - 人">人</span> <em>だけが</em> <span class="jp-word" title="すき - 讨人喜欢的，令人想要的">好きだったんだ</span> 。<br>
<span class="jp-trans">那个人是我唯一喜欢的。</span>

双重助词也是一样，「だけ」必须在前面。

- <span class="jp-word" title="この - 这">この</span> <span class="jp-word" title="はんばいき - 售货机">販売機</span> だけ <em>では</em> 、 <span class="jp-word" title="ごひゃくえんだま - 500 日元硬币">五百円玉</span> が <span class="jp-word" title="つかう - 使用">使えない</span> 。<br>
<span class="jp-trans">只有这台售货机不能用 500 日元硬币。</span>

对于一些副助词例如「から」或「まで」来说，很难讲哪个该放前面。没把握的话，试试 Google 看哪种组合用的人多。然后你就会发现「からだけ」的结果数基本上是「だけから」的两倍了，90,000 vs. 50,000。

- <span class="jp-word" title="こばやし - Kobayashi">小林</span> さん <em>からだけは</em> 、 <span class="jp-word" title="へんじ - 答复">返事</span> が <span class="jp-word" title="くる- 来">来なかった</span> 。<br>
<span class="jp-trans">没有收到的回复 <em>只有来自</em> 小林桑的。</span>

与其他一些助词不同，「だけ」可以直接黏着在动词后面。

1. <span class="jp-word" title="じゅんび - 准备">準備</span> が <span class="jp-word" title="おわる - 结束">終わった</span> から、 <span class="jp-word" title="これから - 从此以后">これから</span> は <em><span class="jp-word" title="たべる - 吃">食べる</span></em> だけだ。<br>
<span class="jp-trans">既然准备好了，现在开始我们只管吃就是了。</span>
2. <span class="jp-word" title="ここ - 这里">ここ</span> に <span class="jp-word" title="なまえ - 名字">名前</span> を <em><span class="jp-word" title="かく - 写">書く</span></em> だけで <span class="jp-word" title="いい - 好的">いい</span> ですか？<br>
<span class="jp-trans">只在这里写（我的）名字是否就可以？</span>

## 用「のみ」作为「だけ」的正式版本 {#part3}

### 词汇学习

1. この - 这个（これの的缩写）
2. 乗車券 【じょう・しゃ・けん】 - 乘车券
3. 発売 【はつ・ばい】 - 销售
4. 当日 【とう・じつ】 - 那一天
5. 有効 【ゆう・こう】 - 有效
6. アンケート - 调查
7. 対象 【たい・しょう】 - 目标
8. 大学生 【だい・がく・せい】 - 大学生

有一个从语法到含义都跟助词「だけ」相同的助词是「のみ」，不过与「だけ」用在普通对话中不同，「のみ」通常只用在书面语。它通常用于在用户手册之类的地方做解释。这个语法其实应该归类到进阶部分，因为正式语言与我们到现在为止所学的从风格到音调都不一样。不过放在这里介绍是因为它跟「だけ」其实是一回事。Google 一下「のみ」你就会发现它都被用在什么地方。

1. <span class="jp-word" title="この - 这">この</span> <span class="jp-word" title="じょうしゃけん - 乘车券">乗車券</span> は <span class="jp-word" title="はつばい - 销售">発売</span> <span class="jp-word" title="とうじつ - 那一天">当日</span> <em>のみ</em> <span class="jp-word" title="ゆうこう - 有效">有効</span> です。<br>
<span class="jp-trans">这种车票 <em>只在</em> 购买当日有效。</span>
2. <span class="jp-word" title="アンケート - 调查">アンケート</span> <span class="jp-word" title="たいしょう - 目标">対象</span> は <span class="jp-word" title="だいがくせい - 大学生">大学生</span> <em>のみ</em> です。<br>
<span class="jp-trans">这个调查的目标 <em>仅限</em> 大学生。</span>

## 用「しか」表示没有别的东西了 {#part4}

### 词汇学习

1. これ - 这
2. ある （う动词） - 存在（无生命）
3. 見る 【み・る】 （る动词） - 看
4. 今日 【きょう】 - 今天
5. 忙しい 【いそが・しい】 （い形容词）- 忙
6. 朝ご飯 【あさ・ご・はん】 - 早饭
7. 食べる 【た・べる】 （る动词） - 吃
8. 全部 【ぜん・ぶ】 - 所有东西
9. 買う 【か・う】 （う动词）- 买
10. ううん - 不（口语）
11. 何【なに】 - 什么
12. もらう - 接受
13. 頑張る 【がん・ば・る】 （う动词） - 尽力
14. こう - 这样
15. なる （う动词） - 成为
16. 逃げる 【に・げる】 （る动词） - 逃走
17. もう - 已经
18. 腐る 【くさ・る】 （う动词） - 腐烂；溺爱
19. 捨てる 【す・てる】 （る动词） - 扔掉

这一小节的标题我特地设计了一下，想告诉大家「しか」必须用来表示<b>没有</b>其他任何东西。换句话说，句子的其他部分必须是否定式的。

- <span class="jp-word" title="これ - 这">これ</span> <em>しか</em> <span class="jp-word" title="ある - 存在（无生命）">ない</span> 。<br>
<span class="jp-trans">只有这个，没别的了。</span>

下面是错误的例句。

- <span class="jp-word" title="これ - 这">これ</span> <em>しか</em> <span class="jp-word" title="ある - 存在（无生命）">ある</span> 。<br>
<span class="jp-trans">（应该用「だけ」才对）</span>

可以看出，「しか」蕴含了否定的意思，而「だけ」则没有暗示什么。

1. <span class="jp-word" title="これ - 这">これ</span> <em>だけ</em> <span class="jp-word" title="みる - 看">見る</span> 。<br>
<span class="jp-trans">只看这个。</span>
2. <span class="jp-word" title="これ - 这">これ</span> <em>だけ</em> <span class="jp-word" title="みる - 看">見ない</span> 。<br>
<span class="jp-trans">只有这个没看。</span>
3. <span class="jp-word" title="これ - 这">これ</span> <em>しか</em> <span class="jp-word" title="みる - 看">見ない</span> 。<br>
<span class="jp-trans">除了这个没看别的。</span>

### 示例

- <span class="jp-word" title="きょう  - 今天">今日</span> は <span class="jp-word" title="いそがしい- 忙">忙しくて</span> 、 <span class="jp-word" title="あさごはん - 早饭">朝ご飯</span> <em>しか</em> <span class="jp-word" title="たべる - 吃">食べられなかった</span> 。<br>
<span class="jp-trans">今天很忙，除了早饭没法吃别的。</span>

注意与「だけ」不同，你必须把句子说完整。

- <span class="jp-word" title="ぜんぶ - 全部">全部</span> <span class="jp-word" title="かう- 买">買う</span> の？<br>
<span class="jp-trans">你打算全买了？</span>

1. <span class="jp-word" title="ううん - 不">ううん</span> 、 <span class="jp-word" title="これ - 这">これ</span> だけ。<br>
<span class="jp-trans">不，就这个。</span>
2. <span class="jp-word" title="ううん - 不">ううん</span> 、 <span class="jp-word" title="これ - 这">これ</span> しか <em><span class="jp-word" title="かう- 买">買わない</span></em><br>
<span class="jp-trans">不，除了这个不会买别的了。</span>
3. <span class="jp-word" title="ううん - 不">ううん</span> 、 <span class="jp-word" title="これ - 这">これ</span>しか 。<br>
<span class="jp-trans">（错误，必须显式的表达出否定的意思）</span>

虽然主要的助词都是放最后，但「しか」必须放在「から」和「まで」后面。Google 一下就会发现「からしか」的结果数对比「しかから」是 60,000 比 600。

- アリス <em>からしか</em> <span class="jp-word" title="なにも - 什么也没有">何も</span> <span class="jp-word" title="もらう - 接受">もらってない</span> 。<br>
<span class="jp-trans">除了 Alice 我没从别人那里收到任何东西。</span>

这个助词也可以跟动词合用

1. <span class="jp-word" title="これから - 从此以后">これから</span> <span class="jp-word" title="がんばる - 尽力">頑張る</span> <em>しか</em> <span class="jp-word" title="ある - 存在（无生命）">ない</span> ！<br>
<span class="jp-trans">除了尽力之外没别的办法了！</span>
2. <span class="jp-word" title="こう - 像这样">こう</span> <span class="jp-word" title="なる - 成为">なったら</span> 、 <em><span class="jp-word" title="にげる - 逃避">逃げる</span></em> しか <span class="jp-word" title="ある - 存在（无生命）">ない</span> 。<br>
<span class="jp-trans">一旦出现这种情况，除了逃跑没有别的选择了。</span>
3. <span class="jp-word" title="もう - 已经">もう</span> <span class="jp-word" title="くさる - to rot">腐って</span> いるから、 <em><span class="jp-word" title="すてる - 扔掉">捨てる</span></em> しか <span class="jp-word" title="ある - 存在（无生命）">ない</span> よ。<br>
<span class="jp-trans">它已经烂掉了，所以除了扔掉没别的办法了。</span>

### 「っきゃ」, 「しか」的另一种说法

### 词汇学习

1. これ - 这
2. 買う 【か・う】 （う动词）- 买
3. ある （う动词） - 存在（无生命）
4. こう - 这样
5. なる （う动词） - 成为
6. もう - 已经
7. やる （う动词） - 做

「っきゃ」是「しか」的另一个版本，意思基本一样，用法则完全一致。把「しか」直接换成「っきゃ」就好啦。这个版本比「しか」强调的意味更多一些，不过用的不多，所以不多做介绍，只是怕万一你碰到，所以这里带一下。

### 示例

1. <span class="jp-word" title="これ - 这">これ</span> は <span class="jp-word" title="かう- 买">買う</span> <em>っきゃ</em> <span class="jp-word" title="ある - 存在（无生命）">ない</span> ！<br>
<span class="jp-trans">除了这个没别的可买的了！</span>
2. <span class="jp-word" title="こう - 像这样">こう</span> <span class="jp-word" title="なる - 成为">なった</span> ら、 <span class="jp-word" title="もう - 已经">もう</span> <span class="jp-word" title="やる - 做">やる</span> <em>っきゃ</em> <span class="jp-word" title="ある - 存在（无生命）">ない</span> ！<br>
<span class="jp-trans">如果事情变成这样，那就只能硬着头皮做了！</span>

## 用「ばかり」表达跟「だけ」相反的意思 {#part5}

### 词汇学习

1. 何 【なに／なん】 - 什么
2. おばさん - 中年妇女
3. 嫌 【いや】 （な形容词） 厌恶；不爽
4. 崇 【たかし】 - 崇（姓）
5. ～君 【～くん】 - ~君
6. 漫画 【まん・が】 - 漫画
7. 読む 【よ・む】 （う动词） - 读
8. かっこ悪い 【かっこ・わる・い】 （い形容词） - 不吸引人；不酷
9. 彼 【かれ】 - 他；男朋友
10. 麻雀【マー・ジャン】 - 麻将
11. 直美 【なお・み】 - 直美（姓）
12. 遊ぶ 【あそ・ぶ】 （う动词） - 玩
13. 最近 【さい・きん】 - 最近
14. 仕事 【し・ごと】 - 工作

「ばかり」用来表示这样一种情况，即某种东西多到了容不下别的东西的地步。注意它跟「しか」有本质上的不同，后者表达的是除了修饰的对象之外其他东西都<b>没有</b>（译者注：fundamental 在哪里，小弟眼拙看不出来——11/21/2013）。在口语里，「ばかり」通常读作「ばっかり」或干脆「ばっか」。比如你去到一个聚会，却发现满屋子都是欧巴桑，这时候你也许就会说：

- <span class="jp-word" title="なに - 什么">何</span> だよ！ <span class="jp-word" title="おばさん - 中年妇女">おばさん</span> <em>ばっかり</em> じゃないか？<br>
<span class="jp-trans">擦？怎么净是欧巴桑？</span>

或者更娘一点：

- <span class="jp-word" title="いや - 不爽的">いや</span> だ。 <span class="jp-word" title="おばさん - 中年妇女">おばさん</span> <em>ばっかり</em> 。<br>
<span class="jp-trans">哎哟，就只有欧巴桑了呀。</span>

### 示例

- <span class="jp-word" title="たかし - 崇">崇</span> <span class="jp-word" title="くん - ~君">君</span> は <span class="jp-word" title="まんが - 漫画">漫画</span> <em>ばっかり</em> <span class="jp-word" title="よむ - 读">読んでて</span> さ。 <span class="jp-word" title="かっこわるい - 不吸引人；不酷">かっこ悪い</span> 。<br>
<span class="jp-trans">崇桑只顾着看漫画书了…没意思。</span>

口语中经常像上面这样话说一半就停了。注意「<span class="jp-word" title="よむ - 读">読んでて</span>」是「<span class="jp-word" title="よむ - 读">読んでいる</span>」（[进行时](/grammar/teform/#part2)）的て形，但去掉了「い」。这里我们就认为另一半话后面会说。

1. <span class="jp-word" title="かれ - 他；男朋友">彼</span> は <span class="jp-word" title="マージャン - 麻将">麻雀</span> <em>ばかり</em> です。<br>
<span class="jp-trans">他除了麻将什么也不是（他除了打麻将不干别的）。</span>
2. <span class="jp-word" title="なおみ - 直美">直美</span> <span class="jp-word" title="ちゃん - ~君">ちゃん</span> と <span class="jp-word" title="あそぶ - 玩">遊ぶ</span> <em>ばっかり</em> でしょう！<br>
<span class="jp-trans">你一门心思跟直美厮混在一起，是不是！</span>
3. <span class="jp-word" title="さいきん - 最近">最近</span> は <span class="jp-word" title="しごと - 工作">仕事</span> <em>ばっか</em> だよ。<br>
<span class="jp-trans">最近只顾工作了。</span>

## 用「<span class="jp-word" title="すぎる - 超过；经过">すぎる</span>」表达太多了的意思 {#part6}

### 词汇学习

1. 過ぎる 【す・ぎる】 （る动词） - 超过；经过
2. 食べる 【た・べる】 （る动词） - 吃
3. 飲む 【の・む】 （う动词） - 喝
4. 太る 【ふと・る】 （う动词） - 变胖
5. 静か 【しず・か】 （な形容词）- 安静
6. 大きい 【おお・きい】 （い形容词） - 大的
7. 面白い 【おも・しろ・い】 （い形容词） - 有趣
8. もったいない （い形容词） - 浪费的
9. 情けない 【なさ・けない】 （い形容词） - 可怜的
10. 危ない 【あぶ・ない】 （い形容词） - 危险的
11. 少ない 【すく・ない】 （い形容词） - few
12. 佐藤 【さ・とう】 - 佐藤 （名）
13. 料理 【りょう・り】 - 料理
14. 上手 【じょう・ず】 （な形容词） - 有技巧的
15. また - 再次
16. お酒 【お・さけ】 - 酒
17. 気 【き】 - 心情；意向
18. つける - 附加
19. 気をつける - 小心
20. トランク - 后备箱
21. 入る 【はい・る】 （う动词） - 进入
22. 罠 【わな】 - 陷阱
23. 時間 【じ・かん】 - 时间
24. 足りる 【た・りる】 （る动词） - 充足（动词）
25. 何【なに】 - 什么
26. 出来る 【で・き・る】 （る动词） - 做好；做出；出现
27. 彼 【かれ】 - 他；男朋友
28. 彼女 【かの・じょ】 - 她；女朋友
29. 昨晩 【さく・ばん】 - 昨晚
30. こと - 事情
31. 全然 【ぜん・ぜん】 - 全然 （用在否定句）

33. それ - 那

「<span class="jp-word" title="すぎる - 超过；经过">すぎる</span>」是一个普通的る动词，汉字写作「<span class="jp-word" title="すぎる - 超过；经过">過ぎる</span>」，意思是「超过」。当你把「<span class="jp-word" title="すぎる - 超过；经过">すぎる</span>」黏着到其他动词和形容词后面时，它的意思就是太多了，或者超过了正常水平。对于动词，你必须直接把「<span class="jp-word" title="すぎる - 超过；经过">すぎる</span>」加到词根后面。比如，「<span class="jp-word" title="たべる - 吃">食べ</span><span class="jp-word" title="すぎる - 超过；经过">すぎる</span>」的意思是「吃太多」，而「<span class="jp-word" title="のむ - 喝">飲み</span><span class="jp-word" title="すぎる - 超过；经过">すぎる</span>」就是「喝太多」。对于形容词，直接加到后面就行，不过い形容词要先去掉「い」。还有一个规则是对于否定式动词及形容词，必须先去掉「ない」的「い」，再换上「さ」，然后才能加「<span class="jp-word" title="すぎる - 超过；经过">すぎる</span>」。这种语法没有时态的区别（现在、过去），且因为「<span class="jp-word" title="すぎる - 超过；经过">すぎる</span>」是个普通的る动词，新的词还是个普通的る动词。

<span class="jp-trans">忙到那种没时间睡觉的程度。</span>

当和条件语合用时，你可以表达类似「你越怎么，就越……」的意思。这种语法必须按照下面的顺序组合：[ 动词的条件语 ] 紧跟 [ 同样的动词 + ほど]

- <span class="jp-word" title="かんこく - 韩国">韓国</span> <span class="jp-word" title="りょうり - cooking">料理</span> は <em><span class="jp-word" title="たべる - 吃">食べれば</span><span class="jp-word" title="たべる - 吃">食べる</span>ほど</em> 、 <span class="jp-word" title="おいしい - 可口">おいしくなる</span> 。<br>
<span class="jp-trans">关于韩料，你吃得越多就会越觉得好吃。</span>

直译过来就是「关于韩料，如果你吃，吃到那种程度，它就变得可口」，意思还是一样。这个例句用了条件词「ば」，不过条件语「たら」也可以这么用。既然这是一个一般性的陈述，上下文条件词「なら」就不能用了。决定性的条件词「と」也不大好用，因为即使到了那个程度，结果也是不一定的。

<span class="jp-trans">你学得越多就会越聪明。</span>

你也可以用条件语「ば」结合い形容词使用这种语法。

1. iPodは、 <span class="jp-word" title="ハードディスク - 硬盘">ハードディスク</span> の <span class="jp-word" title="ようりょう - 容量">容量</span> が <em><span class="jp-word" title="おおきい - 大的">大きければ</span><span class="jp-word" title="おおきい - 大的">大きい</span>ほど</em> <span class="jp-word" title="もっと - 多得多">もっと</span> <span class="jp-word" title="たくさん - 很多">たくさん</span> の <span class="jp-word" title="きょく - 曲子">曲</span> が <span class="jp-word" title="ほぞん - 保存">保存</span> <span class="jp-word" title="できる -  做好；做出；出现">できます</span> 。<br>
<span class="jp-trans">关于 iPod，存储越大，能保存的歌就越多。</span>
2. <span class="jp-word" title="こうくうけん - 机票">航空券</span> は <em><span class="jp-word" title="やすい - 便宜">安ければ</span><span class="jp-word" title="やすい - 便宜">安い</span>ほど</em> <span class="jp-word" title="いい - 好的">いい</span> とは <span class="jp-word" title="かぎる - to impose a limit">限らない</span> 。<br>
