这一节我们学习如何假设以及做出结论，要用到：「と<span class="jp-word" title="する - 做">する</span>」和「<span class="jp-word" title="わけ - 原因；情况">わけ</span>」（<span class="jp-word" title="わけ - 原因；情况">訳</span>）.

## 用「<span class="jp-word" title="わけ - 原因；情况">わけ</span>」得出结论 {#part2}

### 词汇学习

1. 訳 【わけ】 - 意思；原因；可以推导出
2. 直子 【なお・こ】 - 直子（姓）
3. いくら - 多少
4. 英語 【えい・ご】 - 英语
5. 勉強 【べん・きょう】 - 学习
6. する （例外） - 做
7. うまい （い形容词） - 有技巧的；好吃的
8. なる （う动词） - 成为
9. つまり - 简而言之
10. 語学 【ご・がく】 - 语言学习
11. 能力 【のう・りょく】 - 能力
12. ある （う动词） - 存在（无生命）
13. 言う 【い・う】 （う动词）- 说
14. 失礼 【しつ・れい】 - 失礼
15. 中国語 【ちゅう・ごく・ご】 - 中文
16. 読む 【よ・む】 （う动词） - 读
17. 広子 【ひろ・こ】 - 广子（姓）
18. 家 【1) うち; 2) いえ】 - 1）自己家；2）房子
19. 行く 【い・く】 （う动词） - 走
20. こと - 事情
21. 一郎 【いち・ろう】 - 一郎（姓）
22. 微積分 【び・せき・ぶん】 - 微积分
23. 分かる 【わ・かる】 （う动词） - 明白
24. ここ - 这里
25. 試験 【し・けん】 - 考试
26. 合格 【ごう・かく】 - 及格
27. 今度 【こん・ど】 - 这次；另一次
28. 負ける 【ま・ける】 （る动词） - 输掉
29. 来る 【く・る】 （例外）- 来
30. あきらめる （る动词） - 放弃

名词「<span class="jp-word" title="わけ - 原因；情况">わけ</span>」（<span class="jp-word" title="わけ - 原因；情况">訳</span>）不好解释，它的定义是「意思；原因；可以推导出」从下面的简单对话中你可以看到它的用法：

### 例一

<span class="jp-word" title="なおこ - 直子">直子</span>：<span class="jp-word" title="いくら - 多少">いくら</span><span class="jp-word" title="えいご - English">英語</span>を<span class="jp-word" title="べんきょう - 学习">勉強</span><span class="jp-word" title="する - 做">して</span>も、<span class="jp-word" title="うまい - 可口；有技巧的">うまく</span><span class="jp-word" title="なる - 成为">ならない</span>の。<br>
<span class="jp-trans">直子：不管我多努力学习，英语就是学不好。</span>

ジム：<span class="jp-word" title="つまり - 简而言之">つまり</span>、<span class="jp-word" title="ごがく - 语言学习">語学</span>には、<span class="jp-word" title="のうりょく - 能力">能力</span>が<span class="jp-word" title="ある - 存在（无生命）">ない</span>と<span class="jp-word" title="いう- 说">いう</span><em><span class="jp-word" title="わけ - 原因；情况">訳</span></em>か。<br>
<span class="jp-trans">Jim：简而言之，这<em>意味着</em>你的语言能力不行。</span>

<span class="jp-word" title="なおこ - 直子">直子</span>：<span class="jp-word" title="しつれい - 失礼">失礼</span>ね。<br>
<span class="jp-trans">直子：（你）好无礼。</span>

可以看到，Jim 从直子的话中得出了她语言能力不行的结论。这跟解释性的「の」完全不同，后者是用来解释明显或不明显的事情。「<span class="jp-word" title="わけ - 原因；情况">わけ</span>」则是用在任何人根据一定的信息都能推出的结论的地方。

这个语法一个有用之处就是跟「<span class="jp-word" title="ある - 存在（无生命）">ない</span>」合用，来表示无法得出的结论。这样我们就可以说类似「我特么怎么就该知道呢？」之类的话了。

- <span class="jp-word" title="ちゅうごくご - 中文">中国語</span> が <span class="jp-word" title="よむ - 读">読める</span> <em><span class="jp-word" title="わけ - 原因；情况">わけ</span>が<span class="jp-word" title="ある - 存在（无生命）">ない</span></em> 。<br>
<span class="jp-trans">我不可能看得懂中文（字面：（我）能读懂中文这事是没理由的）</span>

按照普通的语法规则，我们必须找一个助词与名词「<span class="jp-word" title="わけ - 原因；情况">わけ</span>」合用来连接动词，但因为这种语法太常用了，助词经常都被省略掉，变成「～<span class="jp-word" title="わけ - 原因；情况">わけ</span><span class="jp-word" title="ある - 存在（无生命）">ない</span>」。

### 例二

<span class="jp-word" title="なおこ - 直子">直子</span>：<span class="jp-word" title="ひろこ - 广子">広子</span>の<span class="jp-word" title="いえ - 家">家</span>に<span class="jp-word" title="いく - 走">行った</span><span class="jp-word" title="こと - 事情">こと</span><span class="jp-word" title="ある - 存在（无生命）">ある</span>？<br>
<span class="jp-trans">直子：你去过广子家吗？</span>

<span class="jp-word" title="いちろう - 一郎">一郎</span>：<span class="jp-word" title="ある - 存在（无生命）">ある</span><em><span class="jp-word" title="わけ - 原因；情况">わけ</span><span class="jp-word" title="ある - 存在（无生命）">ない</span></em>でしょう。<br>
<span class="jp-trans">一郎：我不可能去过她家的，对吧？</span>

### 例三

<span class="jp-word" title="なおこ - 直子">直子</span>：<span class="jp-word" title="びせきぶん - 微积分">微積分</span>は<span class="jp-word" title="わかる - 明白">分かる</span>？<br>
<span class="jp-trans">直子：你会微积分吗？</span>

<span class="jp-word" title="いちろう - 一郎">一郎</span>：<span class="jp-word" title="わかる - 明白">分かる</span><em><span class="jp-word" title="わけ - 原因；情况">わけ</span><span class="jp-word" title="ある - 存在（无生命）">ない</span></em>よ！<br>
<span class="jp-trans">一郎：我怎么可能懂！</span>

有一点要注意，「<span class="jp-word" title="わけ - 原因；情况">わけ</span><span class="jp-word" title="ある - 存在（无生命）">ない</span>」也可以用来表示某件事很容易（字面：无须解释），但你可以很容易分辨出来别人是否要表达这种意思，因为这时候它跟形容词一样用。

- <span class="jp-word" title="ここ - 这里">ここ</span> の <span class="jp-word" title="しけん - 考试">試験</span> に <span class="jp-word" title="ごうかく - 及格">合格</span> <span class="jp-word" title="する - 做">する</span> のは <em><span class="jp-word" title="わけない - 简单，容易">わけない</span></em> 。<br>
<span class="jp-trans">这里的考试很容易过。</span>

最后，虽然不是很常用，但「<span class="jp-word" title="わけ - 原因；情况">わけ</span>」也可以用来非常正式的表达某件事必须不惜一切做到或做不到。这其实就是「～てはいけない」更强硬、更正式的版本。这种语法比较简单，加上「<span class="jp-word" title="わけ - 原因；情况">わけ</span>にはいかない」就可以。这里面的「は」是主题助词，所以发音成「わ」。之所以「いけない」改成了「いかない」，也许是跟自动他动有关系，但我不想深究了。只要记住这里是「い<em><u>か</u></em>ない」而非「い<em><u>け</u></em>ない」。

1. <span class="jp-word" title="こんど - 这次">今度</span> は <span class="jp-word" title="まける - 输掉">負ける</span> <em><span class="jp-word" title="わけ - 原因；情况">わけ</span>にはいかない</em> 。<br>
<span class="jp-trans">这一次我一定不能输了。</span>
2. <span class="jp-word" title="ここ - 这里">ここ</span> まで <span class="jp-word" title="くる- 来">きて</span> 、 <span class="jp-word" title="あきらめる - 放弃">あきらめる</span> <em><span class="jp-word" title="わけ - 原因；情况">わけ</span>にはいかない</em> 。<br>
<span class="jp-trans">到了这个地步，我绝不能放弃。</span>

## 用「と<span class="jp-word" title="する - 做">する</span>」做假设 {#part3}

### 词汇学习

1. する （例外） - 做
2. 明日 【あした】 - 明天
3. 行く 【い・く】 （う动词） - 走
4. 今 【いま】- 现在
5. ～時 【～じ】 - 小时的量词
6. 着く 【つ・く】 （う动词） - 到达
7. 思う 【おも・う】 （う动词） - 思考，认为
8. 観客 【かん・きゃく】 - 观众
9. 参加 【さん・か】 - 参加
10. もらう - 接受
11. 被害者 【ひ・がい・しゃ】 - 受害人
12. 非常 【ひ・じょう】 - 极度
13. 幸い 【さいわ・い】 （な形容词） - 幸运的
14. 朝ご飯 【あさ・ご・はん】 - 早饭
15. 食べる 【た・べる】 （る动词） - 吃
16. もう - 已经
17. 昼 【ひる】 - 下午
18. お腹 【お・なか】 - 胃
19. 空く 【す・く】 （う动词） - 变空

虽然这一小节跟上一小节没有直接关系，但我觉得放一起还是很合适。[之前我们学过](/grammar/try/#part3)如何将意向形与「と<span class="jp-word" title="する - 做">する</span>」合用来表示试图做某个动作。现在我们要学一些「と<span class="jp-word" title="する - 做">する</span>」的其他用法。这里要先说一下，其实「と<span class="jp-word" title="する - 做">する</span>」只不过是引用助词「と」和动词「<span class="jp-word" title="する - 做">する</span>」的组合而已。比如有这么个句子：[ 动词 ]と<span class="jp-word" title="する - 做">する</span>，它表达的就是字面意思，即你像是在做那个动作。可以看到，跟意向形合用的时候，就可以表达「像是要做 [ 动词 ] 的样子」。换句话说，你表现得像是要去做引用的那个动作似的，所以我们的翻译是「尝试做某个动作」。让我们看看如果跟动词字典形合用会怎样。

### 示例

- <span class="jp-word" title="あした - 明天">明日</span> に <span class="jp-word" title="いく - 走">行く</span> <em>と<span class="jp-word" title="する - 做">する</span></em> 。<br>
<span class="jp-trans">假设我们明天去。</span>

上例中说话的人在考虑如果决定明天去的话会发生什么。可以看到，字面翻译「像是要明天去的样子」仍然是有意义的。不过这里与前面学过的使用动词意向形的语法不同，这里是在做假设。另外既然是在做假设，那么可以想象这里用条件语也是合适的，事实确实是这样，你可能会经常看到下面这种说法：

- <span class="jp-word" title="いま- 现在">今</span> から <span class="jp-word" title="いく - 走">行く</span> と <em><span class="jp-word" title="する - 做">したら</span></em> 、 <span class="jp-word" title="くじ - 九点">９時</span> に <span class="jp-word" title="つく - 到达">着く</span> と <span class="jp-word" title="おもう - 思考，认为">思います</span> 。<br>
<span class="jp-trans">如果我们现在走，我想会在九点到达。</span>

可以看到动词「<span class="jp-word" title="する - 做">する</span>」被活用成了「たら」条件语的形式，用来表达考虑<i>如果</i>遇到某种情况。你也可以把「<span class="jp-word" title="する - 做">する</span>」活用为て形（<span class="jp-word" title="する - 做">して</span>），表示为一系列动作：

1. <span class="jp-word" title="かんきゃく - 观众">観客</span> <em>と<span class="jp-word" title="する - 做">して</span></em> <span class="jp-word" title="さんか - 参加">参加</span> <span class="jp-word" title="する - 做">させて</span> <span class="jp-word" title="もらう - 接受">もらった</span> 。<br>
<span class="jp-trans">受人帮忙以观众的身份参加。</span>
2. <span class="jp-word" title="ひがいしゃ - 受害人">被害者</span> <em>と<span class="jp-word" title="する - 做">して</span>は</em> 、 <span class="jp-word" title="ひじょう - 极度">非常</span> に <span class="jp-word" title="さいわい - 幸运的">幸い</span> だった。<br>
<span class="jp-trans">作为受害人，曾经十分幸运。</span>
3. <span class="jp-word" title="あさごはん - 早饭">朝ご飯</span> を <span class="jp-word" title="たべる - 吃">食べた</span> <em>と<span class="jp-word" title="する - 做">して</span>も</em> 、 <span class="jp-word" title="もう - 已经">もう</span> <span class="jp-word" title="ひる - 下午">昼</span> だから <span class="jp-word" title="おなか - 胃">お腹</span> が <span class="jp-word" title="すく - 变空">空いた</span> でしょう。<br>
<span class="jp-trans">虽然猜想你应该吃过早饭了，但现在已经中午，你应该还是饿了吧？</span>

这都是用同样的方法造的句。例一里面，你做得像个「观众」一样；例二里面像个「受害人」一样；例三里面像是吃过早饭一样。这样你应该了解为什么同样的语法适用于这些句子了吧，因为它们在日语里面都是一个意思（当然用到的助词不同，「<span class="jp-word" title="する - 做">する</span>」的活用方式也不同）。
