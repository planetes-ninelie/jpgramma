总的来说，除非绝对有把握，日本人一般不会把话说得太死。体现在语言上，就是大量的使用「～と<span class="jp-word" title="おもう - 思考，认为">思う</span>」，以及各种不同的语法来表示不同程度的确定性。我们下面的介绍从最不确定开始，以最确定结束。

## 用「かもしれない」表达不确定性 {#part2}

### 词汇学习

1. 多分 【た・ぶん】 - 也许
2. 映画 【えい・が】 - 电影
3. 観る 【み・る】 （る动词） - 观看
4. 彼 【かれ】 - 他；男朋友
5. 学生 【がく・せい】 - 学生
6. それ - 那
7. 面白い 【おも・し・ろい】 （い形容词） - 有趣
8. 先生 【せん・せい】 - 老师
9. 退屈 【たい・くつ】 - 厌倦
10. 食堂 【しょく・どう】 - 小饭馆
11. 行く 【い・く】 （う动词） - 走
12. 雨 【あめ】 - 雨
13. 試合 【し・あい】 - 比赛
14. 中止 【ちゅう・し】 - 中止
15. なる （う动词） - 成为
16. この - 这个（これの的缩写）
17. 映画 【えい・が】 - 电影
18. ～回 【～かい】 - 次数的量词
19. こと - 事情
20. ある （う动词） - 存在（无生命）
21. あそこ - 那里
22. 代々木公園 【よ・よ・ぎ・こう・えん】 - 代代木公园
23. もう - 已经
24. 逃げる 【に・げる】 （る动词） - 逃走

「かもしれない」用来表示「也许」或者「可能」，确定性小于「<span class="jp-word" title="たぶん  - 也许">多分</span>」。它黏着在一个完整从句的后面。对于名词和な形容词形成的从句，表状态的「だ」必须去掉。它也可以写成汉字，即「かも知れない」，你可以把它当成一个否定的る动词（但没有对应的肯定形），所以ます形就是「かもしれません」。口语里面可以简写成「かも」，另外还有一个很阳刚的版本是「かもしれん」，但这只是另一种否定式罢了。

<div class="sumbox">
<div class="sumbox-title">用「かもしれない」表达不确定性</div>

- 把「かもしれない」或「かも知れない」加到从句后面就好了<br>
<span class="jp-trans">例</span>
  1. <span class="jp-word" title="えいが - 电影">映画</span> を <span class="jp-word" title="みる - 观看">観た</span> <em>かもしれない</em>
  2. <span class="jp-word" title="かれ - 他；男朋友">彼</span> は <span class="jp-word" title="がくせい - 学生">学生</span> <em>かもしれない</em>
  3. <span class="jp-word" title="それ - 那">それ</span> は <span class="jp-word" title="おもしろい - 有趣">面白い</span> <em>かもしれない</em>
- 名词和な形容词形成的从句 <strong>不能</strong> 带表状态的「だ」<br>
<span class="jp-trans">例</span>
  1. <span class="jp-word" title="せんせい - 老师">先生</span> <em>だ</em> かもしれない → <span class="jp-word" title="せんせい - 老师">先生</span> かもしれない
  2. <span class="jp-word" title="たいくつ - 厌倦">退屈</span> <em>だ</em> かもしれない → <span class="jp-word" title="たいくつ - 厌倦">退屈</span> かもしれない
- 口语里面可以简写成「かも」<br>
<span class="jp-trans">例</span>
  1. <span class="jp-word" title="おもしろい - 有趣">面白い</span> <em>かもしれない</em> → <span class="jp-word" title="おもしろい - 有趣">面白い</span> <em>かも</em>
</div>

### 示例

1. スミスさんは <span class="jp-word" title="しょくどう - 小饭馆; dining hall">食堂</span> に <span class="jp-word" title="いく - 走">行った</span> <em>かもしれません</em> 。<br>
<span class="jp-trans">Smith 桑也许去了食堂。</span>
2. <span class="jp-word" title="あめ - 雨">雨</span> で <span class="jp-word" title="しあい - match; game">試合</span> は <span class="jp-word" title="ちゅうし - 中止">中止</span> に <span class="jp-word" title="なる - 成为">なる</span> <em>かもしれない</em> ね。<br>
<span class="jp-trans">比赛也许会因为下雨取消，你说呢？</span>
3. <span class="jp-word" title="この - 这">この</span> <span class="jp-word" title="えいが - 电影">映画</span> は <span class="jp-word" title="いっかい - 一次">一回</span> <span class="jp-word" title="みる - 观看">観た</span> <span class="jp-word" title="こと - 事情">こと</span> <span class="jp-word" title="ある - 存在（无生命）">ある</span> <em>かも</em> ！<br>
<span class="jp-trans">我也许已经看过一次这部电影了。</span>
4. <span class="jp-word" title="あそこ - 那里">あそこ</span> が <span class="jp-word" title="よよぎ - 代代木">代々木</span> <span class="jp-word" title="こうえん - 公园">公園</span> <em>かもしれない</em> 。<br>
<span class="jp-trans">那边也许就是代代木公园。</span>
5. <span class="jp-word" title="もう - 已经">もう</span> <span class="jp-word" title="にげる - 逃避">逃げられない</span> <em>かもしれん</em> ぞ。<br>
<span class="jp-trans">也许不能再逃避了，是吧。</span>

## 用「でしょう」表示一定程度的确定性（礼貌） {#part3}

### 词汇学习

1. 多分 【た・ぶん】 - 也许
2. 明日 【あした】 - 明天
3. 雨 【あめ】 - 雨
4. 学生 【がく・せい】 - 学生
5. これ - 这
6. どこ - 哪里
7. 行く 【い・く】 （う动词） - 走
8. 休む 【やす・む】 （う动词） - 休息
9. いただく （う动词） - 接受；吃；喝（谦逊语）

「でしょう」用来表示一定程度的确定性，它跟「<span class="jp-word" title="たぶん  - 也许">多分</span>」的意思很接近。类似「～です／～ます」，它必须出现在句尾。它没有别的活用形。你也可以把「～ですか」换成「～でしょうか」，这样加上一点不确定性，问句就显得更为礼貌和无假设前提一点。

### 示例

1. <span class="jp-word" title="あした - 明天">明日</span> も <span class="jp-word" title="あめ - 雨">雨</span> <em>でしょう</em> 。<br>
<span class="jp-trans">明天也许也会下雨。</span>
2. <span class="jp-word" title="がくせい - 学生">学生</span> さん <em>でしょう</em> か。<br>
<span class="jp-trans">（你）是学生吗？</span>
3. <span class="jp-word" title="これから - 从此以后">これから</span> <span class="jp-word" title="どこ - 哪里">どこ</span> へ <span class="jp-word" title="いく - 走">行く</span> ん <em>でしょう</em> か？<br>
<span class="jp-trans">（你）是要去哪里？</span>

如果你想显得特别特别礼貌，甚至可以把「～でしょうか」加到「～ます」后面。

- <span class="jp-word" title="やすむ - 休息">休ませて</span> <span class="jp-word" title="いただく - 接受（谦逊语）">いただけます</span> <em>でしょうか</em> 。- 我可以得到休息的机会吗，也许？

## 用「でしょう」和「だろう」来表示很大程度的确定性（口语） {#part4}

### 词汇学习

1. 遅刻 【ち・こく】 - 延迟
2. する （例外） - 做
3. 時間 【じ・かん】 - 时间
4. ある （う动词） - 存在（无生命）
5. 言う 【い・う】 （う动词）- 说
6. これ - 这
7. 食べる 【た・べる】 （る动词） - 吃
8. 行く 【い・く】 （う动词） - 走
9. 掃除 【そう・じ】 - 扫除
10. 手伝う 【て・つだ・う】 （う动词） - 帮助，协助
11. くれる （る动词） - 给
12. そう - 那样
13. どこ - 哪里
14. もう - 已经
15. 寝る 【ね・る】 （る动词） - 睡
16. 家 【1) うち; 2) いえ】 - 1）自己家；2）房子
17. 帰る 【かえ・る】 （う动词） - 回家

让人惊奇的是，口语里面对应「でしょう」的还是「でしょう」。不过，当你跟人客气说话的时候，「でしょう」的发音是平缓的，而在口语里面，它会有一个升调，还可以简写成「でしょ」。另外，因为人们在口语里倾向于说话更确定，这个词还有一层意思，类似，「看见没，跟你说过了！」

### 例一

Ａ：あっ！<span class="jp-word" title="ちこく - 延迟">遅刻</span><span class="jp-word" title="する - 做">しちゃう</span>！<br>
<span class="jp-trans">A：啊！我们要迟到了！</span>

Ｂ：だから、<span class="jp-word" title="じかん - 时间">時間</span>が<span class="jp-word" title="ある - 存在（无生命）">ない</span>って<span class="jp-word" title="いう- 说">言った</span><em>でしょう</em>！<br>
<span class="jp-trans">B：所以跟你说过没时间了！</span>

### 例二

Ａ：<span class="jp-word" title="これから - 从这里">これから</span><span class="jp-word" title="たべる - 吃">食べ</span>に<span class="jp-word" title="いく - 走">行く</span>ん<em>でしょ</em>。<br>
<span class="jp-trans">A：你现在就要开始吃对吧？</span>

Ｂ：だったら？<br>
<span class="jp-trans">B：是又怎样？</span>

### 例三

Ａ：<span class="jp-word" title="そうじ - 扫除">掃除</span>、<span class="jp-word" title="てつだう - 帮助；协助">手伝って</span><span class="jp-word" title="くれる - 给">くれる</span><em>でしょう</em>。<br>
<span class="jp-trans">A：你会帮我做扫除的，对吧？</span>

Ｂ：え？<span class="jp-word" title="そう - 这样">そう</span>なの？<br>
<span class="jp-trans">B：嗯？是吗？</span>

「だろう」的意思跟「でしょう」基本一样，只不过它听起来更阳刚，通常是男性使用。

### 例四

Ａ：アリスは<span class="jp-word" title="どこ - 哪里">どこ</span>だ？<br>
<span class="jp-trans">A：Alice 在哪里？</span>

Ｂ：<span class="jp-word" title="もう - 已经">もう</span><span class="jp-word" title="ねる - 睡">寝ている</span><em>だろう</em>。<br>
<span class="jp-trans">B：也许已经睡了。</span>

### 例五

Ａ：<span class="jp-word" title="もう - 已经">もう</span><span class="jp-word" title="うち - home; いえ - 家">家</span>に<span class="jp-word" title="かえる - 回家">帰る</span>ん<em>だろう</em>。<br>
<span class="jp-trans">A：你已经要回家了，对吧？</span>

Ｂ：<span class="jp-word" title="そう - 这样">そう</span>よ。<br>
<span class="jp-trans">B：是的。</span>
