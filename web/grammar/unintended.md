有很多有用的表达方式将成为你日常会话的重要组成部分，这里是我们介绍的第一个。我们要学习如何表达自己无意中做出的、通常带来负面结果的动作。这主要用的动词是 「<span class="jp-word" title="しまう - 无意的举动，完全完成">しまう</span>」。我们来看例句。

### 词汇学习

1. 康介 【こう・すけ】 - 康介（姓）
2. 宿題 【しゅく・だい】 - 作业
3. やる （う动词） - 做
4. しまう （う动词） - 无意的举动，完全完成

<span class="jp-word" title="こうすけ - 康介">康介</span>：<span class="jp-word" title="しゅくだい - 作业">宿題</span>を<span class="jp-word" title="やる - 做">やった</span>？<br>
<span class="jp-trans">康介：你做作业了吗？</span>

アリス：<em><span class="jp-word" title="しまう - 无意的举动，完全完成">しまった</span></em>！<br>
<span class="jp-trans">Alice：噢不！（我搞砸了！）</span>

## 与其他动词何用「<span class="jp-word" title="しまう - 无意的举动，完全完成">しまう</span>」 {#part2}

### 词汇学习

1. しまう （う动词） - 无意的举动，完全完成
2. その - 那个 （それの的缩写）
3. ケーキ - 蛋糕
4. 全部 【ぜん・ぶ】 - 所有东西
5. 食べる 【た・べる】 （る动词） - 吃
6. 毎日 【まい・にち】 - 每天
7. キロ - 千克
8. 太る 【ふと・る】 （う动词） - 变胖
9. ちゃんと - 正确的
10. 痩せる 【や・せる】 （る动词） - 瘦身
11. 結局 【けっ・きょく】 - 最终
12. 嫌 【いや】 （な形容词） 厌恶；不爽
13. こと - 事情
14. する （例外） - 做
15. ごめん - 对不起
16. 待つ 【ま・つ】 （う动词） - 等待
17. 金魚 【きん・ぎょ】 - 金鱼
18. もう - 已经
19. 死ぬ 【し・ぬ】 （う动词） - 死

对于上面介绍的「<span class="jp-word" title="しまう - 无意的举动，完全完成">しまう</span>」的用法，很容易想到把它加到动词て形后面来表示无意中发生的事。同样的，这种语法的时态取决于「<span class="jp-word" title="しまう - 无意的举动，完全完成">しまう</span>」的时态。

1. <span class="jp-word" title="その - 那个">その</span> <span class="jp-word" title="ケーキ - 蛋糕">ケーキ</span> を <span class="jp-word" title="ぜんぶ - 全部">全部</span> <em><span class="jp-word" title="たべる - 吃">食べて</span></em> <span class="jp-word" title="しまう - 无意的举动，完全完成">しまった</span> 。<br>
<span class="jp-trans">呀，我把整个蛋糕都吃了。</span>
2. <span class="jp-word" title="まいにち - 每天">毎日</span> <span class="jp-word" title="ケーキ - 蛋糕">ケーキ</span> を <span class="jp-word" title="たべる - 吃">食べて</span> 、 <span class="jp-word" title="にキロ - 2公斤">２キロ</span> <em><span class="jp-word" title="ふとる - 增肥">太って</span></em> <span class="jp-word" title="しまう - 无意的举动，完全完成">しまいました</span> 。<br>
<span class="jp-trans">我每天吃蛋糕，（一不小心）长了两公斤。</span>
3. <span class="jp-word" title="ちゃんと - 正确的">ちゃんと</span> <span class="jp-word" title="たべる - 吃">食べない</span> と、 <em><span class="jp-word" title="やせる - 变瘦">痩せて</span></em> <span class="jp-word" title="しまう - 无意的举动，完全完成">しまいます</span> よ。<br>
<span class="jp-trans">如果饮食不正确的话，你会（无意中）减轻体重的。</span>
4. <span class="jp-word" title="けっきょく - 结果">結局</span> 、 <span class="jp-word" title="いや - 嫌弃">嫌</span> な <span class="jp-word" title="こと - 事情">こと</span> を <em><span class="jp-word" title="する - 做">させて</span></em> <span class="jp-word" title="しまう - 无意的举动，完全完成">しまった</span> 。<br>
<span class="jp-trans">最终，我（无意中）让 [ 某人 ] 做了令人讨厌的事。</span>
5. <span class="jp-word" title="ごめん - 对不起">ごめん</span> 、 <em><span class="jp-word" title="まつ - 等待">待たせて</span></em> <span class="jp-word" title="しまう - 无意的举动，完全完成">しまって</span> ！<br>
<span class="jp-trans">不好意思，（无意中）让你久等了！</span>
6. <span class="jp-word" title="きんぎょ - 金鱼">金魚</span> が <span class="jp-word" title="もう - 已经">もう</span> <em><span class="jp-word" title="しぬ - 死">死んで</span></em> <span class="jp-word" title="しまう - 无意的举动，完全完成">しまった</span> 。<br>
<span class="jp-trans">那条金鱼已经死了（残念）。</span>

## 「～て<span class="jp-word" title="しまう - 无意的举动，完全完成">しまう</span>」的口语版本 {#part3}

### 词汇学习

1. しまう （う动词） - 无意的举动，完全完成
2. 金魚 【きん・ぎょ】 - 金鱼
3. もう - 已经
4. 死ぬ 【し・ぬ】 （う动词） - 死
5. 帰る 【かえ・る】 （う动词） - 回家
6. いい （い形容词） - 好的
7. 皆 【みんな】 - 所有人
8. どっか - 某处（どこか的缩写）
9. 行く 【い・く】 （う动词） - 走
10. そろそろ - 逐渐；即将
11. 遅い 【おそ・い】 （い形容词） - 迟
12. なる （う动词） - 成为
13. また - 再次
14. 遅刻 【ち・こく】 - 延迟
15. する （例外） - 做
16. ごめん - 对不起
17. つい - 刚刚；无意的
18. お前 【お・まえ】 - 你 （口语）
19. 呼ぶ 【よ・ぶ】 （う动词） - 呼叫

口语里面「～て<span class="jp-word" title="しまう - 无意的举动，完全完成">しまう</span>」通常被替换为「～ちゃう」，而「～で<span class="jp-word" title="しまう - 无意的举动，完全完成">しまう</span>」则对应「じゃう」「～ちゃう」和「～じゃう」都可以像普通う动词一样活用。

1. <span class="jp-word" title="きんぎょ - 金鱼">金魚</span> が <span class="jp-word" title="もう - 已经">もう</span> <span class="jp-word" title="しぬ - 死">死ん</span> <em>じゃった</em> 。<br>
<span class="jp-trans">那只金鱼已经死了。</span>
2. <span class="jp-word" title="もう - 已经">もう</span> <span class="jp-word" title="かえる - 回家">帰っ<em>ちゃって</em></span> <span class="jp-word" title="いい - 好的">いい</span> ？<br>
<span class="jp-trans">我现在回家可以吗？</span>
3. <span class="jp-word" title="みんな - 所有人">みんな</span> 、 <span class="jp-word" title="どっか - somewhere (from どこか)">どっか</span> <span class="jp-word" title="いく - 走">行っ<em>ちゃった</em></span> よ。<br>
<span class="jp-trans">每个人都去了某个地方。</span>
4. <span class="jp-word" title="そろそろ - gradually">そろそろ</span> <span class="jp-word" title="おそい - 迟">遅くなっ<em>ちゃう</em></span> よ。<br>
<span class="jp-trans">就快要迟到了。</span>

「～て<span class="jp-word" title="しまう - 无意的举动，完全完成">しまう</span>」和「～で<span class="jp-word" title="しまう - 无意的举动，完全完成">しまう</span>」还有一个更俗的版本，就是「～ちまう」和「～じまう」。与可爱的「～ちゃう」和「～じゃう」不同，这个版本是那种粗俗的中年男子用的。

1. <span class="jp-word" title="また - 再次">また</span> <span class="jp-word" title="ちこく - 延迟">遅刻</span> <em><span class="jp-word" title="する - 做">しちまった</span></em> よ。<br>
<span class="jp-trans">干，又迟到了。</span>
2. <span class="jp-word" title="ごめん - 对不起">ごめん</span> 、 <span class="jp-word" title="つい - unconsciously">つい</span> <span class="jp-word" title="おまえ - 你 (vulgar)">お前</span> を <span class="jp-word" title="よぶ - 呼叫">呼ん<em>じまった</em></span> 。<br>
<span class="jp-trans">不好意思，不小心打了你的电话。</span>

## 「<span class="jp-word" title="しまう - 无意的举动，完全完成">しまう</span>」另外的意思 {#part4}

### 词汇学习

1. しまう （う动词） - 无意的举动，完全完成
2. 宿題 【しゅく・だい】 - 作业
3. やる （う动词） - 做

你也许注意到了，「<span class="jp-word" title="しまう - 无意的举动，完全完成">しまう</span>」的定义里面还有一个意思是「完全完成」，你可以认为这个我们上面介绍过的「<span class="jp-word" title="しまう - 无意的举动，完全完成">しまう</span>」是完全不同的单词。有些时候（但不是很经常），「<span class="jp-word" title="しまう - 无意的举动，完全完成">しまう</span>」的意思是这个，而非无意举动。

- <span class="jp-word" title="しゅくだい - 作业">宿題</span> を <span class="jp-word" title="やる - 做">やって</span> <span class="jp-word" title="しまう - 无意的举动，完全完成">しまいなさい</span> 。<br>
<span class="jp-trans">把你的作业做完。</span>
