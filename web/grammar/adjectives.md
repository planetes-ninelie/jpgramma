## 形容词的属性

学会用助词连接两个名词之后，我们学习用形容词来修饰名词。形容词可以直接放在名词前面，也可以像连接名词那样用助词来连接名词。形容词可以分为<b>な形容词</b>和<b>い形容词</b>两类。

## な形容词 {#naadj}

### 单词学习

1. 静か 【しず・か】 （な形容词） - 安静
2. 人 【ひと】 - 人
3. きれい （な形容词） - 漂亮；干净
4. 友達 【とも・だち】 - 朋友
5. 親切 【しん・せつ】 （な形容词） - 和善
6. 魚 【さかな】 - 鱼
7. 好き 【す・き】 （な形容词） - 招人喜欢的
8. 肉 【にく】 - 肉
9. 野菜 【や・さい】 - 蔬菜

な形容词跟名词基本一样，所以很好学。な形容词和名词的活用规则是一样的，主要区别在于用な形容词修饰名词的时候中间要加一个「な」（这就是な形容词这个名字的由来）。

### 示例

1. <span class="jp-word" title="しずか - 安静">静か</span> <em>な</em> <span class="jp-word" title="ひと - 人">人</span> 。<br>
<span class="jp-trans">安静的人。</span>
2. <span class="jp-word" title="きれい - 漂亮；干净">きれい</span> <em>な</em> <span class="jp-word" title="ひと - 人">人</span> 。<br>
<span class="jp-trans">漂亮的人。</span>

你也可以用助词连接名词和形容词，就跟上一课学的一样。

### 示例

1. <span class="jp-word" title="ともだち - 朋友">友達</span> は <span class="jp-word" title="しんせつ - 和善">親切</span> 。<br>
<span class="jp-trans">朋友是和善的。</span>
2. <span class="jp-word" title="ともだち - 朋友">友達</span> は <span class="jp-word" title="しんせつ - 和善">親切</span> な <span class="jp-word" title="ひと - 人">人</span> だ。<br>
<span class="jp-trans">朋友是和善的人。</span>

如下所示，な形容词的活用规则跟名词是一样的。

### 示例

1. ボブは <span class="jp-word" title="さかな - 鱼">魚</span> が <em><span class="jp-word" title="すき - 喜欢">好き</span></em> だ。<br>
<span class="jp-trans">Bob 喜欢鱼。</span>
2. ボブは <span class="jp-word" title="さかな - 鱼">魚</span> が <em><span class="jp-word" title="すき - 喜欢">好きじゃない</span></em> 。<br>
<span class="jp-trans">Bob 不喜欢鱼。</span>
3. ボブは <span class="jp-word" title="さかな - 鱼">魚</span> が <em><span class="jp-word" title="すき - 喜欢">好きだった</span></em> 。<br>
<span class="jp-trans">Bob 以前喜欢鱼。</span>
4. ボブは <span class="jp-word" title="さかな - 鱼">魚</span> が <em><span class="jp-word" title="すき - 喜欢">好きじゃなかった</span></em> 。<br>
<span class="jp-trans">Bob 以前不喜欢鱼。</span>

「喜欢」在日语里是形容词而不是动词，如果这让你觉得困惑，可以把「<span class="jp-word" title="すき - 喜欢">好き</span>」想成「招人喜欢的」。另外，你也可以从上例看到主题助词和识别助词合用的情形。句子的主题是 "Bob"，而「鱼」标识了 Bob 喜欢的对象。

你也可以用前面学过的三种活用形来直接修饰名词（记住非过去肯定形后面要加「な」）。

### 示例

1. <span class="jp-word" title="さかな - 鱼">魚</span> が <em><span class="jp-word" title="すき - 喜欢">好きな</span></em> <span class="jp-word" title="ひと - 人">人</span> 。<br>
<span class="jp-trans">喜欢鱼的人。</span>
2. <span class="jp-word" title="さかな - 鱼">魚</span> が <em><span class="jp-word" title="すき - 喜欢">好きじゃない</span></em> <span class="jp-word" title="ひと - 人">人</span> 。<br>
<span class="jp-trans">不喜欢鱼的人。</span>
3. <span class="jp-word" title="さかな - 鱼">魚</span> が <em><span class="jp-word" title="すき - 喜欢">好きだった</span></em> <span class="jp-word" title="ひと - 人">人</span> 。<br>
<span class="jp-trans">以前喜欢鱼的人。</span>
4. <span class="jp-word" title="さかな - 鱼">魚</span> が <em><span class="jp-word" title="すき - 喜欢">好きじゃなかった</span></em> <span class="jp-word" title="ひと - 人">人</span> 。<br>
<span class="jp-trans">以前不喜欢鱼的人。</span>

这里，整个「<span class="jp-word" title="さかな - 鱼">魚</span>が<span class="jp-word" title="すき - 喜欢">好き</span>」、「<span class="jp-word" title="さかな - 鱼">魚</span>が<span class="jp-word" title="すき - 喜欢">好きじゃない</span>」等从句都是用来修饰「人」，来谈论喜欢或不喜欢鱼的人。这种句子很有用，因为「<span class="jp-word" title="ひと - 人">人</span>は<span class="jp-word" title="さかな - 鱼">魚</span>が<em><span class="jp-word" title="すき - 喜欢">好き</span></em>だ」 的意思是「人喜欢鱼」，但实际情况并不一定如此。

我们也可以把整个被形容词修饰过的名词从句当作一个单独的名词来看待。比如我们可以把整个从句当作句子的主题，如下所示。

### 示例

1. <em><span class="jp-word" title="さかな - 鱼">魚</span>が<span class="jp-word" title="すき - 喜欢">好き</span>じゃない<span class="jp-word" title="ひと - 人">人</span></em> は、 <span class="jp-word" title="にく - 肉">肉</span> が <span class="jp-word" title="すき - 喜欢">好き</span> だ。<br>
<span class="jp-trans">不喜欢鱼的人喜欢肉。</span>
2. <em><span class="jp-word" title="さかな - 鱼">魚</span>が<span class="jp-word" title="すき - 喜欢">好き</span>な<span class="jp-word" title="ひと - 人">人</span></em> は、 <span class="jp-word" title="やさい - vegetables">野菜</span> も <span class="jp-word" title="すき - 喜欢">好き</span> だ。<br>
<span class="jp-trans">喜欢鱼的人也喜欢蔬菜。</span>

## い形容词 {#iadj}

### 单词学习

1. 嫌い 【きら・い】 （な形容词） - 令人不快的；可恶的
2. 食べ物 【た・べ・もの】 - 食物
3. おいしい （い形容词） - 可口
4. 高い 【たか・い】 （い形容词） - 高；贵
5. ビル - 楼
6. 値段 【ね・だん】 - 价格
7. レストラン - 餐馆
8. あまり／あんまり - 不是非常（与否定式连用）
9. 好き 【す・き】 （な形容词） - 招人喜欢的
10. いい （い形容词） - 好

所有い形容词都是以平假名「い」结尾的。但你也许注意到了，有些な形容词也是以「い」结尾，例如「<span class="jp-word" title="きれい - 漂亮；干净">きれい</span>（な）」。那应该怎么分辨呢？实际上，不是用汉字写的但以「い」结尾的な形容词非常少见。两个最常见的例子是「<span class="jp-word" title="きれい - neat, pretty">きれい</span>」和「<span class="jp-word" title="きらい - 令人不快的；可恶的">嫌い</span>」。所有其他以「い」结尾的な形容词基本都是用汉字写的，所以你可以很容易分辨出来它们不是い形容词。比如，「<span class="jp-word" title="きれい - neat, pretty">きれい</span>」写成汉字就是「<span class="jp-word" title="きれい - neat, pretty">綺麗</span>」或者「<span class="jp-word" title="きれい - neat, pretty">奇麗</span>」。因为「麗」里面包含了「い」，你就能知道它肯定不是い形容词。因为い形容词以「い」结尾的<b>全部意义</b>就在于活用的时候改变「い」而汉字不动。实际上，「<span class="jp-word" title="きらい - 令人不快的；可恶的">嫌い</span>」是极少几个以 「い」而非汉字结尾的な形容词之一，这是因为「<span class="jp-word" title="きらい - 令人不快的；可恶的">嫌い</span>」实际上是从动词「<span class="jp-word" title="きらう - 厌恶">嫌う</span>」派生而来的。

与な形容词不同，你<b>无需</b>在用い形容词修饰名词的时候添加一个「な」。

### 示例

1. <span class="jp-word" title="きらい - 令人不快的；可恶的">嫌い</span> <em>な</em> <span class="jp-word" title="たべもの - 食物">食べ物</span> 。<br>
<span class="jp-trans">不喜欢的食物。</span>
2. <span class="jp-word" title="おいしい - 可口">おいしい</span> <span class="jp-word" title="たべもの - 食物">食べ物</span> 。<br>
<span class="jp-trans">可口的食物</span>

还记得名词的否定活用形也是以「い」结尾的吗（じゃな<em>い</em>）？同样的，永远不要在い形容词后面添加表陈述的「だ」.

> **译者注：** <strong><b>不要在い形容词后面加「だ」！</b></strong>

解释清楚以后，下面是い形容词的活用规则。注意过去否定式和过去形的活用规则是一样的。

<div class="sumbox">
<div class="sumbox-title">い形容词的活用规则</div>

- <b>否定式</b> ：首先把结尾的「い」去掉，然后加上「くない」。
- 例： <span class="jp-word" title="たかい - 高；贵">高<em>い</em></span> → <span class="jp-word" title="たかい - 高；贵">高<em>くない</em></span>
- <b>过去形</b> ：首先把い形容词或否定い形容词结尾的「い」去掉，然后加上「かった」。<br>
<span class="jp-trans">例</span>
  1. <span class="jp-word" title="たかい - 高；贵">高<em>い</em></span> → <span class="jp-word" title="たかい - 高；贵">高<em>かった</em></span>
  2. <span class="jp-word" title="たかい - 高；贵">高くな<em>い</em></span> → <span class="jp-word" title="たかい - 高；贵">高くな<em>かった</em></span>

### 示例

1. <em><span class="jp-word" title="たかい - 高；贵">高い</span></em> <span class="jp-word" title="ビル - 楼">ビル</span> 。<br>
<span class="jp-trans">高的楼。</span>
2. <em><span class="jp-word" title="たかい - 高；贵">高くない</span></em> <span class="jp-word" title="ビル - 楼">ビル</span> 。<br>
<span class="jp-trans">不高的楼。</span>
3. <em><span class="jp-word" title="たかい - 高；贵">高かった</span></em> <span class="jp-word" title="ビル - 楼">ビル</span> 。<br>
<span class="jp-trans">曾经高的楼。</span>
4. <em><span class="jp-word" title="たかい - 高；贵">高くなかった</span></em> <span class="jp-word" title="ビル - 楼">ビル</span> 。<br>
<span class="jp-trans">曾经不高的楼。</span>

你也可以跟な形容词组合出修饰过的名词从句。唯一的区别在于我们不需要用「な」去修饰名词。

### 示例

- <em><span class="jp-word" title="ねだん - 价格">値段</span>が<span class="jp-word" title="たかい - 高；贵">高い</span><span class="jp-word" title="レストラン - 餐馆">レストラン</span></em> は <span class="jp-word" title="あまり - 不是非常（与否定式连用）">あまり</span> <span class="jp-word" title="すき - 喜欢">好きじゃない</span> 。<br>
<span class="jp-trans">不是很喜欢价格高的餐馆。</span>

上例中，描述从句「<span class="jp-word" title="ねだん - 价格">値段</span>が<span class="jp-word" title="たかい - 高；贵">高い</span>」直接修饰了「<span class="jp-word" title="レストラン - restaraunt">レストラン</span>」。

## 烦人的例外

### 单词学习

1. 値段 【ね・だん】 - 价格
2. あまり／あんまり - 不是非常（与否定式连用）
3. いい （い形容词） - 好
4. 彼 【かれ】 - 他；男朋友
5. かっこいい （い形容词） - 酷；帅

只有一个意思是「好」的い形容词跟其他い形容词不一样。日语初学者最痛恨这一点了，因为恰恰是这些常用词会有不寻常的用法。「好」这个词原本是「よい（<span class="jp-word" title="よい - 好">良い</span>）」，但后来逐渐演变成「<span class="jp-word" title="いい - 好">いい</span>」，而写成汉字时，又通常读成「<span class="jp-word" title="よい - 好">よい</span>」，所以「<span class="jp-word" title="いい - 好">いい</span>」基本上总是用平假名写。到此为止一切都没问题。不幸的是，这个词的活用形却又都是从「<span class="jp-word" title="よい - 好">よい</span>」而非「<span class="jp-word" title="いい - 好">いい</span>」派生出来，可以看下面的表格。

还有一个类似的形容词是「<span class="jp-word" title="かっこいい - 好看，酷">かっこいい</span>」，因为它是两个单词合并缩写而成：「<span class="jp-word" title="かっこう - 看上去">格好</span>」和「<span class="jp-word" title="いい - 好">いい</span>」。因为它也包含了「<span class="jp-word" title="いい - 好">いい</span>」，所以你也要遵循同样的活用规则。

注意要以「<span class="jp-word" title="よい - 好">よい</span>」为基础进行活用，而不是「<span class="jp-word" title="いい - 好">いい</span>」。

### 示例

1. <span class="jp-word" title="ねだん - 价格">値段</span> が <span class="jp-word" title="あんまり - 不是非常（与否定式连用）">あんまり</span> <span class="jp-word" title="いい - 好"><em>よくない</em></span> 。<br>
<span class="jp-trans">价格不是很好。</span>
2. <span class="jp-word" title="かれ - 他；男朋友">彼</span> は <span class="jp-word" title="かっこいい - 酷；帅"><em>かっこよかった</em></span> ！<br>
<span class="jp-trans">他看起来真酷！</span>
