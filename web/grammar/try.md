## 试试看这些东西 {#part1}

英语里面 "try" 既可以表示 "to try something out"，也可以表示 "to make an effort to do something"。日语里面，这两种意思要用不同的语法来表达。比如 "I tried the cherry flavor" 和 "I tried to do homework" 的意思差别很大，虽然英语用了同样的动词，但日语可以不一样。

## 试（试试看） {#part2}

### 词汇学习

1. 見る 【み・る】 - 看；观看
2. 切る 【き・る】 （う动词） - 切
3. お好み焼き 【お・この・み・や・き】 - 煎菜饼（日式煎饼）
4. 初めて 【はじ・めて】 - 第一次
5. 食べる 【た・べる】 （る动词） - 吃
6. とても - 非常
7. おいしい （い形容词） - 可口
8. お酒 【お・さけ】 - 酒
9. 飲む 【の・む】 （う动词） - 喝
10. すごい （い形容词） - 很大程度上
11. 眠い 【ねむ・い】（い形容词） - 困
12. なる （う动词） - 成为
13. 新しい 【あたら・しい】（い形容词） - 新的
14. デパート - 百货
15. 行く 【い・く】 （う动词） - 走
16. 広島 【ひろ・しま】 - 广岛

要想说试什么东西很简单，把动词て形后面加上「<span class="jp-word" title="みる - 看">みる</span>」就行。为了方便记忆，你可以把它想象成一先一后两个动作，先做再看结果。实际上，「<span class="jp-word" title="みる - 看">みる</span>」的活用规则跟「<span class="jp-word" title="みる - 看">見る</span>」是一样的。不过跟我们前面学过的[「～て<span class="jp-word" title="ほしい - 要，渴望">ほしい</span>」](/grammar/desire/#part3)语法一样，这是一种固定搭配，「<span class="jp-word" title="みる - 看">みる</span>」也通常写作平假名。

<div class="sumbox">
<div class="sumbox-title">试、试着</div>

- 把动词活用成て形，再加上「 <span class="jp-word" title="みる - 看">みる</span> 」。
- 例： <span class="jp-word" title="きる - 切">切る</span> → <span class="jp-word" title="きる - 切">切<em>って</em></span> → <span class="jp-word" title="きる - 切">切って</span> <em><span class="jp-word" title="みる - 看">みる</span></em>
- 整个组合可以当成跟「 <span class="jp-word" title="みる - 看">見る</span> 」一样的动词来活用。
- 例： <span class="jp-word" title="きる - 切">切って</span> <em><span class="jp-word" title="みる - 看">みる</span></em> 、 <span class="jp-word" title="きる - 切">切って</span> <em><span class="jp-word" title="みる - 看">みた</span></em> 、 <span class="jp-word" title="きる - 切">切って</span> <em><span class="jp-word" title="みる - 看">みない</span></em> 、 <span class="jp-word" title="きる - 切">切って</span> <em><span class="jp-word" title="みる - 看">みなかった</span></em>
</div>

### 示例

1. <span class="jp-word" title="おこのみやき - 跟 pizza 类似用各种食材炸出来的煎饼">お好み焼き</span> を <span class="jp-word" title="はじめて - 第一次">初めて</span> <em><span class="jp-word" title="たべる - 吃">食べて</span><span class="jp-word" title="みる - 看">みた</span></em> けど、 <span class="jp-word" title="とても - 非常">とても</span> <span class="jp-word" title="おいしい - 可口">おいしかった</span> ！<br>
<span class="jp-trans">我第一次试了煎菜饼，很好吃！</span>
2. <span class="jp-word" title="おさけ - 酒">お酒</span> を <em><span class="jp-word" title="のむ - 喝">飲んで</span><span class="jp-word" title="みる - 看">みました</span></em> が、 <span class="jp-word" title="すごい - 令人惊讶的，极度">すごく</span> <span class="jp-word" title="ねむい - 困">眠く</span> <span class="jp-word" title="なる - 成为">なりました</span> 。<br>
<span class="jp-trans">我试了些酒，然后变得非常困。</span>
3. <span class="jp-word" title="あたらしい - 新的">新しい</span> <span class="jp-word" title="デパート - 百货">デパート</span> に <em><span class="jp-word" title="いく - 走">行って</span><span class="jp-word" title="みる - 看">みる</span></em> 。<br>
<span class="jp-trans">我要去看看新开的百货大楼。</span>
4. <span class="jp-word" title="ひろしま - 广岛">広島</span> の <span class="jp-word" title="おこのみやき - 跟 pizza 类似用各种食材炸出来的煎饼">お好み焼き</span> を <em><span class="jp-word" title="たべる - 吃">食べて</span><span class="jp-word" title="みる - 看">みたい</span></em> ！<br>
<span class="jp-trans">我想去试试广岛的菜煎饼！</span>

## 尝试做什么（努力去做） {#part3}

### 词汇学习

1. する （例外） - 做
2. 言う 【い・う】 （う动词）- 说
3. 思う 【おも・う】 （う动词） - 思考，认为
4. 考える 【かんが・える】 （る动词） - 考虑
5. 見る 【み・る】 （る动词） - 看
6. 行く 【い・く】 （う动词） - 走
7. 毎日 【まい・にち】 - 每天
8. 勉強 【べん・きょう】 - 学习
9. 避ける 【さ・ける】 （る动词） - 避开
10. 無理矢理 【む・り・や・り】 - 用力地
11. 部屋 【へ・や】 - 房间
12. 入る 【はい・る】 （う动词） - 进入
13. 早い 【はや・い】 （い形容词） - 快；早
14. 寝る 【ね・る】 （る动词） - 睡
15. 結局 【けっ・きょく】 - 最终
16. 徹夜 【てつ・や】 - 通宵
17. お酒 【お・さけ】 - 酒
18. 飲む 【の・む】 （う动词） - 喝
19. 奥さん【おく・さん】 - 妻子 （礼貌）
20. 止める 【と・める】 （る动词） - 停止
21. なるべく - 尽可能多
22. ジム - 健身房
23. 決める 【き・める】 （る动词） - 决定

我们已经学会用意向形来表达想做某事的意愿。如果你认为下面这种语法也要用到意向形，那么你猜对了。想说尝试做某事的话，只要把动词活用成意向形，再加以引用（这样我们就可以对它用动词了），最后加上「<span class="jp-word" title="する - 做">する</span>」就可以。或者简单说，只要在动词意向形后面加上「と<span class="jp-word" title="する - 做">する</span>」。这是上一课学的引用关系从句的扩展，这次不是说（<span class="jp-word" title="いう- 说">言う</span>）或者想（<span class="jp-word" title="おもう - 思考，认为">思う</span>、<span class="jp-word" title="かんがえる - to consider">考える</span>），而是做「<span class="jp-word" title="する - 做">する</span>」。

<div class="sumbox">
<div class="sumbox-title">尝试做什么</div>

- 把动词活用为意向形，再加上「と <span class="jp-word" title="する - 做">する</span> 」。
- 例
  1. <span class="jp-word" title="みる - 看">見る</span> → <span class="jp-word" title="みる - 看">見<em>よう</em></span> → <span class="jp-word" title="みる - 看">見よう</span> <em>と<span class="jp-word" title="する - 做">する</span></em>
  2. <span class="jp-word" title="いく - 走">行く</span> → <span class="jp-word" title="いく - 走">行<em>こう</em></span> → <span class="jp-word" title="いく - 走">行こう</span> <em>と<span class="jp-word" title="する - 做">する</span></em>
</div>

### 示例

1. <span class="jp-word" title="まいにち - 每天">毎日</span> 、 <span class="jp-word" title="べんきょう - 学习">勉強</span> を <em><span class="jp-word" title="さける - 避开">避けよう</span>と<span class="jp-word" title="する - 做">する</span></em> 。<br>
<span class="jp-trans">她每天都试图逃避学习。</span>
2. <span class="jp-word" title="むりやり - 用力地，违反自己意志的">無理矢理</span> に <span class="jp-word" title="へや - 房间">部屋</span> に <em><span class="jp-word" title="はいる - 进入">入ろう</span>と<span class="jp-word" title="する - 做">している</span></em> 。<br>
<span class="jp-trans">他试着强迫自己进房间。</span>
3. <span class="jp-word" title="はやい - 快；早">早く</span> <em><span class="jp-word" title="ねる - 睡">寝よう</span>と<span class="jp-word" title="する - 做">した</span></em> けど、 <span class="jp-word" title="けっきょく - 结果">結局</span> は <span class="jp-word" title="てつや - 整晚失眠">徹夜</span> <span class="jp-word" title="する - 做">した</span> 。<br>
<span class="jp-trans">我试着早睡，但结果却通宵了。</span>
4. <span class="jp-word" title="おさけ - 酒">お酒</span> を <em><span class="jp-word" title="のむ - 喝">飲もう</span>と<span class="jp-word" title="する - 做">した</span></em> が、 <span class="jp-word" title="おくさん - 妻子">奥さん</span> が <span class="jp-word" title="とめる - 停止">止めた</span> 。<br>
<span class="jp-trans">他尝试喝酒，但他妻子制止了他。</span>

虽然这里我们用了动词「<span class="jp-word" title="する - 做">する</span>」来表达「尝试」，其实还有其他很多动词可用来表示这个意思，比如「<span class="jp-word" title="きめる - 决定">決める</span>」的意思是「决定做某事」。下面是一些例子。

1. <span class="jp-word" title="べんきょう - 学习">勉強</span> を <span class="jp-word" title="なるべく - 尽可能多">なるべく</span> <span class="jp-word" title="さける - 避开">避けよう</span> と <em><span class="jp-word" title="おもう - 思考，认为">思った</span></em> 。<br>
<span class="jp-trans">我想我会尽可能的逃避学习。</span>
2. <span class="jp-word" title="まいにち - 每天">毎日</span> <span class="jp-word" title="ジム - 健身房">ジム</span> に <span class="jp-word" title="いく - 走">行こう</span> と <em><span class="jp-word" title="きめる - 决定">決めた</span></em> 。<br>
<span class="jp-trans">决定每天去健身房。</span>
