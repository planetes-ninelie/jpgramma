我们可以把动词「<span class="jp-word" title="する - 做">する</span>」和「<span class="jp-word" title="なる - 成为">なる</span>」和助词「に」一起用，来表达很丰富的意思。动词「<span class="jp-word" title="する - 做">する</span>」跟对象助词的用法我们已经很熟悉了，因为这些动作总是<u>作用于</u>某物。下面我们会看到如果改用助词「に」的话句意会如何改变。至于「<span class="jp-word" title="なる - 成为">なる</span>」，它总是跟助词「に」一起用，因为「变为」不是作用于什么的动作，而是朝向一个目标进行的改变动作。这里唯一的语法点就是与い形容词以及动词合用「<span class="jp-word" title="なる - 成为">なる</span>」。

## 与名词以及な形容词合用「<span class="jp-word" title="なる - 成为">なる</span>」和「<span class="jp-word" title="する - 做">する</span>」 {#part2}

### 词汇学习

1. 彼 【かれ】 - 他；男朋友
2. 日本語 【に・ほん・ご】 - 日语
3. 上手 【じょう・ず】 （な形容词） - 有技巧的
4. なる （う动词） - 成为
5. 私 【わたし】 - 我；我自己
6. 医者 【い・しゃ】 - 医生
7. 有名 【ゆう・めい】 （な形容词） - 著名
8. 人 【ひと】 - 人
9. ハンバーガー - 汉堡
10. サラダ - 沙拉
11. する （例外） - 做
12. 他 【ほか】 - 其他
13. いい （い形容词） - 好的
14. 物 【もの】 - 东西
15. たくさん - （数量）很多
16. ある （う动词） - 存在（无生命）
17. やはり／やっぱり - 如我所料，果然
18. これ - 这

前面说了，跟名词、な形容词合用「<span class="jp-word" title="なる - 成为">なる</span>」并没有什么新东西，应该跟你想象的用法一样。

1. <span class="jp-word" title="かれ - 他">彼</span> の <span class="jp-word" title="にほんご - 日语">日本語</span> が <span class="jp-word" title="じょうず - 有技巧的">上手</span> <em>に<span class="jp-word" title="なる - 成为">なった</span></em> 。<br>
<span class="jp-trans">他的日语变得越来越厉害了。</span>
2. <span class="jp-word" title="わたし - 我；我自己">私</span> は <span class="jp-word" title="いしゃ - 医生">医者</span> <em>に<span class="jp-word" title="なる - 成为">なった</span></em> 。<br>
<span class="jp-trans">我成了一个医生。</span>
3. <span class="jp-word" title="わたし - 我；我自己">私</span> は <span class="jp-word" title="ゆうめい - 著名">有名</span> な <span class="jp-word" title="ひと - 人">人</span> <em>に<span class="jp-word" title="なる - 成为">なる</span></em> 。<br>
<span class="jp-trans">我将会成为一个名人。</span>

对于形容词来说，跟「<span class="jp-word" title="する - 做">する</span>」和助词「に」一起用只是对副词一课的回顾。但对于名词来说，跟动词「<span class="jp-word" title="する - 做">する</span>」 以及助词「に」一起用的话意味着你将要向着某物做某事。这时候「<span class="jp-word" title="する - 做">する</span>」的意思就变成了「决定要 [X]」。这在点菜之类的场合非常常用。

1. <span class="jp-word" title="わたし - 我；我自己">私</span> は、 <span class="jp-word" title="ハンバーガー - 汉堡">ハンバーガー</span> と <span class="jp-word" title="サラダ - 沙拉">サラダ</span> <em>に<span class="jp-word" title="する - 做">します</span></em> 。<br>
<span class="jp-trans">我打算要汉堡和沙拉（字面：我要朝着汉堡和沙拉做）。</span>
2. <span class="jp-word" title="ほか - 其他">他</span> に <span class="jp-word" title="いい - 好的">いい</span> <span class="jp-word" title="もの - 东西">もの</span> が <span class="jp-word" title="たくさん - 很多">たくさん</span> <span class="jp-word" title="ある - 存在（无生命）">ある</span> けど、 <span class="jp-word" title="やっぱり - 如我所料，果然">やっぱり</span> <em><span class="jp-word" title="これ - 这">これ</span>に</em> <span class="jp-word" title="する - 做">する</span> 。<br>
<span class="jp-trans">好东西还有很多，但对我来说，还是打算要这个。</span>

如果你觉得这种表达方式有点奇怪的话，可以想想说英语 "I'll go with the hamburger." 的时候，不会是说要跟着汉堡去哪里吧？

## 与い形容词合用「<span class="jp-word" title="なる - 成为">なる</span>」 {#part3}

### 词汇学习

1. 去年 【きょ・ねん】 - 去年
2. ～から（助词） - 从…
3. 背 【せ】 - 高度
4. 高い 【たか・い】 （い形容词） - 高
5. なる （う动词） - 成为
6. 運動 【うん・どう】 - 运动
7. する （例外） - 做
8. ～から（助词）- ~所以
9. 強い 【つよ・い】 （い形容词） - 强
10. 勉強 【べん・きょう】 - 学习
11. たくさん - （数量）很多
12. 頭 【あたま】 - 头
13. いい （い形容词） - 好的

因为「に」是一个用于名词以及な形容词的目标助词，为了表示某物正往某个い形容词的方向变化，我们需要别的手段。既然「变为」表示的是一种状态的改变，那么我们用副词来表示这个过程是说得过去的。实际上你可能有注意到，我们在前面已经有介绍过这种用法了。

1. <span class="jp-word" title="きょねん - 去年">去年</span> から <span class="jp-word" title="せ - 高度">背</span> が <em><span class="jp-word" title="たかい - tall, high">高く</span><span class="jp-word" title="なる - 成为">なった</span></em> ね。<br>
<span class="jp-trans">你比去年高多了哈？</span>
2. <span class="jp-word" title="うんどう - 运动">運動</span> <span class="jp-word" title="する - 做">している</span> から、 <em><span class="jp-word" title="つよい - 强">強く</span><span class="jp-word" title="なる - 成为">なる</span></em> 。<br>
<span class="jp-trans">因为锻炼，我会变得更强壮。</span>
3. <span class="jp-word" title="べんきょう - 学习">勉強</span> を <span class="jp-word" title="たくさん - 很多">たくさん</span> <span class="jp-word" title="する - 做">した</span> から、 <span class="jp-word" title="あたま - 头">頭</span> が <em><span class="jp-word" title="いい - 好的">よく</span><span class="jp-word" title="なる - 成为">なった</span></em> 。<br>
<span class="jp-trans">因为我学了很多，我变得更聪明了（字面：头脑变更好了）。</span>

## 合用动词与「<span class="jp-word" title="なる - 成为">なる</span>」和「<span class="jp-word" title="する - 做">する</span>」 {#part4}

### 词汇学习

1. 海外 【かい・がい】 - 海外
2. 行く 【い・く】 （う动词） - 走
3. こと - 事情
4. なる （う动词） - 成为
5. 毎日 【まい・にち】 - 每天
6. 肉 【にく】 - 肉
7. 食べる 【た・べる】 （る动词） - 吃
8. する （例外） - 做
9. 日本 【に・ほん】 - 日本
10. 来る 【く・る】 （例外）- 来
11. 寿司 【すし】 - 寿司
12. 一年間 【いち・ねん・かん】 - 一年间
13. 練習 【れん・しゅう】 - 练习
14. ピアノ - 钢琴
15. 弾く 【ひ・く】 （う动词） - 弹
16. 地下 【ち・か】 - 地下
17. 入る 【はい・る】 （う动词） - 进入
18. 富士山 【ふ・じ・さん】 - 富士山
19. 見える 【み・える】 （る动词） - 变得可见

你也许会想知道如何将「<span class="jp-word" title="なる - 成为">なる</span>」和「<span class="jp-word" title="する - 做">する</span>」与动词合用，总不能直接用一个动词修饰另一个动词吧。解决方法很简单，就是加上一个抽象名词比如<span class="jp-word" title="こと - 事情">こと</span> （<span class="jp-word" title="こと - 事情">事</span>）或者<span class="jp-word" title="よう - 样子">よう</span> （<span class="jp-word" title="よう - 样子">様</span>）。这类名词并不指代具体的对象，用在这里，可以让我们将动词按照名词的用法来用。下面是具体使用「<span class="jp-word" title="する - 做">する</span>」和「<span class="jp-word" title="なる - 成为">なる</span>」的例子。

1. <span class="jp-word" title="かいがい - 海外">海外</span> に <span class="jp-word" title="いく - 走">行く</span> <em><span class="jp-word" title="こと - 事情">こと</span></em> に <span class="jp-word" title="なる - 成为">なった</span> 。<br>
<span class="jp-trans">我去国外的事情已经决定了（字面：去海外成为了那件事）。</span>
2. <span class="jp-word" title="まいにち - 每天">毎日</span> 、 <span class="jp-word" title="にく - 肉">肉</span> を <span class="jp-word" title="たべる - 吃">食べる</span> <em><span class="jp-word" title="よう - 样子">よう</span></em> に <span class="jp-word" title="なる - 成为">なった</span> 。<br>
<span class="jp-trans">我成了每天吃肉的样子（字面：每天吃肉成了样子）。</span>
3. <span class="jp-word" title="かいがい - 海外">海外</span> に <span class="jp-word" title="いく - 走">行く</span> <em><span class="jp-word" title="こと - 事情">こと</span></em> に <span class="jp-word" title="する - 做">した</span> 。<br>
<span class="jp-trans">我决定了要去国外（字面：我朝着去国外这件事做了）。</span>
4. <span class="jp-word" title="まいにち - 每天">毎日</span> 、 <span class="jp-word" title="にく - 肉">肉</span> を <span class="jp-word" title="たべる - 吃">食べる</span> <em><span class="jp-word" title="よう - 样子">よう</span></em> に <span class="jp-word" title="する - 做">する</span> 。<br>
<span class="jp-trans">我会试着每天吃肉（字面：我会朝着每天吃肉的样子做）。</span>

总结一下，你可以这样用「<span class="jp-word" title="なる - 成为">なる</span>」或「<span class="jp-word" title="する - 做">する</span>」来修饰动词：先把它变为名词从句，然后再将它作为普通名词对待。很聪明吧？希望上面的字面翻译能让你明白句子为何表达了那个意思。比如在第四个例句里，「～<span class="jp-word" title="よう - 样子">よう</span>に<span class="jp-word" title="する - 做">する</span>」理论上应该是「向着……努力」，但在日语里面的意思其实只是用某种方式达到某个目标的意思。

因为可能形动词描述的是可能性而非具体动作（记住，所以不能跟助词「を」一起用），所以它常常跟「～<span class="jp-word" title="よう - 样子">よう</span>に<span class="jp-word" title="なる - 成为">なる</span>」一起用，来描述变为某种状态的可能性。趁这个机会我们来复习一下。

1. <span class="jp-word" title="にほん - 日本">日本</span> に <span class="jp-word" title="くる- 来">来て</span> 、 <span class="jp-word" title="すし - 寿司">寿司</span> が <em><span class="jp-word" title="たべる - 吃">食べられる</span><span class="jp-word" title="よう - 样子">よう</span>に<span class="jp-word" title="なる - 成为">なった</span></em> 。<br>
<span class="jp-trans">来日本以后，我变得能吃寿司了。</span>
2. <span class="jp-word" title="いちねんかん - 一年间">一年間</span> <span class="jp-word" title="れんしゅう - 练习">練習</span> <span class="jp-word" title="する - 做">した</span> から、 <span class="jp-word" title="ピアノ - 钢琴">ピアノ</span> が <em><span class="jp-word" title="ひく - 弹">弾ける</span><span class="jp-word" title="よう - 样子">よう</span>に<span class="jp-word" title="なる - 成为">なった</span></em> 。<br>
<span class="jp-trans">因为练习了一年，我会弹钢琴了。</span>
3. <span class="jp-word" title="ちか - 地下">地下</span> に <span class="jp-word" title="はいる - 进入">入って</span> 、 <em><span class="jp-word" title="ふじさん - 富士山">富士山</span>が<span class="jp-word" title="みえる - 变得可见">見えなく</span><span class="jp-word" title="なる - 成为">なった</span></em> 。<br>
<span class="jp-trans">进入地下以后，富士山便看不到了。</span>
