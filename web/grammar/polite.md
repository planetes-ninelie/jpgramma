## 在日本不要粗鲁 {#part1}

### 词汇学习

1. 丁寧語 【てい・ねい・ご】 - 丁宁语
2. 尊敬語 【そん・けい・ご】 - 尊敬语
3. 謙譲語 【けん・じょう・ご】 - 谦逊语
4. はい - 是（礼貌的）
5. いいえ - 不是（礼貌的）

到目前为止我们所学的都是五岁小孩水平的日语。很不幸，成年人在与特定人群对话时应该使用更为礼貌的日语（称为<span class="jp-word" title="ていねいご - 丁宁语">丁寧語</span>）。可能要用<span class="jp-word" title="ていねいご - 丁宁语">丁寧語</span>对话的人包括社会地位高于你，以及与你不熟悉的人。何时使用哪种语言基本靠「感觉」，不过对一个人最好始终用同一种。

后面（也许很后面）我们会学习一种更为礼貌的语言，叫做尊敬语（<span class="jp-word" title="そんけいご - 尊敬语">尊敬語</span>）和谦逊语（<span class="jp-word" title="けんじょうご - 谦逊语">謙譲語</span>）。它可能比你想象的更有用，因为售货员、收银员等人都会用这种语言。但现在让我们还是先学会<span class="jp-word" title="ていねいご - 丁宁语">丁寧語</span>，这是<span class="jp-word" title="そんけいご - 尊敬语">尊敬語</span>和<span class="jp-word" title="けんじょうご - 谦逊语">謙譲語</span>的基础。

幸运的是，从口语变为丁宁语并不难。有些词汇可能要变化（比如「是」和「不是」要改为<span class="jp-word" title="はい- 是的">はい</span>」和「<span class="jp-word" title="いいえ - 不">いいえ</span>」），那些口语化的句尾也不能使用了（后面我们会学到如何结束句子）。两种语言最主要的区别就在句尾。在别人说完一句话之前你甚至无法分别他是否在用丁宁语。

## 动词词根 {#part2}

### 词汇学习

1. 食べる 【た・べる】 （る动词） - 吃
2. 泳ぐ 【およ・ぐ】 （う动词） - 游泳
3. する （例外） - 做
4. 来る 【く・る】 （例外）- 来
5. 怒る 【おこ・る】 （う动词） - 生气
6. 鉄拳 【てっ・けん】 - 拳头
7. 休み 【やす・み】 - 休息，假期
8. 飲む 【の・む】 （う动词） - 喝
9. 明日 【あした】 - 明天
10. 映画 【えい・が】 - 电影
11. 見る 【み・る】 （る动词） - 看
12. 行く 【い・く】 （う动词） - 走
13. 友達 【とも・だち】- 朋友
14. 遊ぶ 【あそ・ぶ】 （う动词） - 玩
15. 楽しむ 【たの・しむ】 （う动词） - 享乐
16. 出す 【だ・す】 （う动词） - 拿出来
17. 走る 【はし・る】 （う动词） - 跑
18. 走り出す 【はし・り・だ・す】 （う动词） - 开始跑
19. 着る 【き・る】 （る动词） - 穿
20. 替える 【か・える】 （る动词） - 更替
21. 着替える 【き・が・える】 （る动词） - 换衣服
22. 付ける 【つ・ける】 （る动词） - 附加
23. 加える 【くわ・える】 （る动词） - 增加
24. 付け加える 【つ・け・くわ・える】 （る动词） - 把一个东西加到另一个东西上
25. 言う 【い・う】 （う动词）- 说
26. 言い出す 【い・い・だ・す】 （う动词） - 开始说话

为了把所有う动词和る动词活用成丁宁语，我们首先要学习动词的词根。通常日语课本里把它称为<i>ます词根</i>，不过我们就叫它<i>词根</i>好了，因为除了ます形它还用在其他很多地方。词根是个好东西，因为它很容易被还原出来，但却在很多语法里面能用上。

<div class="sumbox">
<div class="sumbox-title">从动词还原出词根的规则</div>

- <b>对る动词：</b> 把「る」去掉<br>
<span class="jp-trans">例： <span class="jp-word" title="たべる - 吃">食べ<em>る</em></span> → <span class="jp-word" title="たべる - 吃">食べ</span></span>
- <b>对う动词：</b> 把最后的 / u / 段改为同行 / i / 段假名。<br>
<span class="jp-trans">例： <span class="jp-word" title="およぐ - 游泳">泳<em>ぐ</em></span> → <span class="jp-word" title="およぐ - 游泳">泳<em>ぎ</em></span></span>
- <b>例外：</b>
  1. 「 <span class="jp-word" title="する - 做">する</span> 」的词根是「し」
  2. 「 <span class="jp-word" title="くる- 来">くる</span> 」的词根是「き」
</div>

词根单独使用的时候，可以以一种非常特别而有限的方式（a very specialized and limited way）将动词转变为名词。助词「の」使得我们可以像名词那样使用动词，而词根却可以把动词变为名词。实际上，有时候词根甚至比动词本身还用得多。比如「<span class="jp-word" title="いかる - 生气">怒る</span>」（いかる）的词根就比它本身更常用，电影 "Fists of Fury"（《精武门》）就被翻译成「<span class="jp-word" title="いかり - fury">怒り</span>の<span class="jp-word" title="てっけん - 拳头">鉄拳</span>」而不是「<span class="jp-word" title="おこる - 生气">怒る</span><span class="jp-word" title="てっけん - 拳头">鉄拳</span>」。实际上，「<span class="jp-word" title="おこる - 生气">怒る</span>」更多的是被念成「<span class="jp-word" title="おこる - 生气">おこる</span>」，但汉字和含义都相同！还有一些名词（例如「<span class="jp-word" title="やすみ - 休息，假期">休み</span>」）本身也是动词词根，使用起来跟普通名词无二。但是，总的来说，我们不能把动词拿来就变成名词，比如下面的句子就是错的。

- 飲みを<span class="jp-word" title="する - 做">する</span> 。<br>
<span class="jp-trans">（这句话本身是有意义的，但没人这么说）</span>

然而，一种对大多数动词词根来说都适用的语法是把它作为动作动词（基本上就只有「行く」和「来る」）的目标，意思是「去或者来做（某种动作）」，下面是例子：

1. <span class="jp-word" title="あした - 明天">明日</span> 、 <span class="jp-word" title="えいが - 电影">映画</span> を <em><span class="jp-word" title="みる - 看">見</span>に</em> <span class="jp-word" title="いく - 走">行く</span> 。- 明天，去看电影。

「<span class="jp-word" title="みる - 看">見</span>に」是「<span class="jp-word" title="みる - 看">見る</span>」的词根（見）和目标助词「に」的组合。

方向动词「へ」让人觉得你是要去或来做某事，而助词「に」蕴含了你是冲着某事去或者来的意思。

1. <span class="jp-word" title="きのう - 昨天">昨日</span> 、 <span class="jp-word" title="ともだち- 朋友">友達</span> が <em><span class="jp-word" title="あそぶ - 玩">遊び</span>へ</em> <span class="jp-word" title="くる- 来">きた</span> 。<br>
<span class="jp-trans">昨天，朋友来玩活动了（有点奇怪）。</span>
2. <span class="jp-word" title="きのう - 昨天">昨日</span> 、 <span class="jp-word" title="ともだち- 朋友">友達</span> が <em><span class="jp-word" title="あそぶ - 玩">遊び</span>に</em> <span class="jp-word" title="くる- 来">きた</span> 。<br>
<span class="jp-trans">昨天，朋友来玩了。</span>

短语「<span class="jp-word" title="たのしむ - 享乐">楽しみ</span>に<span class="jp-word" title="する - 做">する</span>」（意思是「期盼着」）也是类似的语法构成，不过是个特例，应该作为一个固定搭配来看待。

还可以把其他动词加到词根上来创造出新的动词。比如把「<span class="jp-word" title="だす - 拿出来">出す</span>」加到「<span class="jp-word" title="はしる - 跑">走る</span>」的词根，也就是「<span class="jp-word" title="はしる - 跑">走り</span>」后面的时候就得到了「<span class="jp-word" title="はしりだす - 开始跑">走り出す</span>」，意思是「开始跑」。其他例子包括「<span class="jp-word" title="きりかえる - 更替">切り替える</span>」，意思是「换成别的东西」，以及「<span class="jp-word" title="つけくわえる - 把一个东西加到另一个东西上">付け加える</span>」，意思是「把一个东西加到另一个东西上」。你可以看到两个不同意思的动词组合成了一个新的动词，例如「<span class="jp-word" title="いいだす - 开始说话">言い出す</span>」——意思是「开始说」——组合了「说」和「拿出来」。这种语法没有一般性的规则，你只能把它们当作单独的词来记。

正式的书面语例如报纸文章会把词根当作活用动词来用。后面说到[正式表达语](/grammar/formal/#part4)的时候我们再说这个话题。

## 用「～ます」来将动词变为丁宁语 {#part3}

### 词汇学习

1. 明日 【あした】 - 明天
2. 大学 【だい・がく】 - 大学
3. 行く 【い・く】 （う动词） - 走
4. 先週 【せん・しゅう】 - 上周
5. 会う 【あ・う】 （う动词） - 会见
6. 晩ご飯 【ばん・ご・はん】 - 晚饭
7. 食べる 【た・べる】 （る动词） - 吃
8. 面白い 【おも・しろ・い】（い形容词） - 有趣
9. 映画 【えい・が】 - 电影
10. 見る 【み・る】 （る动词） - 看

说了半天动词词根就是为了介绍如何把动词活用为丁宁语，也就是所谓的ます形！ます形一定要用在完整句子的末尾，而不能用在修饰用的状语从句里面。后面学了组合句之后你会发现，其中每个子句也可以以ます形结尾。

### 示例

1. <span class="jp-word" title="あした - 明天">明日</span> 、 <span class="jp-word" title="だいがく - 大学">大学</span> に <em><span class="jp-word" title="いく - 走">行きます</span></em> 。<br>
<span class="jp-trans">明天，去大学。</span>
2. <span class="jp-word" title="せんしゅう - 上周">先週</span> 、ボブに <em><span class="jp-word" title="あう - 会见">会いました</span></em> よ。<br>
<span class="jp-trans">知道不，上周见了 Bob。</span>
3. <span class="jp-word" title="ばんごはん - 晚饭">晩ご飯</span> を <em><span class="jp-word" title="たべる - 吃">食べませんでした</span></em> ね。<br>
<span class="jp-trans">没吃晚饭，是吧？</span>
4. <span class="jp-word" title="おもしろい - 有趣">面白くない</span> <span class="jp-word" title="えいが - 电影">映画</span> は <em><span class="jp-word" title="みる - 看">見ません</span></em> 。<br>
<span class="jp-trans">对于没意思的电影，不看。</span>

### 词汇学习

1. かわいい （い形容词） - 可爱
2. 静か 【しず・か】 （な形容词）- 安静
3. 子犬 【こ・いぬ】 - 小狗
4. とても - 非常
5. 好き 【す・き】 （な形容词） - 惹人喜欢的
6. 昨日【きのう】 - 昨天
7. 時間 【じ・かん】 - 时间
8. ある （う动词） - 存在（无生命）
9. その - 那个（「それの」的缩写）
10. 部屋 【へ・や】 - 房间
11. 先週 【せん・しゅう】 - 上周
12. 見る 【み・る】 （る动词） - 看
13. 映画 【えい・が】 - 电影
14. 面白い 【おも・しろ・い】（い形容词） - 有趣

### 示例

1. <span class="jp-word" title="こいぬ - 小狗">子犬</span> は <span class="jp-word" title="とても - 非常">とても</span> <em><span class="jp-word" title="すき- 喜欢">好き</span>です</em> 。<br>
<span class="jp-trans">关于小狗，非常喜欢。（最自然的翻译是某人很喜欢小狗，不过没有足够的上下文，也许本意是小狗很喜欢什么东西。）</span>

<span class="jp-trans">昨天没有时间。</span>
3. <span class="jp-word" title="その - 那个">その</span> <span class="jp-word" title="へや - 房间">部屋</span> は <span class="jp-word" title="あまり - 不是很">あまり</span> <em><span class="jp-word" title="しずか- 安静">静かじゃない</span>です</em> 。<br>
<span class="jp-trans">房间并不是很安静。</span>
4. <span class="jp-word" title="せんしゅう - 上周">先週</span> に <span class="jp-word" title="みる - 看">見た</span> <span class="jp-word" title="えいが - 电影">映画</span> は、 <span class="jp-word" title="とても - 非常">とても</span> <em><span class="jp-word" title="おもしろい - 有趣">面白かった</span>です</em> 。<br>
<span class="jp-trans">上周看过的电影很有意思。</span>

### ※ 现实世界 {#rc}

我在很多场合都听过一种说法，说我介绍否定式丁宁语不是「官方」的正确活用。更加「正确」的活用形应该是把「ないです」部分改为「<span class="jp-word" title="ある - 存在（无生命）">ありません</span>」，理由是动词「<span class="jp-word" title="ある - 存在（无生命）">ある</span>」的丁宁否定式不是「ないです」而是「<span class="jp-word" title="ある - 存在（无生命）">ありません</span>」。所以呢，「<span class="jp-word" title="かわいい - 可爱">かわいくない</span>」应该写作「<span class="jp-word" title="かわいい - 可爱">かわいくありません</span>」，「<span class="jp-word" title="しずか- 安静">静かじゃない</span>」则应该是「<span class="jp-word" title="しずか- 安静">静かじゃありません</span>」。

实际上，这种所谓的「官方」用法听起来太过呆板和正式。在日常对话中，大家基本用的都是我教的用法。虽然你在用丁宁语写作时应该用更加正式的活用形，日常说话你可几乎用不到。对此，我的建议是两种活用形都要熟悉。

## 「です」跟「だ」不！一！样！ {#part5}

### 词汇学习

1. そう - 如此
2. 思う 【おも・う】 （う动词） - 思考，认为
3. はい - 是（丁宁语）
4. 答える 【こた・える】 （る动词） - 回答

你们中很多已经上过日语课的人也许已经被告知「です」是「だ」的丁宁语版本。不过，我在这里要指出二者之间几个关键的区别来说明它们是两个完全不同的概念。因为有些语法还没有讲过，所以我现在无法完整解释为何二者从根本上是不一样的，所以我在这里先针对那些被误导认为「だ」是「です」的口语版本的人做一些解释。对于其他人，你可以跳过这部分内容了。

我想你们大多数人都学过「<span class="jp-word" title="そう - 没错，是这样">そう</span>」了。现在，我们有四种将「<span class="jp-word" title="そう - 没错，是这样">そう</span>」转换为状态表达，来形成一个完整的意思为「没错」的句子的办法。

### 说「没错」的不同方式

1. <span class="jp-word" title="そう - 没错，是这样">そう</span> 。
2. <span class="jp-word" title="そう - 没错，是这样">そう</span> だ。
3. <span class="jp-word" title="そう - 没错，是这样">そう</span> です。
4. <span class="jp-word" title="そう - 没错，是这样">そう</span> でございます。

第一个「<span class="jp-word" title="そう - 没错，是这样">そう</span>」蕴含了状态表述，而「<span class="jp-word" title="そう - 没错，是这样">そう</span>だ」是陈述形。如前所述，「<span class="jp-word" title="そう - 没错，是这样">そう</span>」这种不是很肯定的口吻通常为女性所用，而更为坚定的「<span class="jp-word" title="そう - 没错，是这样">そう</span>だ」则通常为男性所用。

「<span class="jp-word" title="そう - 没错，是这样">そう</span>です」是「<span class="jp-word" title="そう - 没错，是这样">そう</span>」通过在名词后面粘黏「です」转为了丁宁语版本。「<span class="jp-word" title="そう - 没错，是这样">そう</span>です」<b>不是</b>把「<span class="jp-word" title="そう - 没错，是这样">そう</span>だ」里的「だ」换成「です」而转成的丁宁语版本，下面我来解释原因。

也许我们想把这个句子改成问句，「没错吧？」。这又有好多种说法，下面是最常用的（这个语法后面会介绍）

### 问「没错吧？」的不同方式

1. <span class="jp-word" title="そう - 没错，是这样">そう</span> ？
2. <span class="jp-word" title="そう - 没错，是这样">そう</span> か？
3. <span class="jp-word" title="そう - 没错，是这样">そう</span> ですか？

如前所述，「だ」是用来表示一个人所认定的事实的。因此「<span class="jp-word" title="そう - 没错，是这样">そう</span>だか？」就有问题了，因为你不可能即陈述一个事实又把它当作问题来问别人。但「<span class="jp-word" title="そう - 没错，是这样">そう</span>ですか」却是正确的，这里就看出「です」和「だ」之间的本质不同了。「<span class="jp-word" title="そう - 没错，是这样">そう</span>です」是用来表示尊敬和谦逊，它不表达确定的意思，而仅仅是「<span class="jp-word" title="そう - 没错，是这样">そう</span>」的丁宁语版本。

除了上述细微差别之外，「だ」和「です」的另一个关键差异是「だ」在各种不同的语法里面可以用来标识关系从句，而「です」却只能用在句尾，用以礼貌地表达状态。比如下面俩个句子（其中的语法后面会介绍）

- <span class="jp-word" title="そう - 没错，是这样">そう</span> <em>だ</em> と <span class="jp-word" title="おもう - 思考，认为">思います</span><br>
<span class="jp-trans">我想是那样的。</span>
- <span class="jp-word" title="そう - 没错，是这样">そう</span> <em>です</em> と <span class="jp-word" title="おもう - 思考，认为">思います</span><br>
<span class="jp-trans">（语法错误）</span>

「<span class="jp-word" title="そう - 没错，是这样">そう</span><em>だ</em>と<span class="jp-word" title="おもう - 思考，认为">思います</span>」是正确的而「<span class="jp-word" title="そう - 没错，是这样">そう</span><em>です</em>と<span class="jp-word" title="おもう - 思考，认为">思います</span>」 是错的，因为「です」只能放在句尾。「です」出现在关系从句的唯一场景是从句为引用的原话，比如：

- 「はい、 <span class="jp-word" title="そう - 没错，是这样">そう</span> です」と <span class="jp-word" title="こたえる - 回答">答えた</span> 。

综上，认为「です」和「だ」只是丁宁语和口语的对应的话会让你说错话。你最好把二者当成两个完全不同的概念（事实也的确如此）。
