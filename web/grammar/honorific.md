日语大致可以分为三种礼貌层级：口语、丁宁语和尊敬语、谦逊语。到目前为止我们已经学习了用「～です」和「～ます」组合丁宁语。接下来，我们要介绍更为礼貌的尊敬语和谦逊语。这种用语你经常能在商业场合例如饭店、收银台听到。现在你先记住，说话的人总是把自己摆在最卑微的地位，所以他自己的举动都要用谦逊语，而在他眼中其他人做的动作都必须用尊敬语。

### 尊敬语动词的特殊活用规则

### 尊敬语示例

我们已经看到「<span class="jp-word" title="ください - 请给、请做">ください</span>」是「<span class="jp-word" title="くださる（尊敬） - 给">下さる</span>」的特殊活用形，而后者则是「<span class="jp-word" title="くれる - 给">くれる</span>」的尊敬语版本。现在来看一些实例，它们都是直接说给对方的，因此都是用了尊敬语。

1. アリスさん、 <span class="jp-word" title="もう - 已经">もう</span> <em><span class="jp-word" title="めしあがる（尊敬） - 吃">召し上がりました</span></em> か。<br>
<span class="jp-trans">Alice 桑，（你）已经吃了？</span>
2. <span class="jp-word" title="しごと - 工作">仕事</span> で <span class="jp-word" title="なに - 什么">何</span> を <em><span class="jp-word" title="なさる（尊敬） - 做">なさっている</span></em> んですか。<br>
<span class="jp-trans">您上班做什么？</span>
3. <span class="jp-word" title="すいせんじょう - 推荐">推薦状</span> を <span class="jp-word" title="かく - 写">書いて</span> <em><span class="jp-word" title="くださる -（尊敬）给；给予">くださる</span>んです</em> か。<br>
<span class="jp-trans">您可以帮我写一封推荐信吗？</span>
4. <span class="jp-word" title="どちら - 哪里">どちら</span> から <em><span class="jp-word" title="いらっしゃる（尊敬） - 走,，来, 存在（有生命）">いらっしゃいました</span></em> か。<br>
<span class="jp-trans">您从哪里来？</span>
5. <span class="jp-word" title="きょう - 今天">今日</span> は、 <span class="jp-word" title="どちら - 哪里">どちら</span> へ <em><span class="jp-word" title="いらっしゃる（尊敬） - 走,，来, 存在（有生命）">いらっしゃいます</span></em> か。<br>
<span class="jp-trans">您今天去哪儿？</span>

### 谦逊形示例

下例都是说话者自己做的事，所以都用了谦逊形。

1. <span class="jp-word" title="わたし - 我；我自己">私</span> はキムと <em><span class="jp-word" title="もうす（谦逊）- 说">申します</span></em> 。<br>
<span class="jp-trans">至于我，（人们）说 Kim。（我叫 Kim）</span>
2. <span class="jp-word" title="わたし - 我；我自己">私</span> が <span class="jp-word" title="かく - 写">書いた</span> <span class="jp-word" title="レポート - 报告">レポート</span> を <span class="jp-word" title="みる - 看">見て</span> <em><span class="jp-word" title="いただく（谦逊） - 收到">いただけます</span></em> か。<br>
<span class="jp-trans">我可以收到帮我检查报告的荣幸吗？</span>
3. <span class="jp-word" title="しつれい - 失礼">失礼</span> <em><span class="jp-word" title="いたす（谦逊） - 做">致します</span></em> 。<br>
<span class="jp-trans">失陪。</span>

除了这些 固定搭配，还有一些单词也有对应的更加礼貌的形式，这里面最重要的也许就是「<span class="jp-word" title="ある - 存在（无生命）">ある</span>」的礼貌版本：「<span class="jp-word" title="ござる （正式） - 存在（无生命）">ござる</span>」。这个动词可以用在有生命和无生命的对象上。它既不是尊敬语，也不是谦逊语，但礼貌程度在「<span class="jp-word" title="ある - 存在（无生命）">ある</span>」之上。不过除非你想模仿武士口吻，「<span class="jp-word" title="ござる （正式） - 存在（无生命）">ござる</span>」总是以丁宁形出现：「<span class="jp-word" title="ござる （正式） - 存在（无生命）">ございます</span>」.

还有就是「です」的礼貌版本是「で<span class="jp-word" title="ござる （正式） - 存在（无生命）">ございます</span>」这其实就是「で<span class="jp-word" title="ござる （正式） - 存在（无生命）">ござる</span>」的ます形活用，它从「で<span class="jp-word" title="ある - 存在（无生命）">ある</span>」而来，字面意思是「以…的形式存在」（很后面会讲到）。

### 示例

1. <span class="jp-word" title="こちら - 这边，这里">こちら</span> は、 <span class="jp-word" title="わたし - 我；我自己">私</span> の <span class="jp-word" title="へや - 房间">部屋</span> <em>です</em> 。<br>
<span class="jp-trans">这里是我的房间。</span>
2. <span class="jp-word" title="こちら - 这边，这里">こちら</span> は、 <span class="jp-word" title="わたし - 我；我自己">私</span> の <span class="jp-word" title="へや - 房间">部屋</span> <em>で<span class="jp-word" title="ござる （正式） - 存在（无生命）">ございます</span></em> 。<br>
<span class="jp-trans">我的房间在这边。</span>

1. <span class="jp-word" title="おてあらい - 洗手间">お手洗い</span> は <span class="jp-word" title="この - 这">この</span> <span class="jp-word" title="ビル - 楼">ビル</span> の <span class="jp-word" title="にかい - 二楼">二階</span> に <em><span class="jp-word" title="ある - 存在（无生命）">あります</span></em> 。<br>
<span class="jp-trans">盥洗室在这座楼的二层。</span>
2. <span class="jp-word" title="おてあらい - 洗手间">お手洗い</span> は <span class="jp-word" title="この - 这">この</span> <span class="jp-word" title="ビル - 楼">ビル</span> の <span class="jp-word" title="にかい - 二楼">二階</span> に <em><span class="jp-word" title="ござる （正式） - 存在（无生命）">ございます</span></em> 。<br>
<span class="jp-trans">盥洗室在这座楼的二层。</span>

其他的例子包括「<span class="jp-word" title="いい - 好的">いい</span>」，它更正式的表达方法是「<span class="jp-word" title="よろしい - 好的">よろしい</span>」。另外「对不起」六种说法（不包括「<span class="jp-word" title="わるい - 坏">悪い</span>ね」或者稍微做点变化的「<span class="jp-word" title="すいません - 对不起">すいません</span>」）。

越来越礼貌的道歉：

1. <span class="jp-word" title="ごめん - 对不起">ごめん</span> 。
2. <span class="jp-word" title="ごめんなさい - 对不起">ごめんなさい</span> 。
3. <span class="jp-word" title="すみません - 对不起">すみません</span> 。
4. <span class="jp-word" title="もうしわけ - 借口">申し訳</span> <span class="jp-word" title="ある - 存在（无生命）">ありません</span> 。（ <span class="jp-word" title="もうしわけ - 借口">申し訳</span> 是 <span class="jp-word" title="いいわけ - 借口">言い訳</span> 的谦逊形）
5. <span class="jp-word" title="おそれいる - 抱歉">恐れ入ります</span> 。
6. <span class="jp-word" title="きょうしゅく -羞耻">恐縮</span> です。

此外，名字的礼貌后缀是「<span class="jp-word" title="さま - 礼貌的后缀">様</span>」，它比「さん」更进一步。这个后缀在口语里面不常用到，即使你是在说尊敬语或谦逊语。不过，写信的时候一般会用到它，即使对方是熟人也一样。还有服务行业例如收银员或服务员通常会称呼客户为「<span class="jp-word" title="おきゃくさま - 顾客">お客様</span>」。当然了，皇族和神灵一定是会用「<span class="jp-word" title="さま - 礼貌的后缀">様</span>」的，例如「<span class="jp-word" title="かみさま - 神">神様</span>」。

对于其他没有固定搭配的动词，也有活用为尊敬语和谦逊语的规则。它们通常都会先加上一个礼貌的前缀「<span class="jp-word" title="ご - 尊敬形前缀">御</span>」。日语里面有个规矩，就是在特定的名词（并非所有）前面加上尊敬形前缀「<span class="jp-word" title="ご - 尊敬形前缀">御</span>」来表达敬意。实际上，像「<span class="jp-word" title="おさけ - 酒">お酒</span>」、 「<span class="jp-word" title="おちゃ - 茶">お茶</span>」或「<span class="jp-word" title="おかね - 钱">お金</span>」这样带这个前缀的单词用的太多了，以至于已经取代了原先的名词。通常来说，「<span class="jp-word" title="ご - 尊敬形前缀">御</span>」对于<span class="jp-word" title="おんよみ - 音读">音読み</span> 单词（例如ご<span class="jp-word" title="いけん - 意见">意見</span>、<span class="jp-word" title="ごはん - 米，饭">ご飯</span>）会写作平假名「ご」，而对于<span class="jp-word" title="くんよみ - 训读">訓読み</span>单词（例如<span class="jp-word" title="おかね - 钱">お金</span>、 お<span class="jp-word" title="しごと - 工作">仕事</span>）会写作「お」。实际上，也许你都没注意到自己已经在用这个前缀了，例如「<span class="jp-word" title="おこのみやき - 日式煎菜饼">お好み焼き</span>」或「<span class="jp-word" title="おみやげ - 纪念品">お土産</span>」。这个规则也有例外，如「お<span class="jp-word" title="へんじ - 答复">返事</span>」。幸运的是，因为「<span class="jp-word" title="ご - 尊敬形前缀">御</span>」极少写作汉字，你基本用不着区别这些。

### 尊敬形

那些<i>不存在上述固定搭配的</i>动词的尊敬形可以按照下面两种方法之一来活用。

### 尊敬形活用法一：お + 词根 + に + <span class="jp-word" title="なる - 成为">なる</span>

这种其实好理解，你可以想象为一个人变成了动作的那种尊敬状态。接下来你就可以按照う动词 「<span class="jp-word" title="なる - 成为">なる</span>」的活用规则来活用它了。说实在的，人们很少这么造句。

- <span class="jp-word" title="せんせい - 老师">先生</span> は <em>お<span class="jp-word" title="みえる - 变得可见">見え</span>に<span class="jp-word" title="なる - 成为">なります</span></em> か。<br>
<span class="jp-trans">你见着老师了吗？</span>

### 尊敬形活用法二：お + 词根 + です

1. <span class="jp-word" title="もう - 已经">もう</span> <em>お<span class="jp-word" title="かえる - 回家">帰り</span>です</em> か。<br>
<span class="jp-trans">你已经要回家了？</span>
2. <span class="jp-word" title="てんない - 店内">店内</span> で <em>お<span class="jp-word" title="めしあがる（尊敬） - 吃">召し上がり</span>です</em> か。<br>
<span class="jp-trans">您堂食吗？</span>

服务员倾向于极度礼貌，所以他们经常使用这种<span class="jp-word" title="にじゅうけいご - 二重敬语">二重敬語</span>活用（在上面的例句里就是将尊敬形「<span class="jp-word" title="めしあがる（尊敬） - 吃">召し上がる</span>」再次尊敬活用）。当然，这是否必要、语法是否正确就是另一回事了。

### 将「<span class="jp-word" title="ください - 请给、请做">ください</span>」与敬语合用 {#kudasai}

对于尊敬语动词，你也可以用「<span class="jp-word" title="ください - 请给、请做">下さい</span>」替代「に<span class="jp-word" title="なる - 成为">なる</span>」。当你想用尊敬语动词请别人做什么的时候，这种表达很有用。

另一种经常使用的表达方式

- <span class="jp-word" title="しょうしょう - just a bit">少々</span> <em>お<span class="jp-word" title="まつ - 等待">待ち</span><span class="jp-word" title="ください - 请给、请做">ください</span></em> 。- 请稍候。

类似的，对于「<span class="jp-word" title="ごらん（尊敬） - 看">ご覧</span>に<span class="jp-word" title="なる - 成为">なる</span>」，你可以简单的把「に<span class="jp-word" title="なる - 成为">なる</span>」换成「<span class="jp-word" title="ください - 请给、请做">ください</span>」。

- <span class="jp-word" title="こちら - 这边，这里">こちら</span> に <em><span class="jp-word" title="ごらん（尊敬） - 看">ご覧</span><span class="jp-word" title="ください - 请给、请做">下さい</span></em> 。<br>
<span class="jp-trans">请看这边。</span>

对其他名词也可以这么用，例如坐电车时…

- <span class="jp-word" title="しまる - 被关上">閉まる</span> <span class="jp-word" title="ドア - 登机口">ドア</span> に <em>ご<span class="jp-word" title="ちゅうい - 注意">注意</span><span class="jp-word" title="ください - 请给、请做">下さい</span></em> 。<br>
<span class="jp-trans">关门请注意。</span>

### 谦逊形

谦逊形动词按如下规则生成。

<b>谦逊活用：お + 词根 + <span class="jp-word" title="する - 做">する</span></b>

第一个例句你可能已经听过很多次了，但现在你就知道它的来历了。

1. <span class="jp-word" title="よろしい - 好的">よろしく</span> <em>お<span class="jp-word" title="ねがう - 愿">願いします</span></em> 。<br>
<span class="jp-trans">请多关照。</span>
2. <span class="jp-word" title="せんせい - 老师">先生</span> 、 <em>お<span class="jp-word" title="きく - 问；听">聞き</span><span class="jp-word" title="する - 做">したい</span></em> <span class="jp-word" title="こと - 事情">こと</span> が <span class="jp-word" title="ある - 存在（无生命）">あります</span> が。<br>
<span class="jp-trans">老师，我想问您一件事。</span>
3. <span class="jp-word" title="すみません - 对不起">すみません</span> 、 <em>お<span class="jp-word" title="まつ - 等待">待たせ</span><span class="jp-word" title="する - 做">しました</span></em> 。<br>
<span class="jp-trans">抱歉，让您久等了（使役形）。</span>
4. <span class="jp-word" title="せんえん - 1,000 日元">千円</span> から <em>お<span class="jp-word" title="あずかる - 控制住">預かり</span><span class="jp-word" title="いたす（谦逊） - 做">いたします</span></em> 。<br>
<span class="jp-trans">我们拿着您的 1000 日元。</span>

什么时候会听到类似例四的说法呢？比如你付了 1000 日元，等着找零钱。这里又是<span class="jp-word" title="にじゅうけいご - 二重敬语">二重敬語</span>，因为「<span class="jp-word" title="する - 做">する</span>」已经改为谦逊形「<span class="jp-word" title="いたす（谦逊） - 做">致す</span>」，虽然它已经在お + 词根 + <span class="jp-word" title="する - 做">する</span>谦逊形里了。有些日本人抱怨这根本没意义，另外「から」应该是「を」才对。

## 做出尊敬的请求 {#part5}

4. おっしゃる- 说 （尊敬语）

前面我们已经了解如何使用「～<span class="jp-word" title="ください - 请给、请做">ください</span>」做出礼貌的请求，以及如何在请求中使用尊敬动词。不过这里还有一种用尊敬动词做出请求的说法。这种语法只适用于前面介绍过的那些特殊的活用成「～ます」的动词，这里面包括「<span class="jp-word" title="くださる（尊敬） - 给">下さる</span>」、「<span class="jp-word" title="いらっしゃる（尊敬） - 走,，来, 存在（有生命）">いらっしゃる</span>」、「<span class="jp-word" title="なさる（尊敬） - 做">なさる</span>」和「<span class="jp-word" title="おっしゃる（尊敬）- 说">おっしゃる</span>」。我从没见过有人用「<span class="jp-word" title="おっしゃる（尊敬）- 说">おっしゃる</span>」，但语法上来说这是可能的。

### 示例

在日本，你每次走进一个商场估计都能听到这句话。

- <em><span class="jp-word" title="いらっしゃる（尊敬） - 走,，来, 存在（有生命）">いらっしゃいませ</span></em> 。<br>
<span class="jp-trans">请进！</span>

不过，一个中年寿司大厨也许会用缩略形式。

- <em><span class="jp-word" title="いらっしゃる（尊敬） - 走,，来, 存在（有生命）">いらっしゃい</span></em> ！<br>
<span class="jp-trans">请进！</span>

更多例子…

1. <span class="jp-word" title="ありがとうございます (pol) - 谢谢">ありがとうございました</span> 。 <span class="jp-word" title="また - 再次">また</span> お <span class="jp-word" title="こす - 越过">越し</span> <em><span class="jp-word" title="ください - 请给、请做">くださいませ</span></em> 。<br>
<span class="jp-trans">非常感谢，欢迎再次光临。</span>
2. <span class="jp-word" title="どうぞ - 请">どうぞ</span> 、ご <span class="jp-word" title="ゆっくり - 慢慢的">ゆっくり</span> <em><span class="jp-word" title="なさる（尊敬） - 做">なさいませ</span></em> 。<br>
<span class="jp-trans">请别紧张，放松。</span>
