## 用助词来定义语法功能

我们现在要用上一节学到的知识来连接两个名词，这就要用到所谓的「助词」。日语中助词是指黏着到单词末尾来定义其在句中语法功能的平假名。助词在日语中十分重要，因为改变它可能会改变整个句子的意思。例如，改变一个助词就可以把「吃鱼」变成「鱼吃」。

## 表主题的助词「は」 {#wa}

<iframe class="youtube" src="https://www.youtube.com/embed/uZb5IOXBByQ" frameborder="0" allowfullscreen></iframe>

### 单词学习

1. 学生 【がく・せい】 - 学生
2. うん - 是（口语）
3. 明日 【あした】 - 明天
4. ううん - 不是（口语）
5. 今日 【きょう】 - 今天
6. 試験 【しけん】 - 考试

第一个介绍的是表示主题的助词。它指出了你所谈论的对象，也就是句子的主题。比如一个人说「不是学生」，这在日语里是完全没问题的，但却看不出来说话的人指的是什么。主题语助词可以让我们表明句子在谈论什么，它就是「は」。虽然在五十音图里面这个假名读作 / ha /，但当且仅当作为主题语助词的时候它要读作 / wa /。

### 例一

ボブ：アリス<em>は</em><span class="jp-word" title="がくせい - 学生">学生</span>？<br>
<span class="jp-trans">Bob：（你）Alice 是学生吗？</span>

アリス：<span class="jp-word" title="うん - 是，嗯">うん</span>、<span class="jp-word" title="がくせい - 学生">学生</span>。<br>
<span class="jp-trans">Alice：对，（我）是。</span>

这里，Bob 表明了他的问题是关于 Alice 的。请注意一旦明确话题，Alice 回答的时候就不再需要重复话题了。

### 例二

ボブ：ジョン<em>は</em><span class="jp-word" title="あした - 明天">明日</span>？<br>
<span class="jp-trans">Bob：John 是明天吗？</span>

アリス：ううん、<span class="jp-word" title="あした - 明天">明日</span>じゃない。<br>
<span class="jp-trans">Alice：不，不是明天。</span>

因为缺少上下文，我们无从判断对话的含义，John 当然不可能<b>是</b>明天。只需要补充对话的场景，让明天和 John 关联起来，这个句子就有实际意义了。比如他俩可能在谈论 John 何时考试。

### 例三

アリス：<span class="jp-word" title="きょう - 今天">今日</span>は<span class="jp-word" title="しけん - 考试">試験</span>だ。<br>
<span class="jp-trans">Alice：今天是考试。</span>

ボブ：ジョンは？<br>
<span class="jp-trans">Bob：那 John 呢？</span>

アリス：ジョンは<span class="jp-word" title="あした - 明天">明日</span>。<br>
<span class="jp-trans">Alice：John 是明天。（对 John 来说，考试是明天。）</span>

最后一个例子说明了句子主题含义的宽泛性。主题可以指向任何地点的任何动作或对象，甚至可以包含其他句子。比如在上例最后一句里，虽然是在讨论 John 何时考试，但句子里却根本没出现「考试」二字！

本节最后我们还会学到识别助词，它和句子的结合更紧密。

## 包含主题助词「も」 {#mo}

### 单词学习

1. 学生 【がく・せい】 - 学生
2. うん - 是（口语）
3. でも - 但是
4. ううん - 不是（口语）

包含主题助词是一个跟主题助词非常相似的助词。它其实就是一个带了「也」意思的主题助词。基本上，它可以在当前主题之上引入另一个主题。包含主题助词写作「も」，下面我们来看例子。

### 例一

ボブ：アリスは<span class="jp-word" title="がくせい - 学生">学生</span>？<br>
<span class="jp-trans">Bob：（你）Alice 是学生吗？</span>

アリス：うん、トム<em>も</em><span class="jp-word" title="がくせい - 学生">学生</span>。<br>
<span class="jp-trans">Alice：是，Tom <em>也是</em>学生。</span>

使用「も」要注意一致性，比如说「我是学生，Tom 也不是学生。」就很奇怪了。这种情况下，可以用「は」新开一个主题，如下所示。

### 例二

ボブ：アリスは<span class="jp-word" title="がくせい - 学生">学生</span>？<br>
<span class="jp-trans">Bob：（你）Alice 是学生吗？</span>

アリス：うん、でもトム<em>は</em><span class="jp-word" title="がくせい - 学生">学生</span>じゃない。<br>
<span class="jp-trans">Alice：是，但 Tom 不是学生。</span>

下例说明了否定的包含。

### 例三

ボブ：アリスは<span class="jp-word" title="がくせい - 学生">学生</span>？<br>
<span class="jp-trans">Bob：（你）Alice 是学生吗？</span>

アリス：ううん、トム<em>も</em><span class="jp-word" title="がくせい - 学生">学生</span>じゃない。<br>
<span class="jp-trans">Alice：不是，Tom 也不是学生。</span>

## 识别助词「が」 {#ga}

### 单词学习

1. 誰 【だれ】 - 谁
2. 学生 【がく・せい】 - 学生
3. 私 【わたし】 - 我，我自己

现在我们会用「は」和「も」建立话题了，但如果我们不知道话题是什么呢？如果我想问「那个学生是谁？」，那我就需要识别一个对象（identifier），因为我不知道那个学生是谁。如果我仍然使用话题助词，问题就变成了「谁是学生吗？」，这样就很奇怪，因为「谁」不是一个实际的人。

这里我们就要用到助词「が」了。也有人叫它「主语助词」，但我不喜欢这个名字，因为它跟英语语法里面「主语」的意思完全不同。在这个课程里，我用「识别助词」来称呼它，因为这个助词表明了说话的人想要认（identify）清楚的东西。

### 例一

ボブ：<span class="jp-word" title="だれ - 谁">誰</span><em>が</em><span class="jp-word" title="がくせい - 学生">学生</span>？<br>
<span class="jp-trans">Bob：是学生的那个人是谁？</span>

アリス：ジョン<em>が</em><span class="jp-word" title="がくせい - 学生">学生</span>。<br>
<span class="jp-trans">Alice：John 是那个是学生的人。</span>

Bob 想要搞明白众人中哪一个是学生，Alice 回答说 John 就是那个人。注意，Alice 也可以用主题助词来回答，这样意思就变成「说到 John，他是一个学生」（但也许不是 Bob 所指的那个学生）。

### 例二

1. <span class="jp-word" title="だれ - 谁">誰</span> <em>が</em> <span class="jp-word" title="がくせい - 学生">学生</span> ？<br>
<span class="jp-trans">是学生的那个人是谁？</span>
2. <span class="jp-word" title="がくせい - 学生">学生</span> は <span class="jp-word" title="だれ - 谁">誰</span> ？<br>
<span class="jp-trans">（那个）学生是谁？</span>

第一个句子想要在人群中识别出一个学生，而第二个句子只是在谈论学生。第一个句子里的「が」不能换成「は」，否则「谁」就成了主题，而句子意思就成了「谁是一个学生吗？」

「は」和「が」这两个助词也许看起来很相似，因为英语里面都没有直接的对应（译者注：中文也没有吧）。比如下面俩个句子翻译成英语都是一样的。注

### 例三

1. <span class="jp-word" title="わたし - 我，我自己">私</span> <em>は</em> <span class="jp-word" title="がくせい - 学生">学生</span> 。<br>
<span class="jp-trans">我（是）学生。</span>
2. <span class="jp-word" title="わたし - 我，我自己">私</span> <em>が</em> <span class="jp-word" title="がくせい - 学生">学生</span> 。<br>
<span class="jp-trans">我（是）学生。</span>

然而，之所以相似，是因为日语有时候可以去掉一些信息变得很简洁，而英语不行。第一句里面「<span class="jp-word" title="わたし - 我，我自己">私</span>」是主题，所以含义是「至于我呢，我是一个学生」。

1. <span class="jp-word" title="わたし - 我，我自己">私</span> は <span class="jp-word" title="がくせい - 学生">学生</span> 。<br>
<span class="jp-trans"><em>至于</em> 我呢，（我是）学生。</span>
2. <span class="jp-word" title="わたし - 我，我自己">私</span> が <span class="jp-word" title="がくせい - 学生">学生</span> 。<br>
<span class="jp-trans">我（是） <em>那个人</em> （那是个）学生。</span>

如果你按照我的方法来思考就会发现，助词「は」和「が」其实很不一样。助词「が」用来识别某物或某人的属性，而助词「は」只是用来带出一个新的话题。所以在长句子里，人们经常会用逗号来将主题隔开，以免产生歧义，不知道句子的哪个部分是主题。

注技术上来说，在缺少上下文的情况下，这是最接近原意的翻译。
