## （不是！）最后三个助词 {#part1}

我们已经学了很强大的句子结构，足以让我们表达几乎任何意思了。接下来我们会看到助词「の」将赋予我们更为强大的能力来定义一般化、抽象的名词。我们也将学习如何直接用名词修饰名词。下面将介绍的三个助词让我们能用不同的方式组合名词。

这是最后一节专门介绍助词的课了。但这并不意味着助词全都讲完了。我们在后面还会学到很多助词，但不会特别介绍。只要你知道它们的意思和用法，是不是助词也就不再重要了。

## 包含助词「と」 {#part2}

### 词汇学习

1. ナイフ - 刀
2. フォーク - 叉
3. ステーキ - 牛排
4. 食べる 【た・べる】 （る动词） - 吃
5. 本 【ほん】- 书
6. 雑誌 【ざっ・し】 - 杂志
7. 葉書 【はがき】 - 明信片
8. 買う 【か・う】 （う动词）- 买
9. 友達 【とも・だち】- 朋友
10. 話す 【はな・す】 （う动词） - 说
11. 先生 【せん・せい】 - 老师
12. 会う 【あ・う】 （う动词） - 会见

助词「と」的作用跟助词「も」比较像，都是包含的意思。它能把两个以上名词连起来，之间的关系是「和」。

1. <span class="jp-word" title="ナイフ - 刀">ナイフ</span> <em>と</em> <span class="jp-word" title="フォーク - 叉">フォーク</span> で <span class="jp-word" title="ステーキ - 牛排">ステーキ</span> を <span class="jp-word" title="たべる - 吃">食べた</span> 。<br>
<span class="jp-trans">通过刀和叉吃了牛排。</span>
2. <span class="jp-word" title="ほん- 书">本</span> <em>と</em> <span class="jp-word" title="ざっし - 杂志">雑誌</span> <em>と</em> <span class="jp-word" title="はがき - 明信片">葉書</span> を <span class="jp-word" title="かう- 买">買った</span> 。<br>
<span class="jp-trans">买了书、杂志和明信片。</span>

助词「と」的另一个类似的用法是用来表示和某人或某物一起完成了动作。

1. <span class="jp-word" title="ともだち- 朋友">友達</span> <em>と</em> <span class="jp-word" title="はなす -谈话">話した</span> 。<br>
<span class="jp-trans">和朋友聊天了。</span>
2. <span class="jp-word" title="せんせい - 老师">先生</span> <em>と</em> <span class="jp-word" title="あう - 会见">会った</span> 。<br>
<span class="jp-trans">和老师见面了。</span>

## 表示模糊罗列的助词「や」和「とか」 {#part3}

### 词汇学习

1. 飲み物 【の・み・もの】 - 饮料
2. カップ - 杯子
3. ナプキン - 餐巾
4. いる （う动词）- 需要
5. 靴 【くつ】- 鞋子
6. シャツ - 衬衫
7. 買う 【か・う】 （う动词）- 买

助词「や」跟助词「と」一样用来罗列一个或多个名词，但它比助词「と」模糊很多。它暗示也许有其他没有列出的名词，或者清单里的物件并不都适用于上下文。你可以把它想成英语里的 "and/or, etc."。

1. <span class="jp-word" title="のみもの - 饮料">飲み物</span> <em>や</em> <span class="jp-word" title="カップ - 杯子">カップ</span> <em>や</em> <span class="jp-word" title="ナプキン - 餐巾">ナプキン</span> は、 <span class="jp-word" title="いる- 需要">いらない</span> ？<br>
<span class="jp-trans">你不需要饮料、杯子或餐巾等（这些东西）？</span>
2. <span class="jp-word" title="くつ- 鞋子">靴</span> <em>や</em> <span class="jp-word" title="シャツ - 衬衫">シャツ</span> を <span class="jp-word" title="かう- 买">買う</span> 。<br>
<span class="jp-trans">买鞋和衬衫等（这类东西）。</span>

「とか」和「や」的意思一样，但更口语化一些。

1. <span class="jp-word" title="のみもの - 饮料">飲み物</span> <em>とか</em> <span class="jp-word" title="カップ - 杯子">カップ</span> <em>とか</em> <span class="jp-word" title="ナプキン - 餐巾">ナプキン</span> は、 <span class="jp-word" title="いる- 需要">いらない</span> ？<br>
<span class="jp-trans">你不需要饮料、杯子或餐巾等（这些东西）？</span>
2. <span class="jp-word" title="くつ- 鞋子">靴</span> <em>とか</em> <span class="jp-word" title="シャツ - 衬衫">シャツ</span> を <span class="jp-word" title="かう- 买">買う</span> 。<br>
<span class="jp-trans">买鞋和衬衫等（这类东西）。</span>

## 助词「の」 {#part4}

### 词汇学习

1. 本 【ほん】- 书
2. アメリカ - 美国
3. 大学 【だい・がく】 - 大学
4. 学生 【がく・せい】 - 学生
5. それ - 那
6. その - 「それの」的缩写
7. シャツ - 衬衫
8. 誰 【だれ】 - 谁
9. これ - 这
10. この - 「これの」的缩写
11. あれ - 那 （远方）
12. あの - 「あれの」的缩写
13. 白い 【し・ろい】 （い形容词） -
14. かわいい （い形容词） - 可爱
15. 授業 【じゅ・ぎょう】- 课程
16. 行く 【い・く】 （う动词） - 走
17. 忘れる 【わす・れる】 （る动词） - 忘记
18. こと - 事件，事情
19. 毎日 【まい・にち】 - 每天
20. 勉強 【べん・きょう】 - 学习
21. する （例外） - 做
22. 大変 【たい・へん】 （な形容词） - 艰苦，困难时期
23. 同じ 【おな・じ】 - 一样
24. 物 【もの】- 东西
25. 食べる 【た・べる】 （る动词） - 吃
26. 面白い 【おも・し・ろい】 （い形容词） - 有趣的
27. 静か 【しず・か】 （な形容词） - 安静
28. 部屋 【へ・や】 - 房间
29. 人 【ひと】 - 人
30. 学校 【がっ・こう】 - 学校

助词「の」有很多不同的用法，是一个非常有用的助词。因为它跟「と」和「や」一样可以用来连接名词，所以我放在这一节介绍。我们先看几个例子。

1. ボブ <em>の</em> <span class="jp-word" title="ほん- 书">本</span> 。<br>
<span class="jp-trans">Bob 的书。</span>
2. <span class="jp-word" title="ほん- 书">本</span> <em>の</em> ボブ。<br>
<span class="jp-trans">书的 Bob。</span>

第一句意思是「Bob 的书」，第二句的意思是「书的 Bob」，这可能是个错误。我把第一句翻译成了「Bob 的书」，因为助词「の」不一定总是表示从属关系，如下例所示（译者注：英语人士理解「の」可能要转一下脑子，但国人可以忽略这一节的讲解，因为「の」的这种用法和中文是一致的）。

1. ボブは、 <span class="jp-word" title="アメリカ - 美国">アメリカ</span> <em>の</em> <span class="jp-word" title="だいがく - 大学">大学</span> <em>の</em> <span class="jp-word" title="がくせい - 学生">学生</span> だ。<br>
<span class="jp-trans">Bob 是美国的大学的学生。</span>

用正常的英语来翻译应该是 "Bob is a student of an American college"，这里修饰词的顺序是反过来的。「<span class="jp-word" title="がくせい - 学生">学生</span>の<span class="jp-word" title="だいがく - 大学">大学</span>の<span class="jp-word" title="アメリカ - 美国">アメリカ</span>」的意思是「学生的大学的美国」，这应该是错的，意思混乱。

如果上下文可以推断出来的话，被修饰的名词可以被省略掉。下面高亮显示的冗余单词可以被省略掉。

1. <span class="jp-word" title="その - 那">その</span> <span class="jp-word" title="シャツ - 衬衫">シャツ</span> は <span class="jp-word" title="だれ - 谁">誰</span> の <em><span class="jp-word" title="シャツ - 衬衫">シャツ</span></em> ？<br>
<span class="jp-trans">那件衬衫是谁的？</span>
2. ボブの <em><span class="jp-word" title="シャツ - 衬衫">シャツ</span></em> だ。<br>
<span class="jp-trans">它是 Bob 的衬衫。</span>

可以简写为：

1. <span class="jp-word" title="その - 那">その</span> <span class="jp-word" title="シャツ - 衬衫">シャツ</span> は <span class="jp-word" title="だれ - 谁">誰</span> <em>の</em> ？<br>
<span class="jp-trans">那件衬衫是谁的？</span>
2. ボブ <em>の</em> だ。<br>
<span class="jp-trans">它是 Bob 的。</span>

（「<span class="jp-word" title="その - 那">その</span>」是「<span class="jp-word" title="それ - 那">それ</span>+の」 的缩写，因为已经包含了助词「の」，所以它可以直接修饰名词。从「<span class="jp-word" title="これ - 这">これ</span>の」缩写为的「<span class="jp-word" title="この - 这">この</span>」以及从「<span class="jp-word" title="あれ - 那 （远方）">あれ</span>の」缩写为的「<span class="jp-word" title="あの - 那 （远方）">あの</span>」也一样。）

助词「の」的这种用法替代掉了名词，取代它扮演了名词的角色。我们可以将形容词和动词加上「の」来当作名词对待。这样这个助词就成了一个广义名词，可以视作一般的名词来对待。

1. <em><span class="jp-word" title="しろい - 白色">白い</span>の</em> は、 <span class="jp-word" title="かわいい - 可爱">かわいい</span> 。<br>
<span class="jp-trans">白色的东西是可爱的。</span>
2. <em><span class="jp-word" title="じゅぎょう- 课程">授業</span>に<span class="jp-word" title="いく - 走">行く</span>の</em> を <span class="jp-word" title="わすれる - 忘记">忘れた</span> 。<br>
<span class="jp-trans">忘记了去上课这件事。</span>

我们可以将直接对象、话题以及识别助词与动词和形容词合用。这里我们也不一定要用助词「の」。我们也可以用名词「<span class="jp-word" title="もの- 东西">物</span>」——代表广义的东西，或者「<span class="jp-word" title="こと - 事件，事情">こと</span>」——代表广义的事情。比如我们可以说：

1. <span class="jp-word" title="しろい - 白色">白い</span> <em><span class="jp-word" title="もの- 东西">物</span></em> は、 <span class="jp-word" title="かわいい - 可爱">かわいい</span> 。<br>
<span class="jp-trans">白色的东西是可爱的。</span>
2. <span class="jp-word" title="じゅぎょう- 课程">授業</span> に <span class="jp-word" title="いく - 走">行く</span> <em><span class="jp-word" title="こと - 事件，事情">こと</span></em> を <span class="jp-word" title="わすれる - 忘记">忘れた</span> 。<br>
<span class="jp-trans">忘记了去上课这件事。</span>

不过助词「の」的强大之处在于你不必指定一个特别的名词。在下面的例子里，助词「の」并没有替代任何特别的名词，它允许我们像用名词从句那样修饰动词和形容词。关系从句已经高亮显示。

1. <em><span class="jp-word" title="まいにち - 每天">毎日</span><span class="jp-word" title="べんきょう - 学习">勉強</span><span class="jp-word" title="する - 做">する</span></em> のは <span class="jp-word" title="たいへん - 艰苦，困难时期">大変</span> 。<br>
<span class="jp-trans">每天学习太难了。</span>
2. <em><span class="jp-word" title="まいにち - 每天">毎日</span><span class="jp-word" title="おなじ - 一样">同じ</span><span class="jp-word" title="もの- 东西">物</span>を<span class="jp-word" title="たべる - 吃">食べる</span></em> のは、 <span class="jp-word" title="おもしろい - 白色 有趣的">面白くない</span> 。<br>
<span class="jp-trans">每天吃同样的东西没意思。</span>

即使用「の」替代了名词，如果你是用な形容词来修饰它，「な」还是不能省掉的。

- <span class="jp-word" title="しずか - 安静">静か</span> <em>な</em> <span class="jp-word" title="へや - 房间">部屋</span> が、アリスの <span class="jp-word" title="へや - 房间">部屋</span> だ。<br>
<span class="jp-trans">安静的房间是 Alice 的房间。</span>

可以写成：

- <span class="jp-word" title="しずか - 安静">静か</span> <em>な</em> のが、アリスの <span class="jp-word" title="へや - 房间">部屋</span> だ。<br>
<span class="jp-trans">安静的那个是 Alice 的房间。</span>

*警告：别以为这样你就可以用「の」 替代任何名词了。很重要的一件事情是，整个句子的重心应该是从句，而非被替代掉的名词。比如在上一节我们有这么个例句：「<span class="jp-word" title="がくせい - 学生">学生</span>じゃない<span class="jp-word" title="ひと - 人">人</span>は、 <span class="jp-word" title="がっこう - 学校">学校</span>に<span class="jp-word" title="いく - 走">行かない</span>」，也许你会觉得可以把「<span class="jp-word" title="ひと - 人">人</span>」换成「の」来把句子变成「<span class="jp-word" title="がくせい - 学生">学生</span>じゃない<em>の</em>は、<span class="jp-word" title="がっこう - 学校">学校</span>に<span class="jp-word" title="いく - 走">行かない</span>」。实际上这样改完句子就没有意义了，因为现在的重心在「不是学生」，整个句子的意思变成了「不是学生不去学校」，乱了。因为不是学生是一种状态，一种状态去哪里是没有意义的。

## 助词「の」用作解释 {#part5}

### 词汇学习

1. 今 【いま】- 现在
2. 忙しい 【いそが・しい】 （い形容词） - 忙
3. 学生 【がく・せい】 - 学生
4. 飲む 【のむ】 - 喝
5. どこ - 哪里
6. 行く 【い・く】 （う动词） - 走
7. 授業 【じゅ・ぎょう】- 课程
8. ある （う动词） - 存在（无生命）
9. ううん - 「不」的口语说法
10. その - 那 （それの的缩写）
11. 人 【ひと】 - 人
12. 買う 【か・う】 （う动词）- 买
13. 先生 【せん・せい】 - 老师
14. 朝ご飯 【あさ・ご・はん】- 早饭
15. 食べる 【た・べる】 （る动词） - 吃
16. どうして - 为什么

助词「の」加在句子里最后一个从句末尾的时候还可以表达一种解释的口吻。比如有人问你是否有空，你也许会回答「问题是我现在有点忙」，这个抽象的广义名词「问题是…」可以用助词「の」来表达。这种句子隐含了解释的意味。

这个句子可以写为：

- <span class="jp-word" title="いま- 现在">今</span> は <span class="jp-word" title="いそがしい - 忙">忙しい</span> <em>の</em> 。<br>
<span class="jp-trans">问题是（我）现在有点忙。</span>

不过这样听起来很娘。实际生活中成年男性都会再加上一个表陈述的「だ」，除非他们想卖萌。

- <span class="jp-word" title="いま- 现在">今</span> は <span class="jp-word" title="いそがしい - 忙">忙しい</span> <em>のだ</em> 。<br>
<span class="jp-trans">问题是（我）现在有点忙。</span>

不过因为「だ」不能被用在问句里，提问的时候只用「の」是没问题的，这样一点都不娘，不管对男生还是女生。

- <span class="jp-word" title="いま- 现在">今</span> は <span class="jp-word" title="いそがしい - 忙">忙しい</span> <em>の</em> ？<br>
<span class="jp-trans">（你）是不是现在很忙？（中性）</span>

如果要表达状态，且又要用「の」来体现解释的口吻，<u>我们就要添加「な」</u>来和仅表示「的」的「の」区分开。

1. ジムのだ。<br>
<span class="jp-trans">这是 Jim 的。</span>
2. ジム <em>な</em> のだ。<br>
<span class="jp-trans">这是 Jim 的（解释的口吻）。</span>

这是一个特例，其他规则跟前面说的一样。

实际生活里面，虽然这种解释口吻很常用，但通常却都是用「んだ」来替代「のだ」。这也许是因为「んだ」比「のだ」更容易发音。这种语法可以表达很多意思，因为它不仅能和所有形态的形容词、名词和动词合用，它自己<b>也</b>可以像状态表示语那样活用。请参见下面的活用图表。

其实第二张图里面名词 、な形容词的过去形和过去否定式基本上没人用（特别是用「の」时），不过为了图表的完整我还是写上了。

用与不用解释口吻「の」的最关键区别在于前者你是在跟对方说「看，事情是这样的」，而后者仅仅告知对方信息。例如别人问你「忙吗现在？」，你可以简单回答「<span class="jp-word" title="いま- 现在">今</span>は<span class="jp-word" title="いそがしい - 忙">忙しい</span>」。但如果别人问你「你为什么不跟我说话？」，显然你就需要解释一下了，所以应该说「<span class="jp-word" title="いま- 现在">今</span>は<span class="jp-word" title="いそがしい - 忙">忙しい</span>の」或者「<span class="jp-word" title="いま- 现在">今</span>は<span class="jp-word" title="いそがしい - 忙">忙しい</span>んだ」。这种语法在希望对方解释的时候是必须使用的，比如你想问「喂，不觉得很迟了吗？」，就不能只说「<span class="jp-word" title="おそい - 迟">遅くない</span>？」，因为这样意思就成了「现在迟了吗？」。你应该用「<span class="jp-word" title="おそい - 迟">遅い</span>んじゃない？」来表达你想要对方解释的意思。

我们再来看几个适用这种语法的例子。示例中会先直译，让你看到字面意思是相同的；然后再意译，让你看到句子到底传达了怎样的意义。（译者注：下面所谓的直译我就保留原文了，可以看出来想用英语来表达这种期待解释的口吻是挺别扭的，鬼佬不是很习惯双重、三重否定，但换到中文就很自然了）

### 例一

アリス：<span class="jp-word" title="どこ - 哪里">どこ</span>に<span class="jp-word" title="いく - 走">行く</span><em>の</em>？<br>
<span class="jp-trans">Alice：Where is it that (you) are going?</span>

ボブ：<span class="jp-word" title="じゅぎょう- 课程">授業</span>に<span class="jp-word" title="いく - 走">行く</span><em>んだ</em>。<br>
<span class="jp-trans">Bob：It is that (I) go to class.</span>

Alice：Where are you going? （寻求解释）你要去哪里？<br>
<span class="jp-trans">Bob：I'm going to class. （解释）我去上课。</span>

### 例二

アリス：<span class="jp-word" title="いま- 现在">今</span>、<span class="jp-word" title="じゅぎょう- 课程">授業</span>が<span class="jp-word" title="ある - 存在（无生命）">ある</span><em>んじゃない</em>？<br>
<span class="jp-trans">Alice：Isn't it that there is class now?</span>

ボブ：<span class="jp-word" title="いま- 现在">今</span>は、<span class="jp-word" title="ある - 存在（无生命）">ない</span><em>んだ</em>。<br>
<span class="jp-trans">Bob：Now it is that there is no class.</span>

Alice：Don't you have class now? （期待有课）你不是现在要上课吗？<br>
<span class="jp-trans">Bob：No, there is no class now. （解释）没，现在没课。</span>

### 例三

アリス：<span class="jp-word" title="いま- 现在">今</span>、<span class="jp-word" title="じゅぎょう- 课程">授業</span>が<span class="jp-word" title="ある - 存在（无生命）">ない</span><em>んじゃない</em>？<br>
<span class="jp-trans">Alice：Isn't it that there isn't class now?</span>

ボブ：<span class="jp-word" title="ううん - 不">ううん</span>、<span class="jp-word" title="ある - 存在（无生命）">ある</span>。<br>
<span class="jp-trans">Bob：No, there is.</span>

### 例四

アリス：<span class="jp-word" title="その - 那">その</span><span class="jp-word" title="ひと - 人">人</span>が<span class="jp-word" title="かう- 买">買う</span><em>んじゃなかったの</em>？<br>
<span class="jp-trans">Alice：Wasn't it that that person was the one to buy?</span>

ボブ：<span class="jp-word" title="ううん - 不">ううん</span>、<span class="jp-word" title="せんせい - 老师">先生</span>が<span class="jp-word" title="かう- 买">買う</span><em>んだ</em>。<br>
<span class="jp-trans">Bob：No, it is that teacher is the one to buy.</span>

Alice：Wasn't that person going to buy? （期待那个人会买）<br>
<span class="jp-trans">Bob：No, the teacher is going to. （解释）</span>

### 例五

アリス：<span class="jp-word" title="あさごはん- 早饭">朝ご飯</span>を<span class="jp-word" title="たべる - 吃">食べる</span><em>んじゃなかった</em>。<br>
<span class="jp-trans">Alice：It is that breakfast wasn't to eat.</span>

ボブ：<span class="jp-word" title="どうして - 为什么">どうして</span>？<br>
<span class="jp-trans">Bob：Why?</span>

Alice：Should not have eaten breakfast, you know. （解释不该吃早饭）早饭真不该吃<br>
<span class="jp-trans">Bob：何出此言？</span>

如果你现在已经凌乱，别担心，后面还会看到很多例子。一旦你明白了其中的奥秘，就不要再用英语转一道了，因为像例三里面的双重、三重否定会让你绕不过来。不过一旦习惯了，你就会发现这在日语里面这些是再正常不过的了。
