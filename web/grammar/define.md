## 「<span class="jp-word" title="いう- 说">いう</span>」的各种用法 {#part1}

前一课我们学习了如何用「と」来引用关系从句，这样我们就可以谈论人们说的、听到的之类的事情。我们也学习了如何用「と」和「<span class="jp-word" title="いう- 说">言う</span>」来表达「这个用日语怎么说」、「可以叫我 xxx」之类的意思。在这一节里，我们将学习到如何用「と」以及「<span class="jp-word" title="いう- 说">いう</span>」来定义、形容或者一般性的讨论某个东西。我们也将学习如何用口语里的「って」来表达同样的意思，这个词上一课我们已经见过了。

## 用「<span class="jp-word" title="いう- 说">いう</span>」来定义 {#part2}

### 词汇学习

1. 言う 【い・う】 （う动词）- 说
2. これ - 这
3. 何 【なに／なん】 - 什么
4. 魚 【さかな】 - 鱼
5. この - 这个（これの的缩写）
6. 鯛 【たい】 - 鲷（一种鱼）
7. デパート - 百货
8. どこ - 哪里
9. ある （う动词） - 存在（无生命）
10. 知る 【し・る】 （う动词） - 知道
11. 友達 【とも・だち】- 朋友
12. 英語 【えい・ご】 - 英语
13. 意味 【い・み】 - 意思

上一课我们简单介绍了如何用「と」和「<span class="jp-word" title="いう- 说">いう</span>」来自我介绍，例如在下面的例句里 Alice 在自我介绍。

- <span class="jp-word" title="わたし - 我；我自己">私</span> はアリス <em>と<span class="jp-word" title="いう- 说">いいます</span></em> 。<br>
<span class="jp-trans">别人叫我 Alice。（字面：至于我，你可以说 Alice）</span>

这个句型可能是很多日语初学者在教室里学到的第一句话。在这个例句里，动词「<span class="jp-word" title="いう- 说">いう</span>」不代表某人说了什么，Alice 的意思是人们在提到她的时候通常说 "Alice"。虽然用汉字来写「<span class="jp-word" title="いう- 说">いう</span>」是没问题的，但在这里因为并没有人说什么，很多时候就用平假名了。

这种句型可以扩展到物体和地点。基本上，我们可以这样用「という」来定义或识别任何东西。可以想象，这对我们特别有用，比如我们可以问别人用日语怎么说某样东西，或询问某个不认识的词的意思。

### 示例

1. <span class="jp-word" title="これ - 这">これ</span> は、 <em><span class="jp-word" title="なん - 什么">なん</span>と<span class="jp-word" title="いう- 说">いう</span><span class="jp-word" title="さかな - 鱼">魚</span></em> ですか。<br>
<span class="jp-trans">这是什么鱼？</span>
2. <span class="jp-word" title="この - 这">この</span> <span class="jp-word" title="さかな - 鱼">魚</span> は、 <em><span class="jp-word" title="たい - 鲷（鱼）">鯛</span>と<span class="jp-word" title="いう- 说">いいます</span></em> 。<br>
<span class="jp-trans">这种鱼被称为「 <i>鲷</i> 」。</span>
3. <em>ルミネと<span class="jp-word" title="いう- 说">いう</span><span class="jp-word" title="デパート - 百货">デパート</span></em> は <span class="jp-word" title="どこ - 哪里">どこ</span> に <span class="jp-word" title="ある - 存在（无生命）">ある</span> か、 <span class="jp-word" title="しる - 知道">知って</span> <span class="jp-word" title="いる - 存在（有生命）">います</span> か？<br>
<span class="jp-trans">你知道叫 "Lumine" 的百货大楼在哪里吗？</span>
4. 「 <span class="jp-word" title="ともだち- 朋友">友達</span> 」は、 <span class="jp-word" title="えいご - English">英語</span> で <em>「friend」と<span class="jp-word" title="いう- 说">いう</span></em> <span class="jp-word" title="いみ - 意思">意味</span> です。<br>
<span class="jp-trans">「 <i>友達</i> 」英语里的意思是 "friend"。</span>

## 用「<span class="jp-word" title="いう- 说">いう</span>」描述任何东西 {#part3}

### 词汇学习

1. 主人公 【しゅ・じん・こう】 - 主人公
2. 犯人 【はん・にん】 - 罪犯
3. 一番 【いち・ばん】 - 最好；第一
4. 面白い 【おも・しろ・い】（い形容词） - 有趣
5. 日本人 【に・ほん・じん】 - 日本人
6. お酒 【お・さけ】 - 酒
7. 弱い 【よわ・い】（い形容词） - 弱
8. 言う 【い・う】 （う动词）- 说
9. 本当 【ほん・とう】 - 真的
10. 独身 【どく・しん】 - 单身
11. 嘘 【うそ】 - 谎言
12. リブート - 重启
13. パソコン - 电脑
14. こう - 这样
15. そう - 那样
16. ああ - 那样
17. どう - 如何
18. 再起動 【さい・き・どう】 - 重启
19. あんた - 你（俚语）
20. いつも - 总是
21. 時 【とき】 - 时间
22. 来る 【く・る】 （例外）- 来
23. 困る 【こま・る】 （う动词） - 为难
24. 人 【ひと】 - 人
25. 結婚 【けっ・こん】 - 婚姻
26. 出来る 【で・き・る】 （る动词） - 做好；做出；出现
27. 幸せ 【しあわ・せ】 - 幸福
28. なる （う动词） - 成为
29. 思う 【おも・う】 （う动词） - 思考，认为
30. 大学 【だい・がく】 - 大学
31. 行く 【い・く】 （う动词） - 走
32. 意味 【い・み】 - 意思

我们已经学了如何用「と<span class="jp-word" title="いう- 说">いう</span>」来描述某个东西。不过，我们还可以把这种用法进一步扩展到两个关系从句。这时候，「<span class="jp-word" title="いう- 说">いう</span>」的意思就相当抽象了，可以认为没有实际的意思。前面说过当一个关系从句被「と」引用以后，你必须用一个动词来配合它，而这里「<span class="jp-word" title="いう- 说">いう</span>」只是作为一个通用的动词，让我们来谈论被引用的关系从句。这样，我们就可以描述或者解释任何东西了——从一个词到一整个句子都没问题。可以想象，这种结构非常有用，并且在日语里面也是被频繁使用。

### 示例

1. <span class="jp-word" title="しゅじんこう - 主人公">主人公</span> が <span class="jp-word" title="はんにん - 罪犯">犯人</span> だった <em>と<span class="jp-word" title="いう- 说">いう</span></em> のが <span class="jp-word" title="いちばん - 最好；第一">一番</span> <span class="jp-word" title="おもしろい - 有趣">面白かった</span> 。<br>
<span class="jp-trans">有趣的是，主人公就是那个罪犯。</span>
2. <span class="jp-word" title="にほんじん - 日本人">日本人</span> は <span class="jp-word" title="おさけ - 酒">お酒</span> に <span class="jp-word" title="よわい - 弱">弱い</span> <em>と<span class="jp-word" title="いう- 说">いう</span></em> のは <span class="jp-word" title="ほんとう - 真的">本当</span> ？<br>
<span class="jp-trans">日本人真的不能喝酒吗？</span>
3. <span class="jp-word" title="どくしん - single, unmarried">独身</span> だ <em>と<span class="jp-word" title="いう- 说">いう</span></em> のは、 <span class="jp-word" title="うそ - 谎言">嘘</span> だったの？<br>
<span class="jp-trans">你单身的说法是说谎的？</span>
4. <span class="jp-word" title="リブート - 重启">リブート</span> <em>と<span class="jp-word" title="いう- 说">いう</span></em> のは、 <span class="jp-word" title="パソコン - 电脑">パソコン</span> を <span class="jp-word" title="さいきどう - 重启">再起動</span> <span class="jp-word" title="する - 做">する</span> <em>と<span class="jp-word" title="いう- 说">いう</span></em> <span class="jp-word" title="こと - 事情">こと</span> です。<br>
<span class="jp-trans">重启的意思是重新启动你的电脑。</span>

我们还可以进一步抽象，把关系从句换成正在做某事的通用说法。这时候，我们可以用「<span class="jp-word" title="こう - 像这样">こう</span>」、「<span class="jp-word" title="そう - 像那样">そう</span>」、「<span class="jp-word" title="ああ - 像那样">ああ</span>」和「<span class="jp-word" title="どう - 如何">どう</span>」，这些词跟「<span class="jp-word" title="いう- 说">いう</span>」合用的时候表示「这样」、「那样」、「那样（远）」和「怎样」。

### 示例

1. <span class="jp-word" title="あんた - 你">あんた</span> は、 <span class="jp-word" title="いつも - 总是">いつも</span> <em><span class="jp-word" title="こう - 像这样">こう</span><span class="jp-word" title="いう- 说">いう</span></em> <span class="jp-word" title="とき - 时间">時</span> に <span class="jp-word" title="くる- 来">来る</span> んだから、 <span class="jp-word" title="こまる - 为难">困る</span> んだよ。<br>
<span class="jp-trans">你总是在这种时候来，这让我很为难。</span>
2. <em><span class="jp-word" title="そう - 像那样">そう</span><span class="jp-word" title="いう- 说">いう</span></em> <span class="jp-word" title="ひと - 人">人</span> と <span class="jp-word" title="いっしょ - 一起">一緒</span> に <span class="jp-word" title="しごと - 工作">仕事</span> を <span class="jp-word" title="する - 做">する</span> のは、 <span class="jp-word" title="いや - dislike">嫌</span> だよね。<br>
<span class="jp-trans">（会有人）不喜欢跟那种人一起工作吗，嗯？</span>
3. <em><span class="jp-word" title="ああ - 像那样">ああ</span><span class="jp-word" title="いう- 说">いう</span></em> <span class="jp-word" title="ひと - 人">人</span> と <span class="jp-word" title="けっこん - 婚姻">結婚</span> <span class="jp-word" title="できる -  做好；做出；出现">できた</span> ら、 <span class="jp-word" title="しあわせ - 幸福">幸せ</span> に <span class="jp-word" title="なる - 成为">なれる</span> と <span class="jp-word" title="おもう - 思考，认为">思います</span> 。<br>
<span class="jp-trans">我想如果你跟那种人结婚的话会变得幸福的。</span>
4. <span class="jp-word" title="だいがく - 大学">大学</span> に <span class="jp-word" title="いく - 走">行かない</span> って、 <em><span class="jp-word" title="どう - 如何">どう</span><span class="jp-word" title="いう- 说">いう</span></em> <span class="jp-word" title="いみ - 意思">意味</span> なの？<br>
<span class="jp-trans">你说「你不去上大学？」是什么意思？</span>

## 用「と<span class="jp-word" title="いう- 说">いう</span>」改述和总结 {#part4}

### 词汇学习

1. あんた - 你（俚语）
2. 彼女 【かの・じょ】 - 她；女朋友
3. 友達 【とも・だち】- 朋友
4. 言う 【い・う】 （う动词）- 说
5. 何 【なに／なん】 - 什么
6. お酒 【お・さけ】 - 酒
7. 好き 【す・き】 （な形容词） - 讨人喜欢的
8. ある （う动词） - 存在（无生命）
9. 生きる 【い・きる】 （る动词） - 生活
10. 多分 【た・ぶん】 - 也许
11. 行く 【い・く】 （う动词） - 走
12. 思う 【おも・う】 （う动词） - 思考，认为
13. お金 【お・かね】 - 钱
14. もう - 已经
15. 帰る 【かえ・る】 （う动词） - 回家
16. 駄目 【だめ】 - 不好
17. 洋介 【よう・すけ】 - 洋介（姓）
18. 別れる 【わか・れる】 （る动词） - 分开；分手
19. こと - 事情
20. 今 【いま】- 现在
21. 彼氏【かれ・し】 - 男朋友
22. いる （る动词） - 存在（有生命）
23. そう - 那样

我们可以把提问词「か」加到「と<span class="jp-word" title="いう- 说">いう</span>」后面来提问。这种结构用在你想改述或重定义什么东西的时候，如下所示：

### 示例对话

Ａ：みきちゃんは、<span class="jp-word" title="あんた - 你">あんた</span>の<span class="jp-word" title="かのじょ - 她；女朋友">彼女</span>でしょう？<br>
<span class="jp-trans">A：Miki-chan 是你的女朋友吧？</span>

Ｂ：う～ん、<span class="jp-word" title="かのじょ - 她；女朋友">彼女</span><em>と<span class="jp-word" title="いう- 说">いう</span>か</em>、<span class="jp-word" title="ともだち- 朋友">友達</span><em>と<span class="jp-word" title="いう- 说">いう</span>か</em>、<span class="jp-word" title="なん - 什么">なん</span><em>と<span class="jp-word" title="いう- 说">いう</span>か</em>・・・<br>
<span class="jp-trans">B: 唔，可以说是女朋友，也可以是朋友，也可以是别的…</span>

这种结构很常见，特别是在口语里，它可以用来纠正别人，提出异议，甚至用来感叹。

### 示例

1. <span class="jp-word" title="おさけ - 酒">お酒</span> は <span class="jp-word" title="すき - 喜欢，渴望">好き</span> <em>と<span class="jp-word" title="いう- 说">いう</span>か</em> 、 <span class="jp-word" title="ある - 存在（无生命）">ない</span> と <span class="jp-word" title="いきる - 生活">生きて</span> <span class="jp-word" title="いく - 走">いけない</span> 。<br>
<span class="jp-trans">我喜欢喝酒，甚至可以说，没酒活不下去。</span>
2. <span class="jp-word" title="たぶん - 也许">多分</span> <span class="jp-word" title="いく - 走">行かない</span> と <span class="jp-word" title="おもう - 思考，认为">思う</span> 。 <em>と<span class="jp-word" title="いう- 说">いう</span>か</em> 、 <span class="jp-word" title="おかね - 钱">お金</span> が <span class="jp-word" title="ある - 存在（无生命）">ない</span> から、 <span class="jp-word" title="いく - 走">行けない</span> 。<br>
<span class="jp-trans">我想我大概走不了，更确切地说，因为没钱我走不了。</span>
3. <em>と<span class="jp-word" title="いう- 说">いう</span>か</em> 、 <span class="jp-word" title="もう - 已经">もう</span> <span class="jp-word" title="かえる - 回家">帰らない</span> と <span class="jp-word" title="だめ - 不好">だめ</span> ですけど。<br>
<span class="jp-trans">其实我现在必须回家了。</span>

与其用「か」来改述别人的结论，我们也可以只用「<span class="jp-word" title="こと - 事情">こと</span>」来做总结，而不是改述什么。

### 示例对话

Ａ：みきちゃんが<span class="jp-word" title="ようすけ - 洋介">洋介</span>と<span class="jp-word" title="わかれる - 分开">別れた</span>んだって。<br>
<span class="jp-trans">A：我听说 Miki-chan 跟洋介分手了。</span>

Ｂ：<em>と<span class="jp-word" title="いう- 说">いう</span><span class="jp-word" title="こと - 事情">こと</span></em>は、みきちゃんは、<span class="jp-word" title="いま- 现在">今</span><span class="jp-word" title="かれし - 他；男朋友">彼氏</span>が<span class="jp-word" title="いる - 存在（有生命）">いない</span><em>と<span class="jp-word" title="いう- 说">いう</span><span class="jp-word" title="こと - 事情">こと</span></em>？<br>
<span class="jp-trans">B：意思是 Miki-chan 现在没有男朋友了？</span>

Ａ：<span class="jp-word" title="そう - 那's so">そう</span>。<em><span class="jp-word" title="そう - 那里">そう</span><span class="jp-word" title="いう- 说">いう</span><span class="jp-word" title="こと - 事情">こと</span></em>。<br>
<span class="jp-trans">A：是的，是这个意思。</span>

## 用「って」或「て」替代「と<span class="jp-word" title="いう- 说">いう</span>」 {#part5}

### 词汇学习

1. 来年 【らい・ねん】 - 明年
2. 留学 【りゅう・がく】 - 留学
3. する （例外） - 做
4. 言う 【い・う】 （う动词）- 说
5. 智子 【とも・こ】 - 智子（姓）
6. こと - 事情
7. 駄目 【だめ】 - 不好
8. 時間 【じ・かん】 - 时间
9. ある （う动词） - 存在（无生命）
10. 出来る 【で・き・る】 （る动词） - 做好；做出；出现
11. 行く 【い・く】 （う动词） - 走
12. いい （い形容词） - 好的
13. 皆 【みんな】 - 所有人
14. 私 【わたし】 - 我；我自己
15. 今 【いま】- 现在
16. 彼氏【かれ・し】 - 男朋友
17. いる （る动词） - 存在（有生命）
18. もう - 已经
19. 帰る 【かえ・る】 （う动词） - 回家

上一课提过，「って」是一种常见的用以替代「と」的俚语，用它的时候我们假设上下文意思很明确，或者对方能懂，所以把句子的其它部分就都省略掉了。我们也看到「って」可以替代「と<span class="jp-word" title="いう- 说">いう</span>」。不过，因为我们刚刚扩展学习了「と<span class="jp-word" title="いう- 说">いう</span>」的用处，你可能搞不清楚到底可以省略掉哪些部分。但无论如何，「って」不仅允许我们省略掉「<span class="jp-word" title="いう- 说">いう</span>」,同时伴随其出现的助词都可以一并省略掉，如下所示。

### 示例

1. <span class="jp-word" title="らいねん - 明年">来年</span> <span class="jp-word" title="りゅうがく - 留学">留学</span> <span class="jp-word" title="する - 做">する</span> <em>と<span class="jp-word" title="いう- 说">いう</span>のは</em> 、 <span class="jp-word" title="ともこ - 智子">智子</span> の <span class="jp-word" title="こと - 事情">こと</span> ？<br>
<span class="jp-trans">明年留学那件事，是说智子吗？</span>
2. <span class="jp-word" title="らいねん - 明年">来年</span> <span class="jp-word" title="りゅうがく - 留学">留学</span> <span class="jp-word" title="する - 做">する</span> <em>って</em> <span class="jp-word" title="ともこ - 智子">智子</span> の <span class="jp-word" title="こと - 事情">こと</span> ？<br>
<span class="jp-trans">明年留学那件事，是说智子吗？</span>

「だって」是另一个万能的替代词组。习惯上，它用来表示对牢骚、抱怨的不满或异议，或用来制造借口，这都取决于上下文。它应该是从「とは<span class="jp-word" title="いう- 说">いって</span>も」演变出来的，意思是「即便如此」。

### 例一

Ａ：<span class="jp-word" title="する - 做">しない</span>と<span class="jp-word" title="だめ - 不好">だめ</span>だよ。<br>
<span class="jp-trans">A：不得不做啊。</span>

Ｂ：<em>だって</em>、<span class="jp-word" title="じかん - 时间">時間</span>が<span class="jp-word" title="ある - 存在（无生命）">ない</span>から<span class="jp-word" title="できる -  做好；做出；出现">できない</span>よ。<br>
<span class="jp-trans">B：但（即便如此）没时间了，做不了。</span>

### 例二

Ａ：<span class="jp-word" title="いく - 走">行かなくて</span>も<span class="jp-word" title="いい - 好的">いい</span>よ。<br>
<span class="jp-trans">A：不是一定要去的。</span>

Ｂ：<em>だって</em>、<span class="jp-word" title="みんな - 所有人">みんな</span><span class="jp-word" title="いく - 走">行く</span>って。<span class="jp-word" title="わたし - 我；我自己">私</span>も<span class="jp-word" title="いく - 走">行かない</span>と。<br>
<span class="jp-trans">B：但（即便如此）别人都说要去。我也得去。</span>

有时候，小「つ」也会被省略掉，「って」就变成了「て」。这样（通常是俚语）是为了容易发音。通常「て」前面没东西或者前面的假名发音已经不需要用「っ」来跟「て」分隔的时候就会这么说。

### 示例

1. <em>て</em> <span class="jp-word" title="こと - 事情">こと</span> は、みきちゃんは、 <span class="jp-word" title="いま- 现在">今</span> <span class="jp-word" title="かれし - 他；男朋友">彼氏</span> が <span class="jp-word" title="いる - 存在（有生命）">いない</span> <em>て</em> <span class="jp-word" title="こと - 事情">こと</span> ？<br>
<span class="jp-trans">意思是 Miki-chan 现在没有男朋友？</span>
2. <em>て</em> <span class="jp-word" title="いう- 说">いう</span> か、 <span class="jp-word" title="もう - 已经">もう</span> <span class="jp-word" title="かえる - 回家">帰らない</span> と <span class="jp-word" title="だめ - 不好">だめ</span> ですけど。<br>
<span class="jp-trans">其实我应该已经往家走了。</span>

因为俚语是随个人喜好而变的，没有一定的规矩说你必须用「って」或「て」。不过在引述别人说出来的话的时候通常不用「て」，所以上一课我们就没介绍。

- みきちゃんが、 <span class="jp-word" title="あした - 明天">明日</span> <span class="jp-word" title="くる- 来">こない</span> <em>て</em> 。<br>
<span class="jp-trans">（对于已经说出的话不能用「て」）</span>

- みきちゃんが、 <span class="jp-word" title="あした - 明天">明日</span> <span class="jp-word" title="くる- 来">こない</span> <em>って</em> 。<br>
<span class="jp-trans">Miki-chan 说她明天不来了。</span>

## 用「<span class="jp-word" title="ゆう- 说">ゆう</span>」替代「<span class="jp-word" title="いう- 说">いう</span>」 {#part6}

### 词汇学习

1. もう - 已经
2. 帰る 【かえ・る】 （う动词） - 回家
3. そう - 那样
4. こと - 事情

因为「と<span class="jp-word" title="いう- 说">いう</span>」结构太长用了，以致出现各种俚语变种，以及基于它的俚语。我在这里不想做全面介绍，不过你可以看[口语习惯及俚语](/grammar/slang/#part5)一节来了解更多衍生自「と<span class="jp-word" title="いう- 说">いう</span>」的俚语。

最后想提一下用「<span class="jp-word" title="ゆう- 说">ゆう</span>」替代「<span class="jp-word" title="いう- 说">いう</span>」。日常对话里面人们很经常用「<span class="jp-word" title="ゆう- 说">ゆう</span>」代替「<span class="jp-word" title="いう- 说">いう</span>」。「<span class="jp-word" title="ゆう- 说">ゆう</span>」相对更容易发音，因为只是一个长音，不像「<span class="jp-word" title="いう- 说">いう</span>」要发两个元音。

### 示例

1. て <em><span class="jp-word" title="ゆう- 说">ゆう</span></em> か、 <span class="jp-word" title="もう - 已经">もう</span> <span class="jp-word" title="かえる - 回家">帰らない</span> と <span class="jp-word" title="だめ - 不好">だめ</span> ですけど。<br>
<span class="jp-trans">其实我应该已经往家走了。</span>
2. <span class="jp-word" title="そう - 像那样">そう</span> <em><span class="jp-word" title="ゆう- 说">ゆう</span></em> <span class="jp-word" title="こと - 事情">こと</span> じゃないって！<br>
<span class="jp-trans">我说了不是那样的！（字面：不是那样的事）</span>
