## 如何用日语说「如果」 {#part1}

这一整节要专门讲如何用日语说「如果」。哎，要是跟英语一样简单就好了。日语里面有四种（你没看错，四种）说「如果」的方式！幸运的是，活用知识点不多，而且不用考虑时态。

## 用「と」表达自然的结果 {#part2}

### 词汇学习

1. ボール - 球
2. 落とす 【お・とす】 （う动词） - 扔
3. 落ちる 【お・ちる】 （る动词） - 掉落
4. 電気 【でん・き】 - 电气；灯
5. 消す 【け・す】 （う动词） - 擦除
6. 暗い 【くら・い】 （い形容词） - 暗
7. 学校 【がっ・こう】 - 学校
8. 行く 【い・く】 （う动词） - 走
9. 友達 【とも・だち】- 朋友
10. 会う 【あ・う】 （う动词） - 会见
11. たくさん - （数量）很多
12. 太る 【ふと・る】 （う动词） - 变胖
13. 先生 【せん・せい】 - 老师
14. きっと - 当然
15. 年上 【とし・うえ】 - 年长

我们先介绍「如果」最简单的说法，也就是能自然的引出结果的条件语。也就是说，如果 [X] 发生了，那么很自然 [Y] 就会发生，毫无疑问。如果我扔了一个球，它会掉到地上。如果我晚上关了灯，周围会变暗。我们可以用下面的语法表达这类条件。

<div class="sumbox">
<div class="sumbox-title">使用条件语「と」的规则</div>

1. 把「と」加到条件后面，再跟上满足了条件会发生的结果。<br>
<span class="jp-trans"><b>= [ 条件 ] + と + [ 结果 ]</b></span>
2. 状态表示必须显式声明<br>
<span class="jp-trans"><b>= [ 状态 ] + <u>だ</u>と + [ 结果 ]</b></span>
</div>

### 示例

1. <span class="jp-word" title="ボール - 球">ボール</span> を <span class="jp-word" title="おとす - 扔">落とす</span> <em>と</em> <span class="jp-word" title="おちる - 掉落">落ちる</span> 。<br>
<span class="jp-trans">如果扔一个球，它会掉下。</span>
2. <span class="jp-word" title="でんき - electricity, (electric) light">電気</span> を <span class="jp-word" title="けす - 擦除">消す</span> <em>と</em> <span class="jp-word" title="くらい - 暗">暗く</span> <span class="jp-word" title="なる - 成为">なる</span> 。<br>
<span class="jp-trans">如果关了灯，会变暗。</span>

这些例句是特地设计用来说明如何用「と」表示自然结果的。不过即使结果不是自然的，「と」也会传达这么一种意思，即这是说话的人认为会发生的结果。

1. <span class="jp-word" title="がっこう - 学校">学校</span> に <span class="jp-word" title="いく - 走">行かない</span> <em>と</em> <span class="jp-word" title="ともだち- 朋友">友達</span> と <span class="jp-word" title="あう - 会见">会えない</span> よ。<br>
<span class="jp-trans">如果你不去学校，你就见不到朋友们。</span>
2. <span class="jp-word" title="たくさん - 很多">たくさん</span> <span class="jp-word" title="たべる - 吃">食べる</span> <em>と</em> <span class="jp-word" title="ふとる - 变胖">太る</span> よ。<br>
<span class="jp-trans">如果你吃很多，你就会变胖，一定的。</span>
3. <span class="jp-word" title="せんせい - 老师">先生</span> <em>だと</em> 、 <span class="jp-word" title="きっと - 当然">きっと</span> <span class="jp-word" title="としうえ - 年长">年上</span> なんじゃないですか？<br>
<span class="jp-trans">如果他是老师，那他一定年纪比较大，对吧？</span>

（第二句里的）「一定的」是使用「と」后隐含的意思。说话的人想表达一定会发生的意思。另外你可以从最后一个例句看出，如果条件是状态表达，那么必须加上「だ」。对于所有的字典形名词和な形容词来说都是必须这样的，相信你已经比较熟悉。另外这也能避免跟「と」的其他用法搞混。

## 用「なら（ば）」表示前提条件 {#part3}

### 词汇学习

1. 皆 【みんな】 - 所有人
2. 行く 【い・く】 （う动词） - 走
3. 私 【わたし】 - 我；我自己
4. 言う 【い・う】 （う动词）- 说
5. 問題 【もん・だい】 - 问题
6. ある （う动词） - 存在（无生命）
7. 図書館 【と・しょ・かん】 - 图书馆
8. あそこ - 那里

另外一个相对来说容易理解的表示「如果」的说法是前提条件语。你可以用这个助词来表示在一定条件下会发生的事情。比如你想说「好吧，如果大家都去的话我也去」，那就可以用「なら」，因为你去的前提条件是大家都去。用这种条件语的时候总是需要有一个前提。比如你可以表达：「如果你说的是<i>那个</i>的话…」或者「如果<i>那样</i>的话…」。

某种意义上来说，你想表达的是如果某个条件满足了，什么会发生。换句话说，你是想说「在这种情况下，会发生这件事」。英语里面对应的翻译我用的是 "if given"。译者注：中文翻译选用「如果…的话」。并且这里的ならば应该就是从なる的ば形发展来的吧？

「なら」要放在前提条件后面，语法跟用「と」一样。不过，你<u>不能</u>加上表陈述的「だ」。

<div class="sumbox">
<div class="sumbox-title">使用前提条件语「なら」的规则</div>

1. 把「なら」加在前提条件后面<br>
<span class="jp-trans"><b>= [ 假设会发生的前提条件 ] + なら + [ 结果 ]</b></span>
2. <u>不能</u> 加表陈述的「だ」。
</div>

### 示例

1. <span class="jp-word" title="みんな - 所有人">みんな</span> が <span class="jp-word" title="いく - 走">行く</span> <em>なら</em> <span class="jp-word" title="わたし - 我；我自己">私</span> も <span class="jp-word" title="いく - 走">行く</span> 。<br>
<span class="jp-trans">如果大家都去的话我也去。</span>
2. アリス <span class="jp-word" title="さん - 桑">さん</span> が <span class="jp-word" title="いう- 说">言う</span> <em>なら</em> <span class="jp-word" title="もんだい - 问题">問題</span> <span class="jp-word" title="ある - 存在（无生命）">ない</span> よ。<br>
<span class="jp-trans">如果 Alice 都这么说的话，那没问题。</span>

### 对话示例

アリス：<span class="jp-word" title="としょかん - 图书馆">図書館</span>は<span class="jp-word" title="どこ - 哪里">どこ</span>ですか。<br>
<span class="jp-trans">Alice：图书馆在哪里？</span>

ボブ：<span class="jp-word" title="としょかん - 图书馆">図書館</span><em>なら</em>、<span class="jp-word" title="あそこ - 那里">あそこ</span>です。<br>
<span class="jp-trans">Bob：如果你说的是图书馆的话，在那里。</span>

下面的用法是错误的。

- <span class="jp-word" title="としょかん - 图书馆">図書館</span> <em>だ</em> なら <span class="jp-word" title="あそこ - 那里">あそこ</span> です。

你也可以用「なら<em>ば</em>」替代「なら」，两者意思完全一样，但前者让人感觉更正式。

## 一般性条件语「ば」 {#part4}

### 词汇学习

1. 食べる 【た・べる】 （る动词） - 吃
2. 待つ 【ま・つ】 （う动词） - 等待
3. おかしい （い形容词） - 有意思
4. ある （う动词） - 存在（无生命）
5. 学生 【がく・せい】 - 学生
6. 暇 【ひま】 有空
7. 友達 【とも・だち】- 朋友
8. 会う 【あ・う】 （う动词） - 会见
9. 買い物 【か・い・もの】 - 购物
10. 行く 【い・く】 （う动词） - 走
11. お金 【お・かね】 - 钱
12. いい （い形容词） - 好的
13. 楽しい 【たの・しい】 （い形容词） - 有趣
14. 私 【わたし】 - 我；我自己
15. 病気 【びょう・き】 - 疾病
16. なる （う动词） - 成为

下面介绍的这种条件语是通用的，它不预设条件，也没有蕴含的意思。「ば」条件语的活用规则如下，注意名词和な形容词的活用规则其实是在活用「である」里面的动词「ある」，这个是我们[很后面才会学到](/grammar/formal/#part2)的一种正式表达。

<div class="sumbox">
<div class="sumbox-title">「ば」的活用规则</div>

- <b>对动词：</b> 把最后假名从 /u/ 段改为同行 /e/ 段，再加上「ば」<br>
<span class="jp-trans">例</span>
  1. <span class="jp-word" title="たべる - 吃">食べ<em>る</em></span> → <span class="jp-word" title="たべる - 吃">食べ<em>れ</em></span> → <span class="jp-word" title="たべる - 吃">食べれ<em>ば</em></span>
  2. <span class="jp-word" title="まつ - 等待">待<em>つ</em></span> → <span class="jp-word" title="まつ - 等待">待<em>て</em></span> → <span class="jp-word" title="まつ - 等待">待て<em>ば</em></span>
- <b>对い形容词以及「ない」结尾的否定式：</b> 把最后的「い」换成「ければ」。<br>
<span class="jp-trans">例</span>
  1. <span class="jp-word" title="おかしい - 有意思">おかし<em>い</em></span> → <span class="jp-word" title="おかしい - 有意思">おかし<em>ければ</em></span>
  2. <span class="jp-word" title="ある - 存在（无生命）">な<em>い</em></span> → <span class="jp-word" title="ある - 存在（无生命）">な<em>ければ</em></span>
- <b>对名词和な形容词：</b> 加「であれば」<br>
<span class="jp-trans">例</span>
  1. 学生 → 学生 <em>であれば</em>
  2. 暇 → 暇 <em>であれば</em>
</div>

### 示例

1. <span class="jp-word" title="ともだち- 朋友">友達</span> に <span class="jp-word" title="あう - 会见">会えれ<em>ば</em></span> 、 <span class="jp-word" title="かいもの - 购物">買い物</span> に <span class="jp-word" title="いく - 走">行きます</span> 。<br>
<span class="jp-trans">要是能见到我的朋友们，我们就去逛街。</span>
2. <span class="jp-word" title="おかね - 钱">お金</span> が <span class="jp-word" title="ある - 存在（无生命）">あれ<em>ば</em></span> <span class="jp-word" title="いい - 好的">いい</span> ね。<br>
<span class="jp-trans">我要是有钱就好了，哈？</span>
3. <span class="jp-word" title="たのしい - 有趣">楽し<em>ければ</em></span> 、 <span class="jp-word" title="わたし - 我；我自己">私</span> も <span class="jp-word" title="いく - 走">行く</span> 。<br>
<span class="jp-trans">要是有意思我也去。</span>
4. <span class="jp-word" title="たのしい - 有趣">楽しくな<em>ければ</em></span> 、 <span class="jp-word" title="わたし - 我；我自己">私</span> も <span class="jp-word" title="いく - 走">行かない</span> 。<br>
<span class="jp-trans">要是没意思我也不去。</span>
5. <span class="jp-word" title="たべる - 吃">食べな<em>ければ</em></span> <span class="jp-word" title="びょうき - disease, sickness">病気</span> に <span class="jp-word" title="なる - 成为">なる</span> よ。<br>
<span class="jp-trans">你要是不吃就会生病。</span>

## 过去形条件语「たら（ば）」 {#part5}

### 词汇学习

1. 自動 【じ・どう】 - 自动
2. 待つ 【ま・つ】 （う动词） - 等待
3. 読む 【よ・む】 （う动词） - 读
4. 忙しい 【いそが・しい】 （い形容词）- 忙
5. 暇 【ひま】 有空
6. 遊ぶ 【あそ・ぶ】 （う动词） - 玩
7. 行く 【い・く】 （う动词） - 走
8. 学生 【がく・せい】 - 学生
9. 割引 【わり・びき】 - 折扣
10. 買う 【か・う】 （う动词）- 买
11. 友達 【とも・だち】- 朋友
12. 会う 【あ・う】 （う动词） - 会见
13. 買い物 【か・い・もの】 - 购物
14. お金 【お・かね】 - 钱
15. ある （う动词） - 存在（无生命）
16. いい （い形容词） - 好的
17. 家 【1) うち; 2) いえ】 - 1）自己家；2）房子
18. 帰る 【かえ・る】 （う动词） - 回家
19. 誰 【だれ】 - 谁
20. いる （る动词） - 存在（有生命）
21. アメリカ - 美国
22. たくさん - （数量）很多
23. 太る 【ふと・る】 （う动词） - 变胖

我把这种条件语叫做过去形条件语是因为这种语法是在过去形后面加上「ら」，通常也叫做「たら」条件语，因为所有的过去形都是以「た／だ」结尾，所以就组合出「たら／だら」。和「ば」一样，这也是一种通用的条件语。

<div class="sumbox">
<div class="sumbox-title">「たら（ば）」的活用规则</div>

- 首先把名词、形容词或动词活用为过去形，再加上「ら」<br>
<span class="jp-trans">例</span>
  1. <span class="jp-word" title="じどう - 自动">自動</span> → <span class="jp-word" title="じどう - 自动">自動</span> <em>だった</em> → <span class="jp-word" title="じどう - 自动">自動</span> だった <em>ら</em>
  2. <span class="jp-word" title="まつ - 等待">待<em>つ</em></span> → <span class="jp-word" title="まつ - 等待">待<em>った</em></span> → <span class="jp-word" title="まつ - 等待">待った<em>ら</em></span>
  3. <span class="jp-word" title="よむ - 读">読<em>む</em></span> → <span class="jp-word" title="よむ - 读">読<em>んだ</em></span> → <span class="jp-word" title="よむ - 读">読んだ<em>ら</em></span>
  4. <span class="jp-word" title="いそがしい- 忙">忙し<em>い</em></span> → <span class="jp-word" title="いそがしい- 忙">忙し<em>かった</em></span> → <span class="jp-word" title="いそがしい- 忙">忙しかった<em>ら</em></span>
</div>

### 示例

1. <span class="jp-word" title="ひま-  有空，闲暇">暇</span> だっ <em>たら</em> 、 <span class="jp-word" title="あそぶ - 玩">遊び</span> に <span class="jp-word" title="いく - 走">行く</span> よ。<br>
<span class="jp-trans">如果我有空，我就去玩。</span>
2. <span class="jp-word" title="がくせい - 学生">学生</span> だっ <em>たら</em> 、 <span class="jp-word" title="がくせい - 学生">学生</span> <span class="jp-word" title="わりびき - 折扣">割引</span> で <span class="jp-word" title="かう- 买">買えます</span> 。<br>
<span class="jp-trans">如果你是学生，就可以享受学生折扣。</span>

对于い形容词和动词来说，这两种条件语很难区分开来，图省事的话你可以认为他们是一样的。不过二者之间有一个小区别，那就是「たら」的重点在于满足条件后发生的事，这也是我叫它过去形条件语的原因，因为假设条件已经发生了，我们感兴趣的是结果。而「ば」则侧重在条件那一侧。

让我们来体会这微妙的差别：

### 例一

Ａ：<span class="jp-word" title="ともだち- 朋友">友達</span>に<span class="jp-word" title="あう - 会见">会えれ<em>ば</em></span>、<span class="jp-word" title="かいもの - 购物">買い物</span>に<span class="jp-word" title="いく - 走">行きます</span>。<br>
<span class="jp-trans">A：我和朋友们会去逛街，<em>如果</em>我能见到他们。</span>

Ｂ：<span class="jp-word" title="ともだち- 朋友">友達</span>に<span class="jp-word" title="あう - 会见">会え<em>たら</em></span>、<span class="jp-word" title="かいもの - 购物">買い物</span>に<span class="jp-word" title="いく - 走">行きます</span>。<br>
<span class="jp-trans">B：如果我能见到朋友们的话，<em>我们就去逛街</em>。</span>

### 例二

Ａ：<span class="jp-word" title="おかね - 钱">お金</span>が<span class="jp-word" title="ある - 存在（无生命）">あれ<em>ば</em></span><span class="jp-word" title="いい - 好的">いい</span>ね。<br>
<span class="jp-trans">A：情况会很好，<em>如果</em>我有钱，对吧？</span>

Ｂ：<span class="jp-word" title="おかね - 钱">お金</span>が<span class="jp-word" title="ある - 存在（无生命）">あっ<em>たら</em></span><span class="jp-word" title="いい - 好的">いい</span>ね。<br>
<span class="jp-trans">B：如果我有钱的话，<em>那情况会很好，对吧？</em></span>

从上面两组例子的场景来看，用「～たら」显得更自然，因为我们貌似不关心条件本身。我们更感兴趣的，应该是见到朋友后会做什么，或者有了钱会变怎样。

过去形条件语是唯一可以用过去形结果的用法。这听起来也许很奇怪：如果结果已经发生了，还谈何「如果」？实际上这种用法里面就已经没有「如果」了，这只是对某种条件下发生的结果表示惊讶，跟条件没啥关系，不过因为语法一样，所以在这里一并介绍。

1. <span class="jp-word" title="いえ - 家">家</span> に <span class="jp-word" title="かえる - 回家">帰ったら</span> 、 <span class="jp-word" title="だれも - nobody">誰も</span> <span class="jp-word" title="いる - 存在（有生命）">いなかった</span> 。<br>
<span class="jp-trans">我回到家的时候，一个人也没有（意外的结果）。</span>
2. <span class="jp-word" title="アメリカ - 美国">アメリカ</span> に <span class="jp-word" title="いく - 走">行ったら</span> 、 <span class="jp-word" title="たくさん - 很多">たくさん</span> <span class="jp-word" title="ふとる - 变胖">太りました</span> 。<br>
<span class="jp-trans">去了美国的结果是，我胖了（意外的结果）。</span>

你也可以用「たら<em>ば</em>」替代「たら」。类似「ならば」，二者意思完全相同，只是前者感觉更正式。

## 那「<span class="jp-word" title="もし - 如果，假设">もし</span>」算什么？ {#part6}

### 词汇学习

1. もし - 如果，假设
2. いい （い形容词） - 好的
3. 映画 【えい・が】 - 电影
4. 観る 【み・る】 （る动词） - 观看
5. 行く 【い・く】 （う动词） - 走
6. 時間 【じ・かん】 - 时间
7. ある （う动词） - 存在（无生命）
8. 明日 【あした】 - 明天

你们也许有人知道「<span class="jp-word" title="もし - 如果，假设">もし</span>」这个意思也是「如果」的词，它跟上面介绍的四种用法是什么关系呢？其实，如果你想表示假设条件的话，就必须用上面其中一种条件语。「<span class="jp-word" title="もし - 如果，假设">もし</span>」只是一个补充，用来增加一种对假设条件不确定的语气。比如你想发出一个邀请，但又不想预设任何条件的时候，就可以用到它，如下所示。

1. <em><span class="jp-word" title="もし - 如果，假设">もし</span></em> <span class="jp-word" title="いい - 好的">よかった</span> ら、 <span class="jp-word" title="えいが - 电影">映画</span> を <span class="jp-word" title="みる - 观看">観</span> に <span class="jp-word" title="いく - 走">行きます</span> か？<br>
<span class="jp-trans">如果 <em>万一</em> 没问题的话，一起去看电影？</span>
2. <em><span class="jp-word" title="もし - 如果，假设">もし</span></em> <span class="jp-word" title="じかん - 时间">時間</span> が <span class="jp-word" title="ある - 存在（无生命）">ない</span> なら、 <span class="jp-word" title="あした - 明天">明日</span> でも <span class="jp-word" title="いい - 好的">いい</span> よ。<br>
<span class="jp-trans">如果没时间的话，明天也可以（不确定是否有时间）。</span>
